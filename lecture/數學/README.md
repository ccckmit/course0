# 數學

本課程涵蓋電腦科學中涉及的數學基礎，這些數學工具是演算法、機器學習、訊號處理等領域的基礎。

* [代數學](../wiki/數學/代數學.md) - 群論、環、場
* [幾何學](../wiki/數學/幾何學.md) - 計算幾何、凸包
* [微積分](../wiki/數學/微積分.md) - 微分、積分、梯度
* [機率統計](../wiki/數學/機率統計.md) - 機率論、統計學

# lean4py v0.1 規劃

## 目標

模仿 Lean 語言的 mathlib，設計一個 Python 套件 `lean4py`，提供數學定理證明輔助、集合論、代數結構的程式化表示。

## 套件結構

```
lean4py/
├── pyproject.toml          # 安裝設定
├── lean4py/
│   ├── __init__.py
│   ├── logic.py            # 命題邏輯 (Prop, →, ∧, ∨, ¬)
│   ├── sets.py             # 集合論 (Set, ∈, ⊆, ∪, ∩, \, ×)
│   ├── algebra.py          # 代數結構 (Group, Ring, Field, Module)
│   ├── nat.py              # 自然數與數學歸納法
│   └── tactics.py          # 證明策略 (rfl, simp, exact, apply)
├── tests/
│   ├── test_logic.py
│   ├── test_sets.py
│   └── test_algebra.py
├── examples/
│   ├── 01_logic.py         # 命題邏輯範例
│   ├── 02_sets.py          # 集合論範例
│   └── 03_algebra.py       # 代數範例
└── README.md
```

## 核心模組

### logic.py — 命題邏輯

| 類別/函數 | 說明 |
|-----------|------|
| `Prop` | 命題類別 |
| `implies(a, b)` / `a >> b` | 蘊含 (→) |
| `and_(a, b)` / `a & b` | 連言 (∧) |
| `or_(a, b)` / `a \| b` | 選言 (∨) |
| `not_(a)` / `~a` | 否定 (¬) |
| `Theorem(name, prop, proof)` | 定理類別 |
| `prove(prop, tactics)` | 證明函式 |

### sets.py — 集合論

| 類別/函數 | 說明 |
|-----------|------|
| `Set(elems)` | 集合類別 |
| `in_(elem, s)` / `elem ∈ s` | 成員關係 |
| `subset(s1, s2)` / `s1 <= s2` | 子集 (⊆) |
| `union(s1, s2)` / `s1 + s2` | 聯集 (∪) |
| `intersection(s1, s2)` / `s1 * s2` | 交集 (∩) |
| `complement(s)` / `~s` | 補集 |
| `cartesian(s1, s2)` / `s1 × s2` | 卡氏積 (×) |

### algebra.py — 代數結構

| 類別 | 說明 |
|------|------|
| `Magma` | 封閉二元運算 (·) |
| `Semigroup` | 結合律成立 |
| `Monoid` | 含單位元 |
| `Group` | 含逆元 |
| `AbelianGroup` | 交換群 |
| `Ring` | 雙運算 +，· |
| `Field` | 除法封閉 |

## 證明策略 (tactics.py)

| 策略 | 說明 |
|------|------|
| `rfl` | 反射性 (a = a) |
| `exact(p)` | 直接給出命題 p |
| `apply(h)` | 套用假設 h |
| `simp` | 簡化命題 |
| `have(name, prop)` | 引入輔助命題 |
| `assume(name, prop)` | 引入假設 |

## v0.1 範例預覽

### 邏輯範例

```python
from lean4py import Prop, implies, and_, or_, not_, prove, Theorem

p, q, r = Prop('p'), Prop('q'), Prop('r')

theorem = Theorem('modus_ponens',
    implies(and_(p, implies(p, q)), q),
    [assume('h1', and_(p, implies(p, q))),
     have('p', p, from_='h1.left'),
     have('p_implies_q', implies(p, q), from_='h1.right'),
     exact(q)])
```

### 集合範例

```python
from lean4py import Set, union, intersection, complement

A = Set({1, 2, 3})
B = Set({2, 3, 4})

print(union(A, B))    # {1, 2, 3, 4}
print(A * B)          # {2, 3}
print(~A)             # 補集
```

### 代數範例

```python
from lean4py import Group, AbelianGroup, Ring

Z = AbelianGroup('Z', int, lambda a, b: a + b, 0, lambda a: -a)
print(Z.is_abelian())  # True
```

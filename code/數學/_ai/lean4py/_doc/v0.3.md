## 數論模組 (lean4py/number_theory.py)

### 新增類別與函數

| 類別/函數 | 說明 |
|-----------|------|
| `Integer` | 整數類型（擴展 Nat） |
| `Divisibility(a, b)` | 整除關係 `a \| b` |
| `gcd(a, b)` | 最大公因數 |
| `lcm(a, b)` | 最小公倍數 |
| `is_prime(n)` | 質數判定 |
| `prime_factors(n)` | 質因數分解 |
| `mod_exp(a, e, m)` | 模指數運算 |
| `phi(n)` | 歐拉函數 |
| `bezout_identity(a, b)` | 貝茲等式 ( Extended GCD ) |

### 核心定理

```python
# 數論基本定理
gcd(a, b) * lcm(a, b) == |a * b|
bezout_identity(a, b)  # 回傳 (x, y, d) 使得 ax + by = d = gcd(a, b)

# 質數性質
is_prime(p) and p > 1  ⇒  phi(p) = p - 1

# 模運算
mod_exp(a, e, m)  # a^e mod m 的高效計算
```

## 線性代數模組 (lean4py/linear_algebra.py)

### 新增類別與函數

| 類別/函數 | 說明 |
|-----------|------|
| `Vector(n, elements)` | n 維向量 |
| `Matrix(rows, cols)` | 矩陣 |
| `dot_product(u, v)` | 點積 |
| `cross_product(u, v)` | 叉積（3D） |
| `matrix_mul(A, B)` | 矩陣乘法 |
| `det(A)` | 行列式 |
| `rank(A)` | 秩 |
| `nullity(A)` | 零度 |
| `eigenvalues(A)` | 特徵值 |
| `is_linearly_independent(vectors)` | 線性獨立判定 |

### 核心定理

```python
# 行列式性質
det(A * B) == det(A) * det(B)
det(transpose(A)) == det(A)

# 秩-零度定理
rank(A) + nullity(A) == n  # A 是 m×n 矩陣

# 線性獨立
is_linearly_independent([v1, v2, ..., vk])  ⇒  span([v1, ..., vk]) 的維度 = k
```

## 已實現功能

### number_theory.py
- `Integer` 類別：支援負數的整數
- `gcd`, `lcm` - 最大公因數、最小公倍數
- `bezout_identity` - 擴展歐幾里得算法
- `is_prime`, `prime_factors` - 質數判定與分解
- `phi` - 歐拉函數
- `mod_exp`, `mod_inverse` - 模指數與模逆
- `divides`, `coprime` - 整除性與互質判定
- `chinese_remainder` - 中國剩餘定理
- `IntegerInduction` - 數學歸納法

### linear_algebra.py
- `Vector` 類別：n 維向量運算
- `Matrix` 類別：矩陣運算
- `dot_product`, `cross_product` - 向量積
- `matrix_mul`, `matrix_vector_mul` - 矩陣乘法
- `det`, `matrix_inverse`, `trace` - 行列式、矩陣逆、 trace
- `rank`, `nullity` - 秩與零度
- `eigenvalues`, `eigenvectors` - 特徵值與特徵向量
- `is_linearly_independent`, `span` - 線性獨立與生成空間
- `is_orthogonal`, `is_orthonormal` - 正交性判定
- `LinearMap` - 線性變換

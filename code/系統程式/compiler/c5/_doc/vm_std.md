# C4 虛擬機指令手冊

C4 採用基於堆疊的虛擬機器（Stack-based VM），有 30 條指令。

## 指令格式

| 助記符 | 類型 | 說明 |
|--------|------|------|
| LLA | `[n]` | 載入本地位址：a = bp + n |
| IMM | `[n]` | 立即值：a = n |
| JMP | `[n]` | 跳躍：pc = n |
| JSR | `[n]` | 跳至副程式：`*--sp = pc+1; pc = n` |
| BZ | `[n]` | 為零分支：pc = a ? pc+1 : n |
| BNZ | `[n]` | 非零分支：pc = a ? n : pc+1 |
| ENT | `[n]` | 進入函式：`*--sp = bp; bp = sp; sp = sp - n` |
| ADJ | `[n]` | 調整堆疊：sp = sp + n |
| LEV | - | 離開函式：`sp = bp; bp = *sp++; pc = *sp++` |
| LI | - | 載入整數：a = *(long*)a |
| LC | - | 載入字元：a = *(char*)a |
| SI | - | 儲存整數：*sp++ = a |
| SC | - | 儲存字元：a = *(char*)*sp++ = a |
| PSH | - | 推入堆疊：*--sp = a |
| OR | - | 位元 OR：a = *sp++ |
| XOR | - | 位元 XOR：a = *sp++ ^ a |
| AND | - | 位元 AND：a = *sp++ & a |
| EQ | - | 相等：a = *sp++ == a |
| NE | - | 不等：a = *sp++ != a |
| LT | - | 小於：a = *sp++ < a |
| GT | - | 大於：a = *sp++ > a |
| LE | - | 小於等於：a = *sp++ <= a |
| GE | - | 大於等於：a = *sp++ >= a |
| SHL | - | 左移：a = *sp++ << a |
| SHR | - | 右移：a = *sp++ >> a |
| ADD | - | 加法：a = *sp++ + a |
| SUB | - | 減法：a = *sp++ - a |
| MUL | - | 乘法：a = *sp++ * a |
| DIV | - | 除法：a = *sp++ / a |
| MOD | - | 取餘：a = *sp++ % a |

## 系統呼叫

| 指令 | C 對應 | 說明 |
|------|--------|------|
| OPEN | `open((char*)sp[1], *sp)` | 開啟檔案，fd = sp[1] |
| READ | `read(sp[2], (char*)sp[1], *sp)` | 讀取資料 |
| CLOSE | `close(*sp)` | 關閉檔案 |
| PRTF | `printf(t[-1], t[-2], t[-3], t[-4], t[-5], t[-6])` | 格式化輸出 |
| MALC | `malloc(*sp)` | 配置記憶體 |
| FREE | `free((void*)*sp)` | 釋放記憶體 |
| MSET | `memset((char*)sp[2], sp[1], *sp)` | 記憶體設定 |
| MCMP | `memcmp((char*)sp[2], (char*)sp[1], *sp)` | 記憶體比較 |
| EXIT | `return *sp` | 結束程式 |

## 暫存器

| 名稱 | 用途 |
|------|------|
| pc | 程式計數器（指向下一條指令） |
| bp | 框架指標（base pointer） |
| sp | 堆疊指標（stack pointer） |
| a | 累加器（accumulator） |

## 函式呼叫約定

1. 呼叫者將參數推入堆疊
2. JSR 指令：保存返回位址（`pc+1`）到堆疊，跳轉到関数
3. ENT：建立新框架，分配局部變數空間
4. LEV：恢復舊框架，返回呼叫點

## 記憶體佈局

```
+------------------+ _HIGH_
|    程式碼區      |
+------------------+
|    全域變數      |
+------------------+
|    堆疊區        |
+------------------+  <- sp (生長方向)
|    ...           |
+------------------+
|    返回位址      |
+------------------+
|    舊 bp         |
+------------------+  <- bp
|    局部變數      |
+------------------+  _LOW_
```

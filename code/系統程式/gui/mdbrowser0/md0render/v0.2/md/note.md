# 測試筆記：長文本展示

這是一篇用於測試長文本渲染效果的筆記，驗證 md0r 渲染器的換行、連續文本渲染能力。

---

## 第一段
Markdown 是一種輕量級標記語言，它允許人們使用易讀易寫的純文本格式編寫文檔，然後轉換成有效的結構化文檔。

本渲染器實現了 Markdown 最核心的超連結跳轉功能，讓你可以在多個 Markdown 文件之間自由導航，就像一個簡易的離線瀏覽器。

## 第二段
SDL2 是一個跨平台的多媒體開發庫，提供了對音頻、鍵盤、鼠標、遊戲手柄和圖形硬件的低級訪問，被廣泛用於遊戲開發、多媒體應用開發。

本項目基於 SDL2 實現窗口創建、畫面渲染，基於 SDL2_ttf 實現文字渲染，無需依賴複雜的第三方庫，即可完成 Markdown 內容的展示與交互。

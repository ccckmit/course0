# ex1-grad.py — 基礎自動微分運算

## 概述

ex1-grad.py 展示自動微分引擎（nn0.py）的核心功能：如何使用 Value 類建立計算圖並進行反向傳播。

## 執行方式

```bash
python ex1-grad.py
```

## 程式碼詳解

### 1. 建立參數

```python
a = Value(3.0)
b = Value(2.0)
x = Value(4.0)
```

建立三個 `Value` 節點，這些是計算圖的葉節點。

### 2. 建立運算圖

```python
y = a * (x ** 2) + b
```

建立複雜的運算，自動記錄計算圖：
- `x ** 2`：計算 x 的平方
- `a * (x ** 2)`：乘法
- `+ b`：加法

結果：`y = 3 * 4² + 2 = 50`

### 3. 反向傳播

```python
y.backward()
```

沿計算圖反向傳遞梯度，使用鏈式法則：
- $\frac{\partial y}{\partial a} = x^2 = 16$
- $\frac{\partial y}{\partial b} = 1$
- $\frac{\partial y}{\partial x} = 2ax = 2×3×4 = 24$

## 數學原理

### 鏈式法則

若 $y = f(g(x))$，則：
$$\frac{dy}{dx} = \frac{df}{dg} \cdot \frac{dg}{dx}$$

### 運算的局部梯度

| 運算 | 結果 | $\frac{\partial y}{\partial x}$ | $\frac{\partial y}{\partial y}$ |
|------|------|------------------------------|------------------------------|
| $y = a + b$ | $x + y$ | 1 | 1 |
| $y = a \cdot b$ | $x \cdot y$ | $y$ | $x$ |
| $y = a^n$ | $x^n$ | $n \cdot x^{n-1}$ | - |

## 輸出範例

```
=== 1. 基礎自動微分運算 ===
前向計算結果 y.data = 50.0000
dy/da = 16.0000 (理論值 x^2 = 16)
dy/db = 1.0000 (理論值 1)
dy/dx = 24.0000 (理論值 2ax = 2*3*4 = 24)
```

## 學習重點

1. **計算圖**：每個運算自動建立依賴關係
2. **局部梯度**：每個運算知道自己的導數
3. **反向傳播**：沿圖逆向計算梯度
4. **梯度累加**：一個節點被多個父節點依賴時，梯度會累加

# lean4py v0.2 規劃

## 目標

v0.2 著重於**自動定理證明**與**更多 tactics**，讓證明過程更自動化、更多元化。

## 新增模組

### prover.py — 自動定理證明器

```python
lean4py/
├── prover.py  # 新增：自動定理證明
```

| 類別/函數 | 說明 |
|-----------|------|
| `Prover` | 定理證明器類別 |
| `truth_table_prove(prop)` | 真值表法證明（窮舉） |
| `tableau_prove(prop)` | 語義 tablea 法 |
| `is_valid(prop)` | 判斷命題是否普遍有效 |
| `is_satisfiable(prop)` | 判斷命題是否可滿足 |
| `find_counterexample(prop)` | 找反例 |

## 新增 Tactics (tactics.py)

### 現有 tactics
- `rfl` - 反射性
- `exact(p)` - 直接給出命題
- `apply(h)` - 套用假設
- `simp` - 簡化
- `assume(name, prop)` - 引入假設
- `have(name, prop, from_)` - 引入輔助命題

### v0.2 新增 tactics

| 策略 | 說明 | 範例 |
|------|------|------|
| `intros(names)` | 引入多個變數/假設 | `intros(['x', 'h'])` |
| `by_contra(name, prop)` | 反證法 | `by_contra('h', not_(p))` |
| `cases(h, cases)` | 分情況討論 | `cases('h', [p, q])` |
| `induction(var, base, ind)` | 數學歸納法 | `induction('n', base, ind)` |
| `rewrite(eq, sym)` | 使用等式改寫 | `rewrite('h', sym=True)` |
| `split()` | 分割合取或 iff | `split()` |
| `left()` | 左消解（選言） | `left()` |
| `right()` | 右消解（選言） | `right()` |
| `use(witness)` | 存在命題見證 | `use(x)` |
| `show(prop)` | 顯示目標命題 | `show(p)` |
| `by(tactics)` | 嵌套證明 | `by([simp, exact(p)])` |
| `sorry()` | 未完成證明（佔位） | `sorry()` |

## prover.py API

```python
from lean4py.prover import Prover, truth_table_prove, is_valid, is_satisfiable

p, q = Prop('p'), Prop('q')

# 真值表法自動證明
result = truth_table_prove(implies(p, p))
print(result)  # True

# 判斷有效性
print(is_valid(implies(p | ~p, p & ~p)))  # False

# 判斷可滿足性
print(is_satisfiable(p & ~p))  # False

# 使用 Prover 類
prover = Prover()
proof = prover.prove(implies(and_(p, implies(p, q)), q))
print(proof)
```

## 實現要點

### truth_table_prove
1. 枚舉命題變元的所有真值指派
2. 對每個指派計算命題真假
3. 若全為真則命題有效

### tableau_prove
1. 將命題轉為否定形式
2. 試圖封閉所有分支
3. 若能封閉則原命題有效

### 新 tactics 實現
- `intros`: 自動從目標中提取假設
- `by_contra`: 假設否定結論，導出矛盾
- `cases`: 對選言命題進行分支證明
- `induction`: 對自然數進行結構歸納

## 測試新增

```python
# tests/test_prover.py
def test_truth_table_valid():
    p = Prop('p')
    assert is_valid(implies(p, p))

def test_truth_table_invalid():
    p, q = Prop('p'), Prop('q')
    assert not is_valid(implies(p, q))

def test_satisfiable():
    p = Prop('p')
    assert is_satisfiable(p)
    assert not is_satisfiable(p & ~p)
```

## 範例

### 自動證明 p → p

```python

from lean4py.prover import truth_table_prove

p = Prop('p')
result = truth_table_prove(implies(p, p))
print(result)  # True
```

## 實分析/微積分模組 (lean4py/real_analysis.py)

### 新增類別與函數

| 類別/函數 | 說明 |
|-----------|------|
| `Real` | 實數類型（擴展 Integer/Float） |
| `limit(f, x0)` | 函數極限 |
| `derivative(f, x)` | 導數 |
| `partial_derivative(f, var)` | 偏導數 |
| `integral(f, a, b)` | 定積分 |
| `series_sum(a_n, n)` | 級數求和 |
| `converges(series)` | 級數收斂判定 |
| `is_continuous(f, x0)` | 連續性判定 |
| `is_differentiable(f, x)` | 可微性判定 |
| `taylor_series(f, x0, n)` | 泰勒級數 |
| `lhopital_limit(f, g, x0)` | 羅必達法則 |

### 核心概念

```python
# 極限
limit(f, x0)  # lim_{x→x0} f(x)

# 導數
derivative(f, x)  # df/dx at x
partial_derivative(f, 'y')  # ∂f/∂y

# 積分
integral(f, a, b)  # ∫_a^b f(x) dx

# 級數
series_sum(lambda n: 1/n**2, infinity)  # Σ 1/n²
converges(lambda n: 1/n)  #調和級數發散
```

### 實現策略

- 使用數值方法逼近極限、導數、積分
- 符號表達式用於精確計算（可整合 sympy）
- 收斂判定使用比較審斂法、比值審斂法、根值審斂法

## 機率統計模組 (lean4py/probability.py)

### 新增類別與函數

| 類別/函數 | 說明 |
|-----------|------|
| `ProbabilitySpace(Ω, F, P)` | 機率空間 |
| `Event` | 事件 |
| `RandomVariable` | 隨機變數 |
| `ExpectedValue(X)` | 期望值 |
| `Variance(X)` | 變異數 |
| `StandardDeviation(X)` | 標準差 |
| `Covariance(X, Y)` | 共變異數 |
| `Correlation(X, Y)` | 相關係數 |
| `NormalDistribution(μ, σ²)` | 常態分佈 |
| `BinomialDistribution(n, p)` | 二項分佈 |
| `PoissonDistribution(λ)` | 卜瓦松分佈 |
| `hypothesis_test()` | 假設檢定 |
| `confidence_interval()` | 信賴區間 |

### 核心定理

```python
# 期望值與變異數
E[aX + b] = aE[X] + b
Var[aX + b] = a²Var[X]

# 大數法則
P(|X̄_n - μ| > ε) → 0  as n → ∞

# 中央極限定理
(X̄_n - μ) / (σ/√n) → N(0,1)

# 貝葉斯定理
P(A|B) = P(B|A)P(A) / P(B)
```

### 實現要點

- `ProbabilitySpace` 封裝樣本空間、事件域、機率測度
- 隨機變數是從機率空間到實數的可測函數
- 常用分佈使用 scipy.stats 或自実装
- 假設檢定支援 z-test, t-test, chi-square test

```python
# 握手定理

```

## Prover 增強 (lean4py/prover.py)

### 新增功能

| 功能 | 說明 |
|------|------|
| `prove_theorem(name, prop, proof)` | 定理證明器 |
| `auto_prove(prop)` | 自動策略證明 |
| `rewrite_prove(eq, target)` | 重寫證明 |
| `induction_prove(P, n)` | 歸納法證明 |
| `prop_prove_by_cases(cases)` | 分情況證明 |

### 與現有模組整合

```python
# 數論定理可證明
prove_theorem(
    "gcd_lcm_identity",
    implies(And(a != 0, b != 0), gcd(a,b) * lcm(a,b) == abs(a*b)),
    [rewrite(gcd_def), rewrite(lcm_def), algebra()]
)

# 線性代數定理可證明
prove_theorem(
    "rank_nullity_theorem",
    rank(A) + nullity(A) == A.cols,
    [rewrite(rank_def), rewrite(nullity_def), linear_algebra()]
)
```

### 策略改進

- 增加 `algebra()` 策略：自動代數簡化
- 增加 `linear()` 策略：線性代數操作
- 增加 `number_theory()` 策略：數論操作
- 增加 `normalize()` 策略：標準化命題

## 範例

### 實分析範例

```python
from lean4py.real_analysis import limit, derivative, integral, converges

# 極限
f = lambda x: (x**3 - 1) / (x - 1)
print(f"lim_{{x→1}} (x³-1)/(x-1) = {limit(f, 1)}")  # 3

# 導數
g = lambda x: x**2 + 2*x + 1
print(f"g'(3) = {derivative(g, 3)}")  # 8

# 積分
h = lambda x: x**2
print(f"∫₀¹ x² dx = {integral(h, 0, 1)}")  # 1/3

# 級數收斂
series = lambda n: 1 / n**2
print(f"Σ 1/n² converges: {converges(series)}")  # True
```

### 機率統計範例

```python
from lean4py.probability import (
    ProbabilitySpace, RandomVariable,
    ExpectedValue, Variance,
    NormalDistribution, hypothesis_test
)

# 機率空間
Ω = {1, 2, 3, 4, 5, 6}
P = ProbabilitySpace.uniform(Ω)

# 隨機變數（擲骰子）
X = RandomVariable('X', {i: 1/6 for i in range(1, 7)})
print(f"E[X] = {ExpectedValue(X)}")  # 3.5
print(f"Var(X) = {Variance(X)}")  # 35/12

# 常態分佈
N = NormalDistribution(0, 1)
print(f"P(X ≤ 0) = {N.cdf(0)}")  # 0.5

# 假設檢定
result = hypothesis_test('t', sample=[1.2, 2.1, 1.8], mu=2.0)
print(f"p-value = {result['p_value']}")
```

### 圖論範例

```python
from lean4py.graph_theory import Graph, bfs, dfs, shortest_path

# 創建圖
g = Graph(
    vertices=[1, 2, 3, 4, 5],
    edges=[(1,2), (1,3), (2,4), (3,4), (4,5)]
)

# BFS
print(f"BFS from 1: {bfs(g, 1)}")  # [1, 2, 3, 4, 5]

# DFS
print(f"DFS from 1: {dfs(g, 1)}")

# 最短路徑
path = shortest_path(g, 1, 5)
print(f"Shortest path 1→5: {path}")  # [1, 2, 4, 5]

# 圖屬性
print(f"Connected: {is_connected(g)}")  # True
print(f"Bipartite: {is_bipartite(g)}")  # False
```

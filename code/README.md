# code - 課程程式碼範例

本目錄包含金門大學資訊工程系六門課程的程式碼範例。

## 科目列表

### 1. 計算機結構
電腦硬體的組織與設計原理，主要使用 Verilog。

* [計算機結構/eda/](./計算機結構/eda/) - EDA 工具與 Verilog 實作
* [計算機結構/README.md](./計算機結構/README.md) - 課程說明

### 2. 系統程式
電腦系統的底層運作，主要使用 C 語言，輔助 Python、Rust。

* [系統程式/compiler/](./系統程式/compiler/) - 編譯器
* [系統程式/interpreter/](./系統程式/interpreter/) - 直譯器
* [系統程式/irvm/](./系統程式/irvm/) - IR + VM (RISC-V, LLVM IR)
* [系統程式/os/](./系統程式/os/) - 作業系統 (xv6, xv7, mini-riscv-os2)
* [系統程式/network/](./系統程式/network/) - 網路程式
* [系統程式/crpyto/](./系統程式/crpyto/) - 密碼學
* [系統程式/media/](./系統程式/media/) - 媒體處理 (MP4, MPEG, FFT)
* [系統程式/gui/](./系統程式/gui/) - 圖形介面
* [系統程式/textdata/](./系統程式/textdata/) - 文字資料處理 (Git, 區塊鏈, SQL, RegEx)
* [系統程式/tool/](./系統程式/tool/) - 開發工具

### 3. 機器學習
機器學習演算法實作，使用 Python + scikit-learn + PyTorch + Gym。

* [機器學習/ml/regression/](./機器學習/ml/regression/) - 迴歸
* [機器學習/ml/classify/](./機器學習/ml/classify/) - 分類 (KNN, SVM, 決策樹, 隨機森林, GBDT)
* [機器學習/ml/cluster/](./機器學習/ml/cluster/) - 聚類 (K-Means, 層次, DBSCAN, GMM)
* [機器學習/ml/pca/](./機器學習/ml/pca/) - PCA 維度降低
* [機器學習/ml/prob_model/](./機器學習/ml/prob_model/) - 機率模型 (BayesNet, Markov, HMM, EM)
* [機器學習/ml/sklearn/](./機器學習/ml/sklearn/) - scikit-learn 範例
* [機器學習/image/](./機器學習/image/) - 影像辨識 (CNN)
* [機器學習/README.md](./機器學習/README.md) - 課程說明

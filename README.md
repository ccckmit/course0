# course0 - 金門大學資訊工程系課程教材

本專案為金門大學資訊工程系-陳鍾誠的課程教材，包含程式碼範例、課程講稿與知識庫。

## 專案結構

```
course0/
├── code/           # 課程程式碼範例
├── wiki/           # LLM-based 知識庫 (受 Karpathy Wiki 啟發)
├── lecture/        # 課程講稿
└── README.md
```

## 課程列表

### 1. 計算機結構
電腦硬體的組織與設計原理，主要教材為 Nand2Tetris，搭配 Verilog 實作。
* [lecture/計算機結構/README.md](./lecture/計算機結構/README.md) - 課程大綱
* [code/計算機結構/](./code/計算機結構/) - 程式碼範例

### 2. 系統程式
電腦系統的底層運作，包含編譯器、作業系統、網路等核心技術。主要使用 C 語言。
* [lecture/系統程式/README.md](./lecture/系統程式/README.md) - 課程大綱
* [code/系統程式/](./code/系統程式/) - 程式碼範例

### 3. 機器學習
機器學習的核心概念與實作，涵蓋傳統 ML 到深度學習。使用 Python + scikit-learn + PyTorch。
* [lecture/機器學習/README.md](./lecture/機器學習/README.md) - 課程大綱
* [code/機器學習/](./code/機器學習/) - 程式碼範例

### 4. 演算法
電腦科學的核心演算法，包括設計技巧與複雜度分析。使用 Python。
* [lecture/演算法/README.md](./lecture/演算法/README.md) - 課程大綱
* [code/演算法/](./code/演算法/) - 程式碼範例

### 5. 網站設計
網頁開發的完整技術棧，從前端到後端。使用 Node.js + SQLite + HTML/CSS/JavaScript。
* [lecture/網站設計/README.md](./lecture/網站設計/README.md) - 課程大綱
* [code/網站設計/](./code/網站設計/) - 程式碼範例

### 6. 軟體工程
軟體開發的方法論與實踐，特別強調 AI 導向的軟體工程。
* [lecture/軟體工程/README.md](./lecture/軟體工程/README.md) - 課程大綱
* [code/軟體工程/](./code/軟體工程/) - 程式碼範例

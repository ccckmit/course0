# Rust 記憶體管理測試說明

## 執行方式

```bash
rustc 01-memory.rs -o 01-memory
./01-memory
```

## 測試項目

### 1. Box - 堆配置
- **內容**: Box::new, 遞迴資料結構
- **說明**: 在堆上分配資料

### 2. Rc - 參考計數
- **內容**: Rc::new, Rc::clone
- **說明**: 單執行緒共享擁有權

### 3. RefCell - 內部可變性
- **內容**: borrow, borrow_mut
- **說明**: 執行時借用檢查

### 4. Rc + RefCell 組合
- **內容**: 可變的共享資料
- **說明**: 多人可修改

### 5. Arc - 原子參考計數
- **內容**: 多執行緒共享
- **說明**: 執行緒安全

### 6. Mutex 與 Arc
- **內容**: 安全的並發修改
- **說明**: 鎖保護

### 7. Cell - 內部可變性
- **內容**: set, get, replace
- **說明**: Copy 類型專用

### 8. 借用規則
- **內容**: & 和 &mut
- **說明**: 編譯時安全

### 9. Drop 特徵
- **內容**: 自訂釋放邏輯
- **說明**: RAII

### 10. 記憶體安全
- **內容**: 防止常見錯誤
- **說明**: Rust 保證

# Rust 基礎測試說明

## 執行方式

```bash
rustc 01-basics.rs -o 01-basics
./01-basics
```

## 測試項目

### 1. 變數與可變性
- **內容**: let, mut, const
- **說明**: Rust 變數預設不可變

### 2. 資料類型
- **內容**: 標量類型、元組、陣列
- **說明**: i32, f64, bool, char, &str, String

### 3. 所有權與移動
- **內容**: 所有權概念、Copy 特徵
- **說明**: 每個值只有一個所有者

### 4. 借用與引用
- **內容**: &, &mut
- **說明**: 借用而不取得所有權

### 5. 結構體
- **內容**: struct, impl, 方法
- **說明**: 物件導向支援

### 6. 列舉與模式匹配
- **內容**: enum, match
- **說明**: 類型安全的模式匹配

### 7. 錯誤處理
- **內容**: Result, ? 運算符
- **說明**: 可恢復的錯誤

### 8. 泛型
- **內容**: 泛型函數與結構
- **說明**: 代碼重用

### 9. trait
- **內容**: 介面定義
- **說明**: 行為定義

### 10. 生命週期
- **內容**: &'a, 生命周期標註
- **說明**: 引用有效性

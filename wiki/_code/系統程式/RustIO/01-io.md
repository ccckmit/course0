# Rust I/O 測試說明

## 執行方式

```bash
rustc 01-io.rs -o 01-io
./01-io
```

## 測試項目

### 1. 檔案讀取
- **內容**: fs::read_to_string
- **說明**: 讀取整個檔案

### 2. 檔案寫入
- **內容**: fs::write
- **說明**: 寫入檔案

### 3. 逐行讀取
- **內容**: BufReader, lines
- **說明**: 高效逐行讀取

### 4. 標準輸入
- **內容**: stdin::read_line
- **說明**: 讀取使用者輸入

### 5. 命令列參數
- **內容**: env::args
- **說明**: 取得命令列參數

### 6. 環境變數
- **內容**: env::vars, env::var
- **說明**: 存取環境變數

### 7. 網路請求
- **內容**: HTTP 客戶端說明
- **說明**: 需使用外部 crate

### 8. JSON 處理
- **內容**: serde_json 說明
- **說明**: 序列化和反序列化

### 9. 錯誤處理
- **內容**: Result, ? 運算符
- **說明**: 錯誤傳播

### 10. 臨時檔案
- **內容**: temp_dir
- **說明**: 臨時檔案操作

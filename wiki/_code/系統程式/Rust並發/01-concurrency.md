# Rust 並發測試說明

## 執行方式

```bash
rustc 01-concurrency.rs -o 01-concurrency
./01-concurrency
```

## 測試項目

### 1. 基本執行緒
- **內容**: thread::spawn
- **說明**: 建立新執行緒

### 2. Move 閉包
- **內容**: move 關鍵字
- **說明**: 取得所有權

### 3. 通道通訊
- **內容**: mpsc::channel
- **說明**: 執行緒間訊息傳遞

### 4. 共享資料
- **內容**: Arc, Mutex
- **說明**: 安全的共享資料

### 5. 讀寫鎖
- **內容**: RwLock
- **說明**: 多人讀、單人寫

### 6. 條件變數
- **內容**: Condvar
- **說明**: 執行緒同步

### 7. 屏障
- **內容**: Barrier
- **說明**: 等待所有執行緒

### 8. 原子操作
- **內容**: AtomicUsize
- **說明**: lock-free 同步

### 9. 訊號量
- **內容**: Semaphore 實現
- **說明**: 控制並發數

### 10. 延遲初始化
- **內容**: OnceLock
- **說明**: 只執行一次

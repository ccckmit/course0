# 幾何學 - 計算幾何測試說明

## 執行方式

```bash
python 01-computational.py
```

## 測試項目

### 1. 凸包 - Graham Scan
- **輸入**: points=[(0,0),(1,0),(2,0),(1,1),(1,2),(0,2)]
- **預期輸出**: 凸包頂點列表
- **說明**: 求包含所有點的最小凸多邊形

### 2. 凸包 - Jarvis March
- **輸入**: 同上
- **預期輸出**: 同 Graham Scan
- **說明**: 另一種凸包演算法

### 3. 點在多邊形內
- **輸入**: 多邊形 [(0,0),(4,0),(4,3),(0,3)], 測試點
- **預期輸出**: True/False
- **說明**: 射線法

### 4. 點到直線距離
- **輸入**: 點 (1,1), 直線 (0,0)-(2,0)
- **預期輸出**: 1.0
- **說明**: 垂直距離

### 5. 點旋轉
- **輸入**: 點 (1,0), 旋轉 90 度
- **預期輸出**: (0,1)
- **說明**: 逆時針旋轉


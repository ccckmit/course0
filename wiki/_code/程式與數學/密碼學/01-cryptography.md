# 密碼學範例測試文件

## 檔案位置
- 程式碼: `_code/程式與數學/密碼學/01-cryptography.py`
- 測試文件: `_code/程式與數學/密碼學/01-cryptography.md`

## 執行方式
```bash
cd _code/程式與數學/密碼學
python 01-cryptography.py
```

## 測試項目

### 1. 模運算
- [ ] 模指數運算正確計算
- [ ] 模逆元計算正確
- [ ] 最大公因數計算正確

### 2. RSA 公鑰加密
- [ ] 金鑰生成成功
- [ ] 加密後的密文解密正確
- [ ] 驗證解密後訊息與原始訊息相同

### 3. AES SubBytes
- [ ] S-Box 轉換正確
- [ ] AddRoundKey (XOR) 運算正確

### 4. 雜湊函數
- [ ] SHA-256 產生 64 字元雜湊值
- [ ] 不同訊息產生不同雜湊
- [ ] 相同訊息每次產生相同雜湊

### 5. 雜湊碰撞
- [ ] 簡單雜湊函數可找到碰撞
- [ ] SHA-256 一致性驗證通過

### 6. 數位簽章
- [ ] 訊息簽章產生成功
- [ ] 正確驗證原始訊息的簽章
- [ ] 篡改後驗證失敗

### 7. Diffie-Hellman 金鑰交換
- [ ] 雙方公鑰生成成功
- [ ] 共享金鑰計算正確
- [ ] 雙方金鑰匹配

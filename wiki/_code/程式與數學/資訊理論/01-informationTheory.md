## 測試項目

### 1. 香農熵
- [ ] 公平硬幣 (0.5, 0.5) 熵為 1 bit
- [ ] 二元熵函數在 p=0.5 時最大

### 2. 交叉熵與 KL 散度
- [ ] H(P, Q) = H(P) + D(P||Q) 關係成立
- [ ] 當 P=Q 時，D(P||Q) = 0

### 3. 互資訊
- [ ] I(X;Y) = H(X) - H(X|Y)
- [ ] I(X;Y) = I(Y;X) (對稱性)

### 4. 霍夫曼編碼
- [ ] 編碼可正確解碼
- [ ] 平均碼長 ≥ 熵
- [ ] 編碼效率高

### 5. 夏農-哈特利定理
- [ ] 頻寬和 SNR 越大，容量越大

### 6. 編碼效率
- [ ] 效率 η = H/L ≤ 1

### 7. 漢明距離
- [ ] 漢明距離可檢測 d-1 個錯誤
- [ ] 可修正 d/2 個錯誤

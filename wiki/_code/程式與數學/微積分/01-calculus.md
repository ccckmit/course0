## 執行方式
```bash
cd _code/數學/微積分
python 01-calculus.py
```

## 測試項目

### 1. 符號微積分
- [ ] x² 的導數為 2x
- [ ] sin(x) 的導數為 cos(x)
- [ ] eˣ 的積分為 eˣ
- [ ] log(x) 的積分為 x*log(x) - x
- [ ] lim(x→0) sin(x)/x = 1

### 2. 數值微分
- [ ] 中央差分精度最高
- [ ] 各方法的誤差在合理範圍內

### 3. 數值積分
- [ ] 梯形法則近似正確
- [ ] 辛普森法則精度最高
- [ ] Romberg 積分收斂快速

### 4. 微分方程求解
- [ ] dy/dx = -0.5y, y(0)=1 的精確解為 e^(-0.5x)
- [ ] RK4 方法比尤拉法精度高

### 5. 梯度下降
- [ ] x²+2x+1 的最小值在 x = -1
- [ ] x² + y² 的最小值在 (0, 0)

### 6. 偏導數
- [ ] ∂f/∂x 和 ∂f/∂y 計算正確

### 7. 泰勒級數
- [ ] eˣ 的泰勒級數正確展開

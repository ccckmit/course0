## 測試項目

### 1. DFT vs FFT 比較
- [ ] 執行後顯示 DFT 和 FFT 結果相同
- [ ] 驗證 FFT 結果正確性

### 2. 信號分解
- [ ] 成功生成 10Hz 和 50Hz 的混合信號
- [ ] 正確檢測到頻率峰值

### 3. FFT 濾波
- [ ] 原始信號被正確恢復
- [ ] 濾波後的信號接近原始信號

### 4. 信號壓縮
- [ ] 不同壓縮比的誤差計算正確
- [ ] 壓縮比越高，誤差越大

### 5. 方波分解
- [ ] 奇次諧波幅度不為零
- [ ] 符合方波傅立葉級數特性

### 6. 相位譜分析
- [ ] 相位值在合理範圍內 (-π 到 π)
- [ ] 角度轉換正確

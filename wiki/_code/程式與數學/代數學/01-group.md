# 代數學 - 群論測試說明

## 執行方式

```bash
python 01-group.py
```

## 測試項目

### 1. CyclicGroup - 循環群 Z_n
- **輸入**: n=5
- **預期輸出**: 
  - Z_5 是阿貝爾群: True
  - Z_5 的階: 5
  - Z_5 中 2 的階: 5
  - Z_5 中每個元素的反元素: {0: 0, 1: 4, 2: 3, 3: 2, 4: 1}
- **說明**: 驗證循環群的性質

### 2. SymmetricGroup - 對稱群 S_n
- **輸入**: n=3
- **預期輸出**: 
  - S_3 是阿貝爾群: False
  - S_3 的階: 6
- **說明**: 置換群不是阿貝爾群

### 3. Field - 有限體 GF(p)
- **輸入**: p=7
- **預期輸出**: 
  - GF(7) 加法單位: 0
  - GF(7) 乘法單位: 1
- **說明**: 質數階有限體


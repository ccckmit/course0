# 梯度下降測試說明

## 執行方式

```bash
python 01-gradientDescent.py
```

## 測試項目

### 1. 梯度下降 - x² 最小化
- **輸入**: f(x) = x², x0 = 10
- **預期輸出**: x ≈ 0, f(x) ≈ 0
- **說明**: 標準梯度下降

### 2. Momentum - 動量法
- **輸入**: x0 = 10
- **預期輸出**: 收斂更快
- **說明**: 加入動量加速

### 3. Adam - 自適應學習率
- **輸入**: x0 = 10
- **預期輸出**: 穩定收斂
- **說明**: 结合动量和 RMSprop

### 4. SGD - 線性回歸
- **輸入**: 訓練數據
- **預期輸出**: 學習到的權重
- **說明**: 隨機梯度下降

# 分類器測試說明

## 執行方式

```bash
python 01-classifier.py
```

## 測試項目

### 1. KNN - K 近鄰分類器
- **輸入**: X=[[1,2],[2,3],...], y=[0,0,0,1,1,1]
- **預期輸出**: 根據 K 近鄰投票分類
- **說明**: 基於距離的分類

### 2. Naive Bayes - 樸素貝葉斯
- **輸入**: 同上
- **預期輸出**: 貝葉斯分類
- **說明**: 基於機率的分類

### 3. Decision Tree - 決策樹
- **輸入**: 同上
- **預期輸出**: 決策樹分類
- **說明**: 基於資訊增益分裂

## 預期輸出範例

```
=== KNN ===
預測 [[2, 2], [5, 5]]: [0, 1]

=== Naive Bayes ===
預測 [[2, 2], [5, 5]]: [0, 1]

=== Decision Tree ===
預測 [[2, 2], [5, 5]]: [0, 1]
```

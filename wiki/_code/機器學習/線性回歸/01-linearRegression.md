# 線性回歸測試說明

## 執行方式

```bash
python 01-linearRegression.py
```

## 測試項目

### 1. 簡單線性回歸
- **輸入**: X=[1,2,3,4,5], y=[2,4,5,4,5]
- **預期輸出**: 權重, 偏差, 預測值
- **說明**: 最小二乘法閉式解

### 2. 多元線性回歸
- **輸入**: 多特徵數據
- **預期輸出**: 權重向量
- **說明**: 使用矩陣運算

### 3. 嶺回歸
- **輸入**: 加入 L2 正則化的數據
- **預期輸出**: 正則化後的權重
- **說明**: 防止過擬合

## 預期輸出範例

```
=== 簡單線性回歸 ===
權重: 0.6000
偏差: 2.2000
預測 X=[6, 7]: y=[5.8, 6.4]
MSE: 0.7000
R²: 0.4000

=== 多元線性回歸 ===
權重: [1. 1.]
```

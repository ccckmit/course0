# 分治法測試說明

## 執行方式

```bash
python 01-divideConquer.py
```

## 測試項目

### 1. merge_sort - 合併排序
- **輸入**: [64, 34, 25, 12, 22, 11, 90]
- **預期輸出**: [11, 12, 22, 25, 34, 64, 90]
- **說明**: 經典分治排序

### 2. quick_sort - 快速排序
- **輸入**: [64, 34, 25, 12, 22, 11, 90]
- **預期輸出**: [11, 12, 22, 25, 34, 64, 90]
- **說明**: 分區 + 遞迴

### 3. binary_search - 二分搜尋
- **輸入**: arr=[1,2,3,4,5,6,7,8,9], target=5
- **預期輸出**: 4
- **說明**: 分治經典應用

### 4. power - 快速幂
- **輸入**: base=2, exp=10
- **預期輸出**: 1024
- **說明**: 二分幂運算

### 5. closest_pair - 最近點對
- **輸入**: points=[(2,3),(12,30),(40,50),(5,1),(12,10),(3,4)]
- **預期輸出**: 最近點對及其距離
- **說明**: 分治幾何問題

### 6. find_kth_smallest - 第k小元素
- **輸入**: arr=[7,10,4,3,20,15], k=3
- **預期輸出**: 7 (第3小的元素)
- **說明**: 快速選擇演算法

## 預期輸出範例

```
合併排序: [11, 12, 22, 25, 34, 64, 90]
快速排序: [11, 12, 22, 25, 34, 64, 90]
二分搜尋 5: 4
2^10 = 1024
最近點對: (2, 3) 和 (3, 4), 距離: 1.41
第3小: 7
```

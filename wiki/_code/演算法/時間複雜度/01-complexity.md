# 時間複雜度範例測試文件

## 檔案位置
- 程式碼: `_code/演算法/時間複雜度/01-complexity.py`
- 測試文件: `_code/演算法/時間複雜度/01-complexity.md`

## 執行方式
```bash
cd _code/演算法/時間複雜度
python 01-complexity.py
```

## 測試項目

### 1. O(1) 常數時間
- [ ] 取得陣列第一個元素
- [ ] 時間不隨 n 增長

### 2. O(log n) 對數時間
- [ ] 二分搜尋正確找到目標
- [ ] 執行時間遠小於線性搜尋

### 3. O(n) 線性時間
- [ ] 線性搜尋正確找到目標

### 4. O(n log n) 線性對數時間
- [ ] 合併排序結果正確
- [ ] 比 O(n²) 快

### 5. O(n²) 平方時間
- [ ] 氣泡排序結果正確

### 6. O(2ⁿ) 指數時間
- [ ] 費波那契數列計算正確

### 7. 複雜度比較
- [ ] 各複雜度成長率正確顯示

### 8. 實際執行時間
- [ ] 資料量越大，差異越明顯

# 暴力法測試說明

## 執行方式

```bash
python 01-bruteForce.py
```

## 測試項目

### 1. brute_force_search - 字串搜尋
- **輸入**: text = "Hello, World!", pattern = "World"
- **預期輸出**: 位置 = 7
- **說明**: 在 "Hello, World!" 中找 "World" 第一次出現的位置

### 2. brute_force_max - 找最大值
- **輸入**: [3, 1, 4, 1, 5, 9, 2, 6]
- **預期輸出**: (最大值: 9, 索引: 5)
- **說明**: 線性掃描找最大值

### 3. brute_force_two_sum - 兩數之和
- **輸入**: nums = [2, 7, 11, 15], target = 9
- **預期輸出**: [(0, 1)]  # 2 + 7 = 9
- **說明**: 找出所有和為 target 的索引對

### 4. brute_force_subset - 子集列舉
- **輸入**: items = ['a', 'b', 'c']
- **預期輸出**: 8 個子集（包括空集）
- **說明**: 使用位元遮罩列舉所有子集

### 5. brute_force_permutation - 全排列
- **輸入**: items = [1, 2, 3]
- **預期輸出**: 6 種排列
- **說明**: 遞迴生成所有排列

## 複雜度分析

| 演算法 | 時間複雜度 | 空間複雜度 |
|--------|-----------|-----------|
| 搜尋 | O(n*m) | O(1) |
| 找最大值 | O(n) | O(1) |
| 兩數之和 | O(n^2) | O(1) |
| 子集 | O(2^n * n) | O(2^n) |
| 排列 | O(n!) | O(n) |

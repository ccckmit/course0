# 回溯法測試說明

## 執行方式

```bash
python 01-backtracking.py
```

## 測試項目

### 1. n_queens - N皇后問題
- **輸入**: n=4
- **預期輸出**: 2 種解法
- **說明**: 4x4棋盤的皇后擺放

### 2. sudoku_solve - 數獨求解
- **輸入**: 標準數獨盤面
- **預期輸出**: 解答後的盤面
- **說明**: 使用回溯求解

### 3. generate_parentheses - 括號生成
- **輸入**: n=3
- **預期輸出**: 5 種有效組合
- **說明**: 生成所有有效的括號組合

### 4. permutation_backtrack - 全排列
- **輸入**: [1,2,3]
- **預期輸出**: 6 種排列
- **說明**: 回溯生成排列

### 5. combination_sum - 組合總和
- **輸入**: candidates=[2,3,6,7], target=7
- **預期輸出**: 所有和為7的組合
- **說明**: 元素可重複使用

### 6. word_search - 單詞搜索
- **輸入**: 網格 "ABCE"+"SFCS"+"EEDE"+"EAE"+"E", word="ABCCED"
- **預期輸出**: True
- **說明**: 在網格中找單詞路徑

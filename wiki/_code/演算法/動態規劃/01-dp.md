# 動態規劃測試說明

## 執行方式

```bash
python 01-dp.py
```

## 測試項目

### 1. fibonacci_dp - 費波那契數列
- **輸入**: n=10
- **預期輸出**: 55
- **說明**: DP 避免重疊子問題

### 2. coin_change_dp - 硬幣找零
- **輸入**: amount=11, coins=[1,2,5]
- **預期輸出**: 3 (5+2+2+2 或 5+5+1)
- **說明**: 完全背包問題

### 3. longest_common_subsequence - LCS
- **輸入**: s1="AGGTAB", s2="GXTXAYB"
- **預期輸出**: 4 ("GTAB")
- **說明**: 最長公共子序列

### 4. edit_distance - 編輯距離
- **輸入**: s1="sunday", s2="saturday"
- **預期輸出**: 3
- **說明**: Levenshtein 距離

### 5. knapsack_01 - 0/1背包
- **輸入**: values=[60,100,120], weights=[10,20,30], capacity=50
- **預期輸出**: 220 (選擇物品2和3)
- **說明**: 每物品只能選一次

### 6. longest_increasing_subsequence - LIS
- **輸入**: arr=[10,9,2,5,3,7,101,18]
- **預期輸出**: 4 ([2,3,7,101] 或 [2,3,7,18])
- **說明**: 最長遞增子序列

## 預期輸出範例

```
fib(10) = 55
fib(10) optimized = 55
硬幣找零 11: 3
LCS('AGGTAB', 'GXTXAYB') = 4
編輯距離('sunday', 'saturday') = 3
背包最大價值: 220
LIS: 4
```

# 排序演算法測試說明

## 執行方式

```bash
python 01-sorting.py
```

## 測試項目

### 1. bubble_sort - 氣泡排序
- **輸入**: [64, 34, 25, 12, 22, 11, 90]
- **預期輸出**: [11, 12, 22, 25, 34, 64, 90]
- **說明**: 相鄰元素比較交換

### 2. insertion_sort - 插入排序
- **輸入**: [64, 34, 25, 12, 22, 11, 90]
- **預期輸出**: [11, 12, 22, 25, 34, 64, 90]
- **說明**: 類似整理扑克牌

### 3. selection_sort - 選擇排序
- **輸入**: [64, 34, 25, 12, 22, 11, 90]
- **預期輸出**: [11, 12, 22, 25, 34, 64, 90]
- **說明**: 每次選擇最小元素

### 4. merge_sort - 合併排序
- **輸入**: [64, 34, 25, 12, 22, 11, 90]
- **預期輸出**: [11, 12, 22, 25, 34, 64, 90]
- **說明**: 分治+合併，穩定排序

### 5. quick_sort - 快速排序
- **輸入**: [64, 34, 25, 12, 22, 11, 90]
- **預期輸出**: [11, 12, 22, 25, 34, 64, 90]
- **說明**: 分區基準排序

### 6. heap_sort - 堆排序
- **輸入**: [64, 34, 25, 12, 22, 11, 90]
- **預期輸出**: [11, 12, 22, 25, 34, 64, 90]
- **說明**: 使用最大堆

### 7. counting_sort - 計數排序
- **輸入**: [2, 5, 3, 1, 4, 2, 3]
- **預期輸出**: [1, 2, 2, 3, 3, 4, 5]
- **說明**: 非比較排序，適合小範圍整數

### 8. radix_sort - 基數排序
- **輸入**: [329, 457, 657, 839, 436, 720, 355]
- **預期輸出**: [329, 355, 436, 457, 657, 720, 839]
- **說明**: 按位元排序

### 9. bucket_sort - 桶排序
- **輸入**: [0.78, 0.17, 0.39, 0.26, 0.72, 0.94, 0.21, 0.12, 0.23, 0.68]
- **預期輸出**: 排序後的浮點數
- **說明**: 均勻分布時效果好

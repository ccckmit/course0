# 形式文法 (Formal Grammar)

## 概述

形式文法是描述語言的數學工具，定義瞭如何產生合法的字串。

## 形式定義

```
G = (N, Σ, P, S)

N  - 非終結符集合 (Non-terminals)
Σ  - 終結符集合 (Terminals)
P  - 產生式規則集合 (Productions)
S  - 起始符號 (Start Symbol)
```

## 產生式規則

形式：`α → β`

範例：
```
S → aSb
S → ε
```

## Chomsky 語言階層

| 層次 | 名稱 | 規則形式 | 自動機 |
|------|------|----------|--------|
| Type 0 | 無限制 | α → β (α ≠ ε) | 圖靈機 |
| Type 1 | 上下文相關 | αAβ → αγβ | 線性有界自動機 |
| Type 2 | 上下文無關 | A → γ | 下推自動機 |
| Type 3 | 正規 | A → a, A → aB | 有限狀態機 |

## 程式實作

```python
from random import randrange

def gen(G):
    lefts = list(G.keys())
    rule = " S "
    
    while True:
        for left0 in lefts:
            left = f' {left0} '
            i = rule.find(left)
            if i != -1:
                rights = G[left0]
                ri = randrange(0, len(rights))
                right = f' {rights[ri]} '
                rule = rule[0:i] + right + rule[i+len(left):]
                print(rule)
                
                if rule == rule.lower():
                    return rule
                break
```

## 範例：aⁿbⁿ 語言

```python
G = {
    "S": ["a S b", "a b"]  # 遞迴產生對稱的 a...b
}
```

產生過程：
```
S
a S b
a a b b
a a a b b b
```

## 範例：aⁿbⁿcⁿ（上下文相關）

```python
G = {
    "S": ["a B C", "a S B C"],
    "C B": ["C Z"],
    "C Z": ["W Z"],
    "W Z": ["W C"],
    "W C": ["B C"],
    "a B": ["a b"],
    "b B": ["b b"],
    "b C": ["b c"],
    "c C": ["c c"]
}
```

## 範例：英語句子

```python
G = {
    "S": ["NP VP"],
    "NP": ["D N", "D A N"],
    "VP": ["V NP"],
    "N": ["dog", "cat"],
    "V": ["eat", "chase"],
    "D": ["a", "the"],
    "A": ["black", "white", "little"]
}
```


## 自上而下 vs 自下而上解析

| 方法 | 策略 | 問題 |
|------|------|------|
| LL(1) | 自上而下、遞歸下降 | 左遞迴 |
| LR(1) | 自下而上、移位歸約 | 建表複雜 |

# 直譯器 (Interpreter)

## 概述

直譯器是一種程式執行方式，直接閱讀並執行原始程式碼，不需要事先編譯成機器碼。

## 直譯器 vs 編譯器

| 特性 | 直譯器 | 編譯器 |
|------|--------|--------|
| 執行方式 | 逐行翻譯執行 | 完整翻譯後執行 |
| 產出 | 無獨立執行檔 | 機器碼/位元組碼 |
| 執行速度 | 較慢 | 較快 |
| 跨平臺 | 需原始碼 | 需重新編譯 |
| 範例 | Python, JS, Ruby | C, Go, Rust |

## 直譯器架構

```
原始碼
   ↓
┌─────────────────┐
│  詞彙分析器      │  Tokenizer / Lexer
│  (Lexical)      │
└────────┬────────┘
         ↓
┌─────────────────┐
│  語法分析器      │  Parser
│  (Syntax)       │
└────────┬────────┘
         ↓
┌─────────────────┐
│  語意分析器      │  Semantic Analyzer
│  (Semantic)     │
└────────┬────────┘
         ↓
┌─────────────────┐
│     AST         │  抽象語法樹
│(Abstract Syntax) │
└────────┬────────┘
         ↓
┌─────────────────┐
│     執行器      │  Evaluator / Executor
│                 │
└─────────────────┘
```

## 詞彙分析 (Lexing)

```python
import re

class Token:
    def __init__(self, type, value):
        self.type = type
        self.value = value

class Lexer:
    TOKEN_TYPES = [
        ('NUMBER', r'\d+'),
        ('PLUS', r'\+'),
        ('MINUS', r'-'),
        ('MULT', r'\*'),
        ('DIV', r'/'),
        ('LPAREN', r'\('),
        ('RPAREN', r'\)'),
        ('NAME', r'[a-zA-Z_]\w*'),
    ]
    
    def tokenize(self, text):
        tokens = []
        pos = 0
        while pos < len(text):
            match = None
            for type, pattern in self.TOKEN_TYPES:
                match = re.match(pattern, text[pos:])
                if match:
                    tokens.append(Token(type, match.group()))
                    pos += match.end()
                    break
            if not match:
                if not text[pos].isspace():
                    raise SyntaxError(f"Unexpected character: {text[pos]}")
                pos += 1
        return tokens
```

## 語法分析 (Parsing)

### 遞迴下降解析器

```python
class Parser:
    def __init__(self, tokens):
        self.tokens = tokens
        self.pos = 0
    
    def parse(self):
        return self.parse_expression()
    
    def parse_expression(self):
        left = self.parse_term()
        while self.current().type in ('PLUS', 'MINUS'):
            op = self.current().type
            self.advance()
            right = self.parse_term()
            left = ('binop', op, left, right)
        return left
    
    def parse_term(self):
        left = self.parse_factor()
        while self.current().type in ('MULT', 'DIV'):
            op = self.current().type
            self.advance()
            right = self.parse_factor()
            left = ('binop', op, left, right)
        return left
    
    def current(self):
        return self.tokens[self.pos]
    
    def advance(self):
        self.pos += 1
```

## AST 節點定義

```python
class NumberNode:
    def __init__(self, value):
        self.value = value

class BinOpNode:
    def __init__(self, left, op, right):
        self.left = left
        self.op = op
        self.right = right

class VarNode:
    def __init__(self, name):
        self.name = name

class AssignNode:
    def __init__(self, name, value):
        self.name = name
        self.value = value

class IfNode:
    def __init__(self, condition, then_branch, else_branch):
        self.condition = condition
        self.then_branch = then_branch
        self.else_branch = else_branch

class WhileNode:
    def __init__(self, condition, body):
        self.condition = condition
        self.body = body

class FuncDefNode:
    def __init__(self, name, params, body):
        self.name = name
        self.params = params
        self.body = body

class FuncCallNode:
    def __init__(self, name, args):
        self.name = name
        self.args = args
```

## 環境管理

```python
class Environment:
    def __init__(self, parent=None):
        self.vars = {}
        self.parent = parent
        self.functions = {}
    
    def get(self, name):
        if name in self.vars:
            return self.vars[name]
        if self.parent:
            return self.parent.get(name)
        raise NameError(name)
    
    def set(self, name, value):
        self.vars[name] = value
    
    def define(self, name, value):
        self.vars[name] = value
```

## 執行器 (Evaluator)

```python
class Evaluator:
    def __init__(self):
        self.env = Environment()
    
    def eval(self, node):
        if isinstance(node, NumberNode):
            return float(node.value)
        
        if isinstance(node, BinOpNode):
            left = self.eval(node.left)
            right = self.eval(node.right)
            if node.op == '+': return left + right
            if node.op == '-': return left - right
            if node.op == '*': return left * right
            if node.op == '/': return left / right
        
        if isinstance(node, VarNode):
            return self.env.get(node.name)
        
        if isinstance(node, AssignNode):
            value = self.eval(node.value)
            self.env.define(node.name, value)
            return value
        
        if isinstance(node, IfNode):
            if self.eval(node.condition):
                return self.eval(node.then_branch)
            elif node.else_branch:
                return self.eval(node.else_branch)
        
        if isinstance(node, WhileNode):
            result = None
            while self.eval(node.condition):
                result = self.eval(node.body)
            return result
        
        if isinstance(node, FuncDefNode):
            self.env.functions[node.name] = node
            return None
        
        if isinstance(node, FuncCallNode):
            func = self.env.functions[node.name]
            args = [self.eval(arg) for arg in node.args]
            return self.apply(func, args)
        
        raise SyntaxError(f"Unknown node type: {type(node)}")
```

## 直譯器類型

| 類型 | 說明 | 範例 |
|------|------|------|
| Tree-walking | 建立 AST 後遍歷執行 | JS 直譯器 |
| Bytecode VM | 編譯到位元組碼後執行 | Python, JVM |
| JIT | 運行時編譯熱點 | V8, HotSpot |
| Threaded | 直接執行 tokens | 簡單直譯器 |

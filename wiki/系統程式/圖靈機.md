# 圖靈機 (Turing Machine)

## 概述

圖靈機是 Alan Turing 在 1936 年提出的抽象計算模型，被認為是電腦科學的理論基石。

## 形式定義

圖靈機 M = (Q, Σ, Γ, δ, q₀, q_accept, q_reject)

| 符號 | 意義 |
|------|------|
| Q | 狀態集合 |
| Σ | 輸入字母表 |
| Γ | 磁帶字母表（含空白 □） |
| δ | 轉移函數 |
| q₀ | 起始狀態 |
| q_accept | 接受狀態 |
| q_reject | 拒絕狀態 |

## 基本結構

```
┌─────────────────────────────────────────────────────────┐
│                      圖靈機                               │
├─────────────────────────────────────────────────────────┤
│  • 無限長度的磁帶 (Tape)                                 │
│  • 讀寫頭 (Head) - 可讀取、寫入、移動                    │
│  • 狀態寄存器 (State Register)                           │
│  • 轉移函數 (Transition Function)                       │
└─────────────────────────────────────────────────────────┘
```

## 轉移函數

```
δ(q, a) = (p, b, L/R)

當處於狀態 q、讀到符號 a 時，
轉移到狀態 p，寫入 b，向左(L)或向右(R)移動
```

## Python 實作

```python
class TuringMachine:
    def __init__(self, states, ichars, ochars, actionMap, start, accepts, rejects):
        self.states = states
        self.ichars = ichars
        self.ochars = ochars
        self.actionMap = actionMap
        self.start = start
        self.accepts = accepts
        self.rejects = rejects
    
    def run(self, tape):
        self.tape = list(tape + '________')
        state = self.start
        i = 0
        
        while True:
            ichar = self.tape[i]
            acts = self.actionMap.get(f'{state},{ichar}')
            if acts is None: return False
            
            state, ochar, move = acts
            self.tape[i] = ochar
            
            if move == 'L': i -= 1 if i > 0 else 0
            elif move == 'R': i += 1
            
            if state in self.accepts: return True
            if state in self.rejects: return False
```

## 範例：aⁿbⁿcⁿ 語言

```python
tm = TuringMachine(
    states=['0','1','2','3','4','5'],
    ichars=['a','b','c'],
    ochars=['a','b','c','x','_'],
    actionMap={
        '0,a': '1,_,R',
        '1,a': '1,a,R',
        '1,b': '2,x,R',
        '2,c': '5,x,L',
        '5,_': '0,_,R',
        '0,_': '3,_,L',
        '4,_': '3,_,L',
        '0,x': '4,x,R',
    },
    start='0',
    accepts=['3'],
    rejects=[]
)
```

## 圖靈機 vs DFA

| 特性 | DFA | 圖靈機 |
|------|-----|--------|
| 磁帶 | 無 | 無限 |
| 移動 | 只能前進 | 左/右/靜止 |
| 寫入 | 否 | 是 |
| 計算能力 | 正規語言 | 任何可計算問題 |

## 停機問題

> 不存在一個圖靈機 H，能判斷任意圖靈機 M 在輸入 w 上是否會停止。

**證明（反證法）:**
1. 假設存在 H(M, w)
2. 構造悖論機器 P
3. 問 P(P) 是否停機？導致矛盾

## Church-Turing 論點

> 任何可計算的函數都可以由圖靈機計算。

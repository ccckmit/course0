# Noam Chomsky

Noam Chomsky（諾姆·喬姆斯基，1928 年 12 月 7 日出生）是一位美國語言學家、哲學家、認知科學家和政治活動家，被認為是二十世紀最重要的思想家之一。他在語言學、認知科學、資訊理論和哲學領域都做出了深遠的貢獻。

## 生平背景

### 早年與教育

Chomsky 出生於美國費城的一個猶太家庭。父親 William Chomsky 是一位希伯來語學者，母親 Elsie Chomsky 是一位教育工作者。Chomsky 在費城成長，早年受到父親對希伯來語和猶太經典研究的影響。

1945 年，Chomsky 進入賓夕法尼亞大學學習，主修語言學和哲學。在大學期間，他接觸到了結構語言學，並對當時主流的描寫語言學感到不滿。他認為語言學應該解釋語言的生成能力，而非僅僅描寫語言現象。

1951 年，Chomsky 進入哈佛大學研究生院攻讀語言學碩士學位，並於 1955 年在賓夕法尼亞大學獲得博士學位。他的博士論文《言辭的邏輯結構》奠定了他後來革命性語言學理論的基礎。

### 學術生涯

Chomsky 自 1955 年起在麻省理工學院（MIT）任教，先後擔任助理教授、副教授和教授。他在 MIT 度過了超過半個世紀的學術生涯，培養了無數語言學家和認知科學家。

1970 年代，Chomsky 開始將研究重心轉向語法理論的更深層次問題，發展了，最小化計算（Minimalist Program）的框架。他的語法理論對計算語言學、自然語言處理和人工智慧領域產生了重大影響。

## 語言學貢獻

### 轉換生成語法

Chomsky 最著名的學術貢獻是**轉換生成語法**（Transformational-Generative Grammar，簡稱 TG 或 TGG）。這一套理論在 1957 年發表的《句法結構》中首次系統性地提出，並在此後的半個世紀中不斷發展和完善。

#### 核心概念

1. **通用語法**（Universal Grammar，UG）：Chomsky 認為人類天生具有一種語言獲得裝置（Language Acquisition Device， LAD），這使得兒童能夠在有限的輸入下快速學會語言。通用語法假設所有人類語言共享一套核心原則和參數。

2. **表層結構與深層結構**：語句有兩層結構——表層結構（Surface Structure）是實際說出或寫出的形式，深層結構（Deep Structure）則是語義解釋的來源。轉換規則連接這兩層結構。

3. **轉換規則**：轉換規則是將深層結構轉換為表層結構的語法操作。例如，主動句和被動句之間的轉換就是一個典型的轉換規則。

#### 發展階段

1. **古典理論**（1957-1965）：《句法結構》提出的早期理論，側重於短語結構規則和轉換規則。

2. **標準理論**（1965-1971）：《句法理論的若干問題》中提出的擴展版，引入了深層結構和表層結構的區分。

3. **擴展標準理論**（1972-1977）：增加了管轄約束理論（Government and Binding Theory，GB Theory），將語法原則模組化。

4. **最小化計算**（1993-至今）：最新的理論框架，追求以最少的假設解釋語言現象。

### 語言障礙研究

Chomsky 對失語症和語言障礙的研究也做出了重要貢獻。他與神經科學家 Eric Lenneberg 合作，探討了語言能力的生物學基礎。這些研究支持了他的先天論觀點。

## 計算語言學影響

### 形式語法與自然語言處理

Chomsky 的形式語法理論對計算語言學和自然語言處理（NLP）產生了深遠影響：

1. **上下文無關文法**（Context-Free Grammar，CFG）：Chomsky 階層中的第二類文法成為描述自然語言語法的基本工具。

2. **編譯器理論**：形式語法理論直接影響了編譯器和解析器的設計，許多解析算法（如 CYK 算法、EARLEY 算法）都源自於此。

3. **語法分析器生成器**：如 Yacc、Bison 等工具都基於 Chomsky 的理論設計。

### 實例：Chart Parser

```python
# 基於 Chart Parsing 的語法分析算法概念
class ChartParser:
    def __init__(self, grammar):
        self.grammar = grammar
        self.chart = []  # 邊緣集合
        
    def parse(self, sentence):
        # 初始化 chart
        self.chart = [[] for _ in range(len(sentence) + 1)]
        
        # 添加初始假設
        self.chart[0].append(Edge('S', 0, 0))
        
        # 迭代擴展
        while self.has_incomplete_edges():
            # 預測規則
            self.predict()
            # 掃描詞彙
            self.scan(sentence)
            # 完成規則
            self.complete()
        
        return self.chart[-1]
```

## 形式文法與 Chomsky 階層

### Chomsky 語言階層

Chomsky 將形式語言分為四個層次，形成著名的 **Chomsky 階層**（Chomsky Hierarchy）：

| 層次 | 名稱 | 產生式規則 | 識別自動機 | 典型語言 |
|------|------|-----------|-----------|---------|
| Type 0 | 無限制文法 | α → β（α ≠ ε） | 圖靈機 | 任意遞迴可枚舉語言 |
| Type 1 | 上下文相關文法 | αAβ → αγβ | 線性有界自動機 | aⁿbⁿcⁿ |
| Type 2 | 上下文無關文法 | A → γ | 下推自動機 | aⁿbⁿ |
| Type 3 | 正規文法 | A → a, A → aB | 有限狀態機 | a*b |

### 產生式規則範例

```python
# 上下文無關文法範例：aⁿbⁿ
grammar_cf = {
    "S": ["a S b", "ε"]
}

# 上下文相關文法範例：aⁿbⁿcⁿ
grammar_cs = {
    "S": ["a B C", "a S B C"],
    "C B": ["C Z"],
    "C Z": ["W Z"],
    "W Z": ["W C"],
    "W C": ["B C"],
    "a B": ["a b"],
    "b B": ["b b"],
    "b C": ["b c"],
    "c C": ["c c"]
}
```

## 政治活動與公共知識分子

### 政治立場

自 1960 年代起，Chomsky 成為知名的政治活動家和公共知識分子。他自稱無政府主義者，批評美國外交政策和資本主義制度。

1. **越南戰爭**：Chomsky 是反對越南戰爭的主要聲音之一，與 Jean-Paul Sartre 共同編輯了《政治經濟學札記》。

2. **媒體批評**：1971 年發表的《製造共識》論述了媒體如何被權力機構操縱，這本書成為媒介批評的經典。

3. **美國外交政策**：他長期批評美國的中東政策、拉丁美洲干預和軍國主義。

### 代表性著作

1. 《句法結構》（Syntactic Structures，1957）
2. 《句法理論的若干問題》（Aspects of the Theory of Syntax，1965）
3. 《語言與心靈》（Language and Mind，1968）
4. 《製造共識》（Manufacturing Consent，1988，與 Edward Herman 合著）
5. 《9-11》（9-11，2001）

## 哲學觀點

### 認知主義

Chomsky 是認知革命的先驅之一。他認為語言學應該是認知科學的一部分，研究人類心智的內在機制。

1. **先天論**：語言能力部分是天生的，而非完全習得。
2. **心智主義**：語言學應該研究心智中的語法，而非僅僅觀察語言行為。
3. **最小主義**：語法理論應該追求簡潔和一般性。

### 自由意志問題

Chomsky 對決定論和自由意志也有深入思考。他認為即使物理世界是決定論的，人類仍可能擁有某種形式的心智自主性。

## 批評與爭議

### 語言學批評

1. **功能語言學**：M.A.K. Halliday 等功能語言學家批評 Chomsky 過度關注句法結構，忽視語言的功能和社會層面。

2. **認知語言學**：George Lakoff 等人批評 Chomsky 的天賦論過度簡化，語言學習需要更多的環境互動。

### 政治批評

Chomsky 的政治觀點也受到批評：
- 批評者認為他的分析過度意識形態化
- 一些批評者指責他對某些歷史事件（如紅色高棉）的立場

## 遺產與影響

### 學術遺產

1. **語言學**：轉換生成語法仍是語法理論的主流框架之一
2. **認知科學**：為認知科學的誕生做出了重要貢獻
3. **計算語言學**：形式語法理論繼續影響 NLP 領域
4. **人工智慧**：對語言理解和生成研究有深遠影響

### 文化影響

Chomsky 被《時代》雜誌評為「二十世紀最重要的人物」之一，他的思想和著作持續影響著語言學、哲學、政治學和媒體研究領域。

# c4 - C in Four Functions

## 概述

c4 是一個極簡的 C 語言編譯器，只用四個函式實現，展示了編譯器的核心原理。這個專案由 Robert Swier 創作，是理解編譯器運作原理的經典練習。

## 編譯與執行

```bash
# 編譯 c4 編譯器
gcc -o c4 c4.c

# 使用 c4 編譯 hello.c
./c4 hello.c

# 直接執行產生的組譯碼 (-s 參數)
./c4 -s hello.c

# 自己編譯自己
./c4 c4.c hello.c
./c4 c4.c c4.c hello.c
```

## 四個核心函式

c4 的設計哲學是將 C 編譯器簡化到最基本的功能：

### 1. 詞法分析 (Lexing)
```c
// 簡化的詞法分析器
// 識別：整數、字串、關鍵字、運算子
token next_token() {
    // 跳過空白
    skip_whitespace();
    
    // 識別數字（十六進位和十進位）
    if (isdigit(c)) return TOKEN_INT;
    
    // 識別字串
    if (c == '"') return TOKEN_STRING;
    
    // 識別關鍵字和識別符
    if (isalpha(c) || c == '_') return TOKEN_ID;
    
    // 運算子和標點符號
    return c;
}
```

### 2. 語法分析 (Parsing)
```c
// 遞迴下降解析器
// 支援：表達式、陳述式、函式定義
ast_node* parse() {
    // 解析表達式
    if (token == TOKEN_INT) {
        return parse_expression();
    }
    
    // 解析控制流程
    if (token == IF) return parse_if();
    if (token == WHILE) return parse_while();
    if (token == RETURN) return parse_return();
    
    // 解析函式定義
    if (token == INT) return parse_function();
}
```

### 3. 語意分析 (Semantic Analysis)
```c
// 簡化的語意分析
// 處理：類型檢查、作用域、符號表
symbol* lookup(char *name) {
    // 查詢符號表
    for (int i = 0; i < symbol_count; i++) {
        if (strcmp(symbols[i].name, name) == 0)
            return &symbols[i];
    }
    return NULL;
}

void check_types(ast_node *node) {
    // 確保左右類型相容
    if (node->left->type != node->right->type)
        error("type mismatch");
}
```

### 4. 程式碼生成 (Code Generation)
```c
// 生成組合語言輸出
// 目標架構：x86 (GCC 內聯組譯)
void emit_code(ast_node *node) {
    switch (node->type) {
        case ADD:
            emit("    addl    %1, %0", dest, src);
            break;
        case SUB:
            emit("    subl    %1, %0", dest, src);
            break;
        case CALL:
            emit("    call    %s", node->name);
            break;
        // ...
    }
}
```

## 支援的語法

### 基本類型
```c
int x;           // 整數變數
int arr[10];     // 陣列
int *ptr;        // 指標
```

### 控制流程
```c
if (x < y) {
    return x;
} else {
    return y;
}

while (x > 0) {
    x = x - 1;
}
```

### 函式
```c
int max(int a, int b) {
    if (a > b) {
        return a;
    }
    return b;
}
```

## 限制與簡化

c4 為了簡潔做出了以下限制：

| 限制 | 說明 |
|------|------|
| 單一檔案 | 不支援標頭檔或多重檔案 |
| 無結構體 | 不支援 struct |
| 無浮點數 | 只支援整數運算 |
| 有限優化 | 無任何優化 |
| x86 限制 | 只支援 32 位元 x86 |

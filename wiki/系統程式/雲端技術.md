# 雲端技術

雲端運算（Cloud Computing）是透過網際網路提供計算資源的服務模式，它徹底改變了資訊科技的交付方式。從傳統的自行建置和維護資料中心，到現在按需取用的雲端服務，雲端技術讓企業和開發者能夠更專注於業務創新，而非基礎設施管理。雲端技術的發展催生了雲端原生、容器化、微服務等現代軟體架構，成為當代資訊科技的支柱。

## 雲端運算的發展歷程

```
┌─────────────────────────────────────────────────────────────┐
│                  雲端運算發展時間線                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   1960s     分時共享 (Time Sharing)                         │
│             大型主機資源分配給多個使用者                    │
│                                                             │
│   1990s     網路服務 (ASP)                                   │
│             應用服務供應商提供軟體服務                       │
│                                                             │
│   2002     Amazon Web Services 推出                         │
│             S3, EC2 雲端服務正式商業化                      │
│                                                             │
│   2006     Google Cloud Platform 推出                      │
│             GAE (App Engine)                                │
│                                                             │
│   2008     Microsoft Azure 推出                            │
│                                                             │
│   2010s    雲端原生概念興起                                 │
│             容器化、微服務、DevOps                          │
│                                                             │
│   2020s    多雲/混合雲普及                                  │
│             邊緣運算、Serverless                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 雲端服務模型

```
┌─────────────────────────────────────────────────────────────┐
│                    雲端服務模型                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌───────────────────────────────────────────────────┐    │
│   │                   SaaS (軟體即服務)                 │    │
│   │   ┌─────────────────────────────────────────────┐  │    │
│   │   │            應用程式                          │  │    │
│   │   │            資料                             │  │    │
│   │   │            作業系統                         │  │    │
│   │   │            中間件                           │  │    │
│   │   │            執行時間                        │  │    │
│   │   └─────────────────────────────────────────────┘  │    │
│   │                   使用者                             │    │
│   └───────────────────────────────────────────────────┘    │
│                         │                                   │
│   ┌───────────────────────────────────────────────────┐    │
│   │                   PaaS (平臺即服務)                 │    │
│   │   ┌─────────────────────────────────────────────┐  │    │
│   │   │            執行時間                         │  │    │
│   │   │            中間件                           │  │    │
│   │   └─────────────────────────────────────────────┘  │    │
│   │         開發者 ←── 應用程式 + 資料                  │    │
│   └───────────────────────────────────────────────────┘    │
│                         │                                   │
│   ┌───────────────────────────────────────────────────┐    │
│   │                   IaaS (基礎設施即服務)            │    │
│   │   ┌─────────────────────────────────────────────┐  │    │
│   │   │            虛擬機器                         │  │    │
│   │   │            儲存                            │  │    │
│   │   │            網路                            │  │    │
│   │   └─────────────────────────────────────────────┘  │    │
│   │      系統管理員 ←── 作業系統 + 應用程式 + 資料       │    │
│   └───────────────────────────────────────────────────┘    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

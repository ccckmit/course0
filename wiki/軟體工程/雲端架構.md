# 雲端架構：現代雲計算服務

## 概述

雲端架構利用雲服務提供商的基礎設施來部署和運行應用程式。主要雲提供商包括 AWS、Google Cloud Platform 和 Microsoft Azure。雲原生架構強調利用雲的彈性和自動化特性。

## 雲端服務類型

```markdown
# 基礎設施即服務 (IaaS)
- EC2 (AWS), Compute Engine (GCP), VM (Azure)
- 完整的虛擬機控制

# 平台即服務 (PaaS)
- Elastic Beanstalk (AWS), App Engine (GCP), App Service (Azure)
- 自動化的部署平台

# 函數即服務 (FaaS / Serverless)
- Lambda (AWS), Cloud Functions (GCP), Azure Functions
- 按需執行的無伺服器函數

# 容器即服務 (CaaS)
- ECS/EKS (AWS), GKE (GCP), AKS (Azure)
- 容器編排和管理
```

## 總結

雲端架構提供了靈活的基礎設施選擇。選擇合適的服務層級需要考慮成本、控制力和运维复杂度。

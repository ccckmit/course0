# 效能優化 (Performance Optimization)

## 概述

效能優化是提升軟體系統執行效率、降低資源消耗的過程。目標是在合理的成本下提供最佳的使用者體驗和系統回應速度。

## 效能指標

### 前端指標
| 指標 | 說明 | 目標值 |
|------|------|--------|
| FCP | First Contentful Paint | < 1.8s |
| LCP | Largest Contentful Paint | < 2.5s |
| FID | First Input Delay | < 100ms |
| CLS | Cumulative Layout Shift | < 0.1 |
| TTFB | Time To First Byte | < 800ms |

### 後端指標
| 指標 | 說明 | 目標值 |
|------|------|--------|
| 回應時間 | 請求到回應的時間 | < 200ms |
| 吞吐量 | QPS (Queries Per Second) | 根據需求 |
| 錯誤率 | 失敗請求比例 | < 0.1% |
| 可用性 | uptime | > 99.9% |

### JavaScript 優化
```javascript
// 避免重複查詢 DOM
const button = document.getElementById('btn');

// 使用事件委託
document.querySelector('.list').addEventListener('click', (e) => {
    if (e.target.matches('.item')) {
        handleItemClick(e.target);
    }
});

// 防抖和節流
function debounce(fn, delay) {
    let timeout;
    return (...args) => {
        clearTimeout(timeout);
        timeout = setTimeout(() => fn(...args), delay);
    };
}

function throttle(fn, limit) {
    let inThrottle;
    return (...args) => {
        if (!inThrottle) {
            fn(...args);
            inThrottle = true;
            setTimeout(() => inThrottle = false, limit);
        }
    };
}
```

## 後端優化

### 資料庫優化
```sql
-- 建立適當的索引
CREATE INDEX idx_user_email ON users(email);
CREATE INDEX idx_order_date_status ON orders(created_at, status);

-- 查詢優化：避免 SELECT *
SELECT id, name, email FROM users WHERE active = 1;

-- 使用 EXPLAIN 分析查詢
EXPLAIN SELECT * FROM orders WHERE user_id = 123;
```

### 快取策略
```python
# Redis 快取範例
import redis
import json

r = redis.Redis(host='localhost', port=6379)

def get_user_profile(user_id):
    cache_key = f"user:{user_id}"
    
    # 嘗試從快取取得
    cached = r.get(cache_key)
    if cached:
        return json.loads(cached)
    
    # 從資料庫取得
    profile = db.query(f"SELECT * FROM users WHERE id = {user_id}")
    
    # 寫入快取，過期時間 5 分鐘
    r.setex(cache_key, 300, json.dumps(profile))
    
    return profile
```

## 演算法優化

### 複雜度分析
| 資料結構/演算法 | 時間複雜度 | 空間複雜度 |
|-----------------|------------|------------|
| 陣列查詢 | O(1) | O(n) |
| 雜湊表 | O(1) 平均 | O(n) |
| 二元搜尋 | O(log n) | O(1) |
| 合併排序 | O(n log n) | O(n) |
| 圖搜尋 (DFS/BFS) | O(V+E) | O(V) |

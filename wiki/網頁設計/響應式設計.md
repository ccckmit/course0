# 響應式設計

**標籤**: #響應式 #行動優先 #媒體查詢 #CSS
**日期**: 2024-01-15

響應式設計 (Responsive Web Design, RWD) 是一種網頁設計方法論，讓網頁能夠根據不同裝置的螢幕尺寸自動調整版面配置，提供最佳的使用者體驗。這個概念由 Ethan Marcotte 於 2010 年提出，並在行動裝置快速普及的時代變得不可或缺。

## 響應式設計的核心原則

### 彈性網格 (Flexible Grid)

傳統網頁使用固定像素寬度，而響應式設計使用相對單位（如百分比）來定義元素寬度，使版面配置能夠隨容器大小自動調整。

```css
/* 傳統固定寬度 */
.container {
    width: 960px;
}

/* 彈性網格 */
.container {
    width: 90%;
    max-width: 1200px;
    margin: 0 auto;
}

/* 計算彈性寬度 */
.column {
    width: calc(25% - 20px);
    margin: 10px;
}
```

### 彈性圖片 (Flexible Images)

圖片需要能夠縮放而不失真或溢出容器。

```css
/* 基本彈性圖片 */
img {
    max-width: 100%;
    height: auto;
}

/* 背景圖片彈性 */
.hero-bg {
    background-image: url('image.jpg');
    background-size: cover;
    background-position: center;
    min-height: 300px;
}

/* 圖片適配容器 */
figure img {
    width: 100%;
    height: auto;
    display: block;
}
```

### 媒體查詢 (Media Queries)

CSS 媒體查詢允許根據裝置特性應用不同的樣式規則，這是響應式設計的核心技術。

```css
/* 基礎樣式（行動優先） */
body {
    font-size: 16px;
    line-height: 1.5;
    padding: 10px;
}

h1 {
    font-size: 24px;
}

/* 平板及以上 */
@media screen and (min-width: 768px) {
    body {
        padding: 20px;
    }
    
    h1 {
        font-size: 32px;
    }
}

/* 桌面及以上 */
@media screen and (min-width: 1024px) {
    body {
        padding: 40px;
    }
    
    h1 {
        font-size: 48px;
    }
}

/* 大螢幕 */
@media screen and (min-width: 1440px) {
    body {
        max-width: 1400px;
        margin: 0 auto;
    }
}
```

## Viewport 設定

正確設定 viewport 是實現響應式設計的第一步。

```html
<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    
    <!-- 基本的 viewport 設定 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
    <!-- 防止行動裝置自動縮放（通常不推薦） -->
    <!-- <meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no"> -->
    
    <title>響應式網頁</title>
</head>
<body>
    <!-- 網頁內容 -->
</body>
</html>
```

### viewport 屬性說明

| 屬性 | 說明 | 範例值 |
|------|------|--------|
| width | 視窗寬度 | 320, 480, device-width |
| height | 視窗高度 | 480, device-height |
| initial-scale | 初始縮放比例 | 0.5, 1.0, 2.0 |
| minimum-scale | 最小縮放比例 | 0.5, 1.0 |
| maximum-scale | 最大縮放比例 | 1.0, 3.0 |
| user-scalable | 允許使用者縮放 | yes, no |

## 常用斷點策略

### 行動優先 vs 桌面優先

**行動優先 (Mobile-First)**:
從最小螢幕開始設計，然後逐步為更大螢幕添加樣式。這是推薦的方法，因為行動裝置使用量持續增長。

```css
/* 基礎樣式 - 行動裝置 */
.card {
    width: 100%;
    padding: 16px;
    margin-bottom: 16px;
}

/* 平板 */
@media (min-width: 768px) {
    .card {
        width: calc(50% - 16px);
        display: inline-block;
        margin-right: 16px;
    }
}

/* 桌面 */
@media (min-width: 1024px) {
    .card {
        width: calc(33.333% - 16px);
    }
}

/* 大螢幕 */
@media (min-width: 1440px) {
    .card {
        width: calc(25% - 16px);
    }
}
```

**桌面優先 (Desktop-First)**:
從桌面版本開始，然後為較小螢幕覆寫樣式。

```css
/* 桌面優先 */
.card {
    width: 25%;
    float: left;
}

@media (max-width: 1023px) {
    .card {
        width: 50%;
    }
}

@media (max-width: 767px) {
    .card {
        width: 100%;
        float: none;
    }
}
```

### 常見斷點參考

| 斷點 | 裝置類型 | 典型寬度範圍 |
|------|----------|--------------|
| xs | 超小螢幕 | < 576px |
| sm | 大手機 | ≥ 576px |
| md | 平板 | ≥ 768px |
| lg | 小筆電 | ≥ 992px |
| xl | 桌面 | ≥ 1200px |
| xxl | 大螢幕 | ≥ 1400px |

## 常見響應式配置模式

### 單欄佈局 (Single Column)

適合行動裝置和簡單內容頁面。

```css
.container {
    width: 100%;
    padding: 16px;
}

article, .content {
    width: 100%;
    margin-bottom: 24px;
}

.sidebar {
    width: 100%;
}
```

### 雙欄佈局 (Two Columns)

適合中等螢幕尺寸。

```css
@media (min-width: 768px) {
    .container {
        display: flex;
        gap: 24px;
    }
    
    .main-content {
        flex: 2;
    }
    
    .sidebar {
        flex: 1;
    }
}
```

### 栅格系統 (Grid System)

使用 CSS Grid 實現靈活的栅格佈局。

```css
.grid {
    display: grid;
    gap: 16px;
    padding: 16px;
}

/* 自動填充 */
.grid-auto {
    grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
}

/* 自定義欄數 */
.grid-2 {
    grid-template-columns: repeat(2, 1fr);
}

.grid-3 {
    grid-template-columns: repeat(3, 1fr);
}

.grid-4 {
    grid-template-columns: repeat(4, 1fr);
}

/* 響應式栅格 */
@media (max-width: 767px) {
    .grid-2, .grid-3, .grid-4 {
        grid-template-columns: 1fr;
    }
}

@media (min-width: 768px) and (max-width: 1023px) {
    .grid-3, .grid-4 {
        grid-template-columns: repeat(2, 1fr);
    }
}

@media (min-width: 1024px) {
    .grid-4 {
        grid-template-columns: repeat(4, 1fr);
    }
}
```

### 側邊欄佈局 (Sidebar Layout)

```css
.layout {
    display: flex;
    flex-direction: column;
    min-height: 100vh;
}

header {
    background: #333;
    color: white;
    padding: 16px;
}

.main-container {
    display: flex;
    flex: 1;
    flex-direction: column;
}

.main-content {
    flex: 1;
    padding: 16px;
}

.sidebar {
    background: #f5f5f5;
    padding: 16px;
}

footer {
    background: #333;
    color: white;
    padding: 16px;
}

/* 平板及以上 */
@media (min-width: 768px) {
    .main-container {
        flex-direction: row;
    }
    
    .main-content {
        flex: 3;
    }
    
    .sidebar {
        flex: 1;
    }
}

/* 行動裝置隱藏側邊欄 */
@media (max-width: 767px) {
    .sidebar {
        display: none;
    }
}
```

## 響應式圖片策略

### srcset 和 sizes

```html
<img src="image-800.jpg" 
     srcset="image-400.jpg 400w,
             image-800.jpg 800w,
             image-1200.jpg 1200w"
     sizes="(max-width: 600px) 100vw,
            (max-width: 1200px) 50vw,
            33vw"
     alt="響應式圖片">
```

### picture 元素

```html
<picture>
    <source media="(max-width: 767px)" srcset="mobile.jpg">
    <source media="(max-width: 1023px)" srcset="tablet.jpg">
    <source media="(min-width: 1024px)" srcset="desktop.jpg">
    <img src="desktop.jpg" alt="響應式圖片">
</picture>
```

### 藝術方向 (Art Direction)

```html
<picture>
    <source srcset="hero-wide.jpg" media="(min-width: 1024px)">
    <source srcset="hero-medium.jpg" media="(min-width: 768px)">
    <img src="hero-narrow.jpg" alt="Hero 圖片">
</picture>
```

## 效能優化

### 隱藏和顯示內容

```css
/* 僅在行動裝置顯示 */
.mobile-only {
    display: block;
}

.desktop-only {
    display: none;
}

@media (min-width: 768px) {
    .mobile-only {
        display: none;
    }
    
    .desktop-only {
        display: block;
    }
}
```

### 延遲載入圖片

```html
<img src="image.jpg" loading="lazy" alt="圖片">
```

```javascript
// 使用 Intersection Observer
const images = document.querySelectorAll('img[data-src]');

const imageObserver = new IntersectionObserver((entries, observer) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            const img = entry.target;
            img.src = img.dataset.src;
            img.removeAttribute('data-src');
            observer.unobserve(img);
        }
    });
});

images.forEach(img => imageObserver.observe(img));
```

## 常見問題與解決方案

### 100vh 問題

行動裝置的瀏覽器會因為網址列而造成 100vh 不準確。

```css
/* 解決方案 */
.hero {
    min-height: 100vh;
    min-height: 100dvh; /* 動態 viewport height */
}

.modal {
    height: 90vh;
    height: 90dvh;
}
```

### 觸控目標尺寸

```css
/* 確保觸控目標至少 44x44px */
button, 
a.button,
.nav-item {
    min-height: 44px;
    min-width: 44px;
    padding: 12px;
}
```

### 橫向模式支援

```css
/* 橫向模式調整 */
@media (orientation: landscape) and (max-height: 500px) {
    .hero {
        min-height: 200px;
    }
    
    .navbar {
        flex-direction: row;
    }
}
```

# 前端框架

**標籤**: #前端 #框架 #React #Vue #Angular
**日期**: 2024-01-15

前端框架 (Frontend Framework) 是用於開發使用者介面的軟體框架，透過提供元件化架構、狀態管理、路由管理等功能，大幅簡化現代 Web 應用程式的開發過程。

## 前端框架的發展歷程

### 早期時代：原生 JavaScript 和 jQuery

在框架出現之前，開發者主要使用原生 JavaScript 和 jQuery 庫來處理 DOM 操作和事件處理。雖然這種方式靈活，但隨著應用程式規模增長，程式碼變得難以維護。

### 第一代框架：Backbone.js、Ember.js

Backbone.js 引入 Model-View 的概念，提供結構化開發模式。Ember.js 則是全功能框架，強調「慣例優於設定」。

### 現代框架時代：React、Vue、Angular

2013 年 Facebook 發布 React，引入虛擬 DOM 和元件化概念，徹底改變了前端開發。隨後 Google 發布 Angular，Vue.js 作者尤雨溪也發布了 Vue.js，形成現代三大框架鼎立的局面。

## React

React 是由 Facebook（現 Meta）開發的 JavaScript 函式庫，用於建構使用者介面。其核心特色是虛擬 DOM 和宣告式元件設計。

### 核心概念

**元件 (Components)**:
元件是 React 應用程式的建構區塊，可以是函式元件或類別元件。

```jsx
// 函式元件（現代写法）
function Welcome({ name }) {
    return <h1>Hello, {name}!</h1>;
}

// 使用 Arrow Function
const Welcome = ({ name }) => (
    <h1>Hello, {name}!</h1>
);

// 類別元件（舊式写法）
class Welcome extends React.Component {
    render() {
        return <h1>Hello, {this.props.name}!</h1>;
    }
}
```

**狀態 (State)**:
React 使用 useState Hook 來管理元件內部狀態。

```jsx
import React, { useState } from 'react';

function Counter() {
    const [count, setCount] = useState(0);
    const [user, setUser] = useState({ name: '小明', email: '' });
    
    const increment = () => setCount(count + 1);
    const decrement = () => setCount(count - 1);
    const reset = () => setCount(0);
    
    return (
        <div className="counter">
            <h2>計數器：{count}</h2>
            <button onClick={increment}>+</button>
            <button onClick={decrement}>-</button>
            <button onClick={reset}>重設</button>
            
            <div>
                <p>名稱：{user.name}</p>
                <input 
                    value={user.email}
                    onChange={(e) => setUser({...user, email: e.target.value})}
                    placeholder="輸入 Email"
                />
            </div>
        </div>
    );
}

export default Counter;
```

**生命週期與副作用**:
使用 useEffect Hook 處理副作用，如資料獲取、訂閱、DOM 操作等。

```jsx
import React, { useState, useEffect } from 'react';

function UserProfile({ userId }) {
    const [user, setUser] = useState(null);
    const [loading, setLoading] = useState(true);
    const [error, setError] = useState(null);
    
    useEffect(() => {
        async function fetchUser() {
            try {
                setLoading(true);
                const response = await fetch(`/api/users/${userId}`);
                if (!response.ok) {
                    throw new Error('Failed to fetch user');
                }
                const data = await response.json();
                setUser(data);
            } catch (err) {
                setError(err.message);
            } finally {
                setLoading(false);
            }
        }
        
        fetchUser();
    }, [userId]); // userId 改變時重新獲取資料
    
    if (loading) return <div>載入中...</div>;
    if (error) return <div>錯誤：{error}</div>;
    if (!user) return <div>使用者不存在</div>;
    
    return (
        <div>
            <h2>{user.name}</h2>
            <p>{user.email}</p>
            <p>加入日期：{new Date(user.createdAt).toLocaleDateString()}</p>
        </div>
    );
}
```

### 進階模式

**Context API**:
用於跨元件共享資料，避免 prop drilling。

```jsx
import React, { createContext, useContext, useState } from 'react';

const ThemeContext = createContext();

function ThemeProvider({ children }) {
    const [theme, setTheme] = useState('light');
    
    const toggleTheme = () => {
        setTheme(prev => prev === 'light' ? 'dark' : 'light');
    };
    
    return (
        <ThemeContext.Provider value={{ theme, setTheme, toggleTheme }}>
            {children}
        </ThemeContext.Provider>
    );
}

function ThemeButton() {
    const { theme, toggleTheme } = useContext(ThemeContext);
    
    return (
        <button onClick={toggleTheme}>
            切換為 {theme === 'light' ? '深色' : '淺色'} 主題
        </button>
    );
}
```

**Reducer**:
用於複雜狀態邏輯。

```jsx
import React, { useReducer } from 'react';

const initialState = { count: 0, status: 'idle' };

function reducer(state, action) {
    switch (action.type) {
        case 'INCREMENT':
            return { ...state, count: state.count + 1 };
        case 'DECREMENT':
            return { ...state, count: state.count - 1 };
        case 'RESET':
            return { ...state, count: 0 };
        case 'SET_STATUS':
            return { ...state, status: action.payload };
        default:
            return state;
    }
}

function Counter() {
    const [state, dispatch] = useReducer(reducer, initialState);
    
    return (
        <div>
            <p>計數：{state.count}</p>
            <p>狀態：{state.status}</p>
            <button onClick={() => dispatch({ type: 'INCREMENT' })}>+</button>
            <button onClick={() => dispatch({ type: 'DECREMENT' })}>-</button>
            <button onClick={() => dispatch({ type: 'RESET' })}>重設</button>
        </div>
    );
}
```

## Vue

Vue 是由尤雨溪創建的漸進式 JavaScript 框架，以其簡潔的 API 和優秀的文件著稱。

### 核心概念

**模板語法**:
```vue
<template>
  <div class="user-profile">
    <h1>{{ user.name }}</h1>
    <p>{{ user.email }}</p>
    
    <!-- 條件渲染 -->
    <div v-if="user.isActive">使用者已啟用</div>
    <div v-else>使用者未啟用</div>
    
    <!-- 列表渲染 -->
    <ul>
      <li v-for="post in posts" :key="post.id">
        {{ post.title }}
      </li>
    </ul>
    
    <!-- 事件處理 -->
    <button @click="handleClick">點擊</button>
    <input v-model="inputValue" />
    
    <!-- 屬性綁定 -->
    <a :href="linkUrl">連結</a>
    <img :src="imageUrl" :alt="imageAlt" />
    
    <!-- 樣式綁定 -->
    <div :class="{ active: isActive, 'text-error': hasError }"></div>
    <div :style="{ color: textColor, fontSize: fontSize + 'px' }"></div>
  </div>
</template>

<script setup>
import { ref, reactive, computed, onMounted, watch } from 'vue';

const user = reactive({
  name: '王小明',
  email: 'wang@example.com',
  isActive: true
});

const posts = ref([
  { id: 1, title: '第一篇文章' },
  { id: 2, title: '第二篇文章' }
]);

const inputValue = ref('');
const linkUrl = ref('https://example.com');
const imageUrl = ref('/images/photo.jpg');
const imageAlt = ref('圖片');
const isActive = ref(true);
const hasError = ref(false);
const textColor = ref('#333333');
const fontSize = ref(16);

function handleClick() {
  console.log('按鈕被點擊');
}

const fullName = computed(() => {
  return user.name;
});

onMounted(() => {
  console.log('元件已掛載');
});

watch(inputValue, (newValue, oldValue) => {
  console.log(`值從 ${oldValue} 變為 ${newValue}`);
});
</script>

<style scoped>
.user-profile {
  padding: 20px;
}

h1 {
  font-size: 24px;
  color: #333;
}

.active {
  font-weight: bold;
}

.text-error {
  color: red;
}
</style>
```

### Composition API

Vue 3 引入的 Composition API 提供更靈活的程式碼組織方式。

```vue
<script setup>
import { ref, computed, watchEffect } from 'vue';

const count = ref(0);
const doubleCount = computed(() => count.value * 2);

function increment() {
  count.value++;
}

watchEffect(() => {
  console.log(`count 改變為: ${count.value}`);
});
</script>

<template>
  <button @click="increment">Count: {{ count }} (Double: {{ doubleCount }})</button>
</template>
```

### Pinia 狀態管理

```javascript
import { defineStore } from 'pinia';

export const useUserStore = defineStore('user', {
    state: () => ({
        user: null,
        isLoggedIn: false
    }),
    getters: {
        userName: (state) => state.user?.name || '訪客'
    },
    actions: {
        async login(email, password) {
            const response = await fetch('/api/login', {
                method: 'POST',
                body: JSON.stringify({ email, password })
            });
            this.user = await response.json();
            this.isLoggedIn = true;
        },
        logout() {
            this.user = null;
            this.isLoggedIn = false;
        }
    }
});
```

Angular 是 Google 開發的完整平台框架，適合大型企業級應用程式開發。

**元件 (Components)**:
```typescript

@Component({
    selector: 'app-user',
    template: `
        <h1>{{ title }}</h1>
        <p>名稱：{{ user.name }}</p>
```

**服務 (Services)**:
```typescript
import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';

@Injectable({
    providedIn: 'root'
})
export class UserService {
    private apiUrl = 'https://api.example.com/users';
    
    constructor(private http: HttpClient) { }
    
    getUsers(): Observable<User[]> {
        return this.http.get<User[]>(this.apiUrl);
    }
    
    getUser(id: number): Observable<User> {
        return this.http.get<User>(`${this.apiUrl}/${id}`);
    }
    
    createUser(user: User): Observable<User> {
        return this.http.post<User>(this.apiUrl, user);
    }
    
    updateUser(id: number, user: User): Observable<User> {
        return this.http.put<User>(`${this.apiUrl}/${id}`, user);
    }
    
    deleteUser(id: number): Observable<void> {
        return this.http.delete<void>(`${this.apiUrl}/${id}`);
    }
}
```

## 選擇框架的考量因素

### 專案規模
- 小型專案：Vue 學習曲線低，快速開發
- 中型專案：React 彈性高，生態豐富
- 大型專案：Angular 完整解決方案

### 團隊技能
- 團隊熟悉 TypeScript：Angular
- 團隊熟悉 JavaScript：React 或 Vue
- 需要快速上手：Vue

### 生態系統
- 需要豐富的第三方庫：React
- 需要完整官方解決方案：Angular
- 需要輕量解決方案：Vue

### 效能需求
- 高效能場景：React + Vite
- 中等效能：Vue
- 企業級優化：Angular

### TypeScript

TypeScript 是 JavaScript 的超集，提供靜態類型檢查，大幅提升程式碼品質。

```typescript
interface User {
    id: number;
    name: string;
    email: string;
    isActive: boolean;
}

function getUser(id: number): Promise<User> {
    return fetch(`/api/users/${id}`)
        .then(response => {
            if (!response.ok) {
                throw new Error('Failed to fetch user');
            }
            return response.json();
        });
}

function createUser(user: Omit<User, 'id'>): Promise<User> {
    return fetch('/api/users', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(user)
    }).then(response => response.json());
}
```

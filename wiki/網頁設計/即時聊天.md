# 即時聊天

使用 WebSocket 實現的即時聊天系統。

## 功能

- 多用戶即時訊息廣播
- 私人訊息
- 使用者列表
- 即時連線狀態

## 技術棧

- 後端: Node.js + Express + ws (WebSocket)
- 前端: 原生 HTML + CSS + JavaScript

## 程式碼

- [server.js](../_code/網頁設計/即時聊天/server.js) - 後端伺服器
- [index.html](../_code/網頁設計/即時聊天/public/index.html) - 前端頁面
- [app.js](../_code/網頁設計/即時聊天/public/app.js) - 前端邏輯

## API

WebSocket 訊息格式：

| 類型 | 欄位 | 說明 |
|------|------|------|
| chat | text | 廣播訊息 |
| private | to, text | 私人訊息 |
| users | list | 使用者列表 |

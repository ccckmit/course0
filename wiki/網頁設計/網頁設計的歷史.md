# 網頁設計的歷史

網頁設計從 1990 年代初的簡單 HTML 文件發展到今天的複雜 Web 應用，經歷了多個重要階段。每一代技術都解決了特定問題，同時也帶來了新的挑戰，推動著這個領域不斷向前發展。

## 起源：全球資訊網的誕生（1989-1993 年）

### Tim Berners-Lee 的發明

1989 年，Tim Berners-Lee 在歐洲核子研究組織（CERN）工作時，提出了一個名為「World Wide Web」的系統設計。他的目標是讓科學家能夠方便地共享研究文件和數據。

1990 年，Berners-Lee 開發了第一批 Web 技術：
- **HTML**：超文本標記語言，用於描述網頁結構
- **URI**：統一資源標識符，用於定位資源
- **HTTP**：超文本傳輸協定，用於客戶端和伺服器通信

1991 年 8 月，第一個網站在 CERN 上線。

### 早期網頁

最早的網頁非常簡單——純文字、連結、少量格式化：

```html
<header>Welcome to CERN</header>
<p>This is the first website.</p>
<a href="/info">More information</a>
```

沒有圖片、沒有樣式、沒有互動。只有文字和連結。

### Mosaic 瀏覽器

1993 年，NCSA 的 Marc Andreessen 和 Eric Bina 發布了 Mosaic 瀏覽器。這是第一個圖形化 Web 瀏覽器，讓使用者能夠看到圖片和文字混合的網頁。

Mosaic 的發布被視為 Web 普及的轉折點。《紐約時報》稱 Andreessen 為「下一個比爾·蓋茲」。

## 圖形化時代（1994-1999 年）

### Netscape 的崛起

1994 年，Marc Andreessen 離開 NCSA，與 Jim Clark 共同創辦了 Netscape Communications。他們推出了 Netscape Navigator，迅速占據了 80% 的市場份額。

Netscape 創新不斷：
- JavaScript 腳本語言（1995）
- Cookie（用於狀態管理）
- Frames（框架）

### 第一次瀏覽器大戰

微軟感知到威脅，於 1995 年推出了 Internet Explorer。隨後展開了激烈的瀏覽器大戰：

- Netscape 引入新功能，微軟快速跟進
- 兩家公司使用專有擴展，犧牲了標準兼容性
- 微軟最終透過 Windows 捆綁策略獲勝

### HTML 的標準化

1995 年，HTML 2.0 作為第一個正式的 HTML 標準發布。此後：
- HTML 3.2（1997）：表格、 Applet
- HTML 4.0（1997）：樣式表、框架、腳本
- HTML 4.01（1999）：錯誤修正，末代 HTML 4

### CSS 的誕生

1996 年，CSS（Cascading Style Sheets）成為 W3C 推薦標準。CSS 將樣式從 HTML 中分離出來：

```css
/* 終於可以這樣做了！ */
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
}

h1 {
    color: #333;
    text-align: center;
}
```

### 早期 Web 設計特點

這一時期的網頁設計：
- `<table>` 佈局（CSS 出現前）
- 點擊计数器（Page Counter）
- 留言本（Guestbook）
- 動畫 GIF
- `<marquee>` 滾動文字

## Web 2.0 時代（2000-2010 年）

### AJAX 革命

2004-2005 年，AJAX（Asynchronous JavaScript and XML）技術開始普及。這種技術允許網頁在後台與伺服器通信，無需重新載入整個頁面。

典型 AJAX 流程：
```javascript
// Google Maps 風格的地圖滑動
fetch('/api/location?lat=25&lng=121')
    .then(response => response.json())
    .then(data => updateMap(data));
```

Google Maps（2005）和 Gmail（2004）是 AJAX 的先驅應用。

### Web 2.0 的特徵

Tim O'Reilly 在 2004 年提出「Web 2.0」概念：
- **使用者生成內容**：維基百科、YouTube、部落格
- **社交網路**：MySpace、Facebook、Twitter
- **開放平台**：API、第三方應用
- **豐富的用戶體驗**：RIA（Rich Internet Applications）

### jQuery 簡化 DOM 操作

2006 年，John Resig 發布了 jQuery。這個函式庫極大地簡化了 JavaScript 開發：

```javascript
// jQuery 之前
document.getElementById('btn').addEventListener('click', function() {
    var el = document.querySelector('.container');
    el.style.display = 'none';
});

// jQuery 之後
$('#btn').click(function() {
    $('.container').hide();
});
```

jQuery 解決了瀏覽器兼容問題，成為最流行的 JavaScript 庫。

### 部落格與內容管理

2000 年代見證了部落格的爆發：
- Blogger（1999）
- WordPress（2003）
- Movable Type

這些平台降低了網頁發布的門檻，讓任何人都能成為內容創作者。

## 移動互聯網時代（2010-2015 年）

### iPhone 與 Mobile Web

2007 年，Apple 發布 iPhone，重新定義了移動互聯網。iPhone 沒有 Flash、沒有 Java Applet，只有標準的 HTML、CSS 和 JavaScript。

Steve Jobs 的決定「選擇開放標準而非 Flash」被證明是正確的。

### 回應式網頁設計

2010 年，Ethan Marcotte 在《A List Apart》雜誌上發表了「Responsive Web Design」文章，提出了回應式設計概念：

```css
/* 回應式佈局的基礎 */
.container {
    width: 100%;
    max-width: 1200px;
    margin: 0 auto;
}

@media (max-width: 768px) {
    .sidebar {
        display: none;
    }
    .main-content {
        width: 100%;
    }
}
```

回應式設計用一個網址適配所有設備。

### Bootstrap 的流行

2011 年，Twitter 開源了 Bootstrap。這個 CSS 框架讓非設計師也能創建美觀的網頁：

- 12 欄格柵系統
- 預設樣式和組件
- jQuery 插件

Bootstrap 簡化了 Web 開發，但也被批評導致「網頁同質化」。

### PhoneGap 與混合應用

PhoneGap（後來的 Apache Cordova）允許用 HTML/CSS/JS 開發移動應用。Web 開發者可以進入移動應用市場。

但混合應用的效能問題催生了 React Native、Flutter 等新方案。

## JavaScript 框架時代（2013 年-至今）

### AngularJS

2009 年，Google 發布了 AngularJS。這是第一個主流的 MV* JavaScript 框架：

- 雙向資料綁定
- 依賴注入
- 指令系統

```javascript
// AngularJS 控制器
angular.module('app', [])
    .controller('MainController', function($scope) {
        $scope.message = 'Hello, World!';
    });
```

### React 的革命

2013 年，Facebook 開源了 React。React 的創新：
- **虛擬 DOM**：高效更新頁面
- **元件化**：可組合的 UI 組件
- **單向資料流**：更容易預測的狀態管理

```jsx
// React 函式型元件
function Greeting({ name }) {
    return <h1>Hello, {name}!</h1>;
}
```

React 改變了前端開發的範式，催生了 Vue、Angular 等框架的更新。

### Vue 的崛起

2014 年，Evan You 發布了 Vue.js。Vue 結合了 Angular 的指令系統和 React 的元件化思想，同時更加輕量：

```javascript
// Vue 3 Composition API
import { ref, computed } from 'vue';

export default {
    setup() {
        const count = ref(0);
        const doubled = computed(() => count.value * 2);
        return { count, doubled };
    }
}
```

### TypeScript 的普及

2012 年，微軟發布了 TypeScript。TypeScript 為 JavaScript 添加了靜態類型：

```typescript
interface User {
    id: number;
    name: string;
    email: string;
}

function greet(user: User): string {
    return `Hello, ${user.name}!`;
}
```

TypeScript 提高了大型項目的代碼質量和可維護性，被 Angular、Vue 等框架原生支援。

### SSR 與元框架

傳統的客戶端渲染（CSR）有 SEO 和首屏加載問題。服務端渲染（SSR）框架應運而生：

- **Next.js**（2016）：React 元框架
- **Nuxt.js**（2016）：Vue 元框架
- **SvelteKit**：Svelte 的元框架

這些框架結合了 SSR 的 SEO 優勢和 CSR 的交互性。

## 現代 Web 技術棧（2018 年-至今）

### WebAssembly

2017 年，W3C 發布了 WebAssembly（、Wasm）。這是一種可以在瀏覽器中運行的二進制指令格式：

- 幾乎原生的執行效率
- 支持 C、C++、Rust 等語言編譯到瀏覽器
- 用例：遊戲、影片編輯、3D 渲染

### JAMstack 架構

JAMstack（JavaScript、APIs、Markup）是一種現代 Web 架構：

- 預先渲染靜態頁面
- 通過 API 獲取動態內容
- 極快的載入速度和安全性

代表工具：Gatsby、Next.js、Hugo。

### 無服務器架構

Serverless 讓開發者專注於前端邏輯：
- AWS Lambda、Cloudflare Workers
- 邊緣計算
- 函式即服務（FaaS）

### Web 3.0 與區塊鏈

區塊鏈技術催生了 Web3 概念：
- 去中心化應用（DApp）
- 智慧合約
- NFT（非同質化代幣）
- 錢包身份驗證

但 Web3 的實用性和可持續性仍有爭議。

## 台灣與華語網頁設計

### 早期中文網站

台灣是華語網頁設計的重要先驅：
- 新聞網站（PChome、聯合新聞網）
- 電子商務（露天拍賣、Momo）
- 社群媒體（痞客邦、巴哈姆特）

### 響應式設計在華語圈

隨著智慧手機普及，華語網站迅速採用響應式設計。淘寶、京東等電商巨頭的 UI 設計引領了潮流。

### 前端人才的流動

台灣和中國的前端開發者在矽谷和全球科技公司都有重要貢獻。許多流行的開源工具由華人開發或維護。

## 未來趨勢

### 網頁的持續進化

- **WebGPU**：更強大的圖形能力
- **CSS Houdini**：更深入的樣式控制
- **View Transitions API**：原生頁面過渡動畫
- **Declarative Shadow DOM**：元件封裝

### AI 輔助開發

- **GitHub Copilot**：AI 程式碼建議
- **Figma AI**：設計工具的 AI 功能
- **自動生成 UI**：描述即生成介面

### 性能成為焦點

- Core Web Vitals 成為 SEO 因素
- 漸進式 Web 應用（PWA）提供離線能力
- 邊緣計算減少延遲

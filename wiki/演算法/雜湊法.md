# 雜湊法 (Hashing)

## 概述

雜湊法使用雜湊函數將鍵映射到陣列索引，實現 O(1) 平均查找、插入、刪除。

## 雜湊函數

### 基本要求

1. **確定性**：相同輸入產生相同輸出
2. **均勻分佈**：鍵均勻分佈在表中
3. **計算快速**：O(1) 時間複雜度

### 常見雜湊函數

```python
def hash_mod(key, size):
    """除法雜湊法"""
    return key % size

def hash_mult(key, size, A=0.6180339887):
    """乘法雜湊法"""
    return int(size * ((key * A) % 1))

def hash_string(s, size):
    """字串雜湊法"""
    h = 0
    for c in s:
        h = (h * 31 + ord(c)) % size
    return h
```

## 雜湊表實現

```python
class HashTable:
    def __init__(self, size=16, load_factor=0.75):
        self.size = size
        self.count = 0
        self.load_factor = load_factor
        self.table = [[] for _ in range(size)]
    
    def _hash(self, key):
        return hash(key) % self.size
    
    def _resize(self):
        old_table = self.table
        self.size *= 2
        self.table = [[] for _ in range(self.size)]
        self.count = 0
        
        for bucket in old_table:
            for key, value in bucket:
                self[key] = value
    
    def __setitem__(self, key, value):
        if self.count / self.size >= self.load_factor:
            self._resize()
        
        index = self._hash(key)
        bucket = self.table[index]
        
        for i, (k, v) in enumerate(bucket):
            if k == key:
                bucket[i] = (key, value)
                return
        
        bucket.append((key, value))
        self.count += 1
    
    def __getitem__(self, key):
        index = self._hash(key)
        for k, v in self.table[index]:
            if k == key:
                return v
        raise KeyError(key)
    
    def __delitem__(self, key):
        index = self._hash(key)
        bucket = self.table[index]
        
        for i, (k, v) in enumerate(bucket):
            if k == key:
                del bucket[i]
                self.count -= 1
                return
        raise KeyError(key)
    
    def __contains__(self, key):
        index = self._hash(key)
        return any(k == key for k, v in self.table[index])

# 使用
ht = HashTable()
ht['apple'] = 5
ht['banana'] = 3
print(ht['apple'])  # 5
print('banana' in ht)  # True
```

## 碰撞處理

### 鏈接法 (Chaining)

```python
class ChainingHashTable:
    def __init__(self, size=16):
        self.table = [[] for _ in range(size)]
    
    def insert(self, key, value):
        index = hash(key) % len(self.table)
        for i, (k, v) in enumerate(self.table[index]):
            if k == key:
                self.table[index][i] = (key, value)
                return
        self.table[index].append((key, value))
    
    def search(self, key):
        index = hash(key) % len(self.table)
        for k, v in self.table[index]:
            if k == key:
                return v
        return None
```

### 開放定址法 (Open Addressing)

```python
class OpenAddressHashTable:
    def __init__(self, size=16):
        self.size = size
        self.keys = [None] * size
        self.values = [None] * size
    
    def _probe(self, key, i):
        """線性探測"""
        return (hash(key) + i) % self.size
    
    def __setitem__(self, key, value):
        for i in range(self.size):
            index = self._probe(key, i)
            if self.keys[index] is None or self.keys[index] == key:
                self.keys[index] = key
                self.values[index] = value
                return
        raise Exception("Hash table full")
    
    def __getitem__(self, key):
        for i in range(self.size):
            index = self._probe(key, i)
            if self.keys[index] == key:
                return self.values[index]
            if self.keys[index] is None:
                break
        raise KeyError(key)
```

## 雜湊法複雜度

| 操作 | 平均 | 最差 |
|------|------|------|
| 查找 | O(1) | O(n) |
| 插入 | O(1) | O(n) |
| 刪除 | O(1) | O(n) |

## 應用場景

1. **符號表**：編譯器變數管理
2. **字典**：Python dict, Java HashMap
3. **快取**：LRU Cache
4. **去重**：找出重複元素
5. **字串匹配**：Rabin-Karp 演算法

# 暴力法 (Brute Force)

## 概述

暴力法枚舉所有可能情況，從中找出正確解答。雖然簡單，但常作為基準解。

## 設計原則

```
1. 枚舉所有候選解
2. 檢查每個候選解
3. 選取滿足條件的解
```

### 最近點對

```python
import math

def brute_force_closest_pair(points):
    """暴力求解最近點對"""
    min_dist = float('inf')
    closest = None
    
    for i in range(len(points)):
        for j in range(i + 1, len(points)):
            dist = math.sqrt((points[i][0]-points[j][0])**2 + 
                           (points[i][1]-points[j][1])**2)
            if dist < min_dist:
                min_dist = dist
                closest = (points[i], points[j])
    
    return closest, min_dist

points = [(0, 0), (1, 1), (2, 2), (0.5, 0.5)]
print(brute_force_closest_pair(points))
```

### 全排列密碼破解

```python
import itertools

def brute_force_password(target, charset, max_len):
    """暴力破解密碼"""
    for length in range(1, max_len + 1):
        for attempt in itertools.product(charset, repeat=length):
            password = ''.join(attempt)
            if password == target:
                return password
    return None

# 暴力破解 4 位數字密碼
result = brute_force_password("1234", "0123456789", 4)
print(f"Found: {result}")
```

### 0-1 揹包（暴力）

```python
def brute_force_knapsack(weights, values, capacity):
    """暴力求解 0-1 揹包"""
    n = len(weights)
    best_value = 0
    best_items = []
    
    for mask in range(1 << n):
        total_weight = 0
        total_value = 0
        items = []
        
        for i in range(n):
            if mask & (1 << i):
                total_weight += weights[i]
                total_value += values[i]
                items.append(i)
        
        if total_weight <= capacity and total_value > best_value:
            best_value = total_value
            best_items = items
    
    return best_value, best_items

weights = [2, 3, 4, 5]
values = [3, 4, 5, 6]
print(brute_force_knapsack(weights, values, 8))  # (10, [1, 2])
```

### 子集合求和

```python
def brute_force_subset_sum(nums, target):
    """找出和為 target 的子集合"""
    n = len(nums)
    
    for mask in range(1 << n):
        total = 0
        subset = []
        
        for i in range(n):
            if mask & (1 << i):
                total += nums[i]
                subset.append(nums[i])
        
        if total == target:
            return subset
    
    return None

print(brute_force_subset_sum([1, 2, 3, 4, 5], 9))  # [2, 3, 4] 或 [4, 5]
```

## 時間複雜度

| 問題 | 暴力複雜度 |
|------|-----------|
| 字串匹配 | O(n×m) |
| 最近點對 | O(n²) |
| 0-1 揹包 | O(2ⁿ) |
| 子集合求和 | O(2ⁿ) |
| 全排列 | O(n!) |

## 暴力法 vs 優化

| 暴力法 | 優化方法 |
|--------|----------|
| 一定能找到解 | 可能找不到或近似 |
| 時間複雜度高 | 時間複雜度低 |
| 無需特殊技巧 | 需要問題特性 |
| 作為基準對比 | 工程實用 |

## 何時使用

1. 問題規模小
2. 需要找出所有解
3. 作為其他算法的基準
4. 無法找到更好的解法

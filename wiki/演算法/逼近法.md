# 逼近法 (Approximation Algorithm)

## 概述

逼近法在合理時間內找出接近最優解的解答，適用於 NP-困難問題。

## 近似比

```
近似比 = 最差情況解 / 最優解
```

```python
def approximation_ratio(approx, optimal):
    """計算近似比"""
    return approx / optimal if optimal > 0 else float('inf')
```

## 經典問題

### 頂點覆蓋問題

```python
def vertex_cover_approx(graph):
    """2-近似頂點覆蓋：保證不超過 2 倍最優"""
    cover = set()
    edges = set((min(u, v), max(u, v)) for u, v in graph.edges())
    
    while edges:
        u, v = edges.pop()
        cover.add(u)
        cover.add(v)
        
        # 移除所有與 u、v 相連的邊
        edges = {(a, b) for a, b in edges 
                 if a not in (u, v) and b not in (u, v)}
    
    return cover
```

### 集合覆蓋問題

```python
def set_cover_approx(universe, subsets):
    """貪心集合覆蓋：O(log n)-近似"""
    cover = set()
    uncovered = set(universe)
    
    while uncovered:
        best_subset = max(subsets, key=lambda s: len(s & uncovered))
        cover.add(best_subset)
        uncovered -= best_subset
    
    return cover

# 範例
universe = {1, 2, 3, 4, 5}
subsets = [
    {1, 2, 3},
    {2, 4},
    {3, 4},
    {4, 5}
]
print(set_cover_approx(universe, subsets))
```

### 旅行推銷員問題

```python
import heapq

def tsp_approx(graph, start):
    """MST 近似 TSP：2-近似"""
    def prim_mst():
        visited = {start}
        mst_edges = []
        heap = [(0, start, start)]
        
        while len(visited) < len(graph):
            weight, u, v = heapq.heappop(heap)
            if v in visited:
                continue
            
            visited.add(v)
            mst_edges.append((u, v, weight))
            
            for neighbor, w in graph[v]:
                if neighbor not in visited:
                    heapq.heappush(heap, (w, v, neighbor))
        
        return mst_edges
    
    mst_edges = prim_mst()
    
    # DFS 遍歷 MST
    def dfs_tour():
        visited = set()
        tour = []
        
        def dfs(node):
            visited.add(node)
            tour.append(node)
            for u, v, w in mst_edges:
                next_node = v if u == node else u if v == node else None
                if next_node and next_node not in visited:
                    dfs(next_node)
        
        dfs(start)
        tour.append(start)
        return tour
    
    return dfs_tour()
```

### 加權集合覆蓋

```python
def weighted_set_cover(universe, subsets, costs):
    """加權貪心集合覆蓋"""
    cover = set()
    uncovered = set(universe)
    total_cost = 0
    
    while uncovered:
        best_ratio = float('inf')
        best_subset = None
        
        for i, subset in enumerate(subsets):
            new_elements = subset & uncovered
            if new_elements:
                ratio = costs[i] / len(new_elements)
                if ratio < best_ratio:
                    best_ratio = ratio
                    best_subset = i
        
        if best_subset is None:
            break
        
        cover.add(best_subset)
        total_cost += costs[best_subset]
        uncovered -= subsets[best_subset]
    
    return cover, total_cost
```

## PTAS 與 FPTAS

```python
def knapsack_ptas(values, weights, capacity, epsilon):
    """PTAS：多項式時間近似方案"""
    n = len(values)
    
    if n == 0:
        return 0, []
    
    max_value = max(values)
    k = epsilon * max_value / n
    
    rounded_values = [v // k * k for v in values]
    
    # 使用動態規劃（pseudo-polynomial）
    scaled_values = [int(v / k) for v in values]
    dp = [0] * (int(n * max_value / k) + 1)
    
    for i in range(n):
        for w in range(len(dp) - 1, scaled_values[i] - 1, -1):
            dp[w] = max(dp[w], dp[w - scaled_values[i]] + values[i])
    
    max_scaled = int(capacity * max_value / k)
    return dp[max_scaled]
```

## 近似演算法分類

| 類型 | 說明 | 例子 |
|------|------|------|
| 常數近似 | ratio = c | 頂點覆蓋 (2) |
| 多項式近似 | PTAS | 旅行推銷員 |
| FPTAS | 完全多項式 | 揹包問題 |
| 無近似 | 無 PTAS | 組合優化問題 |

## 何時使用

1. 問題為 NP-困難
2. 需要在合理時間內得到可行解
3. 最優解代價太高
4. 可接受近似解的誤差範圍

# 空間複雜度

空間複雜度是演算法分析的另一個核心維度，它描述演算法在執行過程中需要佔用的記憶體空間隨輸入規模增長的趨勢。與時間複雜度同樣重要，空間複雜度直接影響演算法是否能夠在有限的記憶體資源下正常執行。在嵌入式系統、大型資料處理、資源受限的環境中，空間複雜度的優化甚至比時間複雜度更為關鍵。

## 空間複雜度的基本概念

空間複雜度通常使用大O標記法表示，與時間複雜度類似，我們關心的是記憶體使用量隨輸入規模增長的趨勢，而非絕對的位元組數。

### 空間複雜度的組成

演算法的空間複雜度通常由以下幾部分組成：

1. **輸入空間**：儲存輸入資料所需的空間
2. **輔助空間**：演算法執行過程中臨時創建的空間
3. **輸出空間**：儲存輸出結果所需的空間（通常單獨計算）

一般所說的空間複雜度是指輔助空間，即除了輸入資料本身之外，演算法額外使用的空間。

## 常見空間複雜度

### 1. 常數空間 O(1)

常數空間複雜度意味著無論輸入規模如何，演算法只使用固定的記憶體空間。這是最理想的空間複雜度。

```python
# O(1) 空間 - 只使用少數幾個變數
def find_max(arr):
    max_val = arr[0]          # 1個變數
    for i in range(1, len(arr)):
        if arr[i] > max_val:
            max_val = arr[i]
    return max_val

# O(1) 空間 - 兩個數字交換
def swap(a, b):
    temp = a                  # 1個臨時變數
    a = b
    b = temp
    return a, b

# O(1) 空間 - 計算陣列總和
def sum_array(arr):
    total = 0                 # 1個累加器
    for x in arr:
        total += x
    return total
```

### 2. 線性空間 O(n)

線性空間複雜度意味著演算法需要與輸入規模成正比的記憶體空間。

```python
# O(n) 空間 - 創建新陣列
def create_array(n):
    return [0] * n            # n個元素

# O(n) 空間 - 複製陣列
def copy_array(arr):
    return arr[:]              # 完整複製

# O(n) 空間 - 計算所有前綴和
def prefix_sums(arr):
    n = len(arr)
    result = [0] * n           # n個元素的輸出陣列
    for i in range(n):
        if i == 0:
            result[i] = arr[i]
        else:
            result[i] = result[i-1] + arr[i]
    return result
```

### 3. 對數空間 O(log n)

對數空間複雜度通常出現在使用分治策略但只保留少數遞歸狀態的演算法中。

```python
# O(log n) 空間 - 二分搜尋（迭代版本）
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    
    return -1

# O(log n) 空間 - 快速冪（迭代版本）
def power(base, exponent):
    result = 1
    while exponent > 0:
        if exponent % 2 == 1:
            result *= base
        base *= base
        exponent //= 2
    return result
```

### 4. 平方空間 O(n²)

平方空間複雜度通常出現在需要二維資料結構的演算法中。

```python
# O(n²) 空間 - 建立二維矩陣
def create_matrix(n):
    return [[0] * n for _ in range(n)]

# O(n²) 空間 - 動態規劃經典例子：費波那契
def fibonacci_dp(n):
    if n <= 1:
        return n
    
    dp = [0] * (n + 1)         # O(n) 空間
    dp[1] = 1
    
    for i in range(2, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    
    return dp[n]

# O(n²) 空間 - 矩陣乘法
def matrix_multiply(A, B):
    n = len(A)
    C = [[0] * n for _ in range(n)]  # n×n矩陣
    
    for i in range(n):
        for j in range(n):
            for k in range(n):
                C[i][j] += A[i][k] * B[k][j]
    
    return C
```

## 遞迴演算法的空間複雜度

遞迴演算法的空間複雜度特別需要注意，因為每次遞迴調用都會在呼叫堆疊上佔用空間。

### 遞迴深度與空間

```python
# O(n) 遞迴空間 - 遞迴深度為 n
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n - 1)

# 呼叫過程：
# factorial(5) → factorial(4) → factorial(3) → factorial(2) → factorial(1)
# 最多同時存在 5 個未完成的函數呼叫

# O(n) 遞迴空間 - 反轉連結串列
def reverse_linked_list(head):
    if head is None or head.next is None:
        return head
    
    new_head = reverse_linked_list(head.next)
    head.next.next = head
    head.next = None
    
    return new_head
```

### 優化遞迴空間

```python
# 原始版本 - O(n) 空間
def fibonacci_recursive(n):
    if n <= 1:
        return n
    return fibonacci_recursive(n-1) + fibonacci_recursive(n-2)

# 優化版本 - O(1) 空間（迭代）
def fibonacci_iterative(n):
    if n <= 1:
        return n
    
    prev, curr = 0, 1
    for _ in range(2, n + 1):
        prev, curr = curr, prev + curr
    
    return curr
```

### 分治法的空間分析

```python
# 合併排序 - O(n) 空間（因為需要臨時陣列）
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])    # O(n) 空間
    right = merge_sort(arr[mid:])   # O(n) 空間
    
    return merge(left, right)

def merge(left, right):
    result = []                     # O(n) 臨時空間
    i = j = 0
    
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    
    result.extend(left[i:])
    result.extend(right[j:])
    return result

# 就地合併排序 - O(1) 空間（但複雜度較高）
def merge_sort_inplace(arr, left, right):
    if left < right:
        mid = (left + right) // 2
        merge_sort_inplace(arr, left, mid)
        merge_sort_inplace(arr, mid + 1, right)
        merge_inplace(arr, left, mid, right)
```

## 空間複雜度的進階概念

### 原地演算法（In-place Algorithm）

原地演算法是指使用 O(1) 或 O(log n) 額外空間的演算法，即不隨輸入規模增長而增加額外空間。

```python
# 原地反轉陣列 - O(1) 空間
def reverse_array(arr):
    left, right = 0, len(arr) - 1
    
    while left < right:
        arr[left], arr[right] = arr[right], arr[left]
        left += 1
        right -= 1

# 原地冒泡排序 - O(1) 空間
def bubble_sort_inplace(arr):
    n = len(arr)
    
    for i in range(n):
        swapped = False
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        
        if not swapped:
            break
```

### 空間換取時間

有時我們可以通過使用更多空間來減少執行時間。

```python
# 暴力解 - O(1) 空間，O(n²) 時間
def two_sum_brute(nums, target):
    n = len(nums)
    for i in range(n):
        for j in range(i + 1, n):
            if nums[i] + nums[j] == target:
                return [i, j]
    return []

# 雜湊表解 - O(n) 空間，O(n) 時間
def two_sum_hash(nums, target):
    seen = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i
    return []
```

### 資料壓縮與空間優化

```python
# 遊標編碼（Run-Length Encoding）
def rle_encode(s):
    if not s:
        return ""
    
    result = []
    count = 1
    
    for i in range(1, len(s)):
        if s[i] == s[i-1]:
            count += 1
        else:
            result.append(s[i-1] + str(count))
            count = 1
    
    result.append(s[-1] + str(count))
    return ''.join(result)

# 差分編碼
def diff_encode(arr):
    if not arr:
        return []
    
    result = [arr[0]]
    for i in range(1, len(arr)):
        result.append(arr[i] - arr[i-1])
    
    return result
```

## 空間複雜度與時間複雜度的權衡

###經典範例：動態規劃

```python
# 經典理財問題：最大子陣列和

# 方法1：暴力枚舉 - O(1) 空間，O(n³) 時間
def max_subarray_brute(arr):
    max_sum = float('-inf')
    n = len(arr)
    
    for i in range(n):
        for j in range(i, n):
            current_sum = sum(arr[i:j+1])
            max_sum = max(max_sum, current_sum)
    
    return max_sum

# 方法2：前綴和 - O(n) 空間，O(n²) 時間
def max_subarray_prefix(arr):
    n = len(arr)
    max_sum = float('-inf')
    
    # 計算前綴和
    prefix = [0] * (n + 1)
    for i in range(n):
        prefix[i + 1] = prefix[i] + arr[i]
    
    # 枚舉所有子陣列
    for i in range(n):
        for j in range(i, n):
            current_sum = prefix[j + 1] - prefix[i]
            max_sum = max(max_sum, current_sum)
    
    return max_sum

# 方法3：Kadane演算法 - O(1) 空間，O(n) 時間
def max_subarray_kadane(arr):
    max_sum = current_sum = arr[0]
    
    for i in range(1, len(arr)):
        current_sum = max(arr[i], current_sum + arr[i])
        max_sum = max(max_sum, current_sum)
    
    return max_sum
```

### 記憶體受限環境的考量

在嵌入式系統或行動裝置上，記憶體資源有限，需要特別注意：

1. **避免不必要的資料複製**：盡量使用原地操作
2. **選擇合適的資料結構**：例如使用陣列而非連結串列以節省指標空間
3. **延遲計算**：只在需要時計算，而非預先計算所有結果
4. **資料壓縮**：使用適當的壓縮技術減少記憶體使用

## 空間複雜度速查表

| 複雜度 | 名稱 | 常見場景 | 範例 |
|--------|------|----------|------|
| O(1) | 常數 | 原地操作、簡單計算 | 交換兩個變數、找最大值 |
| O(log n) | 對數 | 分治但只留少數狀態 | 迭代二分搜尋、快速冪 |
| O(n) | 線性 | 線性資料結構 | 陣列複製、前綴和 |
| O(n log n) | 線性對數 | 高效排序（需額外空間） | 合併排序（需臨時陣列） |
| O(n²) | 平方 | 二維動態規劃 | 矩陣乘法、 Floyd-Warshall |

## 空間複雜度的測量工具

在實際開發中，我們可以使用各種工具來分析記憶體使用：

- **Python**：memory_profiler、tracemalloc
- **C/C++**：Valgrind（massif）、 AddressSanitizer
- **Java**：VisualVM、 JProfiler

```python
# Python 記憶體分析範例
import tracemalloc

tracemalloc.start()

# 執行你的演算法
result = your_algorithm(data)

current, peak = tracemalloc.get_traced_memory()
print(f"Current memory: {current / 1024 / 1024:.2f} MB")
print(f"Peak memory: {peak / 1024 / 1024:.2f} MB")

tracemalloc.stop()
```

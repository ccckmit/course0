# 迭代法 (Iteration)

## 概述

迭代法透過重複執行某種運算，逐步逼近目標值或收斂到解。

## 基本結構

```python
def iterative_method(initial_guess, tolerance=1e-6, max_iterations=1000):
    """迭代法通用框架"""
    x = initial_guess
    
    for i in range(max_iterations):
        x_new = f(x)  # 迭代函數
        
        if abs(x_new - x) < tolerance:
            return x_new, i + 1
        
        x = x_new
    
    return x, max_iterations  # 未收斂
```

## 收斂類型

### 定點迭代

```python
def fixed_point_iteration(f, x0, tolerance=1e-6, max_iter=100):
    """定點迭代法：x = f(x)"""
    x = x0
    
    for i in range(max_iter):
        x_new = f(x)
        
        if abs(x_new - x) < tolerance:
            return x_new, i + 1
        
        x = x_new
    
    return None, max_iter

# 求解 sqrt(x) = 2 => x = 2²/x
f = lambda x: 4 / x
result, steps = fixed_point_iteration(f, x0=2)
print(f"sqrt(2) ≈ {result}, steps: {steps}")
```

### 牛頓法

```python
def newton_method(f, df, x0, tolerance=1e-6, max_iter=100):
    """牛頓迭代法：x_{n+1} = x_n - f(x)/f'(x)"""
    x = x0
    
    for i in range(max_iter):
        fx = f(x)
        dfx = df(x)
        
        if abs(dfx) < 1e-12:
            break
        
        x_new = x - fx / dfx
        
        if abs(x_new - x) < tolerance:
            return x_new, i + 1
        
        x = x_new
    
    return x, max_iter

# 求解 x² - 2 = 0
f = lambda x: x**2 - 2
df = lambda x: 2*x
result, steps = newton_method(f, df, x0=1)
print(f"sqrt(2) ≈ {result:.10f}, steps: {steps}")
```

### 梯度下降

```python
def gradient_descent(f, grad_f, x0, learning_rate=0.1, max_iter=1000):
    """梯度下降法"""
    x = x0
    
    for i in range(max_iter):
        gradient = grad_f(x)
        x_new = x - learning_rate * gradient
        
        if abs(f(x_new) - f(x)) < 1e-6:
            return x_new, i + 1
        
        x = x_new
    
    return x, max_iter

# 最小化 f(x) = x²
f = lambda x: x**2
grad_f = lambda x: 2*x
result, steps = gradient_descent(f, grad_f, x0=10)
print(f"Minimum x = {result:.6f}, steps: {steps}")
```

## 收斂條件

```python
def check_convergence(old, new, tolerance=1e-6):
    """檢查收斂"""
    return abs(new - old) < tolerance

def check_residual(f, x, tolerance=1e-6):
    """檢查殘差"""
    return abs(f(x)) < tolerance
```

## 收斂速度

| 方法 | 收斂速度 | 說明 |
|------|----------|------|
| 二分法 | 線性 | O(log n) |
| 定點迭代 | 線性/超線性 | 取決於導數 |
| 牛頓法 | 二次收斂 | 每步精度加倍 |
| 弦截法 | 超線性 | 1.618 階 |

## 應用場景

1. **方程求根**：牛頓法、弦截法
2. **數值最佳化**：梯度下降、牛頓法
3. **線性系統**：共軛梯度法
4. **機器學習**：訓練神經網路

## 注意事項

- 初始值選擇影響收斂
- 可能陷入局部最優（非凸函數）
- 需要設計有效的停止準則

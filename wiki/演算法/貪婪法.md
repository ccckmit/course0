# 貪婪法 (Greedy Algorithm)

## 概述

貪婪法是一種每一步都選擇當前最佳選項的演算法策略，期望通過局部最優解得到全域最優解。

## 基本思想

```
貪婪法流程：
1. 選擇函數：選擇下一個元素
2. 可行性檢查：確保選擇有效
3. 目標函數：更新最佳解
```

def greedy_change(money, coins=[100, 50, 10, 5, 1]):
    """最少硬幣數"""
    result = []
    for coin in sorted(coins, reverse=True):
        while money >= coin:
            money -= coin
            result.append(coin)
    return result

### 活動選擇問題

```python
def activity_selection(activities):
    """選擇最多不相衝突的活動"""
    activities.sort(key=lambda x: x[1])
    
    selected = [activities[0]]
    last_end = activities[0][1]
    
    for start, end in activities[1:]:
        if start >= last_end:
            selected.append((start, end))
            last_end = end
    
    return selected

activities = [(1,4), (3,5), (0,6), (5,7), (3,9), (5,9), (6,10), (8,11)]
print(activity_selection(activities))
# 輸出: [(1,4), (5,7), (8,11)]
```

### Huffman 編碼

```python
import heapq
from collections import Counter

def huffman_coding(text):
    """霍夫曼編碼"""
    freq = Counter(text)
    heap = [[weight, [char, '']] for char, weight in freq.items()]
    heapq.heapify(heap)
    
    while len(heap) > 1:
        lo = heapq.heappop(heap)
        hi = heapq.heappop(heap)
        for pair in lo[1:]:
            pair[1] = '0' + pair[1]
        for pair in hi[1:]:
            pair[1] = '1' + pair[1]
        heapq.heappush(heap, [lo[0]+hi[0]] + lo[1:] + hi[1:])
    
    return dict(heap[0][1:])

text = "this is an example"
codes = huffman_coding(text)
print(codes)  # {'a': '110', ' ': '111', ...}
```

## 最小生成樹

### Kruskal 演算法

```python
def kruskal_mst(vertices, edges):
    """Kruskal 最小生成樹"""
    edges.sort(key=lambda x: x[2])
    
    parent = list(range(vertices))
    
    def find(x):
        if parent[x] != x:
            parent[x] = find(parent[x])
        return parent[x]
    
    def union(x, y):
        px, py = find(x), find(y)
        if px != py:
            parent[px] = py
            return True
        return False
    
    mst = []
    for u, v, w in edges:
        if union(u, v):
            mst.append((u, v, w))
            if len(mst) == vertices - 1:
                break
    
    return mst

vertices = 6
edges = [(0,1,4), (0,2,3), (1,2,1), (1,3,2), (2,3,4), (2,4,2), (3,4,3), (3,5,2), (4,5,5)]
print(kruskal_mst(vertices, edges))
```

### Prim 演算法

```python
def prim_mst(graph, start=0):
    """Prim 最小生成樹"""
    vertices = len(graph)
    visited = [False] * vertices
    min_edge = [float('inf')] * vertices
    min_edge[start] = 0
    mst = []
    
    for _ in range(vertices):
        u = min([i for i in range(vertices) if not visited[i]], 
                key=lambda i: min_edge[i])
        visited[u] = True
        mst.append((u, min_edge[u]))
        
        for v in range(vertices):
            if graph[u][v] and not visited[v] and graph[u][v] < min_edge[v]:
                min_edge[v] = graph[u][v]
    
    return mst[1:]
```

## Dijkstra 最短路徑

```python
import heapq

def dijkstra(graph, start):
    """Dijkstra 最短路徑"""
    dist = {v: float('inf') for v in graph}
    dist[start] = 0
    pq = [(0, start)]
    visited = set()
    
    while pq:
        d, u = heapq.heappop(pq)
        if u in visited:
            continue
        visited.add(u)
        
        for v, weight in graph[u]:
            if dist[u] + weight < dist[v]:
                dist[v] = dist[u] + weight
                heapq.heappush(pq, (dist[v], v))
    
    return dist

graph = {
    'A': [('B', 4), ('C', 1)],
    'B': [('A', 4), ('C', 2), ('D', 5)],
    'C': [('A', 1), ('B', 2), ('D', 8)],
    'D': [('B', 5), ('C', 8)]
}
print(dijkstra(graph, 'A'))  # {'A': 0, 'B': 3, 'C': 1, 'D': 8}
```

## 分數揹包問題

```python
def fractional_knapsack(items, capacity):
    """分數揹包（貪婪法可得到最優解）"""
    items.sort(key=lambda x: x[1]/x[0], reverse=True)
    
    total_value = 0
    knapsack = []
    remaining = capacity
    
    for weight, value in items:
        if remaining == 0:
            break
        
        if weight <= remaining:
            total_value += value
            knapsack.append((weight, value))
            remaining -= weight
        else:
            fraction = remaining / weight
            total_value += value * fraction
            knapsack.append((remaining, value * fraction))
            remaining = 0
    
    return total_value, knapsack

items = [(10, 60), (20, 100), (30, 120)]
capacity = 50
print(fractional_knapsack(items, capacity))  # 240.0
```

## 貪婪法 vs 動態規劃

| 特性 | 貪婪法 | 動態規劃 |
|------|--------|-----------|
| 選擇策略 | 局部最優 | 全域考量 |
| 子問題 | 不重疊 | 重疊 |
| 時間複雜度 | 通常 O(n log n) | 通常 O(n²) 或更高 |
| 最優性 | 不一定 | 一定 |

## 何時使用貪婪法

1. **貪婪選擇性質**：全域最優可由局部最優導出
2. **最優子結構**：問題的最優解包含子問題的最優解

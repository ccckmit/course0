# 化簡法 (Reduction)

## 概述

化簡法將複雜問題轉換為已知的簡單問題來求解，是演算法設計的核心思想。

## 基本思想

```
複雜問題 A → 化簡 → 已知問題 B → 求解 → 反向映射 → A 的解
```

### 最大公因數化簡

```python
def gcd_reduce(a, b):
    """GCD 化簡：gcd(a,b) = gcd(b, a mod b)"""
    if b == 0:
        return a
    return gcd_reduce(b, a % b)

# 歐幾裏得算法
def gcd_iterative(a, b):
    while b:
        a, b = b, a % b
    return a

print(gcd_iterative(48, 18))  # 6
```

### 矩陣乘法化簡

```python
def matrix_multiply_reduce(A, B):
    """矩陣乘法可化簡為向量乘法"""
    n = len(A)
    C = [[0] * n for _ in range(n)]
    
    for i in range(n):
        for k in range(n):
            if A[i][k] != 0:
                for j in range(n):
                    C[i][j] += A[i][k] * B[k][j]
    
    return C
```

## 化簡的用途

1. **問題分類**：證明 NP-完全性
2. **演算法設計**：將新問題化簡為已知問題
3. **複雜度分析**：透過已知複雜度推估
4. **實用求解**：借用現成工具

## 化簡範例：電路評估

```python
class Gate:
    pass

class InputGate(Gate):
    def __init__(self, value):
        self.value = value

class AndGate(Gate):
    def __init__(self, left, right):
        self.left = left
        self.right = right
    
    def evaluate(self):
        return self.left.evaluate() and self.right.evaluate()

class OrGate(Gate):
    def __init__(self, left, right):
        self.left = left
        self.right = right
    
    def evaluate(self):
        return self.left.evaluate() or self.right.evaluate()

class NotGate(Gate):
    def __init__(self, input_gate):
        self.input_gate = input_gate
    
    def evaluate(self):
        return not self.input_gate.evaluate()

# 使用：電路評估化簡為布林計算
x, y = InputGate(True), InputGate(False)
circuit = AndGate(OrGate(x, y), NotGate(y))
print(circuit.evaluate())  # True
```

## 化簡複雜度

| 化簡類型 | 複雜度 |
|----------|--------|
| 線性 | O(n) |
| 多項式 | O(n^k) |
| 指數 | O(2^n) |
| 任意 | 可能無效 |

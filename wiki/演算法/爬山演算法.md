# 爬山演算法 (Hill Climbing)

## 概述

爬山演算法是一種局部搜尋演算法，從一個起始點出發，逐步移動到相鄰的較好位置，直到找到局部最優解。

## 基本流程

```
1. 選擇起始點
2. 找出相鄰解
3. 選擇最佳相鄰解
4. 若更好則移動，否則停止
```

## 基本實現

```python
import random
import math

def hill_climbing(f, neighbors, x0, max_iter=1000, tolerance=1e-6):
    """爬山演算法"""
    current = x0
    current_value = f(current)
    
    for i in range(max_iter):
        neighbors_list = neighbors(current)
        best_neighbor = None
        best_value = current_value
        
        for neighbor in neighbors_list:
            value = f(neighbor)
            if value < best_value:  # 最小化
                best_value = value
                best_neighbor = neighbor
        
        if best_neighbor is None:
            break  # 局部最優
        
        current = best_neighbor
        current_value = best_value
        
        if abs(current_value - best_value) < tolerance:
            break
    
    return current, current_value
```

## 爬山法變體

### 簡單爬山法

```python
def simple_hill_climbing(f, neighbors, x0, max_iter=1000):
    """簡單爬山：找到第一個更好的就移動"""
    current = x0
    current_value = f(current)
    
    for _ in range(max_iter):
        neighbors_list = neighbors(current)
        random.shuffle(neighbors_list)
        
        for neighbor in neighbors_list:
            value = f(neighbor)
            if value < current_value:
                current = neighbor
                current_value = value
                break
        else:
            break
    
    return current, current_value
```

### 隨機爬山法

```python
def random_hill_climbing(f, neighbors, x0, max_iter=1000, max_sideways=100):
    """隨機爬山：允許橫向移動"""
    current = x0
    current_value = f(current)
    sideways_count = 0
    
    for _ in range(max_iter):
        neighbors_list = neighbors(current)
        random.shuffle(neighbors_list)
        
        for neighbor in neighbors_list:
            value = f(neighbor)
            
            if value < current_value:
                current = neighbor
                current_value = value
                sideways_count = 0
                break
            elif value == current_value and sideways_count < max_sideways:
                current = neighbor
                sideways_count += 1
                break
        else:
            break
    
    return current, current_value
```

## 爬山法應用

### TSP 問題

```python
def tsp_neighbors(route):
    """產生相鄰解：交換兩個城市"""
    neighbors = []
    n = len(route)
    for i in range(n):
        for j in range(i + 1, n):
            new_route = route.copy()
            new_route[i], new_route[j] = new_route[j], new_route[i]
            neighbors.append(new_route)
    return neighbors

def tsp_distance(route, distances):
    """計算路線總距離"""
    total = 0
    for i in range(len(route)):
        total += distances[route[i]][route[(i+1) % len(route)]]
    return total

def solve_tsp_hill_climbing(distances, max_iter=10000):
    """爬山法求解 TSP"""
    n = len(distances)
    route = list(range(n))
    random.shuffle(route)
    
    def f(route):
        return tsp_distance(route, distances)
    
    def neighbors(route):
        return tsp_neighbors(route)
    
    best, _ = hill_climbing(f, neighbors, route, max_iter)
    return best, tsp_distance(best, distances)
```

### 函數優化

def optimize_function_hill_climbing():
    """求解 Rastrigrin 函數"""
    def rastrigin(x, y):
        A = 10
        return A * 2 + (x**2 - A * math.cos(2 * math.pi * x)) + \
               (y**2 - A * math.cos(2 * math.pi * y))
    
    def neighbors(point):
        x, y = point
        return [
            (x + random.uniform(-0.5, 0.5), y),
            (x, y + random.uniform(-0.5, 0.5)),
            (x + random.uniform(-0.1, 0.1), y + random.uniform(-0.1, 0.1))
        ]
    
    best, value = hill_climbing(rastrigin, neighbors, (random.uniform(-5, 5), random.uniform(-5, 5)))
    return best, value

result = optimize_function_hill_climbing()
print(f"Minimum at: {result[0]}, value: {result[1]:.4f}")

## 爬山法的問題

### 局部最優

```
爬山法可能陷入局部最優而非全域最優：

    全域最優
         ↑
        / \
       /   \
      /局部\     ← 爬山法可能停在這
     /最優  \
    /________\____
```

### 高原

```python
def plateau_hill_climbing(f, neighbors, x0, max_iter=1000, epsilon=1e-6):
    """高原處理：加入小隨機擾動"""
    current = x0
    current_value = f(current)
    
    for _ in range(max_iter):
        neighbors_list = neighbors(current)
        
        best_neighbor = None
        best_value = current_value
        
        for neighbor in neighbors_list:
            value = f(neighbor)
            if value < best_value - epsilon:
                best_value = value
                best_neighbor = neighbor
        
        if best_neighbor is None:
            # 隨機擾動突破高原
            current = (current[0] + random.uniform(-0.1, 0.1),
                      current[1] + random.uniform(-0.1, 0.1))
            current_value = f(current)
        else:
            current = best_neighbor
            current_value = best_value
    
    return current, current_value
```

## 爬山法特點

| 特性 | 說明 |
|------|------|
| 時間複雜度 | O(iterations × neighbors) |
| 空間複雜度 | O(1) |
| 優點 | 簡單、快速、記憶體少 |
| 缺點 | 易陷入局部最優 |

## 與其他方法比較

| 方法 | 局部最優問題 | 跳出局部最優 |
|------|-------------|-------------|
| 爬山法 | 嚴重 | 無 |
| 模擬退火 | 輕微 | 概率接受差解 |
| 遺傳演算法 | 輕微 | 種羣多樣性 |
| 粒子羣優化 | 輕微 | 羣體協作 |

# 分治法 (Divide and Conquer)

## 概述

分治法將問題分割成多個較小的子問題，分別求解後合併結果。

## 基本步驟

```
分治法流程：
1. 分（Divide）：將問題分割成子問題
2. 治（Conquer）：遞迴求解子問題
3. 合（Merge）：合併子問題的解答
```

## 經典問題

### 合併排序

```python
def merge_sort(arr):
    """合併排序"""
    if len(arr) <= 1:
        return arr
    
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    
    return merge(left, right)

def merge(left, right):
    """合併兩個有序陣列"""
    result = []
    i = j = 0
    
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    
    result.extend(left[i:])
    result.extend(right[j:])
    return result
```

### 快速排序

```python
def quicksort(arr, low=0, high=None):
    """快速排序"""
    if high is None:
        high = len(arr) - 1
    
    if low < high:
        pivot_index = partition(arr, low, high)
        quicksort(arr, low, pivot_index - 1)
        quicksort(arr, pivot_index + 1, high)
    
    return arr

def partition(arr, low, high):
    """partition：選 pivot 並分區"""
    pivot = arr[high]
    i = low - 1
    
    for j in range(low, high):
        if arr[j] <= pivot:
            i += 1
            arr[i], arr[j] = arr[j], arr[i]
    
    arr[i + 1], arr[high] = arr[high], arr[i + 1]
    return i + 1
```

### 最大子陣列

```python
def max_subarray(arr, low, high):
    """最大子陣列問題（最大和連續子陣列）"""
    if low == high:
        return low, high, arr[low]
    
    mid = (low + high) // 2
    
    left_low, left_high, left_sum = max_subarray(arr, low, mid)
    right_low, right_high, right_sum = max_subarray(arr, mid + 1, high)
    cross_low, cross_high, cross_sum = max_crossing_subarray(arr, low, mid, high)
    
    if left_sum >= right_sum and left_sum >= cross_sum:
        return left_low, left_high, left_sum
    elif right_sum >= left_sum and right_sum >= cross_sum:
        return right_low, right_high, right_sum
    else:
        return cross_low, cross_high, cross_sum

def max_crossing_subarray(arr, low, mid, high):
    """穿過中點的最大子陣列"""
    left_sum = float('-inf')
    total = 0
    max_left = mid
    
    for i in range(mid, low - 1, -1):
        total += arr[i]
        if total > left_sum:
            left_sum = total
            max_left = i
    
    right_sum = float('-inf')
    total = 0
    max_right = mid + 1
    
    for i in range(mid + 1, high + 1):
        total += arr[i]
        if total > right_sum:
            right_sum = total
            max_right = i
    
    return max_left, max_right, left_sum + right_sum
```

###  Closest Pair of Points

```python
import math

def closest_pair(points):
    """最近點對問題"""
    points_x = sorted(points, key=lambda p: p[0])
    points_y = sorted(points, key=lambda p: p[1])
    
    return closest_pair_rec(points_x, points_y)

def closest_pair_rec(px, py):
    """分治求最近點對"""
    n = len(px)
    
    if n <= 3:
        return brute_force_closest(px)
    
    mid = n // 2
    mid_x = px[mid][0]
    
    pyl = [p for p in py if p[0] <= mid_x]
    pyr = [p for p in py if p[0] > mid_x]
    
    dl, pl = closest_pair_rec(px[:mid], pyl)
    dr, pr = closest_pair_rec(px[mid:], pyr)
    
    d = min(dl, dr)
    pair = pl if dl < dr else pr
    
    strip = [p for p in py if abs(p[0] - mid_x) < d]
    
    for i in range(len(strip)):
        for j in range(i + 1, min(i + 7, len(strip))):
            dy = strip[j][1] - strip[i][1]
            if dy < d:
                d = dy
                pair = (strip[i], strip[j])
    
    return d, pair

def brute_force_closest(points):
    """暴力法求最近點對"""
    min_dist = float('inf')
    pair = None
    
    for i in range(len(points)):
        for j in range(i + 1, len(points)):
            d = math.sqrt((points[i][0]-points[j][0])**2 + 
                         (points[i][1]-points[j][1])**2)
            if d < min_dist:
                min_dist = d
                pair = (points[i], points[j])
    
    return min_dist, pair
```

### 矩陣乘法 Strassen

```python
def strassen(A, B):
    """Strassen 矩陣乘法：O(n^2.807) 而非 O(n^3)"""
    n = len(A)
    
    if n == 1:
        return [[A[0][0] * B[0][0]]]
    
    mid = n // 2
    
    A11 = [row[:mid] for row in A[:mid]]
    A12 = [row[mid:] for row in A[:mid]]
    A21 = [row[:mid] for row in A[mid:]]
    A22 = [row[mid:] for row in A[mid:]]
    
    B11 = [row[:mid] for row in B[:mid]]
    B12 = [row[mid:] for row in B[:mid]]
    B21 = [row[:mid] for row in B[mid:]]
    B22 = [row[mid:] for row in B[mid:]]
    
    M1 = strassen(add(A11, A22), add(B11, B22))
    M2 = strassen(add(A21, A22), B11)
    M3 = strassen(A11, sub(B12, B22))
    M4 = strassen(A22, sub(B21, B11))
    M5 = strassen(add(A11, A12), B22)
    M6 = strassen(sub(A21, A11), add(B11, B12))
    M7 = strassen(sub(A12, A22), add(B21, B22))
    
    C11 = add(sub(add(M1, M4), M5), M7)
    C12 = add(M3, M5)
    C21 = add(M2, M4)
    C22 = add(sub(add(M1, M3), M2), M6)
    
    return combine(C11, C12, C21, C22)

def add(A, B):
    return [[A[i][j] + B[i][j] for j in range(len(A))] for i in range(len(A))]

def sub(A, B):
    return [[A[i][j] - B[i][j] for j in range(len(A))] for i in range(len(A))]

def combine(C11, C12, C21, C22):
    n = len(C11)
    result = [[0] * (2 * n) for _ in range(2 * n)]
    for i in range(n):
        for j in range(n):
            result[i][j] = C11[i][j]
            result[i][j + n] = C12[i][j]
            result[i + n][j] = C21[i][j]
            result[i + n][j + n] = C22[i][j]
    return result
```

## 分治法複雜度分析

| 問題 | 時間複雜度 |
|------|-----------|
| 合併排序 | O(n log n) |
| 快速排序 | O(n log n) 平均 |
| 最大子陣列 | O(n log n) |
| 最近點對 | O(n log n) |
| Strassen 乘法 | O(n^2.807) |

## 何時使用分治法

1. 問題可分割成相似子問題
2. 子問題相互獨立
3. 可有效合併子解答
4. 分割與合併代價合理

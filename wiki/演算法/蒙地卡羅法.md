# 蒙地卡羅法 (Monte Carlo Method)

## 概述

蒙地卡羅法是一種使用隨機取樣來解決問題的計算方法，廣泛應用於數值積分、最佳化、統計估計等領域。

## 基本原理

```
蒙地卡羅法核心思想：
1. 用隨機樣本估計整體
2. 樣本越多，估計越準確
3. 大數定律保證收斂
```

## 數值積分

### 計算 π

```python
import random

def estimate_pi(n_samples):
    """用蒙地卡羅法估計 π"""
    inside = 0
    
    for _ in range(n_samples):
        x = random.random()
        y = random.random()
        if x**2 + y**2 <= 1:
            inside += 1
    
    return 4 * inside / n_samples

for n in [100, 1000, 10000, 100000]:
    print(f"n={n:6d}: π ≈ {estimate_pi(n):.6f}")
# n=   100: π ≈ 3.080000
# n=  1000: π ≈ 3.148000
# n= 10000: π ≈ 3.141592
# n=100000: π ≈ 3.14159265
```

### 多重積分

```python
def integrate_monte_carlo(f, bounds, n_samples):
    """多重積分"""
    volume = 1
    
    for low, high in bounds:
        volume *= (high - low)
    
    total = 0
    for _ in range(n_samples):
        point = [random.uniform(low, high) for low, high in bounds]
        total += f(*point)
    
    return volume * total / n_samples

f = lambda x, y: x**2 + y**2
result = integrate_monte_carlo(f, [(0, 1), (0, 1)], 100000)
print(f"∫∫ x²+y² dxdy = {result:.4f}")  # ≈ 0.6667
```

## 隨機最佳化

### 模擬退火

```python
import math

def simulated_annealing(f, x0, T0, T_min, cooling_rate, n_iter):
    """模擬退火演算法"""
    x = x0
    T = T0
    best_x = x
    best_f = f(x)
    
    while T > T_min:
        for _ in range(n_iter):
            x_new = x + random.uniform(-T, T)
            
            delta = f(x_new) - f(x)
            
            if delta < 0 or random.random() < math.exp(-delta / T):
                x = x_new
                
                if f(x) < best_f:
                    best_x = x
                    best_f = f(x)
        
        T *= cooling_rate
    
    return best_x, best_f

f = lambda x: x**2 + 10*math.sin(x)
result, value = simulated_annealing(
    f, x0=0, T0=10, T_min=0.01, cooling_rate=0.95, n_iter=100
)
print(f"Minimum: x={result:.4f}, f(x)={value:.4f}")
```

## 統計估計

### Bootstrap

```python
import statistics

def bootstrap(data, statistic_func, n_bootstrap):
    """Bootstrap 置信區間"""
    n = len(data)
    samples = []
    
    for _ in range(n_bootstrap):
        sample = [random.choice(data) for _ in range(n)]
        samples.append(statistic_func(sample))
    
    samples.sort()
    lower = samples[int(0.025 * n_bootstrap)]
    upper = samples[int(0.975 * n_bootstrap)]
    
    return {
        'estimate': statistic_func(data),
        'std_error': statistics.stdev(samples),
        'ci_95': (lower, upper)
    }

data = [2.3, 2.5, 2.8, 2.1, 2.6, 2.4, 2.9, 2.2]
result = bootstrap(data, statistics.mean, 10000)
print(f"Estimate: {result['estimate']:.4f}")
print(f"Std Error: {result['std_error']:.4f}")
print(f"95% CI: ({result['ci_95'][0]:.4f}, {result['ci_95'][1]:.4f})")
```

### 馬可夫鏈蒙地卡羅 (MCMC)

```python
def metropolis_hastings(target_pdf, proposal_std, n_samples):
    """Metropolis-Hastings 採樣"""
    samples = []
    x = 0
    
    for _ in range(n_samples):
        x_proposed = random.gauss(x, proposal_std)
        
        acceptance = min(1, target_pdf(x_proposed) / target_pdf(x))
        
        if random.random() < acceptance:
            x = x_proposed
        
        samples.append(x)
    
    return samples

target = lambda x: math.exp(-x**2 / 2) / math.sqrt(2 * math.pi)
samples = metropolis_hastings(target, proposal_std=1, n_samples=10000)
print(f"Mean: {sum(samples)/len(samples):.4f}")  # ≈ 0
print(f"Std: {statistics.stdev(samples):.4f}")  # ≈ 1
```

## 收斂性

| 特性 | 說明 |
|------|------|
| O(1/√n) | 標準誤差隨樣本數量下降 |
| 獨立採樣 | 假設隨機數生成器均勻 |
| 變異數估計 | 可計算估計的不確定性 |

## 優缺點

| 優點 | 缺點 |
|------|------|
| 簡單易實現 | 收斂慢 |
| 適用高維問題 | 需要大量樣本 |
| 可處理複雜邊界 | 變異數可能很大 |

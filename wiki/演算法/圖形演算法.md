# 圖形演算法

**標籤**: #演算法 #圖形 #BFS #DFS #最短路徑 #Dijkstra
**日期**: 2024-01-15

圖形演算法是處理圖結構資料的演算法，廣泛應用於社群網路分析、路徑規劃、網頁排序、網路流等領域。圖是由頂點（Vertex）和邊（Edge）組成的抽象資料結構，能夠有效表示物件之間的複雜關係。

## 圖的基礎概念

### 圖的定義

圖 G 由頂點集合 V 和邊集合 E 組成，記作 G = (V, E)。根據邊的性質，圖可分為多種類型：

| 類型 | 說明 | 範例 |
|------|------|------|
| 有向圖 | 邊有方向 | 網頁連結、道路單行道 |
| 無向圖 | 邊無方向 | 社群關係、地鐵線路 |
| 加權圖 | 邊有權重 | 道路距離、成本 |
| 無權圖 | 邊無權重 | 社交連接、網頁連結 |

### 圖的基本術語

- **頂點 (Vertex)**: 圖中的節點
- **邊 (Edge)**: 連接兩個頂點的線
- **度 (Degree)**: 頂點相連的邊數
- **入度/出度**: 有向圖中進入/離開頂點的邊數
- **路徑 (Path)**: 頂點序列，路徑上相鄰頂點有邊相連
- **環 (Cycle)**: 起點和終點相同的路徑
- **連通圖**: 任意兩頂點間都有路徑相連
- **生成樹 (Spanning Tree)**: 包含所有頂點的樹

## 圖的表示方法

### 鄰接矩陣

鄰接矩陣使用二維陣列表示圖，適合稠密圖。

```python
class GraphMatrix:
    def __init__(self, n):
        self.n = n
        self.matrix = [[float('inf')] * n for _ in range(n)]
        for i in range(n):
            self.matrix[i][i] = 0
    
    def add_edge(self, u, v, weight=1):
        self.matrix[u][v] = weight
        self.matrix[v][u] = weight  # 無向圖
    
    def add_directed_edge(self, u, v, weight=1):
        self.matrix[u][v] = weight
    
    def get_neighbors(self, v):
        return [i for i in range(self.n) if self.matrix[v][i] < float('inf') and i != v]
    
    def print_graph(self):
        for row in self.matrix:
            print(row)
```

### 鄰接表

鄰接表使用字典或串列表示圖，適合稀疏圖。

```python
class GraphAdjList:
    def __init__(self, directed=False):
        self.graph = {}
        self.directed = directed
    
    def add_vertex(self, v):
        if v not in self.graph:
            self.graph[v] = []
    
    def add_edge(self, u, v, weight=1):
        self.add_vertex(u)
        self.add_vertex(v)
        self.graph[u].append((v, weight))
        if not self.directed:
            self.graph[v].append((u, weight))
    
    def get_neighbors(self, v):
        return self.graph.get(v, [])
    
    def vertices(self):
        return list(self.graph.keys())
```

## 圖的遍歷演算法

### BFS (廣度優先搜尋)

BFS 從起點開始，由近及遠層層搜尋，使用佇列輔助。

```python
from collections import deque

def bfs(graph, start, target=None):
    """
    廣度優先搜尋
    時間複雜度: O(V + E)
    空間複雜度: O(V)
    """
    visited = set()
    queue = deque([start])
    visited.add(start)
    order = []
    
    while queue:
        node = queue.popleft()
        order.append(node)
        
        if target and node == target:
            return order
        
        for neighbor in graph.get_neighbors(node):
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)
    
    return order

def bfs_with_distance(graph, start):
    """BFS 計算最短距離"""
    distances = {start: 0}
    queue = deque([start])
    
    while queue:
        node = queue.popleft()
        for neighbor, weight in graph.get_neighbors(node):
            if neighbor not in distances:
                distances[neighbor] = distances[node] + weight
                queue.append(neighbor)
    
    return distances
```

### DFS (深度優先搜尋)

DFS 沿路徑深入搜尋，使用堆疊或遞迴實現。

```python
def dfs_recursive(graph, start, visited=None, order=None):
    """
    深度優先搜尋（遞迴版本）
    時間複雜度: O(V + E)
    空間複雜度: O(V)
    """
    if visited is None:
        visited = set()
    if order is None:
        order = []
    
    visited.add(start)
    order.append(start)
    
    for neighbor in graph.get_neighbors(start):
        if neighbor not in visited:
            dfs_recursive(graph, neighbor, visited, order)
    
    return order

def dfs_iterative(graph, start):
    """DFS 迭代版本（使用堆疊）"""
    visited = set()
    stack = [start]
    order = []
    
    while stack:
        node = stack.pop()
        if node not in visited:
            visited.add(node)
            order.append(node)
            
            for neighbor in graph.get_neighbors(node):
                if neighbor not in visited:
                    stack.append(neighbor)
    
    return order
```

### DFS 應用：拓撲排序

拓撲排序適用於有向無環圖 (DAG)，常用於任務排程。

```python
def topological_sort(graph):
    """
    Kahn 演算法進行拓撲排序
    時間複雜度: O(V + E)
    """
    in_degree = {v: 0 for v in graph.vertices()}
    
    for v in graph.vertices():
        for neighbor, _ in graph.get_neighbors(v):
            in_degree[neighbor] += 1
    
    queue = deque([v for v in graph.vertices() if in_degree[v] == 0])
    result = []
    
    while queue:
        node = queue.popleft()
        result.append(node)
        
        for neighbor, _ in graph.get_neighbors(node):
            in_degree[neighbor] -= 1
            if in_degree[neighbor] == 0:
                queue.append(neighbor)
    
    if len(result) != len(graph.vertices()):
        raise ValueError("圖含有環，無法進行拓撲排序")
    
    return result
```

### DFS 應用：檢測環

```python
def has_cycle(graph, directed=False):
    """檢測圖中是否存在環"""
    WHITE, GRAY, BLACK = 0, 1, 2
    color = {v: WHITE for v in graph.vertices()}
    
    def dfs_has_cycle(node):
        color[node] = GRAY
        
        for neighbor, _ in graph.get_neighbors(node):
            if color[neighbor] == GRAY:
                return True
            if color[neighbor] == WHITE:
                if dfs_has_cycle(neighbor):
                    return True
        
        color[node] = BLACK
        return False
    
    for v in graph.vertices():
        if color[v] == WHITE:
            if dfs_has_cycle(v):
                return True
    
    return False
```

## 最短路徑演算法

### Dijkstra 演算法

Dijkstra 演算法適用於非負權重的有向或無向圖。

```python
import heapq

def dijkstra(graph, start):
    """
    Dijkstra 最短路徑演算法
    時間複雜度: O((V + E) log V)
    適用: 非負權重圖
    """
    dist = {v: float('inf') for v in graph.vertices()}
    dist[start] = 0
    prev = {v: None for v in graph.vertices()}
    pq = [(0, start)]
    
    while pq:
        current_dist, u = heapq.heappop(pq)
        
        if current_dist > dist[u]:
            continue
        
        for neighbor, weight in graph.get_neighbors(u):
            new_dist = dist[u] + weight
            
            if new_dist < dist[neighbor]:
                dist[neighbor] = new_dist
                prev[neighbor] = u
                heapq.heappush(pq, (new_dist, neighbor))
    
    return dist, prev

def reconstruct_path(prev, start, end):
    """重建路徑"""
    path = []
    current = end
    
    while current is not None:
        path.append(current)
        current = prev[current]
    
    path.reverse()
    return path if path[0] == start else []
```

### Bellman-Ford 演算法

Bellman-Ford 演算法可以處理負權重邊，並能檢測負權重環。

```python
def bellman_ford(graph, start):
    """
    Bellman-Ford 演算法
    時間複雜度: O(VE)
    適用: 允許負權重邊
    """
    vertices = list(graph.vertices())
    dist = {v: float('inf') for v in vertices}
    prev = {v: None for v in vertices}
    dist[start] = 0
    
    edges = []
    for u in vertices:
        for v, w in graph.get_neighbors(u):
            edges.append((u, v, w))
    
    # V-1 次鬆弛操作
    for _ in range(len(vertices) - 1):
        updated = False
        for u, v, w in edges:
            if dist[u] != float('inf') and dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
                prev[v] = u
                updated = True
        if not updated:
            break
    
    # 檢測負權重環
    has_negative_cycle = False
    for u, v, w in edges:
        if dist[u] != float('inf') and dist[u] + w < dist[v]:
            has_negative_cycle = True
            break
    
    return dist, prev, has_negative_cycle
```

### Floyd-Warshall 演算法

Floyd-Warshall 演算法計算所有頂點對之間的最短路徑。

```python
def floyd_warshall(vertices, edges):
    """
    Floyd-Warshall 演算法
    時間複雜度: O(V³)
    適用: 計算所有點對最短路徑
    """
    n = len(vertices)
    vertex_to_idx = {v: i for i, v in enumerate(vertices)}
    
    dist = [[float('inf')] * n for _ in range(n)]
    next_node = [[None] * n for _ in range(n)]
    
    for i in range(n):
        dist[i][i] = 0
    
    for u, v, w in edges:
        i, j = vertex_to_idx[u], vertex_to_idx[v]
        dist[i][j] = w
        next_node[i][j] = j
    
    for k in range(n):
        for i in range(n):
            for j in range(n):
                if dist[i][k] + dist[k][j] < dist[i][j]:
                    dist[i][j] = dist[i][k] + dist[k][j]
                    next_node[i][j] = next_node[i][k]
    
    return dist, next_node
```

### A* 演算法

A* 演算法使用啟發式函數加速路徑搜尋。

```python
def astar(graph, start, goal, heuristic):
    """
    A* 最短路徑演算法
    時間複雜度: O(E)
    空間複雜度: O(V)
    """
    open_set = [(0, start)]
    came_from = {}
    g_score = {v: float('inf') for v in graph.vertices()}
    g_score[start] = 0
    
    while open_set:
        _, current = heapq.heappop(open_set)
        
        if current == goal:
            return reconstruct_path_astar(came_from, current)
        
        for neighbor, weight in graph.get_neighbors(current):
            tentative_g = g_score[current] + weight
            
            if tentative_g < g_score[neighbor]:
                came_from[neighbor] = current
                g_score[neighbor] = tentative_g
                f_score = tentative_g + heuristic(neighbor, goal)
                heapq.heappush(open_set, (f_score, neighbor))
    
    return None

def reconstruct_path_astar(came_from, current):
    path = [current]
    while current in came_from:
        current = came_from[current]
        path.append(current)
    path.reverse()
    return path
```

## 最小生成樹 (MST)

### Prim 演算法

```python
def prim(graph, start):
    """
    Prim 演算法
    時間複雜度: O(E log V) - 使用堆疊
    """
    vertices = list(graph.vertices())
    key = {v: float('inf') for v in vertices}
    parent = {v: None for v in vertices}
    in_mst = {v: False for v in vertices}
    
    key[start] = 0
    pq = [(0, start)]
    
    while pq:
        _, u = heapq.heappop(pq)
        
        if in_mst[u]:
            continue
        
        in_mst[u] = True
        
        for v, weight in graph.get_neighbors(u):
            if not in_mst[v] and weight < key[v]:
                key[v] = weight
                parent[v] = u
                heapq.heappush(pq, (weight, v))
    
    mst_edges = []
    for v in vertices:
        if parent[v] is not None:
            mst_edges.append((parent[v], v, key[v]))
    
    return mst_edges
```

### Kruskal 演算法

```python
class UnionFind:
    def __init__(self, vertices):
        self.parent = {v: v for v in vertices}
        self.rank = {v: 0 for v in vertices}
    
    def find(self, x):
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])
        return self.parent[x]
    
    def union(self, x, y):
        px, py = self.find(x), self.find(y)
        if px == py:
            return False
        
        if self.rank[px] < self.rank[py]:
            px, py = py, px
        self.parent[py] = px
        if self.rank[px] == self.rank[py]:
            self.rank[px] += 1
        return True

def kruskal(graph):
    """
    Kruskal 演算法
    時間複�雜度: O(E log E)
    """
    vertices = list(graph.vertices())
    uf = UnionFind(vertices)
    
    edges = []
    for u in vertices:
        for v, w in graph.get_neighbors(u):
            edges.append((w, u, v))
    
    edges.sort()
    mst = []
    
    for w, u, v in edges:
        if uf.union(u, v):
            mst.append((u, v, w))
            if len(mst) == len(vertices) - 1:
                break
    
    return mst
```

## 強連通分量

### Kosaraju 演算法

```python
def kosaraju_scc(graph):
    """
    Kosaraju 演算法找強連通分量
    時間複雜度: O(V + E)
    """
    visited = set()
    finish_order = []
    
    def dfs1(v):
        visited.add(v)
        for neighbor, _ in graph.get_neighbors(v):
            if neighbor not in visited:
                dfs1(neighbor)
        finish_order.append(v)
    
    for v in graph.vertices():
        if v not in visited:
            dfs1(v)
    
    reversed_graph = GraphAdjList(directed=True)
    for u in graph.vertices():
        for v, w in graph.get_neighbors(u):
            reversed_graph.add_directed_edge(v, u, w)
    
    visited.clear()
    sccs = []
    
    def dfs2(v, component):
        visited.add(v)
        component.append(v)
        for neighbor, _ in reversed_graph.get_neighbors(v):
            if neighbor not in visited:
                dfs2(neighbor, component)
    
    for v in reversed(finish_order):
        if v not in visited:
            component = []
            dfs2(v, component)
            sccs.append(component)
    
    return sccs
```

### Tarjan 演算法

```python
def tarjan_scc(graph):
    """
    Tarjan 演算法找強連通分量
    時間複雜度: O(V + E)
    """
    vertices = list(graph.vertices())
    index = {v: -1 for v in vertices}
    low_link = {v: 0 for v in vertices}
    on_stack = {v: False for v in vertices}
    stack = []
    sccs = []
    current_index = 0
    
    def strongconnect(v):
        nonlocal current_index
        index[v] = current_index
        low_link[v] = current_index
        current_index += 1
        stack.append(v)
        on_stack[v] = True
        
        for neighbor, _ in graph.get_neighbors(v):
            if index[neighbor] == -1:
                strongconnect(neighbor)
                low_link[v] = min(low_link[v], low_link[neighbor])
            elif on_stack[neighbor]:
                low_link[v] = min(low_link[v], index[neighbor])
        
        if low_link[v] == index[v]:
            scc = []
            while True:
                w = stack.pop()
                on_stack[w] = False
                scc.append(w)
                if w == v:
                    break
            sccs.append(scc)
    
    for v in vertices:
        if index[v] == -1:
            strongconnect(v)
    
    return sccs
```

## 網路流

### Ford-Fulkerson 演算法

```python
def ford_fulkerson(graph, source, sink):
    """
    Ford-Fulkerson 最大流演算法
    時間複雜度: O(E * max_flow)
    """
    def bfs_find_path(capacity, source, sink):
        visited = {source}
        queue = deque([source])
        parent = {}
        
        while queue:
            u = queue.popleft()
            
            for v, cap in graph.get_neighbors(u):
                if v not in visited and capacity[(u, v)] > 0:
                    visited.add(v)
                    parent[v] = u
                    if v == sink:
                        path = []
                        current = sink
                        while current != source:
                            path.append((parent[current], current))
                            current = parent[current]
                        return path[::-1]
                    queue.append(v)
        return None
    
    capacity = {}
    for u in graph.vertices():
        for v, w in graph.get_neighbors(u):
            capacity[(u, v)] = w
    
    max_flow = 0
    
    while True:
        path = bfs_find_path(capacity, source, sink)
        if not path:
            break
        
        flow = min(capacity[(u, v)] for u, v in path)
        
        for u, v in path:
            capacity[(u, v)] -= flow
            capacity.setdefault((v, u), 0)
            capacity[(v, u)] += flow
        
        max_flow += flow
    
    return max_flow
```

### 社群網路分析

```python
def find_clusters(graph):
    """使用 Union-Find 找出社群"""
    uf = UnionFind(graph.vertices())
    
    for u in graph.vertices():
        for v, _ in graph.get_neighbors(u):
            uf.union(u, v)
    
    clusters = {}
    for v in graph.vertices():
        root = uf.find(v)
        if root not in clusters:
            clusters[root] = []
        clusters[root].append(v)
    
    return clusters
```

### PageRank 演算法

```python
def pagerank(graph, damping=0.85, iterations=100):
    """PageRank 網頁排序演算法"""
    vertices = list(graph.vertices())
    n = len(vertices)
    rank = {v: 1/n for v in vertices}
    
    out_degree = {v: len(graph.get_neighbors(v)) for v in vertices}
    
    for _ in range(iterations):
        new_rank = {v: (1-damping)/n for v in vertices}
        
        for u in vertices:
            if out_degree[u] == 0:
                continue
            for v, _ in graph.get_neighbors(u):
                new_rank[v] += damping * rank[u] / out_degree[u]
        
        rank = new_rank
    
    return rank
```

## 時間複雜度比較

| 演算法 | 時間複雜度 | 空間複雜度 | 適用場景 |
|--------|------------|------------|----------|
| BFS | O(V + E) | O(V) | 最短路徑（無權）、層次遍歷 |
| DFS | O(V + E) | O(V) | 拓撲排序、環檢測、搜尋 |
| Dijkstra | O((V+E) log V) | O(V) | 單源最短路徑 |
| Bellman-Ford | O(VE) | O(V) | 負權重最短路徑 |
| Floyd-Warshall | O(V³) | O(V²) | 全點對最短路徑 |
| Prim | O(E log V) | O(V) | 最小生成樹 |
| Kruskal | O(E log E) | O(V) | 最小生成樹 |
| Kosaraju | O(V + E) | O(V) | 強連通分量 |
| Tarjan | O(V + E) | O(V) | 強連通分量 |

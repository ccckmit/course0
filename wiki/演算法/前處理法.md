# 前處理法 (Preprocessing)

## 概述

前處理法在主要計算前對資料進行準備和轉換，以提高演算法效率。

## 常見前處理技術

### 排序

```python
def preprocess_sort(arr):
    """排序作為前處理"""
    return sorted(arr)

def preprocess_rank(arr):
    """計算排名"""
    sorted_arr = sorted(arr)
    ranks = {v: i for i, v in enumerate(sorted_arr)}
    return [ranks[x] for x in arr]

arr = [64, 25, 12, 22, 11]
print(preprocess_rank(arr))  # [4, 2, 1, 3, 0]
```

### 前綴和

```python
def prefix_sum(arr):
    """前綴和前處理"""
    prefix = [0]
    for x in arr:
        prefix.append(prefix[-1] + x)
    return prefix

def range_sum(prefix, i, j):
    """O(1) 查詢區間和"""
    return prefix[j + 1] - prefix[i]

arr = [1, 2, 3, 4, 5]
prefix = prefix_sum(arr)
print(range_sum(prefix, 1, 3))  # 2+3+4 = 9
```

### 前綴最小/最大值

```python
def prefix_min(arr):
    """前綴最小值"""
    prefix = [arr[0]]
    for x in arr[1:]:
        prefix.append(min(prefix[-1], x))
    return prefix

def prefix_max(arr):
    """前綴最大值"""
    prefix = [arr[0]]
    for x in arr[1:]:
        prefix.append(max(prefix[-1], x))
    return prefix
```

### 雜湊預處理

```python
def hash_preprocess(arr):
    """建立雜湊表加速查詢"""
    hash_table = {}
    for i, x in enumerate(arr):
        if x not in hash_table:
            hash_table[x] = []
        hash_table[x].append(i)
    return hash_table

def fast_lookup(hash_table, target):
    """O(1) 查詢所有匹配位置"""
    return hash_table.get(target, [])
```

### 離散化

```python
def discretize(arr):
    """將連續值離散化"""
    sorted_unique = sorted(set(arr))
    mapping = {v: i for i, v in enumerate(sorted_unique)}
    return [mapping[x] for x in arr], mapping

arr = [1.5, 2.7, 1.5, 0.3, 2.7]
discrete, mapping = discretize(arr)
print(f"Discrete: {discrete}")  # [1, 2, 1, 0, 2]
print(f"Mapping: {mapping}")    # {0.3: 0, 1.5: 1, 2.7: 2}
```

### 正規化

```python
def normalize_minmax(arr):
    """Min-Max 正規化到 [0, 1]"""
    min_val = min(arr)
    max_val = max(arr)
    if max_val == min_val:
        return [0.5] * len(arr)
    return [(x - min_val) / (max_val - min_val) for x in arr]

def normalize_zscore(arr):
    """Z-Score 正規化"""
    import statistics
    mean = statistics.mean(arr)
    std = statistics.stdev(arr)
    if std == 0:
        return [0] * len(arr)
    return [(x - mean) / std for x in arr]

arr = [10, 20, 30, 40, 50]
print(normalize_minmax(arr))  # [0.0, 0.25, 0.5, 0.75, 1.0]
```

## 資料結構前處理

### 並查集初始化

```python
class UnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n
    
    def find(self, x):
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])
        return self.parent[x]
    
    def union(self, x, y):
        px, py = self.find(x), self.find(y)
        if px == py:
            return False
        if self.rank[px] < self.rank[py]:
            px, py = py, px
        self.parent[py] = px
        if self.rank[px] == self.rank[py]:
            self.rank[px] += 1
        return True
```

### 圖的前處理

```python
def preprocess_graph(edges, n):
    """圖的前處理：建立鄰接表"""
    adj = [[] for _ in range(n)]
    for u, v in edges:
        adj[u].append(v)
        adj[v].append(u)
    
    # 計算圖屬性
    in_degree = [0] * n
    out_degree = [0] * n
    
    for u, v in edges:
        out_degree[u] += 1
        in_degree[v] += 1
    
    return {
        'adjacency': adj,
        'in_degree': in_degree,
        'out_degree': out_degree
    }
```

### 字符串前處理

```python
def build_prefix_function(pattern):
    """KMP 前綴函數"""
    m = len(pattern)
    pi = [0] * m
    
    for i in range(1, m):
        j = pi[i - 1]
        while j > 0 and pattern[i] != pattern[j]:
            j = pi[j - 1]
        if pattern[i] == pattern[j]:
            j += 1
        pi[i] = j
    
    return pi

def kmp_search(text, pattern):
    """KMP 字串匹配"""
    n, m = len(text), len(pattern)
    pi = build_prefix_function(pattern)
    
    q = 0
    for i in range(n):
        while q > 0 and pattern[q] != text[i]:
            q = pi[q - 1]
        if pattern[q] == text[i]:
            q += 1
        if q == m:
            return i - m + 1
    return -1

print(kmp_search("ABABDABACDABABCABAB", "ABAB"))  # 0
```

## 前處理複雜度權衡

| 前處理 | 額外空間 | 查詢加速 |
|--------|----------|----------|
| 排序 | O(1) | O(n) → O(log n) |
| 前綴和 | O(n) | O(n) → O(1) |
| 雜湊表 | O(n) | O(n) → O(1) |
| 離散化 | O(n) | 節省空間 |

## 應用場景

1. **資料庫索引**：B-Tree、Hash Index
2. **字串匹配**：KMP、Z-Algorithm
3. **圖論問題**：預計算最短路徑
4. **機器學習**：特徵標準化
5. **影像處理**：直方圖均衡化

# 平行運算 (Parallel Computing)

## 概述

平行運算利用多個處理單元同時執行計算，以提升效能和處理速度。隨著硬體發展，平行化已成為現代計算的主流模式。

## 平行化層級

```
┌────────────────────────────────────────────────────────────┐
│                     平行化層級                              │
├────────────────────────────────────────────────────────────┤
│                                                            │
│  Bit-level    ████  4位 → 8位 → 16位 → 32位  (早期CPU)    │
│                                                            │
│  Instruction  ████  一條指令的多重運算  (SIMD, SSE/AVX)   │
│                                                            │
│  Thread       ████  多執行緒共享記憶體  (多核心CPU)        │
│                                                            │
│  Process      ████  多程序獨立記憶體空間  (叢集)           │
│                                                            │
│  Node-level   ████  多機器分散式計算  (MPI, Hadoop)        │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

## Flynn 分類法

| 分類 | 說明 | 範例 |
|------|------|------|
| SISD | 單一指令流、單一資料流 | 傳統馮紐曼架構 |
| SIMD | 單一指令流、多重資料流 | GPU、SSE/AVX |
| MISD | 多重指令流、單一資料流 | 很少使用 |
| MIMD | 多重指令流、多重資料流 | 多核 CPU、叢集 |

## 共享記憶體平行化

### POSIX Threads
```c
#include <pthread.h>

#define NUM_THREADS 4

typedef struct {
    int thread_id;
    int start;
    int end;
} ThreadData;

int array[1000];
int result = 0;
pthread_mutex_t mutex = PTHREAD_MUTEX_INITIALIZER;

void *partial_sum(void *arg) {
    ThreadData *data = (ThreadData *)arg;
    long sum = 0;
    
    for (int i = data->start; i < data->end; i++) {
        sum += array[i];
    }
    
    pthread_mutex_lock(&mutex);
    result += sum;
    pthread_mutex_unlock(&mutex);
    
    return NULL;
}

int main() {
    pthread_t threads[NUM_THREADS];
    ThreadData thread_data[NUM_THREADS];
    
    // 初始化資料
    for (int i = 0; i < 1000; i++) {
        array[i] = i + 1;
    }
    
    // 建立執行緒
    int chunk = 1000 / NUM_THREADS;
    for (int i = 0; i < NUM_THREADS; i++) {
        thread_data[i].thread_id = i;
        thread_data[i].start = i * chunk;
        thread_data[i].end = (i + 1) * chunk;
        pthread_create(&threads[i], NULL, partial_sum, &thread_data[i]);
    }
    
    // 等待執行緒結束
    for (int i = 0; i < NUM_THREADS; i++) {
        pthread_join(threads[i], NULL);
    }
    
    printf("Sum: %d\n", result);
    return 0;
}
```

### OpenMP
```c
#include <omp.h>

int main() {
    int array[1000];
    long sum = 0;
    
    #pragma omp parallel for reduction(+:sum)
    for (int i = 0; i < 1000; i++) {
        sum += array[i];
    }
    
    #pragma omp parallel sections
    {
        #pragma omp section
        {
            task_a();
        }
        #pragma omp section
        {
            task_b();
        }
    }
    
    return 0;
}
```

編譯：
```bash
gcc -fopenmp -O2 program.c -o program
```

## 分散式記憶體平行化

### MPI (Message Passing Interface)
```c
#include <mpi.h>

int main(int argc, char **argv) {
    MPI_Init(&argc, &argv);
    
    int rank, size;
    MPI_Comm_rank(MPI_COMM_WORLD, &rank);
    MPI_Comm_size(MPI_COMM_WORLD, &size);
    
    int local_data[100];
    int local_sum = 0;
    
    // 每個程序處理自己的資料
    for (int i = 0; i < 100; i++) {
        local_sum += local_data[i];
    }
    
    // 收集所有程序的結果
    int global_sum;
    MPI_Reduce(&local_sum, &global_sum, 1, MPI_INT, MPI_SUM, 0, MPI_COMM_WORLD);
    
    if (rank == 0) {
        printf("Total sum: %d\n", global_sum);
    }
    
    MPI_Finalize();
    return 0;
}
```

執行：
```bash
mpirun -np 4 ./program
```

## SIMD 指令

### x86 SSE/AVX
```c
#include <immintrin.h>

void vector_add(float *a, float *b, float *c, int n) {
    for (int i = 0; i < n; i += 8) {
        __m256 va = _mm256_loadu_ps(&a[i]);
        __m256 vb = _mm256_loadu_ps(&b[i]);
        __m256 vc = _mm256_add_ps(va, vb);
        _mm256_storeu_ps(&c[i], vc);
    }
}
```

### ARM NEON
```c
#include <arm_neon.h>

void vector_add_neon(float32_t *a, float32_t *b, float32_t *c, int n) {
    for (int i = 0; i < n; i += 4) {
        float32x4_t va = vld1q_f32(&a[i]);
        float32x4_t vb = vld1q_f32(&b[i]);
        float32x4_t vc = vaddq_f32(va, vb);
        vst1q_f32(&c[i], vc);
    }
}
```

## 同步機制

### 臨界區 (Critical Section)
```c
// 互斥鎖
pthread_mutex_t mutex = PTHREAD_MUTEX_INITIALIZER;

pthread_mutex_lock(&mutex);
// 臨界區
pthread_mutex_unlock(&mutex);

// OpenMP 臨界區
#pragma omp critical
{
    shared_variable += 1;
}
```

### 屏障同步 (Barrier)
```c
// OpenMP barrier
#pragma omp parallel
{
    // 第一階段
    compute_phase1();
    
    #pragma omp barrier
    
    // 第二階段（等待所有執行緒完成第一階段）
    compute_phase2();
}

// MPI barrier
MPI_Barrier(MPI_COMM_WORLD);
```

### 無鎖同步 (Lock-free)
```c
// 原子操作
#include <stdatomic.h>

atomic_int counter = 0;

void increment() {
    atomic_fetch_add(&counter, 1);
}

// CAS (Compare-And-Swap)
int lock_free_queue_push(queue_t *q, node_t *node) {
    node->next = NULL;
    node_t *old_tail = atomic_load(&q->tail);
    
    while (!atomic_compare_exchange_weak(&q->tail, &old_tail, node)) {
        old_tail = atomic_load(&q->tail);
    }
    
    atomic_store(&old_tail->next, node);
    return 1;
}
```

## 負載平衡

### 靜態負載平衡
```c
// 平均分配
int chunk = N / num_threads;
int start = thread_id * chunk;
int end = (thread_id == num_threads - 1) ? N : start + chunk;
```

### 動態負載平衡 (Work Stealing)
```c
// 佇列任務分配
#pragma omp parallel
{
    #pragma omp single
    {
        while (!task_queue_empty()) {
            task_t *task = pop_task();
            #pragma omp task
            process_task(task);
        }
    }
}
```

## Amdahl 定律

```
Speedup = 1 / (S + (1-S)/N)

S = 可平行化部分的比例
N = 處理器數量
```

| S | N=1 | N=2 | N=4 | N=8 | N=∞ |
|---|-----|-----|-----|-----|-----|
| 50% | 1.33 | 2.67 | 4.00 | 4.00 |
| 75% | 1.14 | 3.20 | 6.40 | 7.11 | 8.00 |
| 90% | 1.11 | 5.26 | 10.0 | 16.0 | 40.0 |

## 平行化效能分析

```c
#include <time.h>

double get_time() {
    struct timespec ts;
    clock_gettime(CLOCK_MONOTONIC, &ts);
    return ts.tv_sec + ts.tv_nsec * 1e-9;
}

void benchmark() {
    double start = get_time();
    // 計算任務
    double elapsed = get_time() - start;
    
    printf("執行時間: %.3f ms\n", elapsed * 1000);
}
```

# 指令集架構

指令集架構（Instruction Set Architecture, ISA）是 CPU 與軟體之間的介面定義，規定了 CPU 支援的指令集合、暫存器組織、記憶體模型和例外處理方式等。ISA 是電腦系統中最關鍵的抽象層，它決定了編譯器可以產生什麼樣的機器碼，也決定了作業系統和應用程式如何與硬體互動。不同的 ISA 適合不同的應用場景，從高效能的伺服器處理器到低功耗的嵌入式系統，都有對應的 ISA 設計。

## 指令格式

### RISC 固定格式

RISC 指令通常使用固定長度格式：

```python
# RISC-V 指令格式 (32 位元)

# R-type (Register) - 運算指令
# | 31-25 | 24-20 | 19-15 | 14-12 | 11-7 | 6-0 |
# | funct7 |  rs2  |  rs1  | funct3|  rd  | opcode |
# 例: add x1, x2, x3

# I-type (Immediate) - 立即數運算/載入
# | 31-20       | 19-15 | 14-12 | 11-7 | 6-0 |
# | imm[11:0]   |  rs1  | funct3|  rd  | opcode |
# 例: addi x1, x2, 100

# S-type (Store) - 儲存指令
# | 31-25 | 24-20 | 19-15 | 14-12 | 11-7 | 6-0 |
# | imm[11:5]|  rs2 |  rs1 | funct3|imm[4:0]| opcode |
# 例: sw x1, 0(x2)

# B-type (Branch) - 分支指令
# | 31-25 | 24-20 | 19-15 | 14-12 | 11-7 | 6-0 |
# | imm[12|10:5] | rs2 | rs1 | funct3 | imm[4:1|11] | opcode |
# 例: beq x1, x2, loop

# U-type (Upper immediate) - 上半部立即數
# | 31-12          | 11-7 | 6-0 |
# | imm[31:12]      |  rd  | opcode |
# 例: lui x1, 0x12345

# J-type (Jump) - 跳躍指令
# | 31-12          | 11-7 | 6-0 |
# | imm[20|10:1|11|19:12] | rd | opcode |
# 例: jal x1, function
```

### CISC 變長格式

x86 指令使用變長編碼：

```python
# x86 指令編碼範例
# 最短: 1 位元組 (如 90h = nop)
# 最長: 15 位元組

# 範例指令：
# 90              ; nop
# 90 90           ; xchg eax, eax
# 48 89 C3         ; mov rbx, rax (REX prefix + opcode + modrm)
# 48 89 04 24      ; mov [rsp], rax

# x86_64 新增 REX 前綴
# REX.B  - Extension for ModRM or SIB.base
# REX.X  - Extension for SIB.index  
# REX.R  - Extension for ModRM.reg
# REX.W  - 64-bit operand size
```

## 指令類別

### 資料傳輸指令

```assembly
# RISC-V 資料傳輸
lw x1, 0(x2)      # load word: x1 = mem[x2+0]
sw x1, 0(x2)      # store word: mem[x2+0] = x1
lb x1, 0(x2)      # load byte (signed)
lbu x1, 0(x2)     # load byte (unsigned)
lh, lhu, sh, sb   # halfword versions

# x86 資料傳輸
mov rax, rbx      # rax = rbx
mov eax, [esi]   # eax = mem[esi]
mov [edi], eax    # mem[edi] = eax
push rax          # push to stack
pop rbx            # pop from stack
```

### 算術指令

```assembly
# RISC-V 算術
add x1, x2, x3    # x1 = x2 + x3
addi x1, x2, 10    # x1 = x2 + 10
sub x1, x2, x3    # x1 = x2 - x3
mul x1, x2, x3    # x1 = x2 * x3 (low)
div x1, x2, x3    # x1 = x2 / x3
rem x1, x2, x3    # x1 = x2 % x3

# x86 算術
add rax, rbx      # rax = rax + rbx
sub rax, rbx      # rax = rax - rbx
imul rbx          # rax = rax * rbx (signed)
idiv rbx          # rax = rax / rbx (signed)
inc rax           # rax++
dec rbx           # rbx--
neg rax           # rax = -rax
```

### 邏輯指令

```assembly
# RISC-V 邏輯
and x1, x2, x3    # x1 = x2 & x3
or  x1, x2, x3    # x1 = x2 | x3
xor x1, x2, x3    # x1 = x2 ^ x3
andi x1, x2, 0xFF # x1 = x2 & 0xFF

# 移位
sll x1, x2, 3     # x1 = x2 << 3 (邏輯左移)
srl x1, x2, 3     # x1 = x2 >> 3 (邏輯右移)
sra x1, x2, 3     # x1 = x2 >> 3 (算術右移)

# x86 邏輯
and rax, rbx      # rax = rax & rbx
or  rax, rbx      # rax = rax | rbx
xor rax, rbx      # rax = rax ^ rbx
not rax           # rax = ~rax
shl rax, 3        # rax = rax << 3
sar rax, 3        # rax = rax >> 3 (算術)
```

### 控制轉移指令

```assembly
# RISC-V 分支
beq x1, x2, label # if (x1 == x2) goto label
bne x1, x2, label # if (x1 != x2) goto label
blt x1, x2, label # if (x1 < x2) goto label (signed)
bge x1, x2, label # if (x1 >= x2) goto label (signed)
bltu x1, x2, label # if (x1 < x2) goto label (unsigned)

# RISC-V 跳躍
jal x1, label    # x1 = PC+4; goto label
jalr x1, 0(x2)   # x1 = PC+4; goto x2

# x86 分支/跳躍
je label          # if (ZF=1) goto label
jne label         # if (ZF=0) goto label
jl label          # if (SF!=OF) goto label
jg label          # if (ZF=1 && SF==OF) goto label

jmp label         # goto label
jmp rax           # goto rax

# 函式呼叫
call label        # push RIP; goto label
ret               # pop RIP
```

### 比較指令

```assembly
# RISC-V
slt x1, x2, x3    # x1 = (x2 < x3) ? 1 : 0 (signed)
sltu x1, x2, x3   # x1 = (x2 < x3) ? 1 : 0 (unsigned)

# x86
cmp rax, rbx      # set flags: rax - rbx
test rax, rbx     # set flags: rax & rbx
```

## 暫存器組織

### RISC-V 暫存器

```assembly
# RISC-V 暫存器 (64 位元)
# x0  - zero (永遠為 0)
# x1  - ra (返回位址)
# x2  - sp (堆疊指標)
# x3  - gp (全域指標)
# x4  - tp (執行緒指標)
# x5-x7 - t0-t2 (臨時)
# x8-x9 - s0-s1 (保存)
# x10-x17 - a0-a7 (引數/返回值)
# x18-x27 - s2-s11 (保存)
# x28-x31 - t3-t6 (臨時)

# 浮點暫存器
# f0-f7 - ft0-ft7 (臨時)
# f8-f9 - fs0-fs1 (保存)
# f10-f17 - fa0-fa7 (引數/返回值)
# f18-f27 - fs2-fs11 (保存)
# f28-f31 - ft8-ft11 (臨時)
```

### x86-64 暫存器

```assembly
# x86-64 通用暫存器 (64 位元)
# rax - 累積器 (返回值)
# rbx - 基址
# rcx - 計數器
# rdx - 資料 (I/O, 乘法/除法)
# rsi - 來源索引
# rdi - 目的索引
# rbp - 基底指標
# rsp - 堆疊指標
# r8-r15 - 新增暫存器

# 子集：
# 32 位元: eax, ebx, ecx, edx...
# 16 位元: ax, bx, cx, dx...
# 8 位元: al, ah, bl, bh...
```

### 位址空間

```python
# 典型 64 位元系統位址空間
# 使用者空間: 0x0000000000400000 - 0x00007FFFFFFFFFFF
# 核心空間:   0xFFFF800000000000 - 0xFFFFFFFFFFFFFFFF

# 分段/分頁
# x86: 段式 + 分頁
```

### 位元組順序

```python
# 大端序 (Big Endian)
# 記憶體: [0x12][0x34][0x56][0x78]
# 解釋為: 0x12345678

# 小端序 (Little Endian) - 大多數 CPU 使用
# 記憶體: [0x78][0x56][0x34][0x12]
# 解釋為: 0x12345678

# x86, RISC-V (小端), ARM (可選)
# PowerPC, MIPS (大端) - 傳統
# ARM (現在小端)
```

## 例外與中斷處理

```python
# 例外類型
# • 同步例外: 除零、頁錯誤、系統呼叫
# • 非同步中斷: 硬體中斷、IO 完成

# 控制轉移
# 發生例外時:

# 2. 設定新 PC 為例外處理常式位址

# mcause (Exception Cause)
# mtval (Exception Value)
# mstatus (Machine Status)
```

## 呼叫約定

### 系統呼叫

```python
# Linux x86_64 系統呼叫
# 系統呼叫號: rax
# 引數: rdi, rsi, rdx, r10, r8, r9
# 返回值: rax

# 範例: write(fd, buf, count)
# mov rax, 1        # write = 1
# mov rdi, 1        # fd = stdout
# mov rsi, msg     # buf
# mov rdx, len     # count
# syscall

# RISC-V Linux 系統呼叫
# a7 = 系統呼叫號
# a0-a5 = 引數
# a0 = 返回值
```

## ABI (Application Binary Interface)

ABI 定義了編譯器和連結器使用的低層約定：

```
ABI 內容：
• 暫存器使用約定
• 函式傳遞方式
• 資料佈局 (對齊)
• 堆疊框架
• 系統呼叫介面
```

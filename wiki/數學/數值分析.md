# 數值分析：計算機上的數學方法

## 概述

數值分析研究在計算機上求解數學問題的近似方法。由於計算機使用有限精度表示數字（通常是 IEEE 754 浮點數），所有計算都會引入誤差。理解這些誤差的來源和傳播對於編寫正確的數值計算程式至關重要。本頁介紹浮點數表示、誤差分析、數值線性代數和數值微積分等基礎方法。

## 浮點數表示

### IEEE 754 標準

現代計算機使用 **IEEE 754 標準**表示浮點數。以雙精度（64 位元）為例：

$$(-1)^s \times (1.f) \times 2^{e-1023}$$

| 欄位 | 符號位 (s) | 指數位 (e) | 尾數位 (f) |
|------|------------|------------|------------|
| 雙精度 | 1 位 | 11 位 | 52 位 |

特殊值表示：
- **零**：$e = 0, f = 0$（正零和負零）
- **無窮大**：$e = 2047, f = 0$
- **NaN（非數字）**：$e = 2047, f \neq 0$

### 浮點數的精度限制

雙精度浮點數的機器精度（machine epsilon）$\epsilon_{\text{mach}} = 2^{-52} \approx 2.22 \times 10^{-16}$。

這意味著：對於雙精度浮點數 $x$，有：
$$\text{fl}(x) = x(1 + \delta), \quad |\delta| \leq \epsilon_{\text{mach}}$$

其中 $\text{fl}(x)$ 表示 $x$ 的浮點表示。

### 常見的精度問題

1. **有限精度表示**：$0.1 + 0.2 \neq 0.3$（因為 0.1, 0.2, 0.3 都無法用二進位精確表示）
2. **大數吃小數**：當 $x$ 很大而 $y$ 很小時，$x + y = x$（因為 $y$ 相對於 $x$ 太小，無法改變 $x$ 的尾數位）
3. **相消現象（Cancellation）**：計算 $x - y$ 當 $x \approx y$ 時，有效位數會嚴重流失

## 誤差分析

### 誤差的來源

1. **截斷誤差（Truncation Error）**：使用有限項近似無窮級數
   $$e^x = 1 + x + \frac{x^2}{2!} + \cdots + \frac{x^n}{n!} + \text{截斷誤差}$$

2. **舍入誤差（Round-off Error）**：將精確值用有限精度表示時的誤差
   $$\text{舍入誤差} = |\text{精確值} - \text{浮點表示值}|$$

3. **累積誤差**：多次運算後誤差的積累

### 誤差傳播

給定 $y = f(x)$，若 $x$ 有誤差 $\Delta x$，則 $y$ 的誤差約為：
$$\Delta y \approx \left|\frac{df}{dx}\right| \Delta x$$

**前向誤差界**：若 $\tilde{x} = x(1 + \delta)$，則 $f(\tilde{x}) - f(x) \approx x f'(x) \delta$。

### 條件數（Condition Number）

問題的條件數衡量輸入誤差對輸出誤差的放大程度。

對於函數 $f(x)$，相對條件數定義為：
$$\kappa = \left|\frac{x f'(x)}{f(x)}\right|$$

- $\kappa \approx 1$：良態問題（well-conditioned）
- $\kappa \gg 1$：病態問題（ill-conditioned）

例如：計算 $f(x) = e^x$ 的條件數為 $\kappa = |x|$，當 $x$ 很大時問題變為病態。

## 數值線性代數

### 矩陣範數與誤差分析

**矩陣範數**用於分析線性系統的誤差：
- **1-範數**：$\|A\|_1 = \max_j \sum_i |a_{ij}|$
- **∞-範數**：$\|A\|_\infty = \max_i \sum_j |a_{ij}|$
- **2-範數（譜範數）**：$\|A\|_2 = \sqrt{\rho(A^T A)}$，其中 $\rho$ 是譜半徑

**矩陣的條件數**：$\kappa(A) = \|A\| \|A^{-1}\|$。

若 $\kappa(A)$ 很大，則線性系統 $Ax = b$ 是病態的：輸入 $b$ 的小誤差會導致解 $x$ 的大誤差。

### LU 分解

**LU 分解**將方陣 $A$ 分解為下三角陣 $L$ 和上三角陣 $U$ 的乘積：$A = LU$。

演算法（Doolitte 方法）：
- 對於 $i \leq j$：$U_{ij} = A_{ij} - \sum_{k=1}^{i-1} L_{ik} U_{kj}$
- 對於 $i > j$：$L_{ij} = \frac{1}{U_{jj}}(A_{ij} - \sum_{k=1}^{j-1} L_{ik} U_{kj})$

求解 $Ax = b$ 的步驟：
1. 求解 $Ly = b$（前向替換）
2. 求解 $Ux = y$（後向替換）

### 迭代法

當矩陣 $A$ 很大且稀疏時，迭代法比直接法更高效。

**Jacobi 迭代法**：將 $A = D + L + U$，其中 $D$ 是對角陣，$L$ 是嚴格下三角陣，$U$ 是嚴格上三角陣。
$$x_i^{(k+1)} = \frac{1}{a_{ii}}\left(b_i - \sum_{j \neq i} a_{ij} x_j^{(k)}\right)$$

**Gauss-Seidel 迭代法**：使用最新計算的值：
$$x_i^{(k+1)} = \frac{1}{a_{ii}}\left(b_i - \sum_{j < i} a_{ij} x_j^{(k+1)} - \sum_{j > i} a_{ij} x_j^{(k)}\right)$$

**收斂條件**：若 $A$ 是嚴格對角佔優（strictly diagonally dominant），則 Jacobi 和 Gauss-Seidel 都保證收斂。

### 共軛梯度法（Conjugate Gradient）

對於對稱正定矩陣 $A$，共軛梯度法能在最多 $n$ 步內收斂（不考慮舍入誤差）：
$$x_{k+1} = x_k + \alpha_k p_k, \quad \alpha_k = \frac{r_k^T r_k}{p_k^T A p_k}$$
其中 $r_k = b - Ax_k$ 是殘差，$p_k$ 是共軛方向。

## 數值微積分

### 數值微分

**前向差分**：$f'(x) \approx \frac{f(x+h) - f(x)}{h}$，誤差 $O(h)$

**中央差分**：$f'(x) \approx \frac{f(x+h) - f(x-h)}{2h}$，誤差 $O(h^2)$

**五點模版**（更高精度）：
$$f'(x) \approx \frac{-f(x+2h) + 8f(x+h) - 8f(x-h) + f(x-2h)}{12h}, \quad \text{誤差 } O(h^4)$$

**二階導數**：
$$f''(x) \approx \frac{f(x+h) - 2f(x) + f(x-h)}{h^2}, \quad \text{誤差 } O(h^2)$$

### 數值積分

**梯形法則**：
$$\int_a^b f(x)\,dx \approx \frac{h}{2}[f(x_0) + 2f(x_1) + \cdots + 2f(x_{n-1}) + f(x_n)], \quad h = \frac{b-a}{n}$$

誤差：$\frac{(b-a)h^2}{12} f''(\xi)$，其中 $\xi \in [a, b]$。

**Simpson 法則**（使用二次多項式插值）：
$$\int_a^b f(x)\,dx \approx \frac{h}{3}[f(x_0) + 4f(x_1) + 2f(x_2) + \cdots + 4f(x_{n-1}) + f(x_n)], \quad n \text{ 為偶數}$$

誤差：$\frac{(b-a)h^4}{180} f^{(4)}(\xi)$。

**自適應積分**：遞迴地將區間細分，直到近似誤差小於容許誤差。基本思想：
$$\text{若 } |S(a, b) - [S(a, c) + S(c, b)]| < \text{tol，則接受 } S(a, c) + S(c, b)$$
否則遞迴細分區間。

## 多項式插值

### Lagrange 插值

給定 $n+1$ 個點 $(x_i, y_i)$，存在唯一的 $n$ 次多項式 $P_n(x)$ 通過這些點：
$$P_n(x) = \sum_{i=0}^n y_i \ell_i(x), \quad \ell_i(x) = \prod_{j \neq i} \frac{x - x_j}{x_i - x_j}$$

**誤差**：若 $f \in C^{n+1}[a, b]$，則對於任意 $x \in [a, b]$，存在 $\xi \in [a, b]$ 使得：
$$f(x) - P_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!} \prod_{i=0}^n (x - x_i)$$

### Runge 現象

高次多項式插值可能在不均勻節點上產生劇烈振盪。解決方法：
- 使用分段低次插值（如樣條插值）
- 使用切比雪夫節點：$x_i = \frac{a+b}{2} + \frac{b-a}{2} \cos\left(\frac{2i+1}{2n+2}\pi\right)$

## 常微分方程式數值解

### Euler 法

最簡單的 ODE 數值解法：$y_{n+1} = y_n + h f(t_n, y_n)$。

全域誤差：$O(h)$，是一階方法。

### 改進 Euler 法（Heun's method）

**預測-校正**方法：
1. 預測：$\tilde{y}_{n+1} = y_n + h f(t_n, y_n)$
2. 校正：$y_{n+1} = y_n + \frac{h}{2}[f(t_n, y_n) + f(t_{n+1}, \tilde{y}_{n+1})]$

全域誤差：$O(h^2)$，是二階方法。

### 四階 Runge-Kutta 法（RK4）

最常用的 ODE 解法：
$$y_{n+1} = y_n + \frac{h}{6}(k_1 + 2k_2 + 2k_3 + k_4)$$
其中：
- $k_1 = f(t_n, y_n)$
- $k_2 = f(t_n + \frac{h}{2}, y_n + \frac{h}{2}k_1)$
- $k_3 = f(t_n + \frac{h}{2}, y_n + \frac{h}{2}k_2)$
- $k_4 = f(t_n + h, y_n + h k_3)$

全域誤差：$O(h^4)$。

## 總結

數值分析處理計算機上的近似計算問題。理解 IEEE 754 浮點數標準、誤差來源和傳播、以及數值方法的穩定性對於編寫正確的科學計算程式至關重要。本頁介紹了浮點數表示、誤差分析、LU 分解、數值積分和微分、多項式插值等基礎方法。實際應用中，應優先使用經過測試的函式庫（如 NumPy、SciPy），除非有特殊原因需要自己實現。

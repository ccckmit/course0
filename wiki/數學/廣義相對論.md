# 廣義相對論 (General Relativity)

## 概述

廣義相對論是阿爾伯特·愛因斯坦於1915年發表的重力理論，其核心思想是：**重力不是力，而是時空本身的幾何彎曲**。這一理論將宇宙理解為一個動態的四維時空實體，物質和能量告訴時空如何彎曲，而時空的彎曲決定物質如何運動。廣義相對論不僅解釋了牛頓重力無法解釋的現象（如水星近日點進動），還預言了黑洞、重力波和宇宙膨脹等驚人效應。

廣義相對論的數學基礎是**微分幾何**——研究曲面和高維流形的數學分支。愛因斯坦場方程式將時空的幾何與物質的分布聯繫起來，形成了一組優美而深刻的方程。雖然其方程在數學上優雅，但在絕大多數情況下難以解析求解，這催生了**數值相對論**這門計算科學。

## 基本原理

### 等價原理

等價原理是廣義相對論的核心基礎，認為重力與加速度在本質上無法區分：

- **弱等價原理**（也稱為 Univsal Equivalence Principle）: 質量與慣性質量相等 ( $m_{inertial} = m_{gravitational}$ )，所有物體在重力場中以相同加速度下落。這保證了自由落體的普遍性。

- **強等價原理**: 在任意引力場中，任意觀察者都能通過選擇局部慣性參考系將重力消除。這意味著所有物理定律在局部都取狹義相對論的形式。

等價原理直接導致了**重力時間膨脹**和**重力透鏡**效應。

### 廣義協變性

物理定律應在任意座標系下具有相同的形式——它們應該是**張量方程**。這確保了物理定律與座標選擇無關，使得理論在任意時空坐標下都有意義。這並非純粹的美學要求，而是等價原理的數學表達。

### 最小作用原理

廣義相對論也可從變分原理導出。愛因斯坦-希爾伯特作用量：

$$S = \frac{1}{16\pi G}\int \sqrt{-g} (R + 2\Lambda) d^4x + S_{matter}$$

其中 $g$ 是度規行列式，$R$ 是標量曲率，$\Lambda$ 是宇宙學常數。對度規變分得到愛因斯坦場方程式。

## 時空幾何

### 度規張量

時空的幾何由**度規張量** $g_{\mu\nu}$ 描述。時空間隔為：

$$ds^2 = g_{\mu\nu}dx^\mu dx^\nu$$

度規是時空最基礎的幾何量，決定了距離、角度和時間的測量方式。

**閔可夫斯基度規**（平坦時空，無重力）：

$$ds^2 = -c^2dt^2 + dx^2 + dy^2 + dz^2$$

**Schwarzschild 度規**（球對稱靜態質量外部）：

$$ds^2 = -(1-\frac{2GM}{c^2r})c^2dt^2 + (1-\frac{2GM}{c^2r})^{-1}dr^2 + r^2d\Omega^2$$

這是廣義相對論第一個精確解，由史瓦西於1916年找到。

### 測地線

自由粒子（不受非重力力）在時空中沿**測地線**運動——這是直線在彎曲時空中的推廣：

$$\frac{d^2x^\lambda}{d\tau^2} + \Gamma^\lambda_{\mu\nu}\frac{dx^\mu}{d\tau}\frac{dx^\nu}{d\tau} = 0$$

其中 $\Gamma^\lambda_{\mu\nu}$ 是**克氏符號** (Christoffel symbols)，由度規及其導數構造：

$$\Gamma^\lambda_{\mu\nu} = \frac{1}{2}g^{\lambda\sigma}(\partial_\mu g_{\sigma\nu} + \partial_\nu g_{\mu\sigma} - \partial_\sigma g_{\mu\nu})$$

測地線方程也可從變分原理導出，作用量為 $\int ds$。

### 里奇曲率與標量曲率

**黎曼曲率張量**描述時空的整體彎曲程度：

$$R^\rho_{\mu\nu\sigma} = \partial_\nu\Gamma^\rho_{\mu\sigma} - \partial_\sigma\Gamma^\rho_{\mu\nu} + \Gamma^\rho_{\lambda\nu}\Gamma^\lambda_{\mu\sigma} - \Gamma^\rho_{\lambda\sigma}\Gamma^\lambda_{\mu\nu}$$

它滿足 **Bianchi 恆等式**：$R_{[\rho\mu\nu]\sigma} = 0$。

**里奇張量** $R_{\mu\nu}$ 是黎曼張量的收縮：

$$R_{\mu\nu} = R^\lambda_{\mu\lambda\nu}$$

**標量曲率** $R$ 是里奇張量的收縮：

$$R = g^{\mu\nu}R_{\mu\nu}$$

這些曲率量在愛因斯坦場方程中扮演核心角色。

## 愛因斯坦場方程式

### 場方程的形式

愛因斯坦場方程式是廣義相對論的核心：

$$G_{\mu\nu} = \frac{8\pi G}{c^4}T_{\mu\nu}$$

其中 $G_{\mu\nu}$ 是**愛因斯坦張量**：

$$G_{\mu\nu} = R_{\mu\nu} - \frac{1}{2}g_{\mu\nu}R$$

$T_{\mu\nu}$ 是**能量-動量張量**，描述物質和能量的分布。

### 能量-動量張量

不同物質的 energy-momentum tensor：

- **理想流體**:
  $$T_{\mu\nu} = (\rho + p/c^2)U_\mu U_\nu + pg_{\mu\nu}$$
  其中 $\rho$ 是質量密度，$p$ 是壓力，$U_\mu$ 是四速度。

- **電磁場**: $T_{\mu\nu} = F_{\mu\alpha}F^\alpha_\nu - \frac{1}{4}g_{\mu\nu}F_{\alpha\beta}F^{\alpha\beta}$

- **真空**: $T_{\mu\nu} = 0$，場方程化為 $R_{\mu\nu} = 0$。

### 宇宙學常數

愛因斯坦最初引入宇宙學常數 $\Lambda$：

$$G_{\mu\nu} + \Lambda g_{\mu\nu} = \frac{8\pi G}{c^4}T_{\mu\nu}$$

後來他稱之為「我最大的錯誤」，但1990年代發現宇宙加速膨脹後，$\Lambda$ 被重新解釋為**暗能量**，是當今宇宙學的核心謎團。

## 重要預言與效應

### 重力時間膨脹

重力越強，時間越慢。在地球表面相對於高空：

$$\frac{dt_{earth}}{dt_{sat}} = \sqrt{\frac{1 - \frac{2GM}{c^2R_{earth}}}{1 - \frac{2GM}{c^2R_{sat}}}}$$

這在 GPS 系統中必須修正，否則每天會累积約 38 微秒的誤差。

### 重力透鏡

光線經過大質量天體時會彎曲。偏折角（對於遠離質量的情况）：

$$\theta = \frac{4GM}{c^2b}$$

其中 $b$ 是碰撞參數。1919 年愛丁頓觀測日食時星光偏折，證實了這一預言。

### 重力波

加速質量產生**重力波**——時空的漣漪，以光速傳播。這是廣義相對論最顯著的預言之一，直到2015年才被 LIGO 直接探測到。雙黑洞合併產生的重力波信號於2015年9月14日被探測，2017年諾貝爾物理學獎頒給了 LIGO 團隊。

### 黑洞

**Schwarzschild 半徑**（事件視界）：

$$r_s = \frac{2GM}{c^2}$$

對於太陽，$r_s \approx 2.95$ km；對於地球，$r_s \approx 8.87$ mm。事件視界內的一切，包括光，都無法逃脫。

黑洞的分類：
- **Schwarzschild 黑洞**: 靜態，無旋轉
- **Kerr 黑洞**: 旋轉，帶角動量
- **Reissner-Nordström 黑洞**: 帶電
- **Kerr-Newman 黑洞**: 旋轉且帶電

## 宇宙學

### Robertson-Walker 度規

均勻且各向同性的宇宙時空度規：

$$ds^2 = -c^2dt^2 + a(t)^2[\frac{dr^2}{1-kr^2} + r^2(d\theta^2 + \sin^2\theta d\phi^2)]$$

其中 $a(t)$ 是宇宙尺度因子，$k$ 決定空間曲率：
- $k = 0$: 平坦宇宙
- $k = 1$: 封閉宇宙
- $k = -1$: 開放宇宙

### Friedmann 方程

宇宙動力學的基本方程：

$$H^2 = \frac{8\pi G}{3}\rho - \frac{kc^2}{a^2} + \frac{\Lambda c^2}{3}$$

$$\frac{\ddot{a}}{a} = -\frac{4\pi G}{3}(\rho + 3p/c^2) + \frac{\Lambda c^2}{3}$$

其中 $H = \dot{a}/a$ 是 Hubble 參數。

### 宇宙學觀測

觀測顯示：
- 宇宙正在加速膨脹（1998年）
- 宇宙微波背景輻射 (CMB) 温度涨落约 $10^{-5}$
- 宇宙由约 68% 暗能量、27% 暗物质、5% 普通物质组成

## 數值相對論

### 數值方法

求解愛因斯坦方程的數值方法：
- **有限差分法**: 離散化時空網格
- **譜方法**: 使用特殊函數展開
- **Cauchy 問題**: 分解為空間和時間
- **3+1 分解**: ADM 形式，空間與時間分開

### 黑洞碰撞模擬

數值相對論在黑洞碰撞模擬中取得重大進展。2005年 Baker 等人首次模擬黑洞合併產生的重力波，與 LIGO 觀測高度一致。

### 引力波數據分析

LIGO 和 Virgo 使用**匹配濾波**技術從噪聲中提取重力波信號。干涉儀輸出與理論模板比較，確定參數（質量、自旋、距離等）。

## 總結

廣義相對論是人類智力的偉大成就之一。它將重力理解為時空的幾何，不僅解釋了經典重力無法解釋的現象，還預言了黑洞、重力波和宇宙膨脹。雖然其方程在數學上優雅，但在實踐中極難求解，這催生了數值相對論這門計算科學。隨著重力波探測的時代來臨，我們有了觀察宇宙的新窗口。廣義相對論與量子力學的結合——量子重力——仍是當代理論物理的核心挑戰。

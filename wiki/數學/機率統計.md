# 機率統計

**標籤**: #機率統計 #機率論 #統計學 #隨機變數 #機率分佈 #期望值 #變異數
**日期**: 2024-01-15

機率統計是數學的重要分支，研究隨機現象的規律性。機率論研究隨機事件的機率規律，統計學從數據中推斷規律。兩者密切相關：機率論是統計學的理論基礎，統計學是機率論的實際應用。

## 機率論基礎

### 樣本空間與事件

樣本空間 $S$ 是所有可能結果的集合。事件是樣本空間的子集。若 $A \cap B = \emptyset$，則稱為互斥事件。

**事件運算**：
- 聯集 $A \cup B$：事件 $A$ 或 $B$ 發生
- 交集 $A \cap B$：事件 $A$ 和 $B$ 同時發生
- 補集 $A^c = S \setminus A$：事件 $A$ 不發生

### 機率公理

機率函數 $P$ 滿足：
1. **非負性**：對任意事件 $A$ 有 $P(A) \geq 0$
2. **規範性**：$P(S) = 1$
3. **可加性**：對於互斥事件序列 $A_1, A_2, \ldots$，有 $P(\cup A_i) = \sum P(A_i)$

### 條件機率與獨立性

**條件機率**：$P(A|B) = \frac{P(A \cap B)}{P(B)}$（當 $P(B) > 0$）

**乘法法則**：$P(A \cap B) = P(A|B)P(B) = P(B|A)P(A)$

**獨立性**：若 $P(A|B) = P(A)$，則 $A$ 與 $B$ 獨立。等價地：$P(A \cap B) = P(A)P(B)$。

**貝葉斯定理**：
$$P(A|B) = \frac{P(B|A)P(A)}{P(B)}$$

其中 $P(B) = \sum_i P(B|A_i)P(A_i)$（全機率定理）。

## 離散隨機變數

### 機率質量函數 (PMF)

離散隨機變數 $X$ 的機率質量函數 $p(x) = P(X = x)$，滿足 $p(x) \geq 0$ 且 $\sum p(x) = 1$。

### 常見離散分佈

**伯努利分佈** $Bernoulli(p)$：$X \in \{0, 1\}$，$P(X=1) = p$

**二項分佈** $Binomial(n, p)$：$P(X=k) = \binom{n}{k}p^k(1-p)^{n-k}$
- 表示 $n$ 次獨立伯努利試驗中成功的次數

**卜瓦松分佈** $Poisson(\lambda)$：$P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}$
- 當 $n$ 很大，$p$ 很小時，$Binomial(n, p) \approx Poisson(np)$

**幾何分佈** $Geometric(p)$：$P(X=k) = (1-p)^{k-1}p$
- 表示首次成功所需的試驗次數

### 期望值與變異數

**期望值**（均值）：$E[X] = \sum x \cdot p(x)$

**變異數**：$Var(X) = E[(X-\mu)^2] = E[X^2] - \mu^2$

**標準差**：$\sigma = \sqrt{Var(X)}$

| 分佈 | 期望值 | 變異數 |
|------|---------|---------|
| Bernoulli($p$) | $p$ | $p(1-p)$ |
| Binomial($n,p$) | $np$ | $np(1-p)$ |
| Poisson($\lambda$) | $\lambda$ | $\lambda$ |
| Geometric($p$) | $\frac{1}{p}$ | $\frac{1-p}{p^2}$ |

## 連續隨機變數

### 機率密度函數 (PDF)

連續隨機變數 $X$ 的機率密度函數 $f(x)$，滿足 $f(x) \geq 0$ 且 $\int_{-\infty}^{\infty} f(x)\,dx = 1$。

注意 $P(X = a) = 0$，因此 $P(a \leq X \leq b) = P(a < X \leq b) = \int_a^b f(x)\,dx$。

### 累積分佈函數 (CDF)

$$F(x) = P(X \leq x) = \int_{-\infty}^x f(t)\,dt$$

性質：$F'(x) = f(x)$，且 $P(a < X \leq b) = F(b) - F(a)$。

### 常見連續分佈

**均勻分佈** $Uniform(a, b)$：$f(x) = \frac{1}{b-a}$，$x \in [a, b]$

**指數分佈** $Exponential(\lambda)$：$f(x) = \lambda e^{-\lambda x}$，$x \geq 0$
- 無記憶性：$P(X > s+t | X > s) = P(X > t)$

**常態分佈** $Normal(\mu, \sigma^2)$：
$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$

標準常態分佈 $N(0, 1)$ 是均值為 $0$、標準差為 $1$ 的常態分佈。

**Gamma 分佈**：$f(x) = \frac{\beta^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\beta x}$，$x > 0$

**Beta 分佈**：$f(x) = \frac{1}{B(\alpha, \beta)} x^{\alpha-1} (1-x)^{\beta-1}$，$x \in [0, 1]$

### 期望值與變異數（連續）

**期望值**：$E[X] = \int_{-\infty}^{\infty} x \cdot f(x)\,dx$

**變異數**：$Var(X) = \int_{-\infty}^{\infty} (x-\mu)^2 f(x)\,dx = E[X^2] - \mu^2$

| 分佈 | 期望值 | 變異數 |
|------|---------|---------|
| Uniform($a, b$) | $\frac{a+b}{2}$ | $\frac{(b-a)^2}{12}$ |
| Exponential($\lambda$) | $\frac{1}{\lambda}$ | $\frac{1}{\lambda^2}$ |
| Normal($\mu, \sigma^2$) | $\mu$ | $\sigma^2$ |

## 多變數機率分佈

### 聯合分佈

**離散**：$P(X=x, Y=y) = p(x, y)$，滿足 $\sum_x \sum_y p(x, y) = 1$

**連續**：$f(x, y)$ 滿足 $\iint f(x, y)\,dx\,dy = 1$

**邊際分佈**：
- 離散：$p_X(x) = \sum_y p(x, y)$
- 連續：$f_X(x) = \int f(x, y)\,dy$

**條件分佈**：
- $p_{X|Y}(x|y) = \frac{p(x, y)}{p_Y(y)}$
- $f_{X|Y}(x|y) = \frac{f(x, y)}{f_Y(y)}$

### 共變異數與相關係數

**共變異數**：
$$Cov(X, Y) = E[(X-\mu_X)(Y-\mu_Y)] = E[XY] - \mu_X \mu_Y$$

**相關係數**：
$$\rho = \frac{Cov(X, Y)}{\sigma_X \sigma_Y}$$

$-1 \leq \rho \leq 1$：
- $\rho > 0$：正相關
- $\rho < 0$：負相關
- $\rho = 0$：不相關（對常態分佈來說，不相關等價於獨立）

### 多變數常態分佈

隨機向量 $\mathbf{X} \sim N(\boldsymbol{\mu}, \Sigma)$，其中 $\boldsymbol{\mu}$ 是均值向量，$\Sigma$ 是共變異數矩陣。

密度函數：
$$f(\mathbf{x}) = \frac{1}{(2\pi)^{n/2}|\Sigma|^{1/2}} \exp\left(-\frac{1}{2}(\mathbf{x}-\boldsymbol{\mu})^T \Sigma^{-1}(\mathbf{x}-\boldsymbol{\mu})\right)$$

## 隨機樣本與估計

### 隨機樣本

設 $X_1, X_2, \ldots, X_n$ 為獨立同分佈 (i.i.d.) 的隨機變數，則樣本均值 $\bar{X} = \frac{1}{n}\sum X_i$ 是對母體均值 $E[X]$ 的估計。

**大數法則**：$\bar{X} \xrightarrow{P} E[X]$（依機率收斂）

**中央極限定理**：$\frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \xrightarrow{d} N(0, 1)$（依分佈收斂）

### 點估計

**最大概似估計 (MLE)**：選擇使觀察到樣本機率最大的參數
$$\hat{\theta}_{MLE} = \arg\max_{\theta} L(\theta; x_1, \ldots, x_n) = \arg\max_{\theta} \prod_{i=1}^n f(x_i; \theta)$$

**動差估計 (Method of Moments)**：使用樣本動差匹配母體動差

**估計量的性質**：
- **不偏性**：$E[\hat{\theta}] = \theta$
- **有效性**：最小變異數
- **一致性**：$\hat{\theta} \xrightarrow{P} \theta$

### 區間估計

**95% 信心區間**：$\bar{X} \pm z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}$（已知母體變異數）

**t 信心區間**：$\bar{X} \pm t_{\alpha/2, n-1} \cdot \frac{s}{\sqrt{n}}$（未知母體變異數）

## 假設檢定

### 基本概念

- **虛無假設** $H_0$：需要檢驗的假設
- **對立假設** $H_1$：$H_0$ 的替代
- **型一錯誤** $\alpha = P(\text{拒絕}H_0|H_0\text{為真})$（偽陽性）
- **型二錯誤** $\beta = P(\text{接受}H_0|H_1\text{為真})$（偽陰性）
- **檢定力** $1 - \beta$：正確拒絕 $H_0$ 的機率

### 常用檢定

**Z 檢定**：已知母體變異數或大樣本時檢驗均值
$$Z = \frac{\bar{X} - \mu_0}{\sigma/\sqrt{n}} \sim N(0, 1)$$

**t 檢定**：未知母體變異數時檢驗均值
$$T = \frac{\bar{X} - \mu_0}{s/\sqrt{n}} \sim t_{n-1}$$

**卡方檢定**：檢驗類別變數的獨立性或適配度
$$\chi^2 = \sum \frac{(O_i - E_i)^2}{E_i} \sim \chi^2_{k-1}$$

**F 檢定**：比較兩個變異數或 ANOVA
$$F = \frac{s_1^2}{s_2^2} \sim F_{n_1-1, n_2-1}$$

### p 值

p 值是在 $H_0$ 為真時，觀察到與樣本相同或更極端結果的機率。若 $p < \alpha$，則拒絕 $H_0$。

## 迴歸分析

### 簡單線性迴歸

$$Y = \beta_0 + \beta_1 X + \varepsilon$$

其中 $\varepsilon \sim N(0, \sigma^2)$ 是誤差項。

**目標**：最小化殘差平方和 $RSS = \sum (Y_i - \hat{Y}_i)^2$。

**估計**：
$$\hat{\beta}_1 = \frac{Cov(X, Y)}{Var(X)} = \frac{\sum (X_i - \bar{X})(Y_i - \bar{Y})}{\sum (X_i - \bar{X})^2}$$
$$\hat{\beta}_0 = \bar{Y} - \hat{\beta}_1 \bar{X}$$

### 多元線性迴歸

$$\mathbf{Y} = \mathbf{X}\boldsymbol{\beta} + \boldsymbol{\varepsilon}$$

最小平方法估計：$\hat{\boldsymbol{\beta}} = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{Y}$

### $R^2$

$$R^2 = 1 - \frac{RSS}{TSS} = 1 - \frac{\sum (Y_i - \hat{Y}_i)^2}{\sum (Y_i - \bar{Y})^2}$$

代表模型解釋的變異比例，取值範圍為 $0$ 到 $1$。

### 變異數分析 (ANOVA)

比較三個或以上組別的均值差異：
- **組間變異**：$SSB = \sum n_i (\bar{Y}_i - \bar{Y})^2$
- **組內變異**：$SSW = \sum \sum (Y_{ij} - \bar{Y}_i)^2$
- **F 統計量**：$F = \frac{SSB/(k-1)}{SSW/(n-k)} \sim F_{k-1, n-k}$

## 非參數統計

當資料不滿足常態分佈假設時，使用非參數方法：

- **符號檢定**：檢驗中位數
- **威爾考克森等級和檢定**：比較兩組中位數
- **克魯斯可-瓦立斯檢定**：比較多組中位數（ANOVA 的非參數版本）
- **曼-惠特尼 U 檢定**：比較兩個獨立樣本

## 貝葉斯統計

**貝葉斯定理**應用於統計推斷：
$$P(\theta|data) = \frac{P(data|\theta)P(\theta)}{P(data)}$$

- **先驗分佈** $P(\theta)$：觀察資料前的信念
- **概似函數** $P(data|\theta)$：給定參數下觀察到資料的機率
- **後驗分佈** $P(\theta|data)$：觀察資料後的信念更新

**貝葉斯估計**：後驗均值 $E[\theta|data]$

**馬可夫鏈蒙地卡羅 (MCMC)**：
- Metropolis-Hastings 演算法
- Gibbs 取樣
- 用於從複雜的後驗分佈中取樣

## 應用領域

### 機器學習

- **貝葉斯分類器**：$P(class|features) \propto P(features|class)P(class)$
- **線性迴歸**：$L_2$ 正則化（嶺回歸）、$L_1$ 正則化（Lasso）
- **高斯混合模型** (GMM)：軟性分群
- **隱馬可夫模型** (HMM)：序列建模

### 資訊論

- **熵**：$H(X) = -\sum p(x)\log p(x)$，衡量不確定性
- **互資訊**：$I(X;Y) = H(X) - H(X|Y)$，衡量相關性
- **Kullback-Leibler 散度**：$D_{KL}(P||Q) = \sum P(x)\log\frac{P(x)}{Q(x)}$

### 生物統計

- **生存分析**：Kaplan-Meier 估計、Cox 比例危險模型
- **遺傳學統計**：關聯分析 (GWAS)、群體遺傳學
- **臨床試驗**：隨機對照試驗 (RCT)、雙盲試驗

### 金融統計

- **風險管理**：風險值 (VaR)、期望虧損 (ES)
- **時間序列**：$AR(p)$、$MA(q)$、$ARIMA(p,d,q)$ 模型
- **波動率模型**：$GARCH$、隨機波動率模型

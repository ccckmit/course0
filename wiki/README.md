# CCC114B LLM Wiki

這是一個根據 Karpathy 的 LLM Wiki 模式建立的六門課程知識庫。

## 核心概念

LLM Wiki 是一種個人知識庫模式，與傳統 RAG 不同：
- **傳統 RAG**: 每次查詢時從原始文檔檢索相關片段
- **LLM Wiki**: LLM 增量建立和維護一個持久的、交叉連結的 Markdown 檔案庫

## 運作方式

1. **攝入 (Ingest)**: 將新來源放入原始收集，LLM 讀取並整合到 Wiki
2. **查詢 (Query)**: 對 Wiki 提問，LLM 綜合答案並引用來源
3. **整理 (Lint)**: 定期檢查 Wiki 健康狀態，修復矛盾、過時資訊

## 為什麼有效

維護知識庫的繁瑣之處不在於閱讀或思考建構，
而在於更新交叉引用、保持摘要最新、標記矛盾之處。
人類放棄 Wiki 是因為維護負擔增長快於價值。
LLM 不會無聊、不會忘記更新交叉引用、可以一次觸及 15 個檔案。

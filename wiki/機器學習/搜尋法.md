# 搜尋法 (Search Algorithms)

## 概述

搜尋法是人工智慧和電腦科學的基礎，用於在問題空間中找到目標解。分為暴力搜尋和啟發式搜尋兩大類。

## 搜尋問題定義

```python
class SearchProblem:
    def __init__(self, initial_state, goal_test, successor_fn, step_cost=1):
        self.initial_state = initial_state
        self.goal_test = goal_test
        self.successor_fn = successor_fn  # 返回 (action, state, cost)
        self.step_cost = step_cost
```

## 無資訊搜尋 (Uninformed Search)

### 廣度優先搜尋 (BFS)

```python
from collections import deque

def bfs(problem):
    frontier = deque([problem.initial_state])
    explored = set()
    
    while frontier:
        state = frontier.popleft()
        
        if problem.goal_test(state):
            return state
        
        explored.add(state)
        
        for action, successor, cost in problem.successor_fn(state):
            if successor not in explored and successor not in frontier:
                frontier.append(successor)
    
    return None
```

**特性：**
| 特性 | 值 |
|------|-----|
| 時間複雜度 | O(b^d) |
| 空間複雜度 | O(b^d) |
| 完整性 | 是 |
| 最佳性 | 是（一致代價時） |

其中 b 為分支因子，d 為深度。

### 深度優先搜尋 (DFS)

```python
def dfs(problem, frontier=None):
    if frontier is None:
        frontier = [problem.initial_state]
    
    while frontier:
        state = frontier.pop()
        
        if problem.goal_test(state):
            return state
        
        if state not in explored:
            explored.add(state)
            for action, successor, cost in problem.successor_fn(state):
                frontier.append(successor)
    
    return None
```

**特性：**
| 特性 | 值 |
|------|-----|
| 時間複雜度 | O(b^m) |
| 空間複雜度 | O(bm) |
| 完整性 | 否（無限深度可能） |
| 最佳性 | 否 |

### 統一代價搜尋 (UCS)

```python
import heapq

def uniform_cost_search(problem):
    frontier = [(0, problem.initial_state)]  # (cost, state)
    explored = {}
    
    while frontier:
        cost, state = heapq.heappop(frontier)
        
        if problem.goal_test(state):
            return state
        
        if state in explored and explored[state] <= cost:
            continue
        
        explored[state] = cost
        
        for action, successor, step_cost in problem.successor_fn(state):
            new_cost = cost + step_cost
            heapq.heappush(frontier, (new_cost, successor))
    
    return None
```

### 迭代加深搜尋 (IDS)

```python
def iterative_deepening_search(problem, max_depth=100):
    for depth in range(max_depth):
        result = depth_limited_search(problem, depth)
        if result is not None:
            return result
    return None

def depth_limited_search(problem, limit):
    def recursive_dls(state, depth):
        if problem.goal_test(state):
            return state
        if depth == 0:
            return 'cutoff'
        
        cutoff = False
        for action, successor, cost in problem.successor_fn(state):
            result = recursive_dls(successor, depth - 1)
            if result == 'cutoff':
                cutoff = True
            elif result is not None:
                return result
        
        return 'cutoff' if cutoff else None
    
    return recursive_dls(problem.initial_state, limit)
```

## 啟發式搜尋 (Heuristic Search)

### A* 搜尋

結合路徑代價和估計代價：

```python
def astar_search(problem, heuristic):
    frontier = [(heuristic(problem.initial_state), 0, problem.initial_state)]
    explored = {}
    
    while frontier:
        f, g, state = heapq.heappop(frontier)
        
        if problem.goal_test(state):
            return state
        
        if state in explored and explored[state] <= g:
            continue
        
        explored[state] = g
        
        for action, successor, cost in problem.successor_fn(state):
            new_g = g + cost
            new_f = new_g + heuristic(successor)
            heapq.heappush(frontier, (new_f, new_g, successor))
    
    return None
```

**A* 的特性：**
```
f(n) = g(n) + h(n)
  │      │       │
  │      │       └── 估計到目標的代價
  │      └── 到節點 n 的實際代價
  └── 節點 n 的總評估值
```

### 貪婪最佳優先搜尋

```python
def greedy_best_first_search(problem, heuristic):
    frontier = [(heuristic(problem.initial_state), problem.initial_state)]
    
    while frontier:
        f, state = heapq.heappop(frontier)
        
        if problem.goal_test(state):
            return state
        
        for action, successor, cost in problem.successor_fn(state):
            heapq.heappush(frontier, (heuristic(successor), successor))
    
    return None
```

### 比較圖

```
搜尋樹視圖：

                    A (起點)
                   /│\
                  B │ C
                 /\│  \
                D  E   F
                
BFS:    A → B → C → D → E → F (層序)
DFS:    A → B → D → E → C → F (深度)
A*:     A → B → D → goal (使用啟發式)
```

## 啟發式函數

### 常見設計方式

```python
# 8-Puzzle 範例
def manhattan_distance(state, goal):
    """曼哈頓距離"""
    distance = 0
    for i, tile in enumerate(state):
        if tile != 0:
            goal_pos = goal.index(tile)
            current_row, current_col = i // 3, i % 3
            goal_row, goal_col = goal_pos // 3, goal_pos % 3
            distance += abs(current_row - goal_row) + abs(current_col - goal_col)
    return distance

def misplaced_tiles(state, goal):
    """錯放方塊數"""
    return sum(1 for i in range(9) if state[i] != goal[i] and state[i] != 0)

def euclidean_distance(state, goal):
    """歐氏距離"""
    distance = 0
    for i, tile in enumerate(state):
        if tile != 0:
            goal_pos = goal.index(tile)
            current_row, current_col = i // 3, i % 3
            goal_row, goal_col = goal_pos // 3, goal_pos % 3
            distance += ((current_row - goal_row)**2 + (current_col - goal_col)**2)**0.5
    return distance
```

### 啟發式性質

| 性質 | 說明 | 條件 |
|------|------|------|
| 可採納性 (Admissible) | 永遠不高估 | h(n) ≤ h*(n) |
| 一致性 (Consistent) | 滿足三角不等式 | h(n) ≤ c(n,a,n') + h(n') |

## 對抗搜尋 (Adversarial Search)

### Minimax 演算法

```python
def minimax(node, depth, is_maximizing, game):
    if depth == 0 or game.is_terminal(node):
        return game.evaluate(node)
    
    if is_maximizing:
        value = float('-inf')
        for child in game.get_children(node):
            value = max(value, minimax(child, depth - 1, False, game))
        return value
    else:
        value = float('inf')
        for child in game.get_children(node):
            value = min(value, minimax(child, depth - 1, True, game))
        return value
```

### Alpha-Beta 剪枝

```python
def alpha_beta(node, depth, alpha, beta, is_maximizing, game):
    if depth == 0 or game.is_terminal(node):
        return game.evaluate(node)
    
    if is_maximizing:
        value = float('-inf')
        for child in game.get_children(node):
            value = max(value, alpha_beta(child, depth - 1, alpha, beta, False, game))
            alpha = max(alpha, value)
            if alpha >= beta:
                break  # Beta 剪枝
        return value
    else:
        value = float('inf')
        for child in game.get_children(node):
            value = min(value, alpha_beta(child, depth - 1, alpha, beta, True, game))
            beta = min(beta, value)
            if alpha >= beta:
                break  # Alpha 剪枝
        return value
```

## 搜尋演算法比較

| 演算法 | 時間 | 空間 | 最佳性 | 完整性 |
|--------|------|------|--------|--------|
| BFS | O(b^d) | O(b^d) | 是 | 是 |
| DFS | O(b^m) | O(bm) | 否 | 否 |
| UCS | O(b^{C*/ε}) | O(b^{C*/ε}) | 是 | 是 |
| IDS | O(b^d) | O(bd) | 是 | 是 |
| Greedy | O(b^m) | O(bm) | 否 | 是 |
| A* | O(b^d) | O(b^d) | 是* | 是 |

*當啟發式可採納時

# 馬可夫鏈

## 概述

馬可夫鏈（Markov Chain）是一種重要的離散時間隨機過程，由俄羅斯數學家安德烈·馬可夫（Andrey Markov）在 1906 年首次系統性研究。馬可夫鏈的核心特性是「無記憶性」（Markov Property）或稱「馬可夫性」：系統的未來狀態只與當前狀態有關，與過去的歷史無關。

馬可夫鏈是機率論、統計物理學、隨機過程理論的基石，在機器學習中扮演著核心角色。它是隱馬可夫模型、馬可夫鏈蒙特卡羅（MCMC）、強化學習等重要方法的理論基礎。

## 馬可夫性的數學定義

### 嚴格馬可夫性

對於任意時間步 t 和狀態序列 s₀, s₁, ..., s_t：

$$P(X_{t+1} = s_{t+1} | X_t = s_t, X_{t-1} = s_{t-1}, ..., X_0 = s_0) = P(X_{t+1} = s_{t+1} | X_t = s_t)$$

這意味著：給定當前狀態，未來條件獨立於過去。

### 時間同質馬可夫鏈

如果轉移機率與時間無關，即：

$$P(X_{t+1} = j | X_t = i) = P_{ij}$$

對於所有 t 成立，則稱為時間同質馬可夫鏈（Time-Homogeneous Markov Chain）。這是最常見的研究對象。

### 時間非齊次馬可夫鏈

轉移概率依賴於時間 t：

$$P(X_{t+1} = j | X_t = i) = P_{ij}(t)$$

這在非平穩環境中更適用，但分析更複雜。

## 轉移矩陣

### 轉移概率矩陣

對於具有 N 個狀態的馬可夫鏈，定義轉移矩陣 P：

$$P = \begin{pmatrix} P_{11} & P_{12} & \cdots & P_{1N} \\ P_{21} & P_{22} & \cdots & P_{2N} \\ \vdots & \vdots & \ddots & \vdots \\ P_{N1} & P_{N2} & \cdots & P_{NN} \end{pmatrix}$$

其中 P_ij = P(X_{t+1} = j | X_t = i)，且每一行之和為 1：

$$\sum_{j=1}^{N} P_{ij} = 1, \quad \forall i$$

### n 步轉移

使用 Chapman-Kolmogorov 方程，k 步後的轉移概率為：

$$P^{(k)} = P^k$$

即轉移矩陣的 k 次冪。更精確地：

$$P(X_{t+k} = j | X_t = i) = (P^k)_{ij}$$

這允許我們計算任意時間跨度後的狀態轉移概率。

## 狀態分類

### 可達性與連通性

如果存在 n ≥ 0 使得 P_ij^(n) > 0，則稱狀態 j 從狀態 i 可達，記為 i → j。

如果 i → j 且 j → i，則稱 i 和 j 互通，記為 i ↔ j。

### 封閉集與不可約性

封閉集（Closed Set）：若對所有 j ∉ C 和所有 i ∈ C，有 Σ_{k∈C} P_{ik} = 1，則 C 是封閉集。從封閉集中的任何狀態都無法到達外部狀態。

不可約（Irreducible）：如果所有狀態互通（對所有 i, j 有 i ↔ j），則馬可夫鏈是不可約的。

### 狀態類型

1. **暫態（Transient）**：從該狀態出發，最終返回的概率嚴格小於 1。

2. **常返態（Recurrent）**：從該狀態出發，以概率 1 返回。進一步分為：
   - 正常返（Positive Recurrent）：平均返回時間有限
   - 零正常返（Null Recurrent）：平均返回時間無限

3. **週期性（Periodic）**：狀態的返回時間只能在特定數的倍數時間步發生。

4. **非週期（Aperiodic）**：返回時間可以是任意整數。

## 平穩分佈

### 定義

一個概率分佈 π = (π₁, π₂, ..., π_N) 稱為平穩分佈（Stationary Distribution），如果：

$$\pi P = \pi$$

即它是轉移矩陣 P 的左特徵向量，特徵值為 1。

### 物理意義

如果馬可夫鏈在時刻 0 以 π 初始化，則在任意時刻 t，狀態分佈仍然是 π。換句話說，π 是一個「平衡點」。

### 平穩分佈的存在性

根據馬可夫鏈定理：

1. 如果馬可夫鏈是不可約且非週期的，則存在唯一的平穩分佈。

2. 如果馬可夫鏈是正則（不可約 + 非週期 + 正常返），則平穩分佈存在且唯一。

### 詳細平衡條件

一個更強的條件是詳細平衡（Detailed Balance）或稱可逆性：

$$\pi_i P_{ij} = \pi_j P_{ji}, \quad \forall i, j$$

如果滿足詳細平衡，則馬可夫鏈是可逆的，且必然以 π 為平穩分佈。這個條件在 MCMC 中經常使用。

## 極限理論

### 收斂到平穩分佈

對於正則馬可夫鏈：

$$\lim_{n \to \infty} P^n = \mathbf{1}^T \pi$$

其中 1 是全 1 向量，π 是唯一的平穩分佈。

換句話說，無論初始分佈如何，經過足夠長的時間後，狀態分佈會收斂到平穩分佈。

### 收斂速度

收斂速度由譜間隙（spectral gap）決定：

$$\gamma = 1 - \lambda_2$$

其中 λ₂ 是第二大的特徵值（按絕對值）。則：

$$\| P^n - \mathbf{1}^T \pi \|_1 \leq 2 |\lambda_2|^n$$

譜間隙越大，收斂越快。

## 馬可夫鏈的應用

### 隨機模擬

馬可夫鏈 Monte Carlo（MCMC）使用馬可夫鏈生成複雜分佈的樣本：
- Metropolis-Hastings 演算法
- Gibbs 採樣
- 粒子濾波

### 自然語言處理

- N-gram 語言模型（馬可夫假設）
- 隱馬可夫模型
- 條件隨機場（CRF）

### 金融建模

- 信用風險轉移矩陣
- 資產價格狀態模型
- 市場微觀結構

### 基因序列分析

- 隱馬可夫模型用於基因預測
- DNA 序列建模

### 排隊理論

- M/M/1 排隊系統
- 服務系統分析

## 馬可夫鏈的擴展

### 高階馬可夫鏈

k 階馬可夫鏈的未來狀態依賴於前 k 個狀態：

$$P(X_{t+1} | X_t, X_{t-1}, ..., X_0) = P(X_{t+1} | X_t, ..., X_{t-k+1})$$

可以透過狀態擴展轉化為一階馬可夫鏈。

### 連續狀態馬可夫鏈

狀態空間是連續的（如 R^n），需要使用轉移核：

$$P(X_{t+1} \in A | X_t = x) = \int_A k(x, y) dy$$

### 馬可夫決策過程（MDP）

加入動作和獎勵，定義 (S, A, P, R, γ)：
- S：狀態空間
- A：動作空間
- P：轉移概率 P(s'|s,a)
- R：獎勵函數
- γ：折扣因子

## 實作範例

### Python 實現基本馬可夫鏈

```python
import numpy as np

class MarkovChain:
    def __init__(self, P):
        self.P = np.array(P)
        self.n_states = len(P)
        
    def steady_state(self):
        """計算平穩分佈"""
        eigenvalues, eigenvectors = np.linalg.eig(self.P.T)
        idx = np.argmin(np.abs(eigenvalues - 1))
        pi = np.abs(eigenvectors[:, idx])
        return pi / pi.sum()
    
    def simulate(self, init_state, n_steps):
        """模擬馬可夫鏈"""
        state = init_state
        trajectory = [state]
        for _ in range(n_steps):
            r = np.random.random()
            cumsum = np.cumsum(self.P[state])
            state = np.searchsorted(cumsum, r)
            trajectory.append(state)
        return trajectory
    
    def n_step_prob(self, n):
        """n步轉移概率"""
        return np.linalg.matrix_power(self.P, n)
```

### 語言模型應用

```python
class MarkovLanguageModel:
    def __init__(self, order=2):
        self.order = order
        self.transitions = {}
        self.context_counts = {}
    
    def train(self, corpus):
        for sentence in corpus:
            tokens = ['<S>'] * self.order + sentence + ['<E>']
            for i in range(len(tokens) - self.order):
                context = tuple(tokens[i:i+self.order])
                next_word = tokens[i+self.order]
                self.context_counts[context] = self.context_counts.get(context, 0) + 1
                if context not in self.transitions:
                    self.transitions[context] = {}
                self.transitions[context][next_word] = self.transitions[context].get(next_word, 0) + 1
    
    def generate(self, max_length=20):
        output = ['<S>'] * self.order
        for _ in range(max_length):
            context = tuple(output[-self.order:])
            if context not in self.transitions:
                break
            probs = self.transitions[context]
            total = sum(probs.values())
            r = np.random.random() * total
            cumsum = 0
            for word, count in probs.items():
                cumsum += count
                if cumsum >= r:
                    output.append(word)
                    break
            if output[-1] == '<E>':
                break
        return output[self.order:]
```

## 數學性質證明

### 馬可夫鏈定理

**定理**：每個不可約、非週期的有限狀態馬可夫鏈恰好有一個平穩分佈，且從任意初始分佈出發，都會收斂到該平穩分佈。

**證明大綱**：
1. 由於不可約，P 中每個元素严格正
2. 由 Perron-Frobenius 定理，P 有一個特徵值 1，且其他特徵值絕對值小於 1
3. 對應特徵值 1 的左特徵向量是唯一的（只差一個倍數），歸一化後得到唯一的平穩分佈
4. P^n 的極限存在且等於 1^T π
5. 收斂速度由 |λ₂| 控制

### 馬可夫不等式

在馬可夫鏈分析中常用：

$$P(X_t = j) \to \pi_j$$

但收斂速度由混合時間（mixing time）量化：

$$t_{\text{mix}}(\epsilon) = \min\{t : \| P^t(i, \cdot) - \pi \|_1 \leq \epsilon \}$$

## 與其他概念的聯繫

### 與馬可夫決策過程

MDP 是馬可夫鏈的决策擴展，加入了智能體的動作選擇和環境獎勵：

$$P(s' | s, a) = \sum_{r} P(s', r | s, a)$$

### 與蒙特卡羅方法

MCMC 通過構建具有目標分佈為平穩分佈的馬可夫鏈來進行採樣。採樣器（如 Metropolis-Hastings）設計轉移矩陣使得：

$$\pi_i P_{ij} = \pi_j P_{ji}$$

### 與隱馬可夫模型

HMM 在馬可夫鏈的基礎上加入觀察層：
- 底層是馬可夫鏈（隱藏狀態）
- 觀察變量由隱藏狀態決定

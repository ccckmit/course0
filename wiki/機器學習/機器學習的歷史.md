# 機器學習的歷史

機器學習是人工智慧的一個分支，讓電腦從資料中自動學習模式並做出預測。這門學科的發展經歷了從符號推理到神經網路，再到大規模深度學習的多次革命性轉變。

## 起源：人工智慧的黎明（1950-1960 年代）

1950 年，艾倫·圖靈提出了著名的「圖靈測試」，為智慧機器的定義設立了基準。同年，克勞德·香農提出了會下西洋棋的機器概念。

1956 年，達特茅斯會議標誌著人工智慧作為獨立學科的誕生。會議汇集了馬文·明斯基、約翰·麥卡錫、艾倫·紐厄爾和赫伯特·西蒙等先驅。他們樂觀地預測：二十年內，機器將能完成任何人類能做的事。

這一時期的主流方法是符號人工智慧（Symbolic AI），基於邏輯推論和專家知識。但很快就遇到了瓶頸——機器無法有效處理現實世界的複雜性和不確定性。

## 第一次寒冬：專家系統的興衰（1970-1980 年代）

1970 年代中期，專家系統成為主流。MYCIN 是最早的專家系統之一，用於診斷血液感染。但專家系統需要人工編碼大量領域知識，成本高昂且難以維護。

1980 年代，專家系統短暫繁榮。日本的第五代電腦計畫投入巨資研發智慧計算機。然而，專家系統的局限性日益明顯：知識獲取困難、難以處理不確定性、無法從資料中學習。

1980 年代末，專家系統熱潮消退，投資者信心喪失。這段時期被稱為「人工智慧寒冬」。

## 機器學習的興起（1980-1990 年代）

### 符號學習

機器學習作為替代方案開始興起。研究者從兩個方向突破：

**概念學習**：Tom Mitchell 提出了機器學習的正式定義：「電腦程式被稱為從經驗 E 中學習，若任務 T 的效能度量 P，隨經驗 E 而提升。」

**決策樹**：ID3、C4.5 等決策樹學習演算法被開發出來。這些方法可以從資料中自動歸納分類規則。

```python
# 決策樹的核心：資訊增益
def information_gain(data, attribute, target):
    # 計算原始熵
    original_entropy = entropy(data[target])
    
    # 計算分割後的加權熵
    weighted_entropy = 0
    for value in unique_values(data[attribute]):
        subset = data[data[attribute] == value]
        weight = len(subset) / len(data)
        weighted_entropy += weight * entropy(subset[target])
    
    return original_entropy - weighted_entropy
```

### 類神經網路的復興

1986 年，David Rumelhart、Geoffrey Hinton 和 Ronald Williams 發表了「透過反向傳播學習內部表示」論文，重新點燃了對神經網路的熱情。

反向傳播演算法解決了多層神經網路的訓練問題。語音辨識和手寫識別開始取得進展。然而，當時的電腦算力有限，網路規模受限。

### 統計學習的崛起

1990 年代，統計學習方法開始主導機器學習領域。Vapnik 的支持向量機（SVM）在分類和回歸任務上表現優異。SVM 的核方法允許處理非線性問題，同時有完善的理論保證。

機器學習開始更加強調機率和統計，推動了貝葉斯方法、隱馬爾可夫模型（HMM)和條件隨機場（CRF）的發展。

## 深度學習革命（2006-2012 年代）

### 深度信念網路

2006 年，Geoffrey Hinton 發表了關於深度信念網路（DBN）的論文，提出了逐層貪婪預訓練（greedy layer-wise pretraining）方法。這是深度學習的關鍵突破，解決了多層網路的訓練困難。

### ImageNet 與電腦視覺的突破

2009 年，李飛飛團隊發布了 ImageNet 資料集，包含 1500 萬張標註圖片。2012 年，Alex Krizhevsky、Ilya Sutskever 和 Geoffrey Hinton 設計的 AlexNet 在 ImageNet 競賽中以巨大優勢奪冠，將錯誤率從 26% 降至 15%。

AlexNet 的關鍵創新：
- 使用 GPU 加速訓練
- ReLU 啟動函數
- Dropout 正則化
- 資料增強

此後，每年 ImageNet 競賽的错误率持续下降。2015 年，ResNet 的錯誤率降至 3.6%，已低於人類的 5%。

### 自然語言處理的變革

傳統 NLP 依賴人工設計的特徵和規則。2013 年，Tomas Mikolov 發明了 Word2Vec，將單詞映射到向量空間，讓機器理解單詞之間的語義關係。

2017 年，Transformer 架構被提出，採用自注意力機制並行處理序列資料。2018 年，BERT 和 GPT 相繼問世，預訓練語言模型徹底改變了 NLP。

## 卷積神經網路的演進

### LeNet：開創者

1998 年，Yann LeCun 設計了 LeNet-5，這是最早的卷積神經網路之一，用於郵政編碼辨識。LeNet 奠定了 CNN 的基本架構：
- 卷積層：提取局部特徵
- 池化層：降低維度
- 全連接層：分類輸出

### AlexNet：深度學習的黎明

2012 年，AlexNet 的成功標誌著深度學習時代的來臨。

```python
# AlexNet 核心結構（簡化）
class AlexNet(nn.Module):
    def __init__(self, num_classes=1000):
        super(AlexNet, self).__init__()
        self.features = nn.Sequential(
            # 5 個卷積層
            nn.Conv2d(3, 64, kernel_size=11, stride=4, padding=2),
            nn.ReLU(inplace=True),
            nn.MaxPool2d(kernel_size=3, stride=2),
            # ... 更多卷積層
        )
        self.avgpool = nn.AdaptiveAvgPool2d((6, 6))
        self.classifier = nn.Sequential(
            # 3 個全連接層
            nn.Dropout(p=0.5),
            nn.Linear(256 * 6 * 6, 4096),
            nn.ReLU(inplace=True),
            nn.Dropout(p=0.5),
            nn.Linear(4096, 4096),
            nn.ReLU(inplace=True),
            nn.Linear(4096, num_classes),
        )
```

### VGG、GoogLeNet、ResNet

- **VGGNet（2014）**：使用更小的 3x3 卷積核，增加網路深度
- **GoogLeNet（2014）**：Inception 模組，多尺度特徵提取
- **ResNet（2015）**：殘差連接，解決深度網路梯度消失問題
- **EfficientNet（2019）**：複合縮放，均衡深度、寬度、解析度

## 循環神經網路與序列處理

### 早期 RNN

循環神經網路（RNN）是最早用於處理序列資料的神經網路：

```python
# 基本 RNN 單元
class RNNCell(nn.Module):
    def __init__(self, input_size, hidden_size):
        super().__init__()
        self.Wxh = nn.Linear(input_size, hidden_size)
        self.Whh = nn.Linear(hidden_size, hidden_size)
    
    def forward(self, x, h):
        h = torch.tanh(self.Wxh(x) + self.Whh(h))
        return h
```

### LSTM 與 GRU

長短期記憶網路（LSTM）和門控循環單元（GRU）解決了長期依賴問題：

```python
# LSTM 的關鍵：門控機制
class LSTMCell(nn.Module):
    def forward(self, x, state):
        h, c = state
        # 輸入門：控制新資訊流入
        i = torch.sigmoid(self.Wii(x) + self.bii + self.Bhi(h))
        # 忘記門：控制舊資訊丟失
        f = torch.sigmoid(self.Wif(x) + self.bif + self.Bhf(h))
        # 輸出門：控制輸出
        o = torch.sigmoid(self.Wio(x) + self.bio + self.Bho(h))
        # 候選記憶
        g = torch.tanh(self.Wig(x) + self.big + self.Bhg(h))
        
        c = f * c + i * g
        h = o * torch.tanh(c)
        return h, c
```

## 生成式 AI 時代（2020 年代）

### 大型語言模型的崛起

2020 年，OpenAI 發布 GPT-3，1750 億參數的大型語言模型展示了驚人的語言能力。此後，模型規模不斷擴大：GPT-4、Claude、Gemini 等相繼問世。

大型語言模型的關鍵能力：
- 零樣本和少樣本學習
- 上下文學習
- 思維鏈推理
- 多模態理解

```python
# Transformer 核心：自注意力機制
class SelfAttention(nn.Module):
    def forward(self, x):
        B, N, C = x.shape
        
        # 計算 Q, K, V
        qkv = self.qkv(x).reshape(B, N, 3, self heads, C // self.heads)
        q, k, v = qkv.unbind(2)
        
        # 注意力權重
        attn = (q @ k.transpose(-2, -1)) * self.scale
        attn = attn.softmax(dim=-1)
        
        # 加權求和
        x = (attn @ v).transpose(1, 2).reshape(B, N, C)
        return x
```

### 生成對抗網路

2014 年，Ian Goodfellow 發明了生成對抗網路（GAN）。GAN 由生成器和判別器組成，透過對抗訓練生成逼真圖像。

```python
# GAN 的對抗訓練
def train_step(generator, discriminator, real_images):
    # 訓練判別器
    fake_images = generator(noise)
    d_loss = adversarial_loss(discriminator(fake_images), True)
    d_loss += adversarial_loss(discriminator(real_images), False)
    
    # 訓練生成器
    fake_images = generator(noise)
    g_loss = adversarial_loss(discriminator(fake_images), True)
    
    return d_loss, g_loss
```

### Diffusion 模型

2020 年代，Diffusion 模型成為生成式 AI 的主流。DDPM、DALL-E、Stable Diffusion、Midjourney 等模型可以根據文字描述生成高質量圖像。

Diffusion 模型相較 GAN 更穩定、更易訓練，開啟了 AI 創作的新時代。

### TD-Gammon 與西洋雙陸棋

1992 年，Gerald Tesauro 開發的 TD-Gammon 在西洋雙陸棋上達到人類專家水平。這是強化學習在遊戲中的首次重大成功。

### AlphaGo 與圍棋

1997 年，IBM 的深藍擊敗國際象棋冠軍卡斯帕洛夫。

2016 年，AlphaGo 擊敗圍棋世界冠軍李世石。AlphaGo 結合了深度學習和蒙地卡羅樹搜索，展示了強化學習在複雜策略遊戲中的潛力。

```python
# 蒙地卡羅樹搜索（MCTS）
class MCTS:
    def search(self, state):
        while self.time_remaining():
            leaf = self.select(state)
            child = self.expand(leaf)
            result = self.simulate(child)
            self.backpropagate(child, result)
        return self.best_child(state)
    
    def select(self, node):
        # UCB1 上置信界線
        return max(node.children, key=lambda n: n.Q + n.U)
    
    def expand(self, node):
        # 擴展所有可能的動作
        for action in node.state.legal_actions():
            node.add_child(action)
        return random.choice(node.children)
    
    def simulate(self, node):
        #  rollout：隨機 rollout 直到遊戲結束
        while not node.state.is_terminal():
            node.state = random.choice(node.state.legal_actions())
        return node.state.reward()
```

## 遷移學習與預訓練模型

遷移學習允許模型從一個任務中學到的知識應用到另一個任務：

```python
# 預訓練模型的遷移學習
import torchvision.models as models

# 載入預訓練模型
model = models.resnet50(pretrained=True)

# 凍結前面層
for param in model.parameters():
    param.requires_grad = False

# 替換最後層
num_features = model.fc.in_features
model.fc = nn.Linear(num_features, num_classes)

# 只需要訓練最後層
optimizer = torch.optim.Adam(model.fc.parameters())
```

## AutoML 與神經架構搜索

AutoML 自動搜索最佳模型架構：

```python
# 簡化的神經架構搜索
class NASController:
    def __init__(self):
        self.search_space = {
            'layers': [1, 2, 3, 4],
            'filters': [16, 32, 64, 128],
            'kernel_size': [3, 5, 7],
        }
    
    def sample_architecture(self):
        return {
            'layers': random.choice(self.search_space['layers']),
            'filters': random.choice(self.search_space['filters']),
            'kernel_size': random.choice(self.search_space['kernel_size']),
        }
    
    def train_and_evaluate(self, architecture):
        model = build_model(architecture)
        train(model)
        return evaluate(model)
    
    def search(self, num_trials=100):
        best_architecture = None
        best_accuracy = 0
        
        for _ in range(num_trials):
            arch = self.sample_architecture()
            accuracy = self.train_and_evaluate(arch)
            if accuracy > best_accuracy:
                best_accuracy = accuracy
                best_architecture = arch
        
        return best_architecture
```

基礎模型（Foundation Models）成為 AI 發展的新範式。單一大型模型透過微調即可適配多種任務，降低了 AI 應用的門檻。

AI 正在加速科學發現：
- **AlphaFold**：蛋白質結構預測

- **能耗問題**：訓練大型模型消耗大量能源（GPT-3 訓練耗電約 1 GWh）

- **偏見與公平**：消除模型中的歧視和偏見

# 反傳遞演算法 (Backpropagation)

## 概述

反向傳播（Backpropagation）是訓練神經網路的核心演算法，透過鏈式法則計算損失函數對每個參數的梯度，實現高效的神經網路學習。

## 計算圖

神經網路的前饋傳播構成一個計算圖：
```
輸入 → Linear → ReLU → Linear → Softmax → Loss
         ↑                    ↑
       W₁, b₁               W₂, b₂
```

### 前饋傳播 (Forward Pass)
```python
import numpy as np

class Linear:
    def __init__(self, in_features, out_features):
        self.W = np.random.randn(in_features, out_features) * 0.01
        self.b = np.zeros((1, out_features))
    
    def forward(self, x):
        self.x = x  # 快取用於反向傳播
        return x @ self.W + self.b

# 前饋
x = input_data                    # (batch, in_features)
h = linear1.forward(x)          # (batch, hidden)
h = relu.forward(h)              # (batch, hidden)
y = linear2.forward(h)           # (batch, out_features)
loss = cross_entropy(y, target)  # scalar
```

### 反向傳播 (Backward Pass)
```python
# 反向計算梯度
dL_dy = cross_entropy_backward(y, target)    # ∂L/∂y
dL_dW2 = h.T @ dL_dy                          # ∂L/∂W₂
dL_db2 = np.sum(dL_dy, axis=0)               # ∂L/∂b₂
dL_dh = dL_dy @ W2.T                          # ∂L/∂h
dL_dh = relu_backward(dL_dh, h)               # ∂L/∂h (through ReLU)
dL_dW1 = x.T @ dL_dh                          # ∂L/∂W₁
dL_db1 = np.sum(dL_dh, axis=0)               # ∂L/∂b₁
```

## 鏈式法則 (Chain Rule)

### 基本原理
對於複合函數 y = f(g(x))：
```
dy/dx = (dy/du) × (du/dx)
```

### 多層鏈式法則
```
L = f(g(h(x)))
∂L/∂x = ∂L/∂f × ∂f/∂g × ∂g/∂h × ∂h/∂x
```

### 實際範例
```python
# 簡單網路: x → W → ReLU → out
# Loss = (out - y)² / 2

out = relu(W @ x)
loss = (out - y)² / 2

# 反向傳播：
# ∂L/∂out = out - y
# ∂out/∂(W@x) = 1 (if W@x > 0) else 0
# ∂(W@x)/∂W = x

dL_dout = out - y
dout_dz = (W @ x > 0).astype(float)  # ReLU derivative
dL_dz = dL_dout * dout_dz
dL_dW = np.outer(x, dL_dz)
```

### 微分的計算圖視圖
```
          Forward                    Backward
          
          x ────→ W ────→ z ────→ ReLU ────→ out
                          ↓                      ↓
                      z = Wx               out = max(0, z)
                         
                          ↑                      ↑
          ←────────── ←────────── ←────────── ←────
                    Backward (反向流動梯度)
                    
∂L/∂W = ∂L/∂out × ∂out/∂z × ∂z/∂W
```

## 反向傳播推導

### 單層網路
```
輸入: x ∈ Rⁿ
權重: W ∈ Rⁿˣᵐ
輸出: y = Wx ∈ Rᵐ
損失: L = ||y - y_target||²

∂L/∂W = ∂L/∂y × ∂y/∂W = 2(y - y_target) × xᵀ
```

### 多層網路
```
x → [W₁] → h → [W₂] → y → L

∂L/∂W₁ = ∂L/∂y × ∂y/∂h × ∂h/∂W₁
        = dL_dy @ W₂ᵀ × ReLU'(h) × xᵀ
```

## 梯度消失與梯度爆炸

### 梯度消失
```python
# 原因：激活函數導數 < 1，連乘後趨近於 0
# ∂L/∂W₁ = ∂L/∂out × ∂out/∂h × ... × ∂h₁/∂W₁
#         = 0.1 × 0.1 × ... × 0.1 ≈ 0

# 解決：ReLU、殘差連接、Batch Normalization
def relu(x):
    return np.maximum(0, x)

def relu_grad(x):
    return (x > 0).astype(float)  # 梯度為 0 或 1
```

### 梯度爆炸
```python
# 原因：權重過大，連乘後趨近無窮大
# ∂L/∂W₁ = 10 × 10 × ... × 10 ≈ ∞

# 解決：梯度裁剪、權重初始化、Xavier/He 初始化
gradient = np.clip(gradient, -1, 1)  # 梯度裁剪
```

## 權重初始化

### Xavier 初始化
```python
# 適用於 sigmoid/tanh
W = np.random.randn(n_in, n_out) * np.sqrt(2.0 / (n_in + n_out))
```

### He 初始化
```python
# 適用於 ReLU
W = np.random.randn(n_in, n_out) * np.sqrt(2.0 / n_in)
```

## 反向傳播演算法步驟

```python
def train_step(model, x, y_target, learning_rate):
    # 1. 前饋傳播
    output = model(x)
    
    # 2. 計算損失
    loss = compute_loss(output, y_target)
    
    # 3. 反向傳播（由後往前）
    dL_dout = compute_loss_gradient(output, y_target)
    
    for layer in reversed(model.layers):
        dL_dout = layer.backward(dL_dout)
    
    # 4. 更新權重
    for layer in model.layers:
        if hasattr(layer, 'W'):
            layer.W -= learning_rate * layer.dL_dW
            layer.b -= learning_rate * layer.dL_db
    
    return loss
```

# 樸素貝葉斯 (Naive Bayes)

## 原理

樸素貝葉斯分類器基於貝葉斯定理，假設特徵之間相互獨立（因此稱為「樸素」）。儘管這個假設在現實中很少成立，但樸素貝葉斯分類器在許多實際應用中表現出人意料的好效果，且具有訓練和預測速度快、對缺失值不敏感等優點。

$$P(y|x) = \frac{P(x|y) P(y)}{P(x)}$$

由於 $P(x)$ 對所有類別相同，預測時選擇使 $P(x|y) P(y)$ 最大的類別：

$$\hat{y} = \arg\max_y P(y) \prod_{i=1}^{n} P(x_i | y)$$

## 高斯樸素貝葉斯

假設特徵服從高斯分佈：

```python
import numpy as np

class GaussianNB:
    """高斯樸素貝葉斯"""
    
    def fit(self, X, y):
        n_samples, n_features = X.shape
        self.classes = np.unique(y)
        n_classes = len(self.classes)
        
        self.class_prior = np.zeros(n_classes)
        self.mean = np.zeros((n_classes, n_features))
        self.var = np.zeros((n_classes, n_features))
        
        for i, c in enumerate(self.classes):
            X_c = X[y == c]
            self.class_prior[i] = X_c.shape[0] / n_samples
            self.mean[i] = X_c.mean(axis=0)
            self.var[i] = X_c.var(axis=0) + 1e-9
        
        return self
    
    def gaussian_pdf(self, x, mean, var):
        return np.exp(-0.5 * ((x - mean) ** 2) / var) / np.sqrt(2 * np.pi * var)
    
    def predict(self, X):
        predictions = []
        for x in X:
            posteriors = []
            for i in range(len(self.classes)):
                prior = np.log(self.class_prior[i])
                likelihood = np.sum(np.log(self.gaussian_pdf(x, self.mean[i], self.var[i])))
                posterior = prior + likelihood
                posteriors.append(posterior)
            predictions.append(self.classes[np.argmax(posteriors)])
        
        return np.array(predictions)
```

## 使用 sklearn

```python
from sklearn.naive_bayes import GaussianNB, MultinomialNB, CategoricalNB

gnb = GaussianNB()
gnb.fit(X_train, y_train)
predictions = gnb.predict(X_test)
```

- `GaussianNB`：連續特徵，假設高斯分佈
- `MultinomialNB`：計數特徵（文本詞頻）
- `CategoricalNB`：類別特徵
- `ComplementNB`： Complement NB，對類別不平衡資料更魯棒

```python
# 文本分類
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.naive_bayes import MultinomialNB

vectorizer = CountVectorizer()
X_train_text = vectorizer.fit_transform(train_texts)

mnb = MultinomialNB(alpha=1.0)  # alpha 為拉普拉斯平滑
mnb.fit(X_train_text, train_labels)
```

## 拉普拉斯平滑

避免零機率問題：

$$P(x_i | y) = \frac{count(x_i, y) + \alpha}{count(y) + \alpha \cdot |V|}$$

其中 $\alpha$ 為平滑參數，通常設為 1。

## 演算法比較

| 演算法 | 優點 | 缺點 | 適用場景 |
|--------|------|------|----------|
| 樸素貝葉斯 | 快速、簡單、對缺失值不敏感 | 特徵獨立假設不成立 | 文本分類、多類別、高維度 |
| KNN | 簡單、無訓練階段 | 預測慢、高維度失效 | 小型資料 |
| 邏輯回歸 | 可解釋、概率輸出 | 只能處理線性 | 基線模型 |

# 隱馬可夫模型

## 概述

隱馬可夫模型（Hidden Markov Model, HMM）是一種統計模型，假設系統是由一個不可直接觀察的馬可夫鏈生成的，同時這個馬可夫鏈的每個狀態會產生可觀察的輸出。HMM 由 Baum 和 Petrie 在 1960 年代提出，是語音識別、自然語言處理、生物資訊學等領域的核心技術。

HMM 的「隱藏」特性使其特別適合以下場景：
- 語音識別：我們聽到聲音（觀察），但想知道說了什麼單詞（狀態）
- 基因序列分析：我們觀察鹼基序列，但想知道基因結構（外顯子、內含子等）
- 自然語言處理：我們觀察詞序列，但想知道語法結構或詞性標注

## 形式化定義

一個 HMM 由五元組 λ = (N, M, A, B, π) 定義：

### 模型參數

1. **N**：隱藏狀態的數量，狀態集合 Q = {q₁, q₂, ..., q_N}
2. **M**：每個狀態可能的觀察值數量，觀察集合 V = {v₁, v₂, ..., v_M}
3. **A**：狀態轉移概率矩陣，A = [a_ij]，其中 a_ij = P(q_{t+1} = j | q_t = i)
4. **B**：觀察概率矩陣，B = [b_j(k)]，其中 b_j(k) = P(o_t = v_k | q_t = j)
5. **π**：初始狀態概率向量，π = [π_i]，其中 π_i = P(q_1 = i)

### 生成過程

給定模型參數，觀察序列 O = o₁, o₂, ..., o_T 的生成過程如下：

1. 根據初始分佈 π 選擇初始狀態 q₁
2. 根據 b_{q₁}(o₁) 生成觀察 o₁
3. 對於每個時間步 t = 2 to T：
   - 根據 a_{q_{t-1}, q_t} 轉移到新狀態 q_t
   - 根據 b_{q_t}(o_t) 生成觀察 o_t

## 三個核心問題

### 問題一：評估（Evaluation）

給定模型 λ 和觀察序列 O，計算 P(O | λ)。

這衡量觀察序列與模型的匹配程度。常用於語音辨識中的候選模型比較。

**前向算法（Forward Algorithm）**

直接計算會有 O(TN^T) 的複雜度（每步都枚舉所有狀態序列）。前向算法利用動態規劃將複雜度降低到 O(TN²)。

定義前向變量：
$$\alpha_t(i) = P(o_1, o_2, ..., o_t, q_t = i | \lambda)$$

遞推公式：
1. **初始化**：α₁(i) = π_i · b_i(o₁)，對於所有 i
2. **遞推**：α_{t+1}(j) = [Σ_i α_t(i) · a_ij] · b_j(o_{t+1})，對於所有 j
3. **終止**：P(O | λ) = Σ_i α_T(i)

時間複雜度：O(TN²)，空間複雜度：O(N)（只需要保存當前和前一時刻的值）。

### 問題二：解碼（Decoding）

給定模型 λ 和觀察序列 O，找到最可能的隱藏狀態序列 Q = {q₁, q₂, ..., q_T}。

**維特比算法（Viterbi Algorithm）**

這是一種動態規劃算法。定義：
$$\delta_t(i) = \max_{q_1, ..., q_{t-1}} P(q_1, ..., q_{t-1}, q_t = i, o_1, ..., o_t | \lambda)$$

遞推公式：
1. **初始化**：δ₁(i) = π_i · b_i(o₁)，ψ₁(i) = 0
2. **遞推**：δ_{t+1}(j) = max_i [δ_t(i) · a_ij] · b_j(o_{t+1})，ψ_{t+1}(j) = argmax_i [δ_t(i) · a_ij]
3. **終止**：P* = max_i δ_T(i)，q_T* = argmax_i δ_T(i)
4. **回溯**：q_t* = ψ_{t+1}(q_{t+1}*)，直到 t = 1

時間複雜度：O(TN²)，空間複雜度：O(TN) 存儲整個路徑，或 O(N) 僅存儲最優路徑。

### 問題三：學習（Learning）

給定觀察序列 O，調整模型參數 λ = (A, B, π) 使得 P(O | λ) 最大。

**鮑姆-韋爾奇算法（Baum-Welch Algorithm）**

這是 EM 演算法在 HMM 中的應用。設觀察序列為 O，隱藏狀態序列未知。

E 步：計算後驗概率 γ_t(i) = P(q_t = i | O, λ) 和 ξ_t(i, j) = P(q_t = i, q_{t+1} = j | O, λ)

M 步：更新參數
- π_i = γ₁(i)
- a_ij = Σ_t ξ_t(i, j) / Σ_t γ_t(i)
- b_j(k) = Σ_{t: o_t = v_k} γ_t(j) / Σ_t γ_t(j)

**前向-後向算法（Forward-Backward Algorithm）**

後向變量定義：
$$\beta_t(i) = P(o_{t+1}, ..., o_T | q_t = i, \lambda)$$

遞推：
1. **初始化**：β_T(i) = 1
2. **遞推**：β_t(i) = Σ_j a_ij · b_j(o_{t+1}) · β_{t+1}(j)

然後：
- γ_t(i) = α_t(i) · β_t(i) / Σ_j α_t(j) · β_t(j)
- ξ_t(i, j) = α_t(i) · a_ij · b_j(o_{t+1}) · β_{t+1}(j) / P(O | λ)

## HMM 的類型

### 離散 HMM

觀察值是離散符號（如詞彙表中的單詞）。需要使用觀察概率矩陣 B。

### 連續 HMM

觀察值是連續向量。通常假設每個狀態的觀察分佈是高斯分佈（高斯 HMM）或高斯混合分佈（GMM-HMM）。

**高斯 HMM 的觀察分佈**：
$$b_j(o) = \mathcal{N}(o; \mu_j, \Sigma_j)$$

**GMM-HMM**：
$$b_j(o) = \sum_{m=1}^{M} c_{jm} \mathcal{N}(o; \mu_{jm}, \Sigma_{jm})$$

### 左到右 HMM（Left-to-Right HMM）

狀態按順序排列，只能保持或向右轉移。常用於語音識別：

$$a_{ij} = 0, \quad \text{若 } j < i \text{ 或 } j > i + \Delta$$

## 實作範例

### Python 實現基本 HMM

```python
import numpy as np
from typing import Tuple, List, Optional

class HiddenMarkovModel:
    def __init__(self, n_states: int, n_observations: int):
        self.n_states = n_states
        self.n_observations = n_observations
        self.A = None  # 轉移矩陣
        self.B = None  # 觀察矩陣
        self.pi = None  # 初始分佈
        
    def initialize(self, strategy='random'):
        """初始化模型參數"""
        if strategy == 'random':
            self.A = np.random.rand(self.n_states, self.n_states)
            self.B = np.random.rand(self.n_states, self.n_observations)
            self.pi = np.random.rand(self.n_states)
        elif strategy == 'uniform':
            self.A = np.ones((self.n_states, self.n_states)) / self.n_states
            self.B = np.ones((self.n_states, self.n_observations)) / self.n_observations
            self.pi = np.ones(self.n_states) / self.n_states
        # 歸一化
        self.A /= self.A.sum(axis=1, keepdims=True)
        self.pi /= self.pi.sum()
        
    def forward(self, observations: List[int]) -> Tuple[float, np.ndarray]:
        """前向算法"""
        T = len(observations)
        alpha = np.zeros((T, self.n_states))
        
        # 初始化
        alpha[0] = self.pi * self.B[:, observations[0]]
        
        # 遞推
        for t in range(1, T):
            alpha[t] = (alpha[t-1] @ self.A) * self.B[:, observations[t]]
        
        # 終止
        prob = alpha[-1].sum()
        return prob, alpha
    
    def viterbi(self, observations: List[int]) -> Tuple[List[int], float]:
        """維特比算法"""
        T = len(observations)
        delta = np.zeros((T, self.n_states))
        psi = np.zeros((T, self.n_states), dtype=int)
        
        # 初始化
        delta[0] = self.pi * self.B[:, observations[0]]
        psi[0] = 0
        
        # 遞推
        for t in range(1, T):
            for j in range(self.n_states):
                trans_probs = delta[t-1] * self.A[:, j]
                delta[t, j] = np.max(trans_probs) * self.B[j, observations[t]]
                psi[t, j] = np.argmax(trans_probs)
        
        # 回溯
        path = np.zeros(T, dtype=int)
        path[-1] = np.argmax(delta[-1])
        for t in range(T-2, -1, -1):
            path[t] = psi[t+1, path[t+1]]
        
        log_prob = np.log(delta[-1]).max()
        return path.tolist(), np.exp(log_prob)
    
    def baum_welch(self, observations: List[int], 
                   n_iter: int = 100, 
                   tol: float = 1e-6):
        """鮑姆-韋爾奇算法"""
        T = len(observations)
        old_prob = -np.inf
        
        for iteration in range(n_iter):
            # 前向
            alpha = np.zeros((T, self.n_states))
            alpha[0] = self.pi * self.B[:, observations[0]]
            for t in range(1, T):
                alpha[t] = (alpha[t-1] @ self.A) * self.B[:, observations[t]]
            
            # 後向
            beta = np.zeros((T, self.n_states))
            beta[-1] = 1
            for t in range(T-2, -1, -1):
                beta[t] = (self.A @ (self.B[:, observations[t+1]] * beta[t+1]))
            
            # E步
            gamma = alpha * beta / (alpha * beta).sum(axis=1, keepdims=True)
            xi = np.zeros((T-1, self.n_states, self.n_states))
            for t in range(T-1):
                denom = (alpha[t].reshape(-1, 1) * self.A * 
                        self.B[:, observations[t+1]].reshape(1, -1) * 
                        beta[t+1].reshape(1, -1)).sum()
                xi[t] = (alpha[t].reshape(-1, 1) * self.A * 
                        self.B[:, observations[t+1]].reshape(1, -1) * 
                        beta[t+1].reshape(1, -1)) / denom
            
            # M步
            self.pi = gamma[0]
            self.A = xi.sum(axis=0) / gamma[:-1].sum(axis=0, keepdims=True)
            
            # 更新B
            for k in range(self.n_observations):
                mask = np.array(observations).reshape(-1, 1) == k
                self.B[:, k] = (gamma * mask).sum(axis=0) / gamma.sum(axis=0)
            
            # 檢查收斂
            prob = alpha[-1].sum()
            if abs(prob - old_prob) < tol:
                break
            old_prob = prob
```

### 應用：詞性標注

```python
class POSTagger:
    def __init__(self):
        self.hmm = HiddenMarkovModel(n_states=len(POS_TAGS), 
                                   n_observations=VOCAB_SIZE)
        self.state_to_idx = {tag: i for i, tag in enumerate(POS_TAGS)}
        self.word_to_idx = {word: i for i, word in enumerate(VOCAB)}
    
    def train(self, sentences: List[List[Tuple[str, str]]], n_iter=100):
        """訓練數據：[(word, pos), ...]"""
        observations = [[self.word_to_idx[w] for w, p in sent] 
                       for sent in sentences]
        self.hmm.initialize()
        self.hmm.baum_welch(observations, n_iter=n_iter)
    
    def tag(self, sentence: List[str]) -> List[str]:
        """標注新句子"""
        obs = [self.word_to_idx.get(w, 0) for w in sentence]
        path, _ = self.hmm.viterbi(obs)
        return [POS_TAGS[i] for i in path]
```

## HMM 的變體

### 層疊 HMM（Layered HMM）

多個 HMM 堆疊，上層 HMM 的觀察來自下層 HMM 的狀態。用於語音理解等層級結構。

### 樹狀 HMM（Tree HMM）

狀態轉移圖是樹狀結構，每個狀態可以有多個父狀態。

### 因子 HMM（Factorial HMM）

每個時間步有多個獨立的隱藏鏈，觀察是它們的函數。用於表示多維狀態。

### 隱馬可夫隨機場（Hidden Markov Random Field）

狀態是二維格點結構，用於圖像建模。

## 實際應用

### 語音識別

HMM 是經典語音識別系統的核心：
1. 語音信號 → 特徵向量（MFCC）
2. 每個音素用一個 HMM 表示
3. 單詞由音素 HMM 串接
4. 使用維特比解碼找到最優音素序列

現代系統可能使用深度神經網路替代 HMM 的觀察分佈（DNN-HMM）。

### 自然語言處理

- **詞性標注**：詞序列 → 詞性序列
- **命名實體識別**：詞序列 → 實體類型序列
- **語音識別**：聲學特徵 → 單詞序列

### 生物資訊學

- **基因預測**：DNA 序列 → 基因結構（外顯子、內含子、啟動子等）
- **蛋白質結構預測**：氨基酸序列 → 二級結構
- **序列比對**：用於局部比對

### 金融時間序列

- 市場狀態識別（牛市、熊市、震蕩）
- 信用風險建模
- 資產價格預測

### 機器人導航

- 位置估計（感測器讀數 → 位置）
- 狀態追蹤

## 局限性與擴展

### HMM 的局限性

1. **獨立性假設**：假設觀察值只在當前狀態下條件獨立，忽略了觀察間的依賴。
2. **狀態數固定**：需要事先指定狀態數。
3. **觀察分佈**：假設觀察分佈屬於某個參數族。

### 擴展方法

1. **條件隨機場（CRF）**：放鬆獨立性假設，考慮全域特徵。
2. **神經 HMM**：用神經網路學習觀察分佈。
3. **Transformer**：用注意力機制替代序列假設。
4. **長短期記憶網路（LSTM）**：處理長期依賴。

## 與其他模型的比較

### vs 馬可夫鏈

| 特性 | 馬可夫鏈 | HMM |
|------|----------|-----|
| 觀察性 | 可觀察狀態 | 隱藏狀態 |
| 觀察生成 | 無 | 每個狀態生成觀察 |
| 應用 | 直接建模可觀察過程 | 建模隱藏結構 |
| 複雜度 | 較低 | 較高 |

### vs CRF

CRF ��� HMM 的無向圖擴展，考慮觀察之間的特徵依賴。

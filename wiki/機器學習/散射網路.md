# 散射網路 (Diffusion Model)

## 概述

散射模型（Diffusion Model）是一類基於機率漸近理論的生成模型，透過逐步添加噪聲再逐步去除噪聲的過程來生成資料。與生成對抗網路（GAN）和變分自編碼器（VAE）相比，散射模型具有更穩定的訓練過程和更好的生成品質，近年來在影像生成領域取得了突破性進展。

散射模型的核心思想源自物理學中的擴散過程：分子會從高濃度區域向低濃度區域均勻擴散，最終達到熱平衡狀態。在機器學習中，我們將這個過程逆轉——從純噪聲開始，逐步去除噪聲，逐步恢復出有意義的資料結構。

## 前向散射過程

散射模型分為兩個主要過程：**前向過程（Forward Process）** 和 **反向過程（Reverse Process）**。

### 前向過程（添加噪聲）

前向過程將真實資料 x₀ 逐步添加高斯噪聲，得到一系列逐漸逼近純噪聲的中間狀態 x₁, x₂, ..., x_T。這個過程可以表示為：

$$q(\mathbf{x}_t | \mathbf{x}_{t-1}) = \mathcal{N}(\mathbf{x}_t; \sqrt{1 - \beta_t} \mathbf{x}_{t-1}, \beta_t \mathbf{I})$$

其中 β_t 是第 t 步的噪聲方差調度（noise schedule），通常隨時間步逐漸增大。這個設計使得：
- 當 t 很小时，x_t 接近原始資料 x₀
- 當 t 很大時（如 t = T ≈ 1000），x_T 接近標準高斯分布

一個關鍵性質是：我們可以直接計算任意時間步 t 的分佈，而不需要逐次迭代。設 α_t = 1 - β_t 且 \bar{\alpha}_t = ∏_{i=1}^{t} α_i，則：

$$q(\mathbf{x}_t | \mathbf{x}_0) = \mathcal{N}(\mathbf{x}_t; \sqrt{\bar{\alpha}_t} \mathbf{x}_0, (1 - \bar{\alpha}_t) \mathbf{I})$$

這個封閉形式使得我們可以在訓練時直接採樣任意時間步的噪聲資料。

### 噪聲調度策略

噪聲調度的設計對模型性能有重要影響：

1. **線性調度（Linear Schedule）**：從 β₁ 到 β_T 線性遞增，簡單但效果一般。

2. **餘弦調度（Cosine Schedule）**：β_t = β_min + (β_max - β_min) × cos²(πt/T)²，表現更好，衰減更平滑。

3. **二次方根調度（Sqrt Schedule）**：β_t = β_min + (β_max - β_min) × √(t/T)，介於線性和餘弦之間。

餘弦調度因其平滑的噪聲衰減曲線，通常能產生更好的生成結果。

## 反向過程（去噪）

反向過程是生成資料的關鍵。我們訓練一個神經網路 ε_θ(x_t, t) 來預測添加到 x_{t-1} 的噪聲。給定 x_t 和 t，網路輸出 ε̂，表示其估計的噪聲成分。

### 訓練目標

標準的去噪散射模型（Denoising Diffusion Probabilistic Model, DDPM）的訓練目標是最小化均方誤差：

$$L = \mathbb{E}_{t, \mathbf{x}_0, \boldsymbol{\epsilon}} \left[ \| \boldsymbol{\epsilon} - \boldsymbol{\epsilon}_\theta(\mathbf{x}_t, t) \|^2 \right]$$

其中：
- t 在 1 到 T 均勻採樣
- ε 是標準高斯噪聲
- x_t = √{\bar{\alpha}_t} x₀ + √{1 - \bar{\alpha}_t} ε

實務上，常用的目標函數形式更穩定。Ho等人在2020年的論文提出使用簡化的目標：

$$L_{\text{simple}} = \mathbb{E}_{t} \left[ w_t \cdot \| \boldsymbol{\epsilon} - \boldsymbol{\epsilon}_\theta(\sqrt{\bar{\alpha}_t} \mathbf{x}_0 + \sqrt{1 - \bar{\alpha}_t} \boldsymbol{\epsilon}, t) \|^2 \right]$$

其中 w_t 是時間步權重，通常設為 1。

### 生成過程

給定純噪聲 x_T ~ N(0, I)，我們迭代執行反向過程：

$$\mathbf{x}_{t-1} = \frac{1}{\sqrt{\alpha_t}} \left( \mathbf{x}_t - \frac{1 - \alpha_t}{\sqrt{1 - \bar{\alpha}_t}} \boldsymbol{\epsilon}_\theta(\mathbf{x}_t, t) \right) + \sigma_t \mathbf{z}$$

其中 z ~ N(0, I) 是隨機噪聲，σ_t 控制每步的隨機性。在推理時，可以選擇是否添加此隨機項。

## 條件生成與引導

### 分類器引導（Classifier Guidance）

為了實現條件生成（如文字到圖像），可以使用外部分類器的梯度來引導生成過程。給定類別 y，分類器引導的預測為：

$$\tilde{\boldsymbol{\epsilon}} = \boldsymbol{\epsilon}_\theta(\mathbf{x}_t | \phi) + s \cdot \nabla_\mathbf{x}_t \log p_\phi(y | \mathbf{x}_t)$$

其中 s 是引導強度（通常 1-10），φ 是分類器參數。較大的 s 值產生更高保真度但多樣性較低的輸出。

### 無分類器引導（Classifier-Free Guidance）

無分類器引導是一種更優雅的方法，不需要額外的分類器。它通過聯合訓練條件和無條件模型來隱式學習引導：

$$\tilde{\boldsymbol{\epsilon}} = (1 + w) \cdot \boldsymbol{\epsilon}_\theta(\mathbf{x}_t | c) - w \cdot \boldsymbol{\epsilon}_\theta(\mathbf{x}_t | \emptyset)$$

其中 w 是引導權重（通常 1-10）。當 w > 1 時，輸出更接近條件但犧牲多樣性。

## 改進架構

### U-Net 主幹網路

早期的散射模型使用 U-Net 作為去噪網路，這是一種對稱編碼器-解碼器架構，具有跳層連接（skip connections）。U-Net 能夠有效捕捉多尺度特徵，對於圖像生成特別有效。

改進點包括：
- 加入自注意力層（self-attention）處理全域依賴
- 使用殘差連接（residual connections）穩定訓練
- 採用時間嵌入（time embedding）編碼擴散時間步

### Transformer 整合

最近的研究將 Transformer 架構引入散射模型，如 DiT（Diffusion Transformer）。DiT 使用潛在表示（latent representation）來減少計算量，並在高位圖生成上取得了優秀效果。

## 加速採樣

### 離散時間加速

由於需要 T 步（通常 T = 1000）的迭代，完整採樣非常耗時。離散時間加速方法包括：

1. **DDIM（Denoising Diffusion Implicit Models）**：重新參數化採樣過程，使其可以跳步採樣而不顯著影響質量。DDIM 可以將採樣步數從 1000 減少到 20-50 步。

2. **快速採樣調度**：如 ProDistiller，訓練專門針對大步長的去噪模型。

### 連續時間模型

連續時間散射模型將離散時間步擴展為連續時間 t ∈ [0, 1]，可以使用更少的離散化步數達到更好的效果。

## 應用領域

### 影像生成

散射模型在高品質影像生成方面表現出色：
- 文字到影像生成（如 DALL-E 2, Stable Diffusion）
- 超解析度重建
- 影像修復和編輯
- 影像風格遷移

### 音頻處理

音頻領域的應用包括：
- 文字轉語音（TTS）合成
- 音頻修復和增強
- 音樂生成

### 3D 建模

最近的 3D 散射模型（如 Point-E, DreamFusion）可以從文字描述生成 3D 模型，雖然品質還不及 2D 生成，但發展迅速。

### 其他領域

- 醫學影像生成
- 異常偵測
- 強化學習中的規劃

## 代表性模型

### DDPM（2020）

Ho 等人提出的原始散射模型，奠定了基礎理論。使用線性噪聲調度，需要在 1000 步迭代中生成樣本。

### DDPM++ / ADM（2021）

改進的架構和訓練策略，使用 U-Net++ 和更大的模型容量。引入了分類器引導技術。

### Stable Diffusion（2022）

結合了潛在散射（Latent Diffusion）和無分類器引導的開源模型。在低維潛在空間執行擴散過程，大幅降低計算需求。

### DALL-E 2 / Imagen（2022）

大型條件散射模型，通過 CLIP 文字編碼器實現文字到影像生成。展示了超大規模型的潛力。

### DiT（2023）

使用 Transformer 架構的散射模型，證明了 Transformer 可以很好地適應散射框架。

## 與其他生成模型比較

### vs GAN

| 特性 | 散射模型 | GAN |
|------|----------|-----|
| 訓練穩定性 | 穩定（基於最大似然） | 不穩定（minimax 對抗） |
| 多樣性 | 較高（趨近數據分佈） | 較低（模式崩潰） |
| 採樣速度 | 慢（需多步迭代） | 快（單步生成） |
| 記憶體 | 較高 | 適中 |
| 生成品質 | 高（細節優異） | 高（但有時有偽影） |

### vs VAE

VAE 通過編碼器-解碼器架構學習潛在空間，但：
- 潛在空間可能不平滑
- 傾向於模糊的生成結果
- 訓練相對簡單

散射模型則提供更清晰的理論基礎和更好的樣本品質。

## 實作要點

### PyTorch 實作框架

```python
class DiffusionModel:
    def __init__(self, T=1000, beta_start=0.0001, beta_end=0.02):
        self.T = T
        self.beta = self.linear_schedule(beta_start, beta_end)
        self.alpha = 1 - self.beta
        self.alpha_bar = torch.cumprod(self.alpha, dim=0)
    
    def linear_schedule(self, beta_start, beta_end):
        return torch.linspace(beta_start, beta_end, self.T)
    
    def forward_process(self, x0, t):
        """添加噪聲"""
        noise = torch.randn_like(x0)
        sqrt_alpha_bar = self.sqrt_alpha_bar[t]
        sqrt_one_minus = self.sqrt_one_minus_alpha_bar[t]
        return sqrt_alpha_bar * x0 + sqrt_one_minus * noise, noise
    
    @abstractmethod
    def model(self, xt, t):
        """去噪網路"""
        pass
    
    @abstractmethod
    def training_loss(self, x0):
        """計算訓練損失"""
        pass
    
    @abstractmethod
    def sampling(self, batch_size):
        """生成樣本"""
        pass
```

### 訓練技巧

1. **時間步採樣**：均勻隨機採樣所有 T 步，確保模型在所有噪聲 level 上訓練。

2. **指數移動平均（EMA）**：對模型權重做 EMA 可以穩定訓練並提高生成品質。

3. **混合精度訓練**：使用 fp16 加速訓練，減少記憶體使用。

4. **梯度累積**：在小 batch size 時累積多個 batch 的梯度。

# 世界模型 (World Models)

## 概述

世界模型是能夠學習和理解環境動態規律的 AI 系統，讓代理能夠預測未來狀態、進行想像模擬和長期規劃。

## 核心概念

```
┌─────────────────────────────────────────────────────────────┐
│                    世界模型架構                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  觀察 O_t ──→ 感知模組 ──→ 壓縮表示 z_t                    │
│                         ↓                                   │
│                    世界模型                                  │
│  z_t ──→ 預測 ──→ z_{t+1}, r_{t+1}, done               │
│                         ↓                                   │
│                    想像 rollout                              │
│  模擬未來軌跡，評估動作價值                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## World Model 架構

### 組成部分

1. **Vision Model (V)**: 將原始觀察編碼為壓縮表示
2. **Memory (M)**: 儲存和檢索歷史資訊
3. **World Model (M)**: 預測未來狀態和獎勵
4. **Controller (C)**: 根據表示選擇動作

## Dreamer 算法

```python
class Dreamer:
    def __init__(self, obs_dim, action_dim, hidden_dim=200):
        # 世界模型：RSSM (Recurrent State-Space Model)
        self.dynamics = RSSM(hidden_dim, action_dim)
        
        # 觀察解碼器
        self.obs_decoder = ObservationDecoder(hidden_dim, obs_dim)
        
        # 獎勵預測器
        self.reward_predictor = RewardPredictor(hidden_dim)
        
        # 價值網路
        self.value網絡 = ValueNetwork(hidden_dim)
        
        # 動作策略
        self.policy = ActorCritic(hidden_dim, action_dim)
    
    def imagine_rollout(self, state, horizon=10):
        """想像未來軌跡"""
        states = [state]
        rewards = []
        
        for _ in range(horizon):
            action = self.policy(state)
            next_state, reward = self.dynamics.sample(state, action)
            states.append(next_state)
            rewards.append(reward)
        
        return states, rewards
```

### RSSM (Recurrent State-Space Model)

```python
class RSSM(nn.Module):
    def __init__(self, hidden_dim, action_dim, stochastic_dim=30):
        super().__init__()
        self.stochastic_dim = stochastic_dim
        self.deterministic_dim = hidden_dim - stochastic_dim
        
        # 轉移動力學
        self.transition = nn.GRUCell(
            hidden_dim, self.deterministic_dim
        )
        
        # 後驗分佈
        self.posterior_net = nn.Sequential(
            nn.Linear(hidden_dim + obs_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, stochastic_dim * 2)
        )
        
        # 先驗分佈
        self.prior_net = nn.Sequential(
            nn.Linear(self.deterministic_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, stochastic_dim * 2)
        )
    
    def forward(self, prev_state, action, obs=None):
        deterministic = self.transition(
            torch.cat([prev_state['det'], action], dim=-1),
            prev_state['det']
        )
        
        # 先驗
        prior_dist = self.prior_net(deterministic)
        prior_mean, prior_logstd = prior_dist.chunk(2, dim=-1)
        prior = prior_mean + torch.randn_like(prior_mean) * prior_logstd.exp()
        
        # 後驗（如果提供觀察）
        if obs is not None:
            posterior_dist = self.posterior_net(
                torch.cat([deterministic, obs], dim=-1)
            )
            posterior_mean, posterior_logstd = posterior_dist.chunk(2, dim=-1)
            posterior = posterior_mean + torch.randn_like(posterior_mean) * posterior_logstd.exp()
        else:
            posterior = prior
        
        return {
            'det': deterministic,
            'stoch': posterior
        }
```

## Model-Based RL

```python
class ModelBasedRL:
    def __init__(self, env, model, planner):
        self.env = env
        self.model = model
        self.planner = planner
    
    def collect_trajectories(self, n=100):
        """收集訓練資料"""
        trajectories = []
        for _ in range(n):
            obs = self.env.reset()
            done = False
            episode = []
            
            while not done:
                action = self.model.policy(torch.tensor(obs))
                next_obs, reward, done, _ = self.env.step(action)
                episode.append((obs, action, reward, next_obs))
                obs = next_obs
            
            trajectories.append(episode)
        return trajectories
    
    def train_model(self, trajectories):
        """訓練世界模型"""
        for epoch in range(100):
            for traj in trajectories:
                obs, actions, rewards, next_obs = zip(*traj)
                
                # 預測未來
                pred_next = self.model.predict(obs, actions)
                
                # 更新模型
                loss = mse_loss(pred_next, next_obs)
                loss.backward()
                self.optimizer.step()
    
    def plan(self, obs, horizon=10):
        """使用世界模型規劃"""
        return self.planner.imagine(self.model, obs, horizon)
```

## World Model 的優勢

| 優勢 | 說明 |
|------|------|
| 樣本效率 | 想像中訓練，減少真實環境互動 |
| 長期規劃 | 預測未來，進行多步規劃 |
| 風險評估 | 模擬不同動作的後果 |
| 泛化能力 | 學習環境動態，而非特定任務 |

## 挑戰

| 挑戰 | 說明 |
|------|------|
| 累積誤差 | 想像 rollout 越長，誤差越大 |
| 隨機性 | 環境可能是隨機的 |
| 部分觀察 | 觀察可能不完整 |
| 計算成本 | 世界模型本身的訓練成本 |

# 爬山演算法 (Hill Climbing)

## 概述

爬山演算法是一種簡單的局部搜尋演算法，從一個初始解出發，逐步向目標函數值增加的方向移動，直到找到局部最佳解。

## 基本演算法

```python
def hill_climbing(objective, neighbor_fn, initial_state):
    current = initial_state
    current_value = objective(current)
    
    while True:
        neighbors = neighbor_fn(current)
        best_neighbor = max(neighbors, key=objective)
        best_value = objective(best_neighbor)
        
        if best_value <= current_value:
            return current, current_value  # 局部最佳解
        
        current = best_neighbor
        current_value = best_value
```

## 爬山演算法變體

### 1. 簡單爬山法 (Simple Hill Climbing)

```python
def simple_hill_climbing(problem):
    current = problem.initial_state()
    
    while True:
        neighbors = problem.successors(current)
        best = max(neighbors, key=problem.value)
        
        if problem.value(best) <= problem.value(current):
            return current
        current = best
```

### 2. 最陡爬山法 (Steepest-Ascent Hill Climbing)

```python
def steepest_ascent_hill_climbing(problem):
    current = problem.initial_state()
    
    while True:
        best_neighbor = None
        best_value = problem.value(current)
        
        for neighbor in problem.successors(current):
            if problem.value(neighbor) > best_value:
                best_neighbor = neighbor
                best_value = problem.value(neighbor)
        
        if best_neighbor is None or best_value <= problem.value(current):
            return current
        
        current = best_neighbor
```

### 3. 隨機爬山法 (Random Hill Climbing)

```python
import random

def random_hill_climbing(problem, max_iterations=1000):
    current = problem.initial_state()
    
    for _ in range(max_iterations):
        neighbors = problem.successors(current)
        neighbor = random.choice(neighbors)
        
        if problem.value(neighbor) > problem.value(current):
            current = neighbor
    
    return current
```

## 問題與解決

### 1. 區域最大 (Local Maximum)

```
         ╱╲
        ╱  ╲
 Local ─╱    ╲    ╱╲
 Max   ╱      ╲  ╱  ╲
      ╱        ╲╱    ╲ Global
     ╱                ╲ Max
    ╱                  ╲
```

**解決：隨機重新開始爬山 (Random Restart)**
```python
def random_restart_hill_climbing(problem, num_restarts=10):
    best_state = None
    best_value = float('-inf')
    
    for _ in range(num_restarts):
        state = hill_climbing(problem, random_initial=True)
        value = problem.value(state)
        
        if value > best_value:
            best_state = state
            best_value = value
    
    return best_state
```

### 2. 高原 (Plateaus)

```
         ─────────────
        │   Plateau    │ ← 所有鄰居相同，無法前進
         ─────────────
```

**解決：側向移動 (Lateral Move)**
```python
def hill_climbing_with_lateral(problem, max_lateral=100):
    current = problem.initial_state()
    lateral_moves = 0
    
    while True:
        neighbors = problem.successors(current)
        best = max(neighbors, key=problem.value)
        
        if problem.value(best) > problem.value(current):
            current = best
            lateral_moves = 0
        elif lateral_moves < max_lateral:
            current = random.choice(neighbors)  # 側向移動
            lateral_moves += 1
        else:
            return current
```

### 3. 山脊 (Ridges)

```
    ╱╲      ╱╲      ╱╲
   ╱  ╲    ╱  ╲    ╱  ╲
  ╱    ╲  ╱    ╲  ╱    ╲
```

**解決：使用動量或模擬退火**

## 模擬退火 (Simulated Annealing)

結合隨機性的爬山法：

```python
import math
import random

def simulated_annealing(problem, schedule):
    current = problem.initial_state()
    
    for t in range(sys.maxsize):
        T = schedule(t)  # 溫度參數
        
        if T == 0:
            return current
        
        neighbors = problem.successors(current)
        next_state = random.choice(neighbors)
        
        delta = problem.value(next_state) - problem.value(current)
        
        if delta > 0:
            current = next_state
        else:
            probability = math.exp(delta / T)
            if random.random() < probability:
                current = next_state
```

### 退火排程
```python
# 指數衰減
def exponential_schedule(T0=100, decay=0.95):
    t = 0
    while True:
        yield T0 * (decay ** t)
        t += 1

# 對數衰減（收斂保證）
def logarithmic_schedule(k=1):
    t = 1
    while True:
        yield k / math.log(t + 1)
        t += 1
```

## TSP 範例

```python
import random

def tsp_hill_climbing(cities, max_iterations=10000):
    # 初始解：隨機排列
    current = list(range(len(cities)))
    random.shuffle(current)
    
    def distance(route):
        return sum(
            cities[route[i]][route[(i+1) % len(route)]]
            for i in range(len(route))
        )
    
    for _ in range(max_iterations):
        # 產生鄰居：交換兩個城市
        neighbor = current.copy()
        i, j = random.sample(range(len(cities)), 2)
        neighbor[i], neighbor[j] = neighbor[j], neighbor[i]
        
        if distance(neighbor) < distance(current):
            current = neighbor
    
    return current, distance(current)
```

## 爬山法 vs 窮舉搜尋

| 特性 | 爬山法 | 窮舉搜尋 |
|------|--------|----------|
| 時間複雜度 | O(k × n) | O(n^k) |
| 記憶體 | O(n) | O(k) |
| 解的品質 | 局部最佳 | 全域最佳 |
| 保證 | 無 | 有 |

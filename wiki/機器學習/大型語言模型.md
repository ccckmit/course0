# 大型語言模型

大型語言模型（Large Language Model，簡稱 LLM）是指基於深度學習技術，特別是 Transformer 架構，在海量文字資料上進行預訓練而形成的語言模型。LLM 的出現標誌著自然語言處理領域的重大突破，它們不僅能夠理解和生成人類語言，還展現出推理、程式碼撰寫、翻譯、問答等多種能力。從 GPT-3 到 GPT-4，從 LLaMA 到 Claude，LLM 的規模和能力持續快速增長，正在深刻改變人工智慧的發展方向和應用格局。

## Transformer 架構

Transformer 是 LLM 的核心架構，其基於自注意力機制：

```python
import torch
import torch.nn as nn
import math

class SelfAttention(nn.Module):
    """自注意力機制"""
    def __init__(self, embed_size, heads):
        super(SelfAttention, self).__init__()
        self.embed_size = embed_size
        self.heads = heads
        self.head_dim = embed_size // heads
        
        self.W_q = nn.Linear(embed_size, embed_size)
        self.W_k = nn.Linear(embed_size, embed_size)
        self.W_v = nn.Linear(embed_size, embed_size)
        self.fc_out = nn.Linear(embed_size, embed_size)
    
    def forward(self, x, mask=None):
        batch_size, seq_len, _ = x.shape
        
        # 線性變換
        Q = self.W_q(x)
        K = self.W_k(x)
        V = self.W_v(x)
        
        # 分割成多個 head
        Q = Q.view(batch_size, seq_len, self.heads, self.head_dim).transpose(1, 2)
        K = K.view(batch_size, seq_len, self.heads, self.head_dim).transpose(1, 2)
        V = V.view(batch_size, seq_len, self.heads, self.head_dim).transpose(1, 2)
        
        # 計算注意力分數
        scores = torch.matmul(Q, K.transpose(-2, -1)) / math.sqrt(self.head_dim)
        
        # Masking (可選)
        if mask is not None:
            scores = scores.masked_fill(mask == 0, float('-inf'))
        
        # 注意力權重
        attention = torch.softmax(scores, dim=-1)
        
        # 應用注意力到 V
        out = torch.matmul(attention, V)
        
        # 合併 heads
        out = out.transpose(1, 2).contiguous().view(batch_size, seq_len, self.embed_size)
        
        return self.fc_out(out)

class TransformerBlock(nn.Module):
    """Transformer 區塊"""
    def __init__(self, embed_size, heads, dropout, forward_expansion):
        super(TransformerBlock, self).__init__()
        
        self.attention = SelfAttention(embed_size, heads)
        self.norm1 = nn.LayerNorm(embed_size)
        self.norm2 = nn.LayerNorm(embed_size)
        
        self.feed_forward = nn.Sequential(
            nn.Linear(embed_size, forward_expansion * embed_size),
            nn.ReLU(),
            nn.Linear(forward_expansion * embed_size, embed_size)
        )
        
        self.dropout = nn.Dropout(dropout)
    
    def forward(self, x, mask=None):
        # Multi-Head Attention + Residual
        attention = self.attention(x, mask)
        x = self.norm1(x + self.dropout(attention))
        
        # Feed Forward + Residual
        ff = self.feed_forward(x)
        x = self.norm2(x + self.dropout(ff))
        
        return x

class GPT(nn.Module):
    """GPT 模型簡化版"""
    def __init__(self, vocab_size, embed_size, num_layers, heads, 
                 dropout, max_length):
        super(GPT, self).__init__()
        
        self.embed_size = embed_size
        self.position_embedding = nn.Embedding(max_length, embed_size)
        self.token_embedding = nn.Embedding(vocab_size, embed_size)
        self.dropout = nn.Dropout(dropout)
        
        self.layers = nn.ModuleList([
            TransformerBlock(embed_size, heads, dropout, forward_expansion=4)
            for _ in range(num_layers)
        ])
        
        self.fc_out = nn.Linear(embed_size, vocab_size)
    
    def forward(self, x, mask=None):
        batch_size, seq_len = x.shape
        
        # Token 和 Position 嵌入
        token_emb = self.token_embedding(x)
        pos_emb = self.position_embedding(torch.arange(seq_len, device=x.device))
        x = self.dropout(token_emb + pos_emb)
        
        # Transformer 層
        for layer in self.layers:
            x = layer(x, mask)
        
        return self.fc_out(x)
```

### 4. 人類回饋強化學習（RLHF）

```python
# RLHF 三步驟：
# 1. 訓練 reward model
# 2. 用 PPO 優化 policy
# 3. 反覆迭代

# Step 1: Reward Model
class RewardModel(nn.Module):
    def __init__(self, base_model):
        super().__init__()
        self.base_model = base_model
        self.scorer = nn.Linear(hidden_size, 1)
    
    def forward(self, input_ids, attention_mask):
        outputs = self.base_model(input_ids, attention_mask)
        # 使用最後一個 token 的表示作為評分
        hidden = outputs.hidden_states[-1][:, -1, :]
        return self.scorer(hidden)

# Step 2: PPO 優化
from trl import PPOTrainer, PPOConfig

ppo_config = PPOConfig(
    batch_size=1,
    learning_rate=1.4e-5,
    log_dir="./logs"
)

ppo_trainer = PPOTrainer(
    config=ppo_config,
    model=model,
    reward_model=reward_model,
    tokenizer=tokenizer
)

# 訓練 loop
for batch in dataloader:
    query_tensors = batch['query']

    # 生成回應

        response = generate(query)
```

### 蒸餾

```python
# 知識蒸餾
class DistillationLoss(nn.Module):
    def __init__(self, temperature=2, alpha=0.5):
        super().__init__()
        self.temperature = temperature
        self.alpha = alpha
    
    def forward(self, student_logits, teacher_logits, labels):
        # KL 散度蒸餾損失
        soft_teacher = F.softmax(teacher_logits / self.temperature, dim=-1)
        soft_student = F.log_softmax(student_logits / self.temperature, dim=-1)
        distillation_loss = F.kl_div(soft_student, soft_teacher, reduction='batchmean') * (self.temperature ** 2)
        
        # 標準交叉熵損失
        ce_loss = F.cross_entropy(student_logits, labels)
        
        # 合併
        return self.alpha * distillation_loss + (1 - self.alpha) * ce_loss
```

## 相關挑戰與未來方向

### 當前挑戰

1. **事實準確性**：模型可能產生幻覺
2. **推理複雜度**：複雜推理仍有限制
3. **計算成本**：訓練和部署成本高
4. **安全對齊**：確保模型行為符合人類價值觀
5. **上下文長度**：長文本處理能力
6. **多語言平衡**：非英語能力落後

### 未來發展方向

1. **多模態整合**：文字、圖像、音頻、影片統一處理
2. **長上下文**：百萬級 token 上下文
3. **工具使用**：自動使用外部工具和 API
4. **自主代理**：自主規劃和執行任務
5. **高效模型**：更小的模型，更好的性能
6. **持續學習**：在運行中學習新知識

# 生成對抗網路 (GAN)

## 概述

生成對抗網路（Generative Adversarial Networks，簡稱 GAN）是由 Ian Goodfellow 等人於 2014 年提出的深度學習架構，被譽為近年來最重要的生成模型之一。GAN 的核心思想源自博弈論中的「零和遊戲」，透過兩個神經網路相互對抗的方式，使生成器能夠學習產生高度逼真的資料。

GAN 的出現徹底改變了生成式 AI 的發展方向，使得合成圖像、音訊、影片等任務達到了前所未有的品質水平。從藝術創作到醫學影像，從遊戲開發到資料增強，GAN 的應用範圍極為廣泛。

## 歷史背景

### 起源與發展

```
2014 - GAN 誕生
    Ian Goodfellow 在論文 "Generative Adversarial Nets" 中首次提出
    奠定生成對抗的理論基礎
    
2015 - DCGAN
    Deep Convolutional GAN
    首次將卷積網路應用於 GAN，大幅提升圖像品質
    
2016 - Conditional GAN
    加入條件約束，控制生成屬性
    實現類別指定的圖像生成
    
2017 - Wasserstein GAN (WGAN)
    改進訓練穩定性，解決模式崩潰問題
    引入 Earth Mover's Distance
    
2018 - StyleGAN
    實現高品質人臉生成
    可控制生成屬性和細節層次
    
2020 - StyleGAN2/3
    進一步提升品質和解析度
    更精細的屬性控制
```

## 核心概念

### 對抗訓練機制

GAN 由兩個核心组件組成：

1. **生成器（Generator）**：負責從隨機噪音中生成假的樣本，目標是欺騙判別器使其認為生成的樣本是真實的。

2. **判別器（Discriminator）**：負責區分輸入的樣本是來自真實資料還是生成器產生的假樣本，目標是正確識別真假。

這兩個網路在訓練過程中相互競爭，形成一種「貓捉老鼠」的動態平衡：

```python
# PyTorch 實現基本 GAN 架構
import torch
import torch.nn as nn

class Generator(nn.Module):
    """生成器網路：將隨機噪音轉換為逼真樣本"""
    def __init__(self, latent_dim, img_shape):
        super().__init__()
        self.img_shape = img_shape
        
        # 逐步上採樣的生成網路
        self.model = nn.Sequential(
            # 輸入層：latent_dim -> 1024
            nn.Linear(latent_dim, 1024),
            nn.ReLU(inplace=True),
            nn.BatchNorm1d(1024),
            
            # 隱藏層：1024 -> 512
            nn.Linear(1024, 512),
            nn.ReLU(inplace=True),
            nn.BatchNorm1d(512),
            
            # 隱藏層：512 -> 256
            nn.Linear(512, 256),
            nn.ReLU(inplace=True),
            nn.BatchNorm1d(256),
            
            # 輸出層：256 -> 像素數
            nn.Linear(256, int(torch.prod(torch.tensor(img_shape)))),
            nn.Tanh()  # 輸出範圍 [-1, 1]
        )
    
    def forward(self, z):
        """前向傳播：z -> 生成圖像"""
        img = self.model(z)
        return img.view(img.size(0), *self.img_shape)


class Discriminator(nn.Module):
    """判別器網路：區分真假樣本"""
    def __init__(self, img_shape):
        super().__init__()
        
        # 逐步下採樣的判別網路
        self.model = nn.Sequential(
            # 輸入層：像素數 -> 256
            nn.Linear(int(torch.prod(torch.tensor(img_shape))), 256),
            nn.LeakyReLU(0.2, inplace=True),
            nn.Dropout(0.3),
            
            # 隱藏層：256 -> 512
            nn.Linear(256, 512),
            nn.LeakyReLU(0.2, inplace=True),
            nn.Dropout(0.3),
            
            # 隱藏層：512 -> 1024
            nn.Linear(512, 1024),
            nn/LeakyReLU(0.2, inplace=True),
            nn.Dropout(0.3),
            
            # 輸出層：1024 -> 1
            nn.Linear(1024, 1),
            nn.Sigmoid()  # 輸出機率 [0, 1]
        )
    
    def forward(self, img):
        """前向傳播：圖像 -> 真實機率"""
        img_flat = img.view(img.size(0), -1)
        validity = self.model(img_flat)
        return validity
```

### 損失函數

GAN 的訓練目標可以用下面的迷你最大博弈來描述：

```python
def gan_loss(d_real, d_fake):
    """
    標準 GAN 損失函數（Binary Cross-Entropy）
    
    判別器目標：最大化 log(D(x)) + log(1 - D(G(z)))
    生成器目標：最小化 log(1 - D(G(z)))，即最大化 log(D(G(z)))
    """
    # 判別器損失
    d_loss = -torch.mean(torch.log(d_real) + torch.log(1 - d_fake))
    
    # 生成器損失
    g_loss = -torch.mean(torch.log(d_fake))
    
    return d_loss, g_loss


# 完整訓練循環
def train_gan(generator, discriminator, dataloader, epochs, latent_dim):
    optimizer_g = torch.optim.Adam(generator.parameters(), lr=0.0002, betas=(0.5, 0.999))
    optimizer_d = torch.optim.Adam(discriminator.parameters(), lr=0.0002, betas=(0.5, 0.999))
    
    for epoch in range(epochs):
        for batch in dataloader:
            real_images = batch
            batch_size = real_images.size(0)
            
            # 真實標籤為 1，假資料標籤為 0
            real_labels = torch.ones(batch_size, 1)
            fake_labels = torch.zeros(batch_size, 1)
            
            # ===== 訓練判別器 =====
            optimizer_d.zero_grad()
            
            # 對真實圖像進行分類
            d_real = discriminator(real_images)
            d_loss_real = nn.BCELoss()(d_real, real_labels)
            
            # 生成假圖像
            noise = torch.randn(batch_size, latent_dim)
            fake_images = generator(noise)
            
            # 對假圖像進行分類
            d_fake = discriminator(fake_images.detach())
            d_loss_fake = nn.BCELoss()(d_fake, fake_labels)
            
            # 判別器總損失
            d_loss = d_loss_real + d_loss_fake
            d_loss.backward()
            optimizer_d.step()
            
            # ===== 訓練生成器 =====
            optimizer_g.zero_grad()
            
            # 生成假圖像並計算損失
            noise = torch.randn(batch_size, latent_dim)
            fake_images = generator(noise)
            d_fake = discriminator(fake_images)
            
            # 生成器希望判別器認為生成的圖像是真的
            g_loss = nn.BCELoss()(d_fake, real_labels)
            g_loss.backward()
            optimizer_g.step()
            
            if epoch % 100 == 0:
                print(f"Epoch {epoch}: D_loss={d_loss:.4f}, G_loss={g_loss:.4f}")
```

## GAN 變體

### DCGAN（Deep Convolutional GAN）

將卷積網路應於 GAN，是首個成功生成高品質圖像的架構：

```python
class DCGANGenerator(nn.Module):
    """DCGAN 生成器使用轉置卷積進行上採樣"""
    def __init__(self, latent_dim, channels):
        super().__init__()
        
        self.init_size = 4  # 初始 feature map 大小
        self.lfc = nn.Linear(latent_dim, 512 * self.init_size ** 2)
        
        self.conv_blocks = nn.Sequential(
            # 4x4 -> 8x8
            nn.BatchNorm2d(512),
            nn.ReLU(inplace=True),
            nn.ConvTranspose2d(512, 256, 4, 2, 1),
            
            # 8x8 -> 16x16
            nn.BatchNorm2d(256),
            nn.ReLU(inplace=True),
            nn.ConvTranspose2d(256, 128, 4, 2, 1),
            
            # 16x16 -> 32x32
            nn.BatchNorm2d(128),
            nn.ReLU(inplace=True),
            nn.ConvTranspose2d(128, 64, 4, 2, 1),
            
            # 32x32 -> 64x64
            nn.BatchNorm2d(64),
            nn.ReLU(inplace=True),
            nn.ConvTranspose2d(64, channels, 4, 2, 1),
            
            nn.Tanh()
        )
    
    def forward(self, z):
        out = self.lfc(z)
        out = out.view(out.size(0), 512, self.init_size, self.init_size)
        img = self.conv_blocks(out)
        return img
```

### Conditional GAN（cGAN）

加入條件資訊，實現可控生成：

```python
class ConditionalGenerator(nn.Module):
    """條件生成器：同時輸入噪音和類別標籤"""
    def __init__(self, latent_dim, num_classes, img_channels, img_size):
        super().__init__()
        
        self.label_emb = nn.Embedding(num_classes, latent_dim)
        
        self.init_size = img_size // 8
        
        # 合併噪音和類別嵌入
        self.model = nn.Sequential(
            nn.Linear(latent_dim * 2, 128 * self.init_size ** 2),
            nn.LeakyReLU(0.2),
            nn.BatchNorm1d(128 * self.init_size ** 2),
            
            nn.ConvTranspose2d(128, 64, 4, 2, 1),
            nn.BatchNorm2d(64),
            nn.LeakyReLU(0.2),
            
            nn.ConvTranspose2d(64, img_channels, 4, 2, 1),
            nn.Tanh()
        )
    
    def forward(self, noise, labels):
        # 類別嵌入
        label_emb = self.label_emb(labels)
        # 合併噪音和類別
        gen_input = torch.cat([noise, label_emb], dim=1)
        img = self.model(gen_input)
        return img
```

### Wasserstein GAN（WGAN）

使用 Earth Mover's Distance 替代 KL 散度，訓練更穩定：

```python
class WGANGenerator(nn.Module):
    """WGAN 生成器：移除了 Sigmoid"""
    def __init__(self, latent_dim, img_shape):
        super().__init__()
        self.model = nn.Sequential(
            nn.Linear(latent_dim, 256),
            nn.ReLU(inplace=True),
            nn.Linear(256, 512),
            nn.ReLU(inplace=True),
            nn.Linear(512, 1024),
            nn.ReLU(inplace=True),
            nn.Linear(1024, int(torch.prod(torch.tensor(img_shape)))),
            # 移除 Sigmoid
        )
        self.img_shape = img_shape
    
    def forward(self, z):
        img = self.model(z)
        return img.view(img.size(0), *self.img_shape)


class WGANDiscriminator(nn.Module):
    """WGAN 判別器：輸出不受限的得分"""
    def __init__(self, img_shape):
        self.model = nn.Sequential(
            nn.Linear(int(torch.prod(torch.tensor(img_shape))), 512),
            nn.LeakyReLU(0.2),
            nn.Linear(512, 256),
            nn.LeakyReLU(0.2),
            nn.Linear(256, 1),
            # 移除 Sigmoid
        )
    
    def forward(self, img):
        img_flat = img.view(img.size(0), -1)
        validity = self.model(img_flat)
        return validity


def wgan_training_step(generator, discriminator, real_data, latent_dim):
    """WGAN 訓練步驟：使用 Gradient Penalty"""
    batch_size = real_data.size(0)
    
    # 訓練判別器（多次迭代）
    for _ in range(5):
        noise = torch.randn(batch_size, latent_dim)
        fake_data = generator(noise)
        
        # 計算 W 距離
        d_real = discriminator(real_data)
        d_fake = discriminator(fake_data)
        d_loss = -(torch.mean(d_real) - torch.mean(d_fake))
        
        # Gradient Penalty
        alpha = torch.rand(batch_size, 1, 1, 1)
        interpolated = alpha * real_data + (1 - alpha) * fake_data
        interpolated.requires_grad = True
        d_inter = discriminator(interpolated)
        
        gradients = torch.autograd.grad(
            outputs=d_inter,
            inputs=interpolated,
            grad_outputs=torch.ones(d_inter.size()),
            create_graph=True
        )[0]
        
        gp_loss = ((gradients.norm(2, dim=1) - 1) ** 2).mean()
        d_loss += 10 * gp_loss
        
        d_loss.backward()
        optimizer_d.step()
        clip_weights(discriminator, 0.01)
    
    # 訓練生成器
    noise = torch.randn(batch_size, latent_dim)
    fake_data = generator(noise)
    g_loss = -torch.mean(discriminator(fake_data))
    g_loss.backward()
    optimizer_g.step()
```

### StyleGAN

引入了映射網路和自適應實例正則化（AdaIN），實現細節控制：

```python
class StyleGANGenerator(nn.Module):
    """StyleGAN 生成器：精細控制屬性"""
    def __init__(self, latent_dim, w_dim, img_channels):
        super().__init__()
        
        # 映射網路：latent code -> w space
        self.mapping = nn.Sequential(
            nn.Linear(latent_dim, w_dim),
            nn.ReLU(),
            nn.Linear(w_dim, w_dim),
            nn.ReLU(),
            nn.Linear(w_dim, w_dim),
        )
        
        # 合成網路
        self.synthesis = SynthesisNetwork(w_dim, img_channels)
    
    def forward(self, z, styles=None):
        # 映射到 w 空間
        w = self.mapping(z)
        
        if styles is not None:
            w = styles
            
        # 生成圖像
        img = self.synthesis(w)
        return img


class SynthesisBlock(nn.Module):
    """StyleGAN 合成模組：AdaIN + 卷積"""
    def __init__(self, in_channels, out_channels, w_dim):
        super().__init__()
        self.conv = nn.Conv2d(in_channels, out_channels, 3, 1, 1)
        self.adaIN = AdaptiveInstanceNorm(w_dim, out_channels)
    
    def forward(self, x, w):
        x = self.conv(x)
        x = self.adaIN(x, w)
        x = nn.LeakyReLU(0.2)(x)
        return x
```

## 訓練技巧與挑戰

### 模式崩潰（Mode Collapse）

生成器只生成少數幾種樣本，缺乏多樣性：

```python
# 解決方法：小批次多樣性損失
def minibatch_diversity(generated_images):
    """計算生成圖像間的距離，鼓勵多樣性"""
    # 計算特徵向量
    features = encoder(generated_images)
    # 計算兩兩之間的距離
    dist_matrix = torch.cdist(features, features)
    # 最大化平均距離
    diversity = dist_matrix.mean()
    return diversity
```

### 訓練不穩定

解決方法包括：
- 使用標籤平滑（Label Smoothing）
- 批次正則化（Batch Normalization）
- 學習率調度
- 使用 TTUR（Two Time-Scale Update Rule）

### 評估指標

```python
def calculate_fid(real_features, fake_features):
    """Frechet Inception Distance"""
    mu1, sigma1 = real_features.mean(dim=0), torch.cov(real_features)
    mu2, sigma2 = fake_features.mean(dim=0), torch.cov(fake_features)
    
    diff = mu1 - mu2
    covmean = torch.sqrt(sigma1 * sigma2)
    
    fid = diff.dot(diff) + torch.trace(sigma1 + sigma2 - 2 * covmean)
    return fid


def calculate_inception_score(predictions, num_splits=10):
    """Inception Score"""
    scores = []
    for i in range(num_splits):
        part = predictions[i::num_splits]
        kl = part * (torch.log(part) - torch.log(part.mean(dim=0, keepdim=True)))
        kl = kl.sum(dim=1).mean()
        scores.append(torch.exp(kl))
    return torch.stack(scores).mean()
```

### 風格轉換

```python
# CycleGAN：無配對的風格轉換
class CycleGAN:
    """
    使用雙向 GAN 實現風格轉換
    X -> Y -> X' 應該與 X 相似
    Y -> X -> Y' 應該與 Y 相似
    """
    def __init__(self):
        self.G_XtoY = Generator()  # X -> Y
        self.G_YtoX = Generator()  # Y -> X
        self.D_X = Discriminator()  # 判別 X
        self.D_Y = Discriminator()  # 判別 Y
    
    def cycle_consistency_loss(self, x, y):
        # X -> Y -> X'
        fake_y = self.G_XtoY(x)
        recovered_x = self.G_YtoX(fake_y)
        loss_x = nn.L1Loss()(recovered_x, x)
        
        # Y -> X -> Y'
        fake_x = self.G_YtoX(y)
        recovered_y = self.G_XtoY(fake_x)
        loss_y = nn.L1Loss()(recovered_y, y)
        
        return loss_x + loss_y
```

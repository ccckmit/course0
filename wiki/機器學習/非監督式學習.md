# 非監督式學習

非監督式學習（Unsupervised Learning）是機器學習中不使用標籤資料進行訓練的學習範式。在非監督式學習中，演算法需要自動發現資料中的規律、模式和結構。由於不需要人類標註標籤，非監督式學習可以處理大規模未標記的資料，這在現實世界中非常常見，因為收集和標記資料往往需要大量的人力資源。非監督式學習的核心任務是探索資料的內在結構，找出隱藏的模式，這使得它在資料探索、異常檢測、資料壓縮等場景中非常有用。

## 非監督式學習的基本概念

### 與監督式學習的比較

```
┌─────────────────────────────────────────────────────────────────┐
│                  監督式 vs 非監督式學習                         │
├─────────────────────────────────────────────────────────────────┤
│                                                              │
│  監督式學習：                                                  │
│  • 需要標籤資料 (x, y)                                        │
│  • 學習輸入到輸出的映射                                       │
│  • 需要「老師」指導                                           │
│  • 例：分類、回歸                                             │
│                                                              │
│  非監督式學習：                                                │
│  • 不需要標籤資料 x                                           │
│  • 學習資料的內在結構                                         │
│  • 自主探索發現                                               │
│  例：聚類、降維、關聯規則                                     │
│                                                              │
└─────────────────────────────────────────────────────────────────┘
```

## 主要類型

非監督式學習主要分為三大類型：聚類、降維和關聯規則學習。每種類型解決不同的問題。

### 聚類（Clustering）

聚類是將相似的資料點分組的任務，同一組內的資料點彼此相似，不同組之間差異較大：

```
┌─────────────────────────────────────────────────────────────────┐
│                        聚類範例                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                              │
│  • 客戶細分：將客戶按購買行為分群                              │
│  • 文件分類：將新聞文章按主題分組                              │
│  • 影像分割：將圖像按區域分組                                  │
│  • 異常檢測：將正常資料點分群，異常點單獨                      │
│                                                              │
└─────────────────────────────────────────────────────────────────┘
```

### 降維（Dimensionality Reduction）

降維將高維度資料投影到低維度空間，同時盡量保留原始資料的重要資訊：

```
┌─────────────────────────────────────────────────────────────────┐
│                        降維範例                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                              │
│  • 視覺化：將高維資料降到 2-3 維進行視覺化                      │
│  • 加速：減少特徵數量以加速模型訓練                            │
│  • 去除雜訊：移除不重要或冗餘的特徵                            │
│  • 緩解維度災難：高維資料的稀疏問題                            │
│                                                              │
└─────────────────────────────────────────────────────────────────┘
```

### 關聯規則學習（Association Rule Learning）

關聯規則學習發現資料項目之間有趣的非監督式學習：

```

├─────────────────────────────────────────────────────────────────┤

│  • 網站行為分析：                                            │

│                                                              │

```

## 聚類演算法

### K-Means

K-Means 是最經典的聚類演算法之一，將資料劃分為 K 個群組：

```python
from sklearn.cluster import KMeans
import numpy as np

# K-Means 聚類
kmeans = KMeans(
    n_clusters=3,          # 群組數量
    init='k-means++',      # 初始化方法
    n_init=10,            # 初始化次數
    max_iter=300,         # 最大迭代次數
    random_state=42
)

kmeans.fit(X)

# 取得標籤
labels = kmeans.labels_

# 取得中心點
centers = kmeans.cluster_centers_

# 預測新資料
new_labels = kmeans.predict(new_X)
```

### K-Means 運作原理

```python
# K-Means 演算法步驟：
# 1. 隨機選擇 K 個中心點
# 2. 計算每個點到各中心的距離，分配到最近的中心
# 3. 重新計算每個群組的中心點
# 4. 重複步驟 2-3 直到收斂

# 找最佳 K 值（肘部法）
inertias = []
K_range = range(1, 11)
for k in K_range:
    km = KMeans(n_clusters=k, random_state=42)
    km.fit(X)
    inertias.append(km.inertia_)

# 選擇肘部位置的 K 值
```

### DBSCAN

DBSCAN（Density-Based Spatial Clustering of Applications with Noise）是一種基於密度的聚類方法：

```python
from sklearn.cluster import DBSCAN

# DBSCAN 聚類
dbscan = DBSCAN(
    eps=0.5,              # 鄰域半徑
    min_samples=5,       # 核心點最小鄰居數
    metric='euclidean'   # 距離度量
)

labels = dbscan.fit_predict(X)

# labels 中的 -1 表示噪聲點
```

### DBSCAN 優點

```
DBSCAN 特色：
• 不需要指定群組數量
• 可以識別任意形狀的群組
• 可以識別噪聲點
• 對離群值魯棒
```

### 階層式聚類

階層式聚類建立一個樹狀結構（ dendrogram）表示資料的層次關係：

```python
from sklearn.cluster import AgglomerativeClustering

# 階層式聚類
agg = AgglomerativeClustering(
    n_clusters=3,
    linkage='ward'       # 連接方法: 'ward', 'complete', 'average'
)

labels = agg.fit_predict(X)
```

### 連接方法比較

| 方法 | 說明 | 適用場景 |
|------|------|----------|
| Ward | 最小化方差增量 | 緊湊球形群組 |
| Complete | 最小化最大距離 | 距離所有點最遠 |
| Average | 最小化平均距離 | 一般用途 |

### 高斯混合模型

高斯混合模型（GMM）是一種機率模型，假設資料由多個高斯分布混合而成：

```python
from sklearn.mixture import GaussianMixture

# GMM 聚類
gmm = GaussianMixture(
    n_components=3,
    covariance_type='full',  # 'full', 'tied', 'diag', 'spherical'
    random_state=42
)

gmm.fit(X)

# 取得標籤
labels = gmm.predict(X)

# 取得機率
probas = gmm.predict_proba(X)
```

## 降維演算法

### PCA（主成分分析）

PCA 將資料投影到正交的新座標系統，保留最大變異量：

```python
from sklearn.decomposition import PCA

# PCA 降維
pca = PCA(n_components=2)       # 降到 2 維
X_pca = pca.fit_transform(X)

# 取得解釋變異量比例
print(f"解釋變異量比例: {pca.explained_variance_ratio_}")
print(f"累積解釋變異量: {sum(pca.explained_variance_ratio_)}")

# 自動選擇維度（保留 95% 變異量）
pca = PCA(n_components=0.95)
X_reduced = pca.fit_transform(X)
```

### t-SNE

t-SNE（t-Distributed Stochastic Neighbor Embedding）特別適合視覺化：

```python
from sklearn.manifold import TSNE

# t-SNE 降維（常用於視覺化）
tsne = TSNE(
    n_components=2,        # 降到 2 維
    perplexity=30,         # 鄰居數
    learning_rate='auto',  # 學習率
    n_iter=1000,          # 迭代次數
    random_state=42
)

X_tsne = tsne.fit_transform(X)
```

### t-SNE 特點

```
t-SNE 特色：
• 保留局部結構
• 適合高維資料視覺化
• 計算較慢
• 隨機性高，每次結果不同
• 不適合保持全局結構
```

### UMAP

UMAP（Uniform Manifold Approximation and Projection）是一種較新的降維方法：

```python
import umap.umap_ as umap

# UMAP 降維
reducer = umap.UMAP(
    n_components=2,
    n_neighbors=15,
    min_dist=0.1,
    random_state=42
)

X_umap = reducer.fit_transform(X)
```

### UMAP 優點

```
UMAP 特色：
• 保留全域和局部結構
• 比 t-SNE 快
• 結果更穩定
• 可用於監督式學習的特徵增強
```

## 關聯規則學習

### Apriori 演算法

Apriori 是經典的關聯規則演算法：

```python
# 安裝 mlxtend
# pip install mlxtend

from mlxtend.frequent_patterns import apriori, association_rules

# 轉換為交易格式
transactions = [
    ['牛奶', '麵包', '雞蛋'],
    ['牛奶', '麵包'],
    ['麵包', '雞蛋'],
    ['牛奶', '雞蛋'],
]

from mlxtend.preprocessing import TransactionEncoder
te = TransactionEncoder()
te_array = te.fit_transform(transactions)
df = pd.DataFrame(te_array, columns=te.columns_)

# 找頻繁項目集
frequent_itemsets = apriori(df, min_support=0.5)

# 產生關聯規則
rules = association_rules(frequent_itemsets, metric="confidence", min_threshold=0.7)
```

### 關鍵指標

```
關聯規則指標：
• 支援度 (Support): 項目出現的頻率
  sup(X) = count(X) / N

• 信心度 (Confidence): 規則正確率
  conf(X → Y) = sup(X∪Y) / sup(X)

• 提升度 (Lift): 規則相關性
  lift(X → Y) = conf(X → Y) / sup(Y)
```

## 評估方法

### 聚類評估

```python
from sklearn.metrics import (
    silhouette_score,      # 輪廓系數
    calinski_harabasz_score, # CH 指數
    davies_bouldin_score    # DB 指數
)

# 輪廓系數（-1 到 1）
silhouette = silhouette_score(X, labels)

# CH 指數（越高越好）
ch_score = calinski_harabasz_score(X, labels)

# DB 指數（越低越好）
db_score = davies_bouldin_score(X, labels)
```

### 無標籤評估

對於沒有真實標籤的情況，可以使用：

```python
# 群組內凝聚度
inertia = kmeans.inertia_

# Davies-Bouldin 指數
db_score = davies_bouldin_score(X, labels)
```

## 實際應用

### 客戶細分

```python
# 客戶資料分析
# 假設有購買金額、購買頻率、年齡等特徵
customer_features = df[['purchase_amount', 'purchase_frequency', 'age']]

# 標準化
from sklearn.preprocessing import StandardScaler
scaler = StandardScaler()
X_scaled = scaler.fit_transform(customer_features)

# 聚類
kmeans = KMeans(n_clusters=4, random_state=42)
df['segment'] = kmeans.fit_predict(X_scaled)

# 分析每個群組
for i in range(4):
    segment = df[df['segment'] == i]
    print(f"群組 {i}: {len(segment)} 客户")
    print(segment.mean())
```

### 異常檢測

```python
# 使用 DBSCAN 識別異常
dbscan = DBSCAN(eps=0.5, min_samples=5)
labels = dbscan.fit_predict(X)

# 標記為 -1 的點是異常
anomalies = X[labels == -1]

# 或使用 GMM
gmm = GaussianMixture(n_components=2, random_state=42)
gmm.fit(X)

# 機率低於閾值的點為異常
probas = gmm.score_samples(X)
threshold = np.percentile(probas, 5)  # 最低 5%
anomalies = X[probas < threshold]
```

### 維度減少

```python
# 降維後視覺化
pca = PCA(n_components=2)
X_2d = pca.fit_transform(X)

import matplotlib.pyplot as plt
plt.scatter(X_2d[:, 0], X_2d[:, 1])
plt.show()

# 為監督式學習準備特徵
pca = PCA(n_components=50)
X_reduced = pca.fit_transform(X)

# 然後使用 X_reduced 訓練監督式模型
```

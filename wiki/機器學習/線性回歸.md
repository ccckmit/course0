# 線性回歸 (Linear Regression)

## 概述

線性回歸是機器學習中最基本且廣泛使用的監督式學習演算法，用於建立輸入特徵與連續目標變數之間的線性關係。線性回歸的歷史可追溯到 19 世紀，當時 Francis Galton 研究父子身高關係時發現了「回歸」現象。作為機器學習的入門算法，線性回歸不僅可以直接應用於許多實際問題，也是理解更複雜模型的基礎。

線性回歸的核心假設是目標變數與特徵之間存在線性組合關係，這種簡單的假設使得模型具有很好的可解釋性，也為我們提供了對機器學習更深入理解的起點。

## 數學基礎

### 簡單線性回歸

簡單線性回歸處理單一輸入變數的場景，其數學表示為：

$$y = wx + b$$

其中 $y$ 是預測值（目標變數），$x$ 是輸入特徵，$w$ 是權重（斜率），$b$ 是偏差（截距）。模型的目标是找到最優的 $w$ 和 $b$ 使得預測值與真實值之間的誤差最小。

### 多元線性回歸

當有多個輸入特徵時，扩展为多元線性回歸：

$$y = w_1 x_1 + w_2 x_2 + \cdots + w_n x_n + b$$

用矩陣表示：

$$y = \mathbf{X} \mathbf{w} + b$$

其中 $\mathbf{X} \in \mathbb{R}^{m \times n}$ 是 $m$ 個樣本、$n$ 個特徵的矩陣，$\mathbf{w} \in \mathbb{R}^n$ 是權重向量，$b$ 是偏差。

### 向量化表示

```python
import numpy as np

class LinearRegression:
    """向量化實現的線性回歸"""
    
    def __init__(self):
        self.weights = None
        self.bias = None
    
    def fit(self, X, y):
        """
        使用正規方程式擬合模型
        X: (n_samples, n_features)
        y: (n_samples,)
        """
        n_samples, n_features = X.shape
        
        # 添加偏置列
        X_b = np.c_[np.ones((n_samples, 1)), X]
        
        # 正規方程式: θ = (X^T X)^(-1) X^T y
        # 使用 numpy 實現
        theta = np.linalg.inv(X_b.T @ X_b) @ X_b.T @ y
        
        self.bias = theta[0]
        self.weights = theta[1:]
        
        return self
    
    def predict(self, X):
        """預測"""
        return X @ self.weights + self.bias
```

## 損失函數

### 均方誤差 (MSE)

線性回歸使用均方誤差（Mean Squared Error，MSE）作為損失函數：

$$MSE = \frac{1}{m} \sum_{i=1}^{m} (y_i - \hat{y}_i)^2$$

MSE 的優點包括處處可微、對大誤差給予更大懲罰等特性。

```python
def mse(y_true, y_pred):
    """計算均方誤差"""
    return np.mean((y_true - y_pred) ** 2)

def rmse(y_true, y_pred):
    """計算均方根誤差"""
    return np.sqrt(mse(y_true, y_pred))

def mae(y_true, y_pred):
    """計算平均絕對誤差"""
    return np.mean(np.abs(y_true - y_pred))
```

### 平均絕對誤差 (MAE)

對於對離群值更魯棒的场景，可以使用 MAE：

$$MAE = \frac{1}{m} \sum_{i=1}^{m} |y_i - \hat{y}_i|$$

### R² 分數

R²（決定係數）用於衡量模型解釋變異的能力：

$$R^2 = 1 - \frac{\sum(y_i - \hat{y}_i)^2}{\sum(y_i - \bar{y})^2}$$

其中 $\bar{y}$ 是目標變數的平均值。

```python
def r2_score(y_true, y_pred):
    """計算 R² 分數"""
    ss_res = np.sum((y_true - y_pred) ** 2)
    ss_tot = np.sum((y_true - np.mean(y_true)) ** 2)
    return 1 - ss_res / ss_tot
```

## 參數估計方法

### 正規方程式 (Closed-form Solution)

當特徵數量不大時，可以使用正規方程式直接求解：

$$\mathbf{\theta} = (\mathbf{X}^T \mathbf{X})^{-1} \mathbf{X}^T \mathbf{y}$$

```python
class LinearRegressionClosedForm:
    """正規方程式實現"""
    
    def fit(self, X, y):
        # 添加偏置項
        X_with_bias = np.column_stack([np.ones(len(X)), X])
        
        # 計算正規方程式
        self.theta = np.linalg.inv(X_with_bias.T @ X_with_bias) @ X_with_bias.T @ y
        
        self.bias = self.theta[0]
        self.weights = self.theta[1:]
        
        return self
    
    def predict(self, X):
        return X @ self.weights + self.bias
```

正規方程式的優點是直接求解、一次計算；缺點是矩陣逆運算複雜度為 $O(n^3)$，不適合大規模特徵。

### 梯度下降法

對於大規模資料，使用梯度下降法進行迭代優化：

```python
class LinearRegressionGD:
    """梯度下降實現"""
    
    def __init__(self, learning_rate=0.01, n_iterations=1000):
        self.lr = learning_rate
        self.n_iterations = n_iterations
        self.weights = None
        self.bias = None
    
    def fit(self, X, y):
        n_samples, n_features = X.shape
        
        # 初始化參數
        self.weights = np.zeros(n_features)
        self.bias = 0
        
        # 梯度下降迭代
        for _ in range(self.n_iterations):
            # 預測
            y_pred = X @ self.weights + self.bias
            
            # 計算梯度
            dw = (1 / n_samples) * X.T @ (y_pred - y)
            db = (1 / n_samples) * np.sum(y_pred - y)
            
            # 更新參數
            self.weights -= self.lr * dw
            self.bias -= self.lr * db
        
        return self
    
    def predict(self, X):
        return X @ self.weights + self.bias
```

梯度下降的更新規則：

$$w_j = w_j - \alpha \frac{\partial}{\partial w_j} MSE$$

$$b = b - \alpha \frac{\partial}{\partial b} MSE$$

其中 $\alpha$ 是學習率。

### 隨機梯度下降 (SGD)

```python
class LinearRegressionSGD:
    """隨機梯度下降實現"""
    
    def __init__(self, learning_rate=0.01, n_epochs=100, batch_size=32):
        self.lr = learning_rate
        self.n_epochs = n_epochs
        self.batch_size = batch_size
        self.weights = None
        self.bias = None
    
    def fit(self, X, y):
        n_samples, n_features = X.shape
        self.weights = np.zeros(n_features)
        self.bias = 0
        
        for epoch in range(self.n_epochs):
            # 隨機打亂資料
            indices = np.random.permutation(n_samples)
            X_shuffled = X[indices]
            y_shuffled = y[indices]
            
            # 小批量訓練
            for i in range(0, n_samples, self.batch_size):
                X_batch = X_shuffled[i:i+self.batch_size]
                y_batch = y_shuffled[i:i+self.batch_size]
                
                y_pred = X_batch @ self.weights + self.bias
                
                dw = (1 / len(X_batch)) * X_batch.T @ (y_pred - y_batch)
                db = (1 / len(X_batch)) * np.sum(y_pred - y_batch)
                
                self.weights -= self.lr * dw
                self.bias -= self.lr * db
        
        return self
```

## 正則化

### Ridge 回歸 (L2 正則化)

Ridge 回歸在損失函數中添加 L2 正則化項：

$$L_{ridge} = \sum_{i=1}^{m} (y_i - \hat{y}_i)^2 + \lambda \sum_{j=1}^{n} w_j^2$$

```python
class RidgeRegression:
    """Ridge 回歸實現"""
    
    def __init__(self, alpha=1.0):
        self.alpha = alpha
        self.weights = None
        self.bias = None
    
    def fit(self, X, y):
        n_samples, n_features = X.shape
        
        # 正規方程式 with L2 正則化
        # θ = (X^T X + λI)^(-1) X^T y
        X_b = np.c_[np.ones((n_samples, 1)), X]
        
        # 添加正則化項（除了偏置）
        reg = self.alpha * np.eye(n_features + 1)
        reg[0, 0] = 0  # 不正則化偏置
        
        theta = np.linalg.inv(X_b.T @ X_b + reg) @ X_b.T @ y
        
        self.bias = theta[0]
        self.weights = theta[1:]
        
        return self
```

### Lasso 回歸 (L1 正則化)

Lasso 使用 L1 正則化，可以實現特徵選擇：

$$L_{lasso} = \sum_{i=1}^{m} (y_i - \hat{y}_i)^2 + \lambda \sum_{j=1}^{n} |w_j|$$

```python
def soft_threshold(x, threshold):
    """軟閾值函數 - Lasso 的近端梯度"""
    return np.sign(x) * np.maximum(np.abs(x) - threshold, 0)

class LassoRegression:
    """Lasso 回歸實現 - 使用座標下降法"""
    
    def __init__(self, alpha=1.0, n_iterations=1000):
        self.alpha = alpha
        self.n_iterations = n_iterations
        self.weights = None
        self.bias = None
    
    def fit(self, X, y):
        n_samples, n_features = X.shape
        
        # 先用最小二乘法估計偏置
        self.weights = np.zeros(n_features)
        
        # 中心化 y
        y_mean = np.mean(y)
        y_centered = y - y_mean
        
        # 座標下降優化權重
        for _ in range(self.n_iterations):
            for j in range(n_features):
                # 計算殘差
                residual = y_centered - X @ self.weights + X[:, j] * self.weights[j]
                
                # 更新權重（軟閾值）
                rho_j = np.dot(X[:, j], residual)
                self.weights[j] = soft_threshold(rho_j, self.alpha) / (np.dot(X[:, j], X[:, j]) + 1e-8)
        
        # 計算偏置
        self.bias = y_mean - np.dot(X, self.weights)
        
        return self
```

### Elastic Net

結合 L1 和 L2 正則化：

$$L_{elastic} = \sum_{i=1}^{m} (y_i - \hat{y}_i)^2 + \lambda (\rho \sum |w_j| + \frac{1-\rho}{2} \sum w_j^2)$$

## 實際應用

### 房價預測

```python
import numpy as np
from sklearn.model_selection import train_test_split

def house_price_prediction():
    """房價預測示例"""
    # 模擬資料：房屋面積、臥室數量、年齡 -> 價格
    np.random.seed(42)
    n_samples = 1000
    
    area = np.random.uniform(500, 5000, n_samples)
    bedrooms = np.random.randint(1, 6, n_samples)
    age = np.random.randint(0, 50, n_samples)
    
    # 價格 = 200 * area + 10000 * bedrooms - 500 * age + 50000 + noise
    price = (200 * area + 10000 * bedrooms - 500 * age + 50000 + 
             np.random.normal(0, 10000, n_samples))
    
    X = np.column_stack([area, bedrooms, age])
    y = price
    
    # 分割資料
    X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)
    
    # 訓練模型
    model = LinearRegression()
    model.fit(X_train, y_train)
    
    # 預測
    y_pred = model.predict(X_test)
    
    # 評估
    from sklearn.metrics import r2_score, mean_squared_error
    print(f"R²: {r2_score(y_test, y_pred):.4f}")
    print(f"RMSE: {np.sqrt(mean_squared_error(y_test, y_pred)):.2f}")
    print(f"權重: {model.weights}")
    print(f"偏置: {model.bias:.2f}")
```

### 時間序列預測

```python
class TimeSeriesLinearRegression:
    """時間序列預測 - 使用滑動窗口"""
    
    def __init__(self, window_size=7):
        self.window_size = window_size
        self.model = LinearRegression()
    
    def create_dataset(self, data):
        """創建滑動窗口資料集"""
        X, y = [], []
        for i in range(len(data) - self.window_size):
            X.append(data[i:i+self.window_size])
            y.append(data[i+self.window_size])
        return np.array(X), np.array(y)
    
    def fit(self, data):
        X, y = self.create_dataset(data)
        self.model.fit(X, y)
        return self
    
    def predict(self, data):
        X, _ = self.create_dataset(data)
        return self.model.predict(X)
```

### 多重共線性檢測

```python
def check_multicollinearity(X):
    """檢查多重共線性 - 使用 VIF"""
    from scipy import stats
    
    vif = []
    for i in range(X.shape[1]):
        X_others = np.delete(X, i, axis=1)
        r_squared = stats.pearsonr(X[:, i], X_others[:, 0])[0] ** 2
        vif.append(1 / (1 - r_squared))
    
    return vif
```

## 擴展與變體

### 多項式回歸

```python
class PolynomialRegression:
    """多項式回歸"""
    
    def __init__(self, degree=2):
        self.degree = degree
        self.model = LinearRegression()
    
    def transform(self, X):
        """多項式特徵擴展"""
        X_poly = []
        for i in range(self.degree + 1):
            X_poly.append(X ** i)
        return np.column_stack(X_poly)
    
    def fit(self, X, y):
        X_poly = self.transform(X)
        self.model.fit(X_poly, y)
        return self
    
    def predict(self, X):
        X_poly = self.transform(X)
        return self.model.predict(X_poly)
```

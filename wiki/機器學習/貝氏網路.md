# 貝氏網路

## 概述

貝氏網路（Bayesian Network），又稱信念網路（Belief Network）或因果網路（Causal Network），是一種用於表示隨機變量之間條件獨立關係的有向無環圖（DAG）模型。由 Judea Pearl 在 1980 年代中期提出，貝氏網路結合了圖論和概率論，提供了一種直觀且計算上可行的方式來表示和推理複雜的聯合概率分佈。

貝氏網路的核心思想是：利用變量之間的條件獨立性，將高維聯合概率分佈分解為多個低維條件概率分佈的乘積。這種分解大大簡化了表示、學習和推論的複雜度。

## 形式化定義

### 圖結構

貝氏網路由以下組成：
1. **節點（Nodes）**：每個節點代表一個隨機變量 X_i
2. **有向邊（Edges）**：連接節點，表示變量之間的依賴關係
3. **有向無環圖（DAG）**：確保網路沒有有向環，形成層次結構

如果存在從節點 i 到節點 j 的有向邊，則稱節點 i 是節點 j 的父節點（parent），節點 j 是節點 i 的子節點（child）。

### 條件概率表（CPT）

每個節點 X 關聯一個條件概率表，指定：

$$P(X | \text{Parents}(X))$$

對於沒有父節點的節點，則只有邊緣概率 P(X)。

### 聯合分佈分解

給定圖結構 G 和 CPT，網路表示的聯合分佈為：

$$P(X_1, X_2, ..., X_n) = \prod_{i=1}^{n} P(X_i | \text{Parents}(X_i))$$

這個分解利用了條件獨立性：每個變量在給定其父節點後，條件獨立於它的非子孫節點。

## 條件獨立性

### D-分隔（D-Separation）

D-分隔是判定貝氏網路中條件獨立性的圖形化方法。

**定義**：節點集合 A 和 B 在給定節點集合 E 的情況下是 D-分隔的，如果存在一條連接 A 和 B 的路徑，使得該路徑被 E 阻斷。

**阻斷條件**：
1. 路徑上某個節點 c 是「串聯��（chain）或「分叉」（fork）節點，且 c ∈ E
2. 路徑上某個節點 c 是「碰撞」（collider）節點（即 c 的兩個父節點都有邊指向 c），且 c ∉ E

**圖形化理解**：
- **串聯節點** (A → B → C)：B 阻斷路徑
- **分叉節點** (A ← B → C)：B 阻斷路徑
- **碰撞節點** (A → B ← C)：只有 B 在條件集內才阻斷

### 全域馬可夫性

如果 A 和 B 在給定 C 時 D-分隔，則：

$$P(A, B | C) = P(A | C) \cdot P(B | C)$$

這保證了網路表示的獨立性與實際概率分佈的一致性。

## 推論算法

### 精確推論

#### 變量消元（Variable Elimination）

最基礎的精確推論算法。基本思想：按順序邊際化，避免重複計算。

對於查詢 P(X | E=e)：
1. 選擇消除順序
2. 從因子表中選擇相關因子
3. 依次邊際化不需要的變量
4. 最後剩下的因子包含 X

複雜度：取決於消除順序和網路樹寬（treewidth）。

#### 連接樹算法（Junction Tree Algorithm）

將貝氏網路轉換為樹結構的消息傳遞算法：
1. **道德化（Moralization）**：結婚父母節點，去除有向邊
2. **三角化（Triangulation）**：添加弦使圖為弦圖
3. **構建連接樹**：形成叢（clique）的樹結構
4. **消息傳遞**：在連接樹上執行信念傳播

時間複雜度：O(N^w)，其中 w 是樹寬。

#### 信念傳播（Belief Propagation）

對於樹狀結構的網路，可以在 O(N) 時間內完成推論。對於一般圖，使用連接樹或近似方法。

**正向傳遞**（由根向葉）：
$$m_{i \to j}(x_j) = \sum_{x_i} \psi_i(x_i) \prod_{k \in N(i) \setminus \{j\}} m_{k \to i}(x_i)$$

**反向傳遞**（由葉向根）：
$$m_{i \to j}(x_j) = \sum_{x_i} \psi_i(x_i) \left[ \prod_{k \in N(i) \setminus \{j\}} m_{k \to i}(x_i) \right]$$

### 近似推論

當精確推論不可行時，使用近似算法。

#### 重要性採樣（Importance Sampling）

對重要性分佈 q(X) 採樣，估計後驗概率。

權重計算：
$$w^{(m)} = \frac{P(X^{(m)}, E)}{q(X^{(m)})}$$

後驗估計：
$$\hat{P}(X | E) \approx \frac{1}{M} \sum_{m=1}^{M} w^{(m)} \delta_{X^{(m)}}(X)$$

#### 吉布斯採樣（Gibbs Sampling）

馬可夫鏈蒙特卡羅的一種特例，在條件分佈上迭代採樣。

循環採樣：
$$X_1^{(t+1)} \sim P(X_1 | X_2^{(t)}, ..., X_n^{(t)}, E)$$
$$X_2^{(t+1)} \sim P(X_2 | X_1^{(t+1)}, X_3^{(t)}, ..., E)$$
$$...$$
$$X_n^{(t+1)} \sim P(X_n | X_1^{(t+1)}, ..., X_{n-1}^{(t+1)}, E)$$

#### 變分推論（Variational Inference）

使用易於處理的分佈族 Q 近似後驗分佈 P，通過最小化 KL 散度：

$$\min_{q \in Q} KL(q(X) || P(X | E))$$

## 參數學習

### 完整數據學習

當所有變量的數據都可觀察時，學習問題相對簡單。

**最大似然估計（MLE）**：
- 每個 CPT 參數可以獨立估計
- 頻率估計：
  $$\hat{P}(X_i = x_i | \text{Parents}(X_i) = pa) = \frac{N(x_i, pa)}{N(pa)}$$

**貝氏估計（Bayesian Estimation）**：
- 引入狄利克雷先驗：
  $$\hat{P}(X_i = x_i | pa) = \frac{N(x_i, pa) + \alpha_{x_i,pa}}{N(pa) + \sum_{x'_i} \alpha_{x'_i,pa}}$$

### 不完整數據學習

當存在隱藏變量或缺失數據時，使用 EM 演算法。

**EM 步驟**：
1. **E步**：計算隱藏變量的期望
2. **M步**：使用這些期望更新參數

### 結構學習

學習網路結構比學習參數更困難。

**基於約束的方法**：
- 使用條件獨立性測試識別父子關係
- PC 算法：從完整圖開始，逐一移除條件獨立的邊

**基於分數的方法**：
- 定義網路的分數（如 BIC、AIC、貝氏分數）
- 搜索最大化分數的結構
- 貪心搜索、模擬退火、遺傳算法

**混合方法**：
- 結合約束和分數方法
- 先用約束方法識別必備結構，再優化

## 實作範例

### Python 實現基本貝氏網路

```python
import numpy as np
from typing import Dict, List, Tuple, Optional
from collections import defaultdict

class BayesianNetwork:
    def __init__(self):
        self.nodes: List[str] = []  # 節點列表
        self.parents: Dict[str, List[str]] = {}  # 父節點映射
        self.cpt: Dict[str, np.ndarray] = {}  # 條件概率表
        self.domains: Dict[str, List[int]] = {}  # 變量取值域
        self.cardinalities: Dict[str, int] = {}  # 變量基數
        
    def add_node(self, node: str, values: List[str], 
                 parents: List[str] = None):
        """添加節點"""
        self.nodes.append(node)
        self.domains[node] = list(range(len(values)))
        self.cardinalities[node] = len(values)
        self.parents[node] = parents if parents else []
        
        if parents:
            # 計算 CPT 大小
            n_parents = np.prod([self.cardinalities[p] for p in parents])
            self.cpt[node] = np.ones((n_parents, len(values)))
        else:
            self.cpt[node] = np.ones(len(values))
    
    def set_probability(self, node: str, values: np.ndarray, 
                       parent_values: List = None):
        """設置概率值"""
        if parent_values is None:
            self.cpt[node] = values / values.sum()  # 歸一化
        else:
            # 計算索引
            idx = self._compute_parent_idx(node, parent_values)
            self.cpt[node][idx] = values / values.sum()
    
    def _compute_parent_idx(self, node: str, parent_values: List) -> int:
        """計算父節點組合的索引"""
        idx = 0
        multiplier = 1
        for i, parent in enumerate(self.parents[node]):
            idx += parent_values[i] * multiplier
            multiplier *= self.cardinalities[parent]
        return idx
    
    def joint_probability(self, evidence: Dict[str, int]) -> float:
        """計算聯合概率"""
        prob = 1.0
        for node in self.nodes:
            if node in evidence:
                parent_vals = [evidence[p] for p in self.parents[node]] if self.parents[node] else []
                idx = self._compute_parent_idx(node, parent_vals) if parent_vals else 0
                prob *= self.cpt[node][idx, evidence[node]]
        return prob
    
    def marginal(self, query: str, evidence: Dict[str, int] = None) -> np.ndarray:
        """計算邊緣概率"""
        evidence = evidence or {}
        
        if query in evidence:
            result = np.zeros(self.cardinalities[query])
            result[evidence[query]] = 1.0
            return result
        
        # 枚舉所有可能的完全賦值
        result = np.zeros(self.cardinalities[query])
        
        # 構建需要枚舉的變量
        hidden = [n for n in self.nodes 
                 if n != query and n not in evidence]
        
        n_hidden = len(hidden)
        total = np.prod([self.cardinalities[n] for n in hidden]) if hidden else 1
        
        for i in range(int(total)):
            assignment = evidence.copy()
            temp = i
            for j, node in enumerate(hidden):
                val = temp % self.cardinalities[node]
                temp //= self.cardinalities[node]
                assignment[node] = val
            
            prob = self.joint_probability(assignment)
            result[assignment[query]] += prob
        
        # 歸一化
        return result / result.sum()
```

### 使用 pgmpy 庫

pgmpy 是一個專業的 Python 貝氏網路庫：

```python
from pgmpy.models import BayesianNetwork
from pgmpy.estimators import MaximumLikelihoodEstimator
from pgmpy.inference import VariableElimination

# 定義網路結構
model = BayesianNetwork([
    ('D', 'G'),  # 課程難度 → 學生評分
    ('I', 'G'),  # 智力 → 學生評分
    ('G', 'L')   # 評分 → 推薦信
])

# 估計參數
model.fit(data, estimator=MaximumLikelihoodEstimator)

# 推論
inference = VariableElimination(model)
prob = inference.query(variables=['G'], evidence={'D': 1, 'I': 0})
```

## 經典應用案例

### 醫療診斷

**示例：肺炎診斷網路**

```
症狀 (S) ← 疾病 (D)
化驗結果 (T) ← 疾病 (D)
疾病 (D) ← 誘因 (F)
```

利用貝氏網路可以：
- 計算後驗診斷概率 P(疾病 | 症狀)
- 評估化驗結果的信息價值
- 指導臨床決策

### 故障診斷

用於複雜系統（如航空引擎）的故障定位：
- 感測器讀數是觀察節點
- 組件狀態是隱藏節點
- 推斷最可能的故障原因

### 風險評估

金融領域的信用風險評估：
- 申請人特徵 → 信用評分 → 違約風險
- 宏觀經濟因素 → 行業風險

### 垃圾郵件過濾

```
發件人特徵 → 郵件內容 → 是否垃圾郵件
```

結合文本分類和貝葉斯推論，計算 P(垃圾郵件 | 內容)。

## 與馬可夫網路的比較

| 特性 | 貝氏網路 | 馬可夫網路 |
|------|----------|------------|
| 圖結構 | 有向無環圖（DAG） | 無向圖 |
| 獨立性 | 條件獨立 | 成對獨立 |
| 分解形式 | 條件概率乘積 | 因子乘積 |
| 適合場景 | 因果關係明確 | 對稱關聯關係 |
| 推論難度 | 通常較易 | 通常較難 |

## 深度學習時代的貝氏網路

### 神經網路的不確定性

貝氏網路提供了一種量化不確定性的框架，可以與深度學習結合：

- **貝葉斯深度學習**：將網路權重視為隨機變量
- **貝葉斯神經網路**：使用貝葉斯推論進行權重學習
- **變分推論**：用變分方法近似貝葉斯推論

### 應用

- **不確定性感知預測**：同時預測值和置信區間
- **主動學習**：選擇最有信息量的樣本
- **知識蒸餾**：將大模型知識蒸餾到小模型
- **模型解釋性**：理解決策的不確定性來源

## 理論基礎

### 因子圖（Factor Graph）

將貝氏網路轉換為無向因子圖，有助於理解和推論：

```
每個 CPT 是一個因子
每個變量是變量節點
連接 CPT 和其涉及的所有變量
```

因子圖上的消息傳遞算法是推論的核心。

### 圖模型的一般理論

貝氏網路是概率圖模型的一個分支，屬於有向圖模型（Directed Graphical Models）類別。

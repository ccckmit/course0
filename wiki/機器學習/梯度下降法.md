# 梯度下降法 (Gradient Descent)

## 概述

梯度下降法是最優化的核心演算法，通過沿著函數梯度的反方向迭代更新參數，以找到函數的局部或全域最小值。

## 基本原理

```
      Loss
        │    ╱╲
        │   ╱  ╲        ← 梯度方向
        │  ╱    ╲
        │ ╱      ╲
        │╱        ╲
        └──────────────→ 參數 θ
              ↑
           最佳點
```

對於函數 J(θ)，梯度下降更新規則：
```
θ_new = θ_old - α · ∇J(θ_old)
```
其中 α 為學習率 (learning rate)。

## 梯度計算

### 單變量函數
```python
# J(θ) = θ² 的梯度
# ∇J(θ) = 2θ

theta = 5.0
learning_rate = 0.1

for _ in range(100):
    gradient = 2 * theta  # ∇J(θ) = 2θ
    theta = theta - learning_rate * gradient
    
print(theta)  # → 接近 0
```

### 多變量函數
```python
# J(θ₁, θ₂) = θ₁² + θ₂²
# ∇J(θ) = [2θ₁, 2θ₂]

theta = np.array([5.0, 5.0])
learning_rate = 0.1

for _ in range(100):
    gradient = 2 * theta  # [2θ₁, 2θ₂]
    theta = theta - learning_rate * gradient

print(theta)  # → [0, 0]
```

## 梯度下降類型

### 批量梯度下降 (Batch GD)
```python
# 使用全部資料計算梯度
for _ in range(epochs):
    gradients = compute_full_gradient(X, y, theta)
    theta = theta - learning_rate * gradients
```
- 缺點：每次迭代計算量大
- 優點：收斂穩定

### 隨機梯度下降 (SGD)
```python
# 使用單一範例計算梯度
for _ in range(epochs * len(X)):
    idx = random.randint(0, len(X) - 1)
    gradient = compute_gradient(X[idx], y[idx], theta)
    theta = theta - learning_rate * gradient
```
- 優點：速度快，可逃離局部最小值
- 缺點：收斂震盪，不穩定

### 小批量梯度下降 (Mini-batch GD)
```python
# 使用小批量資料計算梯度
batch_size = 32

for _ in range(epochs):
    indices = random.sample(range(len(X)), batch_size)
    X_batch = X[indices]
    y_batch = y[indices]
    gradient = compute_gradient(X_batch, y_batch, theta)
    theta = theta - learning_rate * gradient
```
- 平衡速度和穩定性
- 這是深度學習的主流方法

## 學習率影響

```
學習率過大 (α = 1.0)           學習率適中 (α = 0.1)
    Loss                              Loss
      │╲                               │   ╱╲
      │ ╲      ← 震盪發散             │  ╱  ╲
      │  ╲                            │ ╱    ╲
      │   ╲                           │╱      ╲
      └──────→ θ                      └──────────→ θ

學習率過小 (α = 0.001)           學習率正確
    Loss                              Loss
      │                               │    
      │   ╱╲                          │     ╲
      │  ╱  ╲                         │      ╲
      │ ╱    ╲                        │       ╲
      │╱      ╲                       │        ╲
      └──────────→ θ                  └─────────╲→ θ
        (收斂太慢)                              (適中)
```

## 學習率排程

### 學習率衰減
```python
initial_lr = 0.1
decay_rate = 0.95

for epoch in range(epochs):
    lr = initial_lr * (decay_rate ** epoch)
    theta = theta - lr * gradient
```

### 常見衰減策略
| 策略 | 更新規則 |
|------|----------|
| Step Decay | lr = lr₀ × γ^⌊epoch/steps⌋ |
| Exponential | lr = lr₀ × γ^epoch |
| Cosine | lr = lr₀ × (1 + cos(π × epoch/total))/2 |
| Warmup | lr 先增加後衰減 |

## 動量 (Momentum)

```python
# 動量梯度下降
momentum = 0.9
velocity = 0

for _ in range(epochs):
    gradient = compute_gradient(X, y, theta)
    velocity = momentum * velocity - learning_rate * gradient
    theta = theta + velocity
```

動量累積過去的梯度方向，加速收斂並減少震盪：
```
不使用動量              使用動量
    ↓                      ↓
   ╱╲                    ╱╲
  ╱  ╲      →          ╱──╲     →
 ╱    ╲                ╱    ╲
```

## AdaGrad

```python
# 適應性學習率（針對稀疏特徵）
epsilon = 1e-8
grad_squared = 0

for _ in range(epochs):
    gradient = compute_gradient(X, y, theta)
    grad_squared += gradient ** 2
    theta = theta - learning_rate * gradient / (np.sqrt(grad_squared) + epsilon)
```

## RMSProp

```python
# 解決 AdaGrad 學習率過早衰減的問題
epsilon = 1e-8
decay_rate = 0.9
grad_squared = 0

for _ in range(epochs):
    gradient = compute_gradient(X, y, theta)
    grad_squared = decay_rate * grad_squared + (1 - decay_rate) * gradient ** 2
    theta = theta - learning_rate * gradient / (np.sqrt(grad_squared) + epsilon)
```

## 收斂判斷

```python
# 監控 loss 變化
prev_loss = float('inf')
tolerance = 1e-6

for epoch in range(epochs):
    loss = compute_loss(X, y, theta)
    if abs(prev_loss - loss) < tolerance:
        print(f"Converged at epoch {epoch}")
        break
    prev_loss = loss
```

## 限制與挑戰

| 問題 | 說明 | 解決方法 |
|------|------|----------|
| 局部最小值 | 可能困於局部最小 | 動量、隨機性 |
| 鞍點 | 梯度為零但非極值 | 動量、Adam |
| 震盪 | 峽谷地形來回震盪 | 動量、Nesterov |
| 學習率選擇 | 影響收斂速度和品質 | 學習率排程 |

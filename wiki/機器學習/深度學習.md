# 深度學習

深度學習是機器學習的一個分支，專注於使用多層神經網路（稱為深度神經網路）來學習資料的複雜表示。深度學習在過去十多年中徹底改變了人工智慧領域，從影像識別、自然語言處理到語音識別，都取得了突破性的進展。與傳統機器學習方法相比，深度學習能夠自動從原始資料中學習層次化的特徵表示，無需人工設計特徵工程，這使其在處理大規模、高維度資料時具有顯著優勢。

## 深度學習的發展歷程

### 早期發展（1940s-1990s）

- **1943年**：Warren McCulloch 和 Walter Pitts 提出神經元的數學模型
- **1958年**：Frank Rosenblatt 發明感知機（Perceptron）
- **1986年**：Geoffrey Hinton 推廣反向傳播算法（Backpropagation）
- **1989年**：Yann LeCun 實現卷積神經網路用於手寫辨識

### 深度學習的突破（2000s-至今）

- **2006年**：Geoffrey Hinton 提出深度信念網路（DBN），開啟深度學習熱潮
- **2012年**：AlexNet 在 ImageNet 競賽中大幅領先，震撼電腦視覺界
- **2014年**：GAN（生成對抗網路）開創生成模型新領域
- **2017年**：Transformer 架構革命性地改變了自然語言處理
- **2020年**：GPT-3 展示大型語言模型的強大能力

## 神經網路基礎

### 神經元（Neuron）

神經元是神經網路的基本單位，其核心運算是輸入的加權求和加上偏差：

```
輸出 = 激活函數(Σ(輸入i × 權重i) + 偏差)
```

```python
import numpy as np

class Neuron:
    def __init__(self, input_size):
        # 初始化權重和偏差
        self.weights = np.random.randn(input_size) * 0.01
        self.bias = 0
    
    def forward(self, inputs):
        """
        前向傳播計算
        inputs: 輸入向量
        """
        self.inputs = inputs
        self.z = np.dot(inputs, self.weights) + self.bias
        return self.activation(self.z)
    
    def activation(self, z):
        """激活函數：ReLU"""
        return max(0, z)
```

### 激活函數

激活函數為神經網路引入非線性，使其能夠學習複雜的模式：

```python
import numpy as np

def sigmoid(z):
    """Sigmoid 函數：將值映射到 (0, 1)"""
    return 1 / (1 + np.exp(-z))

def tanh(z):
    """Tanh 函數：將值映射到 (-1, 1)"""
    return np.tanh(z)

def relu(z):
    """ReLU 函數：max(0, z)"""
    return np.maximum(0, z)

def leaky_relu(z, alpha=0.01):
    """Leaky ReLU：避免 ReLU 的神經元死亡問題"""
    return np.where(z > 0, z, alpha * z)

def softmax(z):
    """Softmax 函數：用於多分類輸出層"""
    exp_z = np.exp(z - np.max(z))  # 數值穩定性
    return exp_z / np.sum(exp_z)
```

```
激活函數圖形：

Sigmoid:        Tanh:           ReLU:
  1 ┤       ╱    1 ┤     ╱        ∞ ┤    ╱
    │      ╱       │    ╱           │   ╱
  0 ┼───╱         0 ┼─╱            0 ┼──
    │ ╱           │╱               0 ┤
    └────────     └────────         └────────
    -3    3       -3    3           -3    3
```

### 多層神經網路

深度學習的「深度」指的是網路中有多層神經元：

```python
import numpy as np

class Layer:
    def __init__(self, input_size, output_size, activation='relu'):
        self.weights = np.random.randn(input_size, output_size) * 0.01
        self.bias = np.zeros((1, output_size))
        self.activation = activation
    
    def forward(self, inputs):
        self.inputs = inputs
        self.z = np.dot(inputs, self.weights) + self.bias
        return self.apply_activation(self.z)
    
    def apply_activation(self, z):
        if self.activation == 'sigmoid':
            return 1 / (1 + np.exp(-z))
        elif self.activation == 'tanh':
            return np.tanh(z)
        elif self.activation == 'relu':
            return np.maximum(0, z)
        elif self.activation == 'softmax':
            exp_z = np.exp(z - np.max(z))
            return exp_z / np.sum(exp_z, axis=1, keepdims=True)
        return z

class NeuralNetwork:
    def __init__(self):
        self.layers = []
    
    def add_layer(self, layer):
        self.layers.append(layer)
    
    def forward(self, X):
        """前向傳播"""
        for layer in self.layers:
            X = layer.forward(X)
        return X
    
    def predict(self, X):
        """預測"""
        return self.forward(X)
```

## 深度神經網路架構

### 1. 全連接神經網路（Fully Connected Network / MLP）

全連接網路中，每個神經元與上一層的所有神經元相連：

```python
import torch
import torch.nn as nn

class MLP(nn.Module):
    """多層感知機"""
    def __init__(self, input_size, hidden_sizes, output_size):
        super(MLP, self).__init__()
        
        layers = []
        prev_size = input_size
        
        for hidden_size in hidden_sizes:
            layers.append(nn.Linear(prev_size, hidden_size))
            layers.append(nn.ReLU())
            prev_size = hidden_size
        
        layers.append(nn.Linear(prev_size, output_size))
        
        self.network = nn.Sequential(*layers)
    
    def forward(self, x):
        return self.network(x)

# 使用範例
model = MLP(input_size=784, hidden_sizes=[256, 128], output_size=10)

# 784 -> 256 -> 128 -> 10
# MNIST 手寫數字辨識
x = torch.randn(32, 784)  # 批次大小 32，輸入維度 784
output = model(x)
print(output.shape)  # torch.Size([32, 10])
```

### 2. 卷積神經網路（CNN）

卷積神經網路專門處理網格結構資料（如圖像），詳細內容見 [卷積神經網路](./卷積神經網路.md)。

```python
class CNN(nn.Module):
    def __init__(self, num_classes=10):
        super(CNN, self).__init__()
        
        # 卷積層
        self.conv1 = nn.Conv2d(3, 32, kernel_size=3, padding=1)
        self.conv2 = nn.Conv2d(32, 64, kernel_size=3, padding=1)
        self.conv3 = nn.Conv2d(64, 128, kernel_size=3, padding=1)
        
        # 池化層
        self.pool = nn.MaxPool2d(2, 2)
        
        # 全連接層
        self.fc1 = nn.Linear(128 * 4 * 4, 512)
        self.fc2 = nn.Linear(512, num_classes)
        
        self.relu = nn.ReLU()
        self.dropout = nn.Dropout(0.5)
    
    def forward(self, x):
        # Conv Block 1: 32x32 -> 16x16
        x = self.relu(self.conv1(x))
        x = self.pool(x)
        
        # Conv Block 2: 16x16 -> 8x8
        x = self.relu(self.conv2(x))
        x = self.pool(x)
        
        # Conv Block 3: 8x8 -> 4x4
        x = self.relu(self.conv3(x))
        x = self.pool(x)
        
        # Flatten
        x = x.view(x.size(0), -1)
        
        # 全連接層
        x = self.dropout(self.relu(self.fc1(x)))
        x = self.fc2(x)
        
        return x
```

### 3. 循環神經網路（RNN）

RNN 適合處理序列資料，詳細內容見 [循環神經網路](./循環神經網路.md)。

```python
class SimpleRNN(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super(SimpleRNN, self).__init__()
        
        self.hidden_size = hidden_size
        
        self.rnn = nn.RNN(input_size, hidden_size, batch_first=True)
        self.fc = nn.Linear(hidden_size, output_size)
    
    def forward(self, x):
        # x: (batch, seq_len, input_size)
        out, hidden = self.rnn(x)
        # out: (batch, seq_len, hidden_size)
        # hidden: (1, batch, hidden_size)
        
        out = self.fc(out[:, -1, :])  # 取最後一個時間步
        return out
```

### 4. Transformer

Transformer 架構基於自注意力機制，是當前最主流的深度學習架構：

```python
class TransformerBlock(nn.Module):
    def __init__(self, embed_size, heads, dropout, forward_expansion):
        super(TransformerBlock, self).__init__()
        
        self.attention = nn.MultiheadAttention(embed_size, heads, dropout=dropout)
        self.norm1 = nn.LayerNorm(embed_size)
        self.norm2 = nn.LayerNorm(embed_size)
        
        self.feed_forward = nn.Sequential(
            nn.Linear(embed_size, forward_expansion * embed_size),
            nn.ReLU(),
            nn.Linear(forward_expansion * embed_size, embed_size)
        )
        
        self.dropout = nn.Dropout(dropout)
    
    def forward(self, x, mask=None):
        # Multi-Head Attention
        attn_output, _ = self.attention(x, x, x, attn_mask=mask)
        x = self.norm1(x + self.dropout(attn_output))
        
        # Feed Forward
        ff_output = self.feed_forward(x)
        x = self.norm2(x + self.dropout(ff_output))
        
        return x

class Transformer(nn.Module):
    def __init__(self, vocab_size, embed_size, heads, num_layers, 
                 forward_expansion, dropout, max_length):
        super(Transformer, self).__init__()
        
        self.embedding = nn.Embedding(vocab_size, embed_size)
        self.positional_encoding = PositionalEncoding(embed_size, max_length)
        
        self.layers = nn.ModuleList([
            TransformerBlock(embed_size, heads, dropout, forward_expansion)
            for _ in range(num_layers)
        ])
        
        self.fc_out = nn.Linear(embed_size, vocab_size)
        self.dropout = nn.Dropout(dropout)
    
    def forward(self, x, mask=None):
        x = self.dropout(self.embedding(x) + self.positional_encoding(x))
        
        for layer in self.layers:
            x = layer(x, mask)
        
        return self.fc_out(x)
```

## 訓練深度神經網路

### 反向傳播算法

反向傳播是訓練神經網路的核心算法，它計算梯度並更新參數：

```python
import numpy as np

class Layer:
    def __init__(self, input_size, output_size, activation='relu'):
        self.weights = np.random.randn(input_size, output_size) * np.sqrt(2.0 / input_size)
        self.bias = np.zeros((1, output_size))
        self.activation = activation
        
        # 用於反向傳播的中間值
        self.inputs = None
        self.output = None
        self.z = None
    
    def forward(self, inputs):
        self.inputs = inputs
        self.z = np.dot(inputs, self.weights) + self.bias
        self.output = self.activate(self.z)
        return self.output
    
    def backward(self, dOutput, learning_rate):
        """反向傳播"""
        # 計算激活函數的梯度
        if self.activation == 'relu':
            dZ = dOutput * (self.z > 0)
        elif self.activation == 'sigmoid':
            dZ = dOutput * self.output * (1 - self.output)
        elif self.activation == 'tanh':
            dZ = dOutput * (1 - self.output ** 2)
        
        # 計算參數梯度
        m = self.inputs.shape[0]
        dWeights = np.dot(self.inputs.T, dZ) / m
        dBias = np.sum(dZ, axis=0, keepdims=True) / m
        
        # 更新參數
        self.weights -= learning_rate * dWeights
        self.bias -= learning_rate * dBias
        
        # 傳回上一層的梯度
        dInputs = np.dot(dZ, self.weights.T)
        return dInputs
    
    def activate(self, z):
        if self.activation == 'relu':
            return np.maximum(0, z)
        elif self.activation == 'sigmoid':
            return 1 / (1 + np.exp(-z))
        elif self.activation == 'tanh':
            return np.tanh(z)
        return z
```

### 完整訓練迴圈

```python
import torch
import torch.nn as nn
import torch.optim as optim
from torch.utils.data import DataLoader

def train_model(model, train_loader, num_epochs, device):
    model = model.to(device)
    criterion = nn.CrossEntropyLoss()
    optimizer = optim.Adam(model.parameters(), lr=0.001)
    
    for epoch in range(num_epochs):
        model.train()
        running_loss = 0.0
        correct = 0
        total = 0
        
        for batch_idx, (inputs, labels) in enumerate(train_loader):
            inputs, labels = inputs.to(device), labels.to(device)
            
            # 前向傳播
            optimizer.zero_grad()
            outputs = model(inputs)
            loss = criterion(outputs, labels)
            
            # 反向傳播
            loss.backward()
            optimizer.step()
            
            # 統計
            running_loss += loss.item()
            _, predicted = outputs.max(1)
            total += labels.size(0)
            correct += predicted.eq(labels).sum().item()
        
        # 每個 epoch 結束後的指標
        epoch_loss = running_loss / len(train_loader)
        epoch_acc = 100.0 * correct / total
        
        print(f'Epoch [{epoch+1}/{num_epochs}] '
              f'Loss: {epoch_loss:.4f} '
              f'Acc: {epoch_acc:.2f}%')
    
    return model
```

## 正則化與防止過擬合

深度神經網路很容易過擬合，需要各種正則化技術：

```python
import torch.nn as nn

class RegularizedModel(nn.Module):
    def __init__(self):
        super(RegularizedModel, self).__init__()
        
        self.conv1 = nn.Conv2d(3, 32, 3, padding=1)
        self.conv2 = nn.Conv2d(32, 64, 3, padding=1)
        
        # Dropout - 隨機失活
        self.dropout1 = nn.Dropout(0.25)
        self.dropout2 = nn.Dropout(0.5)
        
        # Batch Normalization - 批次正規化
        self.bn1 = nn.BatchNorm2d(32)
        self.bn2 = nn.BatchNorm2d(64)
        
        self.fc1 = nn.Linear(64 * 8 * 8, 256)
        self.fc2 = nn.Linear(256, 10)
        
        self.relu = nn.ReLU()
        self.pool = nn.MaxPool2d(2, 2)
    
    def forward(self, x):
        # Conv Block 1
        x = self.conv1(x)
        x = self.bn1(x)
        x = self.relu(x)
        x = self.pool(x)
        x = self.dropout1(x)
        
        # Conv Block 2
        x = self.conv2(x)
        x = self.bn2(x)
        x = self.relu(x)
        x = self.pool(x)
        x = self.dropout1(x)
        
        # Flatten
        x = x.view(x.size(0), -1)
        
        # FC Layers
        x = self.fc1(x)
        x = self.relu(x)
        x = self.dropout2(x)
        x = self.fc2(x)
        
        return x
```

### 常見正則化技術

| 技術 | 描述 | 參數 |
|------|------|------|
| L2 正則化 | 懲罰大的權重 | weight_decay |
| Dropout | 隨機失活神經元 | p |
| Batch Normalization | 標準化層輸入 | - |
| Early Stopping | 驗證集不再改善時停止 | patience |
| Data Augmentation | 資料增強 | - |

## PyTorch 完整範例：MNIST 手寫辨識

```python
import torch
import torch.nn as nn
import torch.optim as optim
import torch.nn.functional as F
from torch.utils.data import DataLoader
from torchvision import datasets, transforms

# 設備配置
device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')

# 超參數
batch_size = 64
num_epochs = 10
learning_rate = 0.001

# 資料轉換
transform = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize((0.1307,), (0.3081,))
])

# 載入資料
train_dataset = datasets.MNIST('./data', train=True, download=True, transform=transform)
test_dataset = datasets.MNIST('./data', train=False, transform=transform)

train_loader = DataLoader(train_dataset, batch_size=batch_size, shuffle=True)
test_loader = DataLoader(test_dataset, batch_size=batch_size, shuffle=False)

# 定義模型
class Net(nn.Module):
    def __init__(self):
        super(Net, self).__init__()
        self.conv1 = nn.Conv2d(1, 32, 3, 1)
        self.conv2 = nn.Conv2d(32, 64, 3, 1)
        self.dropout1 = nn.Dropout(0.25)
        self.dropout2 = nn.Dropout(0.5)
        self.fc1 = nn.Linear(9216, 128)
        self.fc2 = nn.Linear(128, 10)
    
    def forward(self, x):
        x = self.conv1(x)
        x = F.relu(x)
        x = self.conv2(x)
        x = F.relu(x)
        x = F.max_pool2d(x, 2)
        x = self.dropout1(x)
        x = torch.flatten(x, 1)
        x = self.fc1(x)
        x = F.relu(x)
        x = self.dropout2(x)
        x = self.fc2(x)
        return F.log_softmax(x, dim=1)

# 初始化模型
model = Net().to(device)
optimizer = optim.Adam(model.parameters(), lr=learning_rate)
criterion = nn.CrossEntropyLoss()

# 訓練
def train(epoch):
    model.train()
    for batch_idx, (data, target) in enumerate(train_loader):
        data, target = data.to(device), target.to(device)
        
        optimizer.zero_grad()
        output = model(data)
        loss = criterion(output, target)
        loss.backward()
        optimizer.step()
        
        if batch_idx % 100 == 0:
            print(f'Train Epoch: {epoch} [{batch_idx * len(data)}/{len(train_loader.dataset)}]')

# 測試
def test():
    model.eval()
    test_loss = 0
    correct = 0
    
    with torch.no_grad():
        for data, target in test_loader:
            data, target = data.to(device), target.to(device)
            output = model(data)
            test_loss += criterion(output, target).item()
            pred = output.argmax(dim=1, keepdim=True)
            correct += pred.eq(target.view_as(pred)).sum().item()
    
    print(f'Test set: Average loss: {test_loss/len(test_loader):.4f}, '
          f'Accuracy: {correct}/{len(test_loader.dataset)}')

# 執行訓練
for epoch in range(1, num_epochs + 1):
    train(epoch)
    test()
```

## 深度學習的挑戰與未來

### 當前挑戰

1. **計算資源需求**：大型模型需要大量 GPU/TPU 資源
2. **資料依賴**：需要大量標註資料
3. **可解釋性**：模型決策過程不透明
4. **泛化能力**：模型可能對分佈外資料失效
5. **能耗問題**：訓練大型模型消耗大量能源

### 未來發展方向

1. **更高效的模型**：蒸餾、剪枝、量化
2. **自監督學習**：減少對標註資料的依賴
3. **多模態學習**：整合文字、圖像、音頻等多種資料
4. **持續學習**：模型終身學習新知識
5. **神經網路與符號 AI 結合**

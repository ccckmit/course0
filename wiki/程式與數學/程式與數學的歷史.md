# 程式與數學的歷史

程式與數學的關係如同硬幣的兩面——電腦科學本質上是應用數學的分支，而數學的抽象化與形式化為計算奠定了堅實的理論基礎。從古希臘的几何學到現代的拓撲學，數學的發展為程式設計提供了工具和思想。

## 古代數學：計算的起源（公元前 3000 年 - 公元 500 年）

### 美索不達米亞與埃及

最早的數學起源於實際需求：
- **巴比倫人**（公元前 1800 年）：解決二次方程，擁有 60 進制
- **埃及人**（公元前 1650 年）：《萊因德紙草書》記錄了數學問題和求解方法

這些文明發展了最早的「演算法」概念——解決特定問題的機械化步驟。

### 古希臘：公理化方法

公元前 300 年，歐幾里得的《幾何原本》建立了公理化方法：
- **公設**：不證自明的基本假設
- **定理**：從公設推導出的結論
- **證明**：嚴密的邏輯推導

這種公理化思想深刻影響了後世的數學和電腦科學。

### 印度的數學創新

5-12 世紀的印度數學家有諸多重要貢獻：
- **婆羅摩笈多**（598-668）：定義了零，提出了二次方程解法
- **婆什迦羅二世**（1114-1185）：系統化了算術運算規則
- **零的數學**：從占位符到真正的數字概念

### 中國古代數學

中國數學強調計算實用性：
- **《九章算術》**（東漢）：分為九類的數學問題集
- **劉徽**（3 世紀）：割圓術，計算圓周率
- **祖沖之**（5 世紀）：精確計算圓周率至小數點後 7 位
- **秦九韶**（13 世紀）：求解高次方程的秦九韶演算法（Horner's method）

秦九韶演算法是中國對世界數學的重要貢獻，在多項式求值中效率極高：

```python
def horner(coefficients, x):
    """秦九韶演算法：高效多項式求值"""
    result = coefficients[0]
    for coef in coefficients[1:]:
        result = result * x + coef
    return result
```

## 中世紀與文藝復興（500-1600 年）

### 阿拉伯數學

花剌子模（約 780-850）的《代數學》系統化了代數運算：
- 引入代數方程的一般解法
- 命名了「代數學」（Algebra）一詞
- 奠定了求解線性和二次方程的基礎

「演算法」（Algorithm）一詞正是源於他的名字。

### 費波那契與西方數學復興

1202 年，費波那契（Leonardo of Pisa）發表了《算盤書》：
- 引進阿拉伯數字到歐洲
- 提出了費波那契數列：
  ```
  F(0) = 0, F(1) = 1
  F(n) = F(n-1) + F(n-2)
  ```
- 解決了兔子繁殖問題

費波那契數列在自然界和電腦科學中都有重要應用。

## 近代數學的興起（1600-1850 年）

### 笛卡爾與解析幾何

1637 年，René Descartes 發明了解析幾何：
- 將幾何問題轉化為代數問題
- 建立了座標系統
- 為牛頓和萊布尼茲的微積分奠定基礎

### 牛頓與萊布尼茲：微積分的發明

1665-1666 年，牛頓獨立發明了微積分。萊布尼茲也在同時期發展了類似的理論，並引入了現代微積分符號。

微積分的核心概念：
- **微分**：瞬時變化率（導數）
- **積分**：面積和體積的計算
- **微分方程**：描述自然規律的數學語言

微積分是現代科學和工程的數學基礎，也是機器學習中梯度下降法的理論依據。

### 機率論的誕生

1654 年，帕斯卡和費馬通信討論擲骰子問題，標誌著機率論的誕生。

1657 年，惠更斯發表了《關於骰子遊戲的推理》，這是第一部機率論著作。此後：
- 1713 年：雅各布·伯努利的大數定律
- 1733 年：棣莫弗的中央極限定理
- 1812 年：拉普拉斯的《機率分析論》

機率論是統計學、機器學習和信息論的數學基礎。

## 形式數學的建立（1850-1950 年）

### 集合論

1873 年，Georg Cantor 創立了集合論：
- 定義了無窮集合和基數
- 對角線論證法
- 為現代數學提供了統一語言

1908 年，Ernst Zermelo 提出了集合論的公理系統（後經 Fraenkel 補充）。

### 布爾代數

1854 年，George Boole 發表了《思維定律的研究》：
- 為邏輯提供了數學工具
- 命題邏輯的代數化
- 後來成為數位電路設計的理論基礎

1938 年，Claude Shannon 將布林代數應用於電話交換電路的分析，開創了數位電路設計。

### 數理邏輯

1879 年，Frederick Gottlob Frege 的《概念文字》建立了現代形式邏輯。1921 年，Ludwig Wittgenstein 的《邏輯哲學論叢》影響深遠。

1930 年代，Herbert Gödel 的不完備定理（1931）震動了數學界——任何足夠強大的形式系統都有無法證明的真理。

## 電腦科學的數學基礎（1930-1960 年）

### 圖靈機

1936 年，Alan Turing 的圖靈機是第一個通用的計算模型：
- 定義了「可計算」的數學定義
- 解決了 Hilbert 的決定性問題
- 為電腦的誕生提供了理論基礎

### λ 演算

1936 年，Alonzo Church 的 λ 演算是另一個等價的計算模型：
- 函數的抽象和應用
- 為函數式程式語言提供了理論基礎
- Lisp、Haskell 等語言的數學根源

Church-Turing 論點斷言：圖靈機和 λ 演算計算能力等价。

### 自動機理論

1943 年，Warren McCulloch 和 Walter Pitts 提出了第一個人工神經網路模型。1956 年，Stephen Kleene 正式確立了有限自動機理論。

### 形式語言

1956 年，Noam Chomsky 提出了形式語言的 Chomsky 階層：
- Type 0：任意文法（圖靈機）
- Type 1：上下文相關文法
- Type 2：上下文無關文法
- Type 3：正規文法

形式語言理論是編譯器設計和程式語言語法的理論基礎。

## 現代應用數學（1950-1990 年）

### 線性代數的復興

矩陣理論和線性代數在電腦圖學、最佳化、機器學習中變得重要：
- **奇異值分解（SVD）**：資料壓縮和降維
- **特徵值分解**：主成分分析（PCA）
- **矩陣分解**：推薦系統、協同過濾

### 機率與統計的電腦化

1950-60 年代，貝氏統計、蒙地卡羅方法得到快速發展。John von Neumann 和 Stanislaw Ulam 發展了蒙地卡羅模擬。

### 資訊理論

1948 年，Claude Shannon 發表了《通信的數學理論》：
- **熵（Entropy）**：資訊量的度量
- **頻寬**：信道容量
- **資料壓縮**：無失真和有失真編碼

Shannon 的資訊理論是數位通信、密碼學和機器學習的重要基礎。

### 離散數學

電腦科學的數學需要離散的結構：
- **圖論**：網路、流、匹配問題
- **組合學**：排列、組合、計數問題
- **數論**：密碼學的基礎

## 數學與 AI 的交匯（1990 年代-至今）

### 機器學習的數學基礎

機器學習是應用數學的集大成者：

- **線性代數**：神經網路的矩陣運算
- **微積分**：梯度下降和反向傳播
- **機率統計**：貝氏學習、機率模型
- **最佳化理論**：目標函數、約束條件

### 深度學習的數學

2010 年代，深度學習的突破建立在數學之上：

- **自動微分**：高效計算複雜函數的導數
- **非凸最佳化**：理解深度網路的訓練動態
- **資訊理論**：變分自編碼器（VAE）
- **微分幾何**：幾何深度學習

### 密碼學的數學

現代密碼學建立在數論和代數之上：

- **RSA**：大整數分解的困難性
- **橢圓曲線密碼學**：橢圓曲線上的離散對數
- **同態加密**：允許在加密資料上計算

### 量子計算的數學

量子計算承諾了指數級加速：

- **線性代數**：量子態是向量，測量是投影
- **群論**：量子閘是么正矩陣
- **拓撲學**：拓撲量子計算

## 台灣與中國的數學傳統

### 中國數學的輝煌

中國古代數學有獨立的發展體系：
- **十進位制**：領先世界
- **方程術**：多元方程的解法
- **天元術**：符號代數的萌芽
- **割圓術**：無窮級數思想

### 台灣的數學教育

台灣在數學教育和研究方面成就顯著：
- 中央研究院數學研究所
- 台灣大學數學系
- 在 combinatorics、數論、代數幾何等領域有國際級學者

### 兩岸數學交流

改革開放後，中美數學交流恢復。中國在基礎數學研究方面快速追趕，在數論、代數幾何、微分幾何等領域達到國際一流水平。

## 未來展望

### 數學的電腦化

電腦證明和形式化數學正在興起：
- **Lean 定理證明器**：數學的形式化驗證
- **自動定理證明**：AI 輔助數學研究
- **反例搜尋**：發現猜想中的漏洞

### AI for Mathematics

DeepMind 的 AlphaProof 和 AlphaGeometry 展示了 AI 解決數學問題的潛力。數學研究可能因此加速。

### 交叉學科的深化

- **生物數學**：基因網路建模
- **金融數學**：風險管理和演算法交易
- **量子數學**：為量子計算奠定基礎

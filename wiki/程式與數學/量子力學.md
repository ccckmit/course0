# 量子力學 (Quantum Mechanics)

## 概述

量子力學是描述微觀粒子行為的物理理論，20世紀初由普朗克、波耳、海森堡、薛丁格、狄拉克等人發展而成。與經典物理學不同，量子力學的核心特徵包括：**態疊加原理**、**測量坍縮**、**不確定性原理**、**波粒二象性**等。這些概念徹底改變了我們對物質本質的理解，也奠定了現代物理學的基礎，從半導體到雷射，從核能到量子計算，無不依賴量子力學的理論框架。

量子力學的數學基礎是**希爾伯特空間**——一個配備內積的完整向量空間。量子態表示為這個空間中的向量，物理量表示為作用在這些向量上的**厄米算符**，測量結果則是算符的**本徵值**。這種數學結構使得量子力學具有精確的預測能力，並且與實驗結果高度一致。

## 波函數與態向量

### 波函數的物理意義

量子系統的狀態由**波函數** $\psi(\mathbf{r}, t)$ 完全描述。波函數是一個複數函數，其模平方給出粒子在位置 $\mathbf{r}$ 處被發現的**機率密度**：

$$P(\mathbf{r}, t) = |\psi(\mathbf{r}, t)|^2$$

波函數必須滿足**歸一化條件**：

$$\int |\psi(\mathbf{r}, t)|^2 d^3\mathbf{r} = 1$$

這意味著粒子在整個空間中被發現的機率為1。波函數的**相位**雖然不直接對應可觀測量，但相位差會產生**干涉效應**，這是量子力學最顯著的特征之一。

### 態向量表示

更一般的量子態可用**態向量** $|\psi\rangle$ 表示，這是**狄拉克符號**的核心概念。態向量位於希爾伯特空間 $\mathcal{H}$ 中，任意量子態可表示為一組**基底向量**的線性組合：

$$|\psi\rangle = \sum_n c_n |n\rangle$$

其中 $c_n = \langle n|\psi\rangle$ 是複振幅，滿足 $\sum_n |c_n|^2 = 1$。這種表示方式特別適合處理**離散**的量子態，例如自旋向上的態 $|0\rangle$ 和自旋向下的態 $|1\rangle$ 的疊加。

### 疊加原理

量子態的**疊加原理**是量子計算的數學基礎。與經典機率不同，量子疊加允許態向量同時處於多個基態的**相干疊加**：

$$|\psi\rangle = \alpha|0\rangle + \beta|1\rangle$$

其中 $|\alpha|^2 + |\beta|^2 = 1$。這種疊加在測量時會**坍縮**到其中一個基態，測量結果 $0$ 的機率為 $|\alpha|^2$，結果 $1$ 的機率為 $|\beta|^2$。

## 薛丁格方程式

### 時間相關薛丁格方程

描述量子態時間演化的基本方程是**薛丁格方程式**：

$$i\hbar\frac{\partial}{\partial t}|\psi(t)\rangle = \hat{H}|\psi(t)\rangle$$

其中 $\hbar = h/2\pi \approx 1.054 \times 10^{-34}$ J·s 是約化普朗克常數，$\hat{H}$ 是**哈密頓算符**，代表系統的總能量。對於單粒子系統：

$$\hat{H} = \frac{\hat{p}^2}{2m} + V(\mathbf{r}, t)$$

其中 $\hat{\mathbf{p}} = -i\hbar\nabla$ 是動量算符，$V(\mathbf{r}, t)$ 是勢能函數。

### 時間獨立薛丁格方程

對於**定態**系統（勢能與時間無關），波函數可分離為 $\psi(\mathbf{r}, t) = \psi(\mathbf{r})e^{-iEt/\hbar}$，代入薛丁格方程得到**時間獨立薛丁格方程**：

$$\hat{H}\psi(\mathbf{r}) = E\psi(\mathbf{r})$$

這是求解量子系統能量本徵值和本徵函數的根本方程。其形式類似於線性代數中的**特徵值問題**：哈密頓算符作用在本徵函數上，只是得到一個常數（本徵能量）乘以本徵函數本身。

### 例子：無限深勢阱

考慮一維無限深勢阱，粒子被限制在 $0 < x < a$ 的區域內，邊界條件為 $\psi(0) = \psi(a) = 0$。時間獨立薛丁格方程的解為：

$$\psi_n(x) = \sqrt{\frac{2}{a}}\sin\left(\frac{n\pi x}{a}\right)$$

對應的能量本徵值為：

$$E_n = \frac{n^2\pi^2\hbar^2}{2ma^2}, \quad n = 1, 2, 3, \ldots$$

這顯示能量是**量子化**的，只能取離散的值。隨著量子數 $n$ 增加，能量間距變大，波函數的節點數增加。

## 算符與測量

### 基本算符

量子力學中，每個可觀測物理量對應一個**厄米算符**：

- **位置算符**：$\hat{\mathbf{r}}\psi = \mathbf{r}\psi$
- **動量算符**：$\hat{\mathbf{p}}\psi = -i\hbar\nabla\psi$
- **角動量算符**：$\hat{\mathbf{L}} = \hat{\mathbf{r}} \times \hat{\mathbf{p}}$
- **哈密頓算符**：$\hat{H} = \frac{\hat{p}^2}{2m} + V(\hat{\mathbf{r}})$

厄米算符的**本徵值**是實數，對應實際測量可能得到的結果。其**本徵向量**構成希爾伯特空間的**正交基**，任意量子態可展開為這些本徵態的疊加。

### 測量理論

對於算符 $\hat{A}$ 的本徵方程 $\hat{A}|a_n\rangle = a_n|a_n\rangle$，測量量子態 $|\psi\rangle$ 得到結果 $a_n$ 的**機率**為：

$$P(a_n) = |\langle a_n|\psi\rangle|^2$$

測量後，量子態**坍縮**到對應的本徵態：

$$|\psi\rangle \rightarrow |a_n\rangle$$

這是量子測量的核心特征，稱為**投影假設**。多次測量的期望值為：

$$\langle A\rangle = \langle\psi|\hat{A}|\psi\rangle$$

### 不確定性原理

海森堡**不確定性原理**是量子力學的基本限制：

$$\Delta x \cdot \Delta p \geq \frac{\hbar}{2}$$

其中 $\Delta x$ 和 $\Delta p$ 分別是位置和動量的**標準差**。這不是測量技術的限制，而是量子力學本身的核心特征——粒子沒有同時確定的位置和動量。

更一般地，對於任意兩個厄米算符 $\hat{A}$ 和 $\hat{B}$：

$$\Delta A \cdot \Delta B \geq \frac{1}{2}|\langle[\hat{A}, \hat{B}]\rangle|$$

其中 $[\hat{A}, \hat{B}] = \hat{A}\hat{B} - \hat{B}\hat{A}$ 是**對易子**。如果兩個算符**對易** ($[\hat{A}, \hat{B}] = 0$)，則它們可以同時精確測量。

## 核心定理

###  Ehrenfest 定理

 Ehrenfest 定理連接量子力學與經典力學：

$$m\frac{d}{dt}\langle\mathbf{r}\rangle = \langle\mathbf{p}\rangle$$
$$\frac{d}{dt}\langle\mathbf{p}\rangle = -\langle\nabla V(\mathbf{r})\rangle$$

在某些近似下，量子期望值遵循經典運動方程。

### 態疊加原理與干涉

量子力學的**態疊加**與經典疊加完全不同。考慮雙縫實驗，電子通過兩個狹縫後的波函數是通過每個狹縫波函數的疊加：

$$\psi = \psi_1 + \psi_2$$

機率密度展現**干涉項**：

$$|\psi|^2 = |\psi_1|^2 + |\psi_2|^2 + 2\text{Re}(\psi_1^*\psi_2)$$

這是量子力學最驚人的預言之一，已被無數實驗證實。

### EPR 悖論與貝爾定理

愛因斯坦-波多爾斯基-羅森 (EPR) 悖論 (1935) 質疑量子力學的完整性，認為量子力學是不完整的「隱變量理論」。1964 年，**貝爾定理**證明：如果隱變量理論預測某些局域性結果，則會違反**貝爾不等式**。1982 年， Aspect 實驗證實量子力學違反貝爾不等式，支持**量子糾纏**的真实性。

**量子糾纏**是兩個或多個粒子之間的強相關性，測量一個粒子會瞬間影響另一個粒子的狀態，無論它們相距多遠。這是量子資訊科學的基礎資源。

## 量子力學與程式設計

### 量子計算

量子計算機使用**量子位元** (qubit) 作為基本資訊單元。與經典位元不同，量子位元可處於 $|0\rangle$ 和 $|1\rangle$ 的疊加態：

$$|\psi\rangle = \alpha|0\rangle + \beta|1\rangle$$

量子計算利用疊加和糾纏，可以同時處理多個計算路徑，在某些問題上提供指數級加速。

### 數值方法

求解薛丁格方程需要數值方法：
- **變分法**：用於估算基態能量
- **蒙特卡羅方法**：路徑積分量子蒙地卡羅
- **有限差分法**：離散化波函數
- **譜方法**：使用特殊函數展開

### 量子模擬

用傳統電腦模擬量子系統面臨**指數牆**問題——模擬 $N$ 個自旋需要 $2^N$ 維度的向量。這催生了**量子模擬器**和**量子電腦**的發展。

## 總結

量子力學是20世紀最偉大的科學成就之一。它不僅解釋了原子和分子的行為，還奠定了現代科技的核心基礎。從半導體物理到雷射技術，從核磁共振到量子計算，量子力學的應用無處不在。其數學框架——希爾伯特空間中的向量和算符——提供了一種精確且強大的語言來描述微觀世界。量子力學的深刻含義——態疊加、測量坍縮、糾纏——仍然吸引著物理學家和哲學家進行深入探討。

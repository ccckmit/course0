# 變分學 (Calculus of Variations)

## 概述

變分學是數學中處理**泛函極值問題**的分支，所謂泛函是「函數的函數」——輸入是一個函數，輸出是一個數值。變分學的核心問題是：給定一個泛函，找到使其取得極值（通常是最小值）的函數。這種「優化函數」的思想在物理學、工程學和經濟學中有著廣泛的應用。

變分學的起源可以追溯到最速降線問題（Brachistochrone Problem）：在給定兩點之間，找到一條使小球滾動時間最短的曲線。這個問題在 1696 年由約翰·伯努利提出，解決過程中發展出了變分法的基本思想，並最終導致了歐拉-拉格朗日方程的發現。

變分學與物理學有著深刻的聯繫。**最小作用原理**告訴我們，自然界中物理系統的真實軌跡是使「作用量」取極值的軌跡。這一原理不僅統一了牛頓力學、拉格朗日力學和哈密頓力學，更是現代理論物理學的核心思想。

## 泛函與變分

### 泛函的定義

**泛函**（Functional）是從函數空間到實數的映射。形式上，若 $J: \mathcal{F} \to \mathbb{R}$，其中 $\mathcal{F}$ 是函數空間，則 $J$ 是一個泛函。

**例子**：

1. **曲線長度**：對於曲線 $y(x)$，從 $x=a$ 到 $x=b$，長度為：
   $$J[y] = \int_a^b \sqrt{1 + (y')^2} dx$$

2. **作用量**：$S[y] = \int_{t_1}^{t_2} L(y, y', t) dt$

3. **面積**：對於曲面 $z(x, y)$，在區域 $D$ 上的面積為：
   $$J[z] = \iint_D \sqrt{1 + z_x^2 + z_y^2} dx dy$$

### 變分的概念

**變分**（Variation）類似於微分，但作用於函數空間。對於泛函 $J[y]$，其變分 $\delta J$ 是泛函在方向 $\eta(x)$ 上的線性近似：

$$J[y + \varepsilon \eta] = J[y] + \varepsilon \delta J[y, \eta] + O(\varepsilon^2)$$

其中 $\eta(x)$ 是任意光滑函數（稱為**變分方向**），滿足 $\eta(a) = \eta(b) = 0$（固定端點）。

變分的關鍵性質是：$J[y]$ 在 $y$ 處取得極值當且僅當 $\delta J[y, \eta] = 0$ 對所有變分方向 $\eta$ 成立。

這個條件類似於普通微積分中的極值條件：導數為零。只不過這裡「導數」是作用在函數空間上的線性泛函。

## 歐拉-拉格朗日方程

### 方程的推導

變分學的核心結果是**歐拉-拉格朗日方程**，它給出了使泛函取極值的函數所滿足的微分方程。

設泛函為：
$$J[y] = \int_a^b F(y, y', x) dx$$

其中 $F$ 是給定的光滑函數。令 $y(x)$ 為極值函數，考慮：
$$J(y + \varepsilon \eta) = \int_a^b F(y + \varepsilon \eta, y' + \varepsilon \eta', x) dx$$

對 $\varepsilon$ 求導並在 $\varepsilon = 0$ 處取值：
$$\frac{d}{d\varepsilon} J(y + \varepsilon \eta)\bigg|_{\varepsilon=0} = \int_a^b \left(\frac{\partial F}{\partial y} \eta + \frac{\partial F}{\partial y'} \eta'\right) dx$$

對第二項使用分部積分，並利用邊界條件 $\eta(a) = \eta(b) = 0$：
$$\int_a^b \left(\frac{\partial F}{\partial y} - \frac{d}{dx}\frac{\partial F}{\partial y'}\right) \eta dx$$

極值條件要求這對所有 $\eta$ 都為零，因此被積函數必須恆為零：
$$\frac{\partial F}{\partial y} - \frac{d}{dx}\frac{\partial F}{\partial y'} = 0$$

這就是**歐拉-拉格朗日方程**。

### 例子：最速降線問題

**最速降線問題**：在重力作用下，求兩點之間使小球滾動時間最短的曲線。

時間泛函為：
$$J[y] = \int_{x_0}^{x_1} \frac{\sqrt{1 + y'^2}}{\sqrt{2g(y - y_0)}} dx$$

這裡 $F = \frac{\sqrt{1 + y'^2}}{\sqrt{2g(y - y_0)}}$。歐拉-拉格朗日方程給出：
$$\frac{d}{dx}\left(\frac{y'}{\sqrt{1 + y'^2}\sqrt{2g(y - y_0)}}\right) + \frac{1}{2}\frac{\sqrt{1 + y'^2}}{(2g(y - y_0))^{3/2}} = 0$$

這個方程的解是**擺線**（Cycloid）：
$$x = r(\theta - \sin\theta), \quad y = r(1 - \cos\theta)$$

擺線是歷史上第一個非平凡的變分問題解答。

### 例子：測地線

在曲面上，兩點之間的**測地線**是短程線。設曲面由參數方程給出，測地線問題的泛函為：
$$J[\mathbf{r}] = \int_{t_0}^{t_1} \sqrt{\mathbf{r}' \cdot \mathbf{r}'} dt$$

歐拉-拉格朗日方程給出測地線方程，這是微分幾何中的核心問題。

## 多變數情況

### 多變數泛函

對於多變數函數 $u(x, y)$，泛函形式為：
$$J[u] = \iint_D F(u, u_x, u_y, x, y) dx dy$$

歐拉-拉格朗日方程推廣為：
$$\frac{\partial F}{\partial u} - \frac{\partial}{\partial x}\frac{\partial F}{\partial u_x} - \frac{\partial}{\partial y}\frac{\partial F}{\partial u_y} = 0$$

**例子**：極小曲面問題（面積最小的曲面）

面積泛函為：
$$J[u] = \iint_D \sqrt{1 + u_x^2 + u_y^2} dx dy$$

歐拉-拉格朗日方程為：
$$\frac{\partial}{\partial x}\left(\frac{u_x}{\sqrt{1 + u_x^2 + u_y^2}}\right) + \frac{\partial}{\partial y}\left(\frac{u_y}{\sqrt{1 + u_x^2 + u_y^2}}\right) = 0$$

這稱為**極小曲面方程**。

### 自然邊界條件

在變分問題中，如果不固定邊界條件，會得到**自然邊界條件**。例如，對於泛函：
$$J[y] = \int_a^b F(y, y') dx$$

若 $y(a)$ 固定但 $y(b)$ 自由，則自然邊界條件為：
$$\frac{\partial F}{\partial y'}\bigg|_{x=b} = 0$$

這稱為**紐曼邊界條件**。

## 變分學在物理學的應用

### 最小作用原理

**最小作用原理**是物理學中最深刻的思想之一：**自然選擇的路徑使作用量取極值**。

作用量定義為：
$$S[q] = \int_{t_1}^{t_2} L(q, \dot{q}, t) dt$$

其中 $L$ 是拉格朗日量。最小作用原理給出：
$$\delta S = 0$$

這直接導出歐拉-拉格朗日方程：
$$\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}^i}\right) - \frac{\partial L}{\partial q^i} = 0$$

這正是拉格朗日力學的基本方程！

### 哈密頓原理

**哈密頓原理**是最小作用原理的更精確表述：不僅作用量的變分為零，而且是真實軌跡使作用量取**平穩值**（極值或鞍點）。

對於保守系統，哈密頓量 $H = T + V$，作用量為：
$$S = \int_{t_1}^{t_2} (T - V) dt$$

這連接了拉格朗日力學和哈密頓力學。

### 費瑪原理

**費瑪原理**（Fermat's Principle）是幾何光學的基本原理：光選擇時間最短的路徑。

對於從點 $A$ 到點 $B$ 的光線，若介質折射率為 $n(\mathbf{r})$，路徑時間為：
$$T = \int_A^B \frac{n(\mathbf{r})}{c} ds$$

費瑪原理 $\delta T = 0$ 導出光的折射定律——斯涅爾定律：
$$\frac{\sin\theta_1}{\sin\theta_2} = \frac{n_2}{n_1}$$

## 直接方法

### 里茲方法

**里茲方法**（Ritz Method）是求解變分問題的數值方法。其基本思想是：

1. 選擇一組基函數 $\phi_1, \phi_2, \ldots, \phi_n$
2. 用線性組合近似解：$y \approx \sum_{i=1}^n c_i \phi_i$
3. 將係數 $c_i$ 作為變分參數
4. 令 $\delta J = 0$ 得到 $n$ 個代數方程

這將無限維的函數空間問題轉化為有限維的優化問題。

**例子**：對於量子諧振子，使用高斯型試探波函數：
$$\psi(x) = e^{-\alpha x^2}$$

通過變分優化參數 $\alpha$，可以得到基態能量的良好近似。

### 有限元方法

**有限元方法**（Finite Element Method, FEM）是工程中最重要的數值方法之一，與變分學密切相關。

基本步驟：
1. 將區域分割成小的「有限元」（如三角形、四面體）
2. 在每個元上使用低次多項式近似
3. 組裝整體方程
4. 求解大型線性方程組

有限元方法廣泛應用於結構力學、流體力學和熱傳導等問題。

## 約束變分問題

### 約束條件

現實中的變分問題往往帶有約束條件：

1. **等周約束**：曲線有一定長度，求包圍最大面積的形狀
2. **固定端點**：曲線必須經過給定點
3. **等時约束**：運動時間固定

**例子**：等周問題——在固定周長下，求面積最大的曲線。

答案是圓。這可以用變分法證明，約束條件是周長 $L = \int_a^b \sqrt{1 + y'^2} dx = \text{常數}$。

使用拉格朗日乘數法，引入約束條件：
$$J[y] = \iint_D dx dy - \lambda \int_a^b \sqrt{1 + y'^2} dx$$

極值條件給出微分方程，其解為圓。

### 拉格朗日乘數法

**拉格朗日乘數法**是處理約束變分問題的標準方法。

對於問題：
$$\min J[y] \text{ subject to } G[y] = 0$$

構造拉格朗日泛函：
$$\mathcal{L}[y] = J[y] - \lambda G[y]$$

然後求解 $\delta \mathcal{L} = 0$。

**例子**：在固定體積下求表面積最小的形狀——這給出球形。

## 總結

變分學是處理「函數優化」的數學工具。從最速降線問題到測地線，從最小作用原理到費瑪原理，變分法提供了一套統一的方法來處理物理學中的優化問題。歐拉-拉格朗日方程是這個理論的核心，它將泛函極值問題轉化為微分方程。通過拉格朗日乘數法和里茲方法，變分學也成為數值求解實際問題的強大工具。

# 代數學

**標籤**: #代數學 #抽象代數 #群論 #環 #體 #有限體 #伽羅瓦 #密碼學
**日期**: 2024-01-15

代數學是數學的重要分支，主要研究群、環、體等代數結構。這些抽象結構是現代密碼學、編碼理論與電腦科學的數學基礎。

## 抽象代數概述

### 為什麼要抽象？

1. **統一化與簡化**：當我們證明某個定理適用於「群」這個概念時，自動就證明了這個定理適用於所有符合「群」規則的數學系統。這就像發現了「所有鳥類都會飛」這個規則後，當你看到麻雀、老鷹，你就知道它們可以飛。

2. **擴展應用**：透過抽象化，可以將原本適用於數字的理論應用到完全不同的領域，例如對稱性研究（分子結構、晶體結構、物理粒子）。

## 群 (Group)

群是抽象代數中最基本的結構。一個非空集合 $G$ 必須配備一個二元運算 $*$，同時滿足以下四個法則：

### 群的四個基本法則

1. **閉合性 (Closure)**：若 $a, b \in G$，則 $a * b \in G$。
2. **結合律 (Associativity)**：$(a * b) * c = a * (b * c)$。
3. **單位元素 (Identity Element)**：存在 $e \in G$，使得 $a * e = e * a = a$。
4. **反元素 (Inverse Element)**：對每個 $a \in G$，存在 $a^{-1} \in G$，使得 $a * a^{-1} = a^{-1} * a = e$。

若還滿足交換律 $a * b = b * a$，則稱為**阿貝爾群 (Abelian Group)**。

### 群的例子

| 例子 | 集合 | 運算 | 單位元素 |
|------|------|------|----------|
| 整數加法 | $\mathbb{Z}$ | $+$ | $0$ |
| 非零實數乘法 | $\mathbb{R} \setminus \{0\}$ | $\times$ | $1$ |
| $n$ 階可逆矩陣 | $GL(n, \mathbb{R})$ | 矩陣乘法 | 單位矩陣 |

## 有限體 (Finite Field)

有限體又稱**伽羅瓦體 (Galois Field)**，是只包含有限個元素的體。

### 關鍵特性

1. **元素個數有限**：元素個數只能是質數的冪次方 $p^n$，其中 $p$ 為質數，$n$ 為正整數。

### 質數階有限體 $GF(p)$

當 $n = 1$ 時，有限體的階就是質數 $p$：
- 元素：$\{0, 1, 2, \ldots, p-1\}$
- 運算：所有加法和乘法取模 $p$

例如 $GF(5) = \{0, 1, 2, 3, 4\}$：
- 加法：$3 + 4 = 7 \equiv 2 \pmod{5}$
- 乘法：$3 \times 4 = 12 \equiv 2 \pmod{5}$

### 擴展有限體 $GF(p^n)$

當 $n > 1$ 時：
- 元素：用多項式表示
- 運算：多項式運算後對不可約多項式取模

例如 $GF(2^8)$ 是 AES 加密的數學基礎。

## 有限體的應用

### 密碼學

- **RSA 加密**：基於大整數分解的困難性
- **橢圓曲線密碼學 (ECC)**：基於橢圓曲線上的離散對數問題
- **AES**：基於 $GF(2^8)$ 的位元運算

### 密碼學中的有限體運算

```python
# GF(2^8) 加法 (XOR)
def gf_add(a, b):
    return a ^ b

# GF(2^8) 乘法 (多項式乘法)
def gf_mul(a, b):
    result = 0
    while b:
        if b & 1:
            result ^= a
        a = gf_xtime(a)
        b >>= 1
    return result
```

## 同態與同構

**同態**是保持運算結構的映射。若 $\phi: G \to H$ 是群同態，則對所有 $a, b \in G$：
$$\phi(ab) = \phi(a)\phi(b)$$

**同構**是雙射的同態。兩個同構的群在結構上完全相同，只是標記不同。

## 環 (Ring)

環是配備兩種二元運算（加法和乘法）的代數結構。

### 環的法則

1. $(\mathbb{S}, +)$ 是阿貝爾群
2. 乘法封閉性：$a \times b \in \mathbb{S}$
3. 乘法結合律：$(a \times b) \times c = a \times (b \times c)$
4. 乘法對加法的分配律：$a \times (b + c) = a \times b + a \times c$

## 體 (Field)

體是比環更強的代數結構。

### 定義

一個集合 $F$ 必須配備兩種二元運算：加法 $(+)$ 和乘法 $(\cdot)$，並滿足：
1. $(F, +)$ 是阿貝爾群
2. $(F \setminus \{0\}, \cdot)$ 是阿貝爾群
3. 乘法對加法的分配律：$a \cdot (b + c) = a \cdot b + a \cdot c$

常見的體包括：有理數 $\mathbb{Q}$、實數 $\mathbb{R}$、複數 $\mathbb{C}$。

## 伽羅瓦理論

伽羅瓦理論建立了多項式方程式根的對稱性與群論之間的深刻聯繫。伽羅瓦群描述了多項式根的置換群，這些置換保持了所有有理係數多項式關係。

# 幾何學 (Geometry)

## 概述

幾何學是數學中最古老且最實用的分支之一，其研究對象包括空間中點、線、面、體的性質與關係。在電腦科學領域，幾何學特別應用於**計算幾何（Computational Geometry）**、電腦圖學、機器人路徑規劃、地理資訊系統（GIS）、碰撞檢測等領域。

計算幾何的核心問題是如何有效地表示和處理幾何物件，以及設計解決幾何問題的演算法。這些問題通常涉及大量的點、線或多邊形，需要在時間和空間複雜度之間取得平衡。經典的計算幾何問題包括凸包建構、點定位、最近點對、線段交點計算等。

幾何學與代數學有著深刻的聯繫。透過**解析幾何**，我們可以將幾何問題轉化為代數問題；透過**向量空間**和**矩陣運算**，我們可以處理高維度的幾何變換。這些工具構成了現代電腦圖學和機器視覺的數學基礎。

## 基礎概念

### 點與向量

在二維平面中，點可以用座標 $(x, y)$ 表示；在三維空间中，则用 $(x, y, z)$ 表示。點之間的運算可以透過向量來定義：

- **向量加法**：$\mathbf{u} + \mathbf{v} = (u_x + v_x, u_y + v_y)$
- **向量純量乘法**：$k \cdot \mathbf{v} = (k v_x, k v_y)$
- **向量內積（點積）**：$\mathbf{u} \cdot \mathbf{v} = |\mathbf{u}||\mathbf{v}|\cos\theta = u_x v_x + u_y v_y$
- **向量外積（叉積）**：$\mathbf{u} \times \mathbf{v} = u_x v_y - u_y v_x$（在二維中為純量）

向量內積可用於計算兩個向量之間的夾角和投影；向量外積（在三維中為向量，在二維中為有向面積）可用於判斷點的相對位置。

### 有向面積與行列式

三個點 $A(x_1, y_1)$、$B(x_2, y_2)$、$C(x_3, y_3)$ 組成的三角形有向面積為：

$$\text{Area}(A,B,C) = \frac{1}{2} \begin{vmatrix} x_1 & y_1 & 1 \\ x_2 & y_2 & 1 \\ x_3 & y_3 & 1 \end{vmatrix} = \frac{1}{2}[(x_2-x_1)(y_3-y_1) - (x_3-x_1)(y_2-y_1)]$$

這個行列式的值為正時，表示點 $C$ 在有向直線 $AB$ 的左側；值為負時在右側；值為零時三點共線。這個性質是許多幾何演算法的基礎。

### 距離計算

不同類型的距離適用於不同的幾何問題：

1. **歐氏距離（Euclidean Distance）**：$d(\mathbf{p}, \mathbf{q}) = \sqrt{\sum_{i=1}^{n}(p_i - q_i)^2}$

2. **曼哈頓距離（Manhattan Distance）**：$d(\mathbf{p}, \mathbf{q}) = \sum_{i=1}^{n}|p_i - q_i|$

3. **切比雪夫距離（Chebyshev Distance）**：$d(\mathbf{p}, \mathbf{q}) = \max_{i}|p_i - q_i|$

4. **閔可夫斯基距離（Minkowski Distance）**：$d(\mathbf{p}, \mathbf{q}) = \left(\sum_{i=1}^{n}|p_i - q_i|^p\right)^{1/p}$

## 凸包演算法

**凸包（Convex Hull）**是計算幾何中最基礎也是最重要的問題之一。给定平面上的一組點，凸包是包含所有點的最小凸多邊形。這個多邊形的頂點是輸入點集的子集，其內部任意兩點的連線完全落在多邊形內部。

凸包問題的數學定義如下：設點集為 $S = \{p_1, p_2, \ldots, p_n\}$，則凸包 $H(S)$ 是滿足以下條件的最小凸集：
$$H(S) = \bigcap \{C \subseteq \mathbb{R}^2 : C \text{ 是凸集且 } S \subseteq C\}$$

### Graham 掃描演算法

**Graham 掃描（Graham Scan）**是一個時間複雜度為 $O(n \log n)$ 的凸包建構演算法。其基本思想是：

1. 找到最下方的點（y 座標最小，若有多个则取 x 座標最小）
2. 按極角（相對於起始點）排序其餘所有點
3. 使用堆疊逐步構建凸包

```python
import math

def graham_scan(points):
    """
    Graham 掃描演算法求凸包
    
    參數:
        points: 二維點列表 [(x1,y1), (x2,y2), ...]
    
    返回:
        凸包上的點列表（按順時針或逆時針順序）
    """
    # 找到最下方的點
    start = min(points, key=lambda p: (p[1], p[0]))
    points.remove(start)
    
    # 按極角排序（相對於起始點）
    def polar_angle(p):
        return math.atan2(p[1] - start[1], p[0] - start[0])
    
    points.sort(key=polar_angle)
    points = [start] + points
    
    # 使用堆疊構建凸包
    def cross(o, a, b):
        return (a[0] - o[0]) * (b[1] - o[1]) - (a[1] - o[1]) * (b[0] - o[0])
    
    hull = []
    for p in points:
        while len(hull) >= 2 and cross(hull[-2], hull[-1], p) <= 0:
            hull.pop()
        hull.append(p)
    
    return hull

# 測試
points = [(0, 0), (1, 0), (2, 0), (1, 1), (1, 2), (0, 2)]
hull = graham_scan(points)
print("凸包頂點:", hull)
```

### Jarvis march 演算法

**Jarvis march**（又稱 gift wrapping）的時間複雜度為 $O(nh)$，其中 $h$ 是凸包上的點數。當點分佈均勻時，這可能比 Graham 掃描慢；但當凸包上的點很少時，它可能更快。

```python
def jarvis_march(points):
    """
    Jarvis  марш 演算法（禮物包裝法）
    
    思想：從最左邊的點開始，每次找到當前點的下一個凸包點
    """
    def cross(o, a, b):
        return (a[0] - o[0]) * (b[1] - o[1]) - (a[1] - o[1]) * (b[0] - o[0])
    
    # 找到最左邊的點
    start = min(points, key=lambda p: p[0])
    
    hull = []
    point_on_hull = start
    
    while True:
        hull.append(point_on_hull)
        next_point = points[0]
        
        for p in points:
            if next_point == point_on_hull:
                next_point = p
            elif cross(point_on_hull, next_point, p) < 0:
                next_point = p
        
        point_on_hull = next_point
        
        if point_on_hull == start:
            break
    
    return hull
```

### 凸包應用

凸包在許多實際問題中有廣泛應用：

1. **碰撞檢測**：凸包可用於快速排除不可能發生碰撞的物體
2. **形態識別**：分析物體的輪廓特征
3. **路徑規劃**：在機器人導航中尋找可行的運動邊界
4. **統計學**：用於描述點集的分布範圍

## 點定位問題

**點定位（Point Location）**是另一個經典的計算幾何問題：给定一個平面分割（一組多邊形），判斷一個查詢點落在哪個區域內。

### 射线法

對於簡單多邊形，可以使用**射線法（Ray Casting）**判斷點是否在多邊形內部。從查詢點發出一條射線，計算其與多邊形邊的交點數量：

- 奇數個交點：點在多邊形內部
- 偶數個交點：點在多邊形外部

這個方法的數學依據是：從外部進入內部需要穿過邊界奇數次，從內部到外部也需要穿過邊界奇數次。

```python
def point_in_polygon(point, polygon):
    """
    射線法判斷點是否在多邊形內
    
    參數:
        point: (x, y) 查詢點
        polygon: [(x1,y1), (x2,y2), ...] 多邊形頂點
    
    返回:
        True 如果點在多邊形內
    """
    x, y = point
    n = len(polygon)
    inside = False
    
    j = n - 1
    for i in range(n):
        xi, yi = polygon[i]
        xj, yj = polygon[j]
        
        # 檢查射線是否與邊 (i,j) 相交
        if ((yi > y) != (yj > y)) and (x < (xj - xi) * (y - yi) / (yj - yi) + xi):
            inside = not inside
        
        j = i
    
    return inside
```

### 點到直線的距離

點到直線的最短距離是幾何計算中的基本運算。给定點 $P(x_0, y_0)$ 和直線 $Ax + By + C = 0$，距離為：

$$d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$$

如果直線通過兩點 $P_1(x_1, y_1)$ 和 $P_2(x_2, y_2)$，則：
$$A = y_2 - y_1, \quad B = x_1 - x_2, \quad C = x_2 y_1 - x_1 y_2$$

## 最近點對問題

**最近點對（Closest Pair）**問題要求在 $n$ 個點中找到距離最近的兩個點。使用**分治法**可以在 $O(n \log n)$ 時間內解決：

1. 將點按 x 座標排序
2. 從中間將點集分成兩半
3. 遞迴解決左右兩半的最近點對問題
4. 合併：在中間條帶內檢查可能的候選點對

這個演算法的關鍵優化在於：條帶內每個點只需要與最多 7 個相鄰點進行比較。這是因為在寬度為 $d$ 的條帶內，一個 $d \times d$ 的正方形最多能容納 6 個點而不會有兩個點的距離小於 $d$。

## 線段交點問題

**線段交點（Line Segment Intersection）**問題要求找出一組線段中的所有交點。這在電腦圖學（畫線消除）、地理資訊系統（道路網分析）中都有重要應用。

### Bentley-Ottmann 演算法

對於 $n$ 條線段，Bentley-Ottmann 演算法可以在 $O((n + k) \log n)$ 時間內找出所有 $k$ 個交點，複雜度優於暴力的 $O(n^2)$。

## 極值問題

### 包絡線（Envelope）

一組直線的**包絡線**是這些直線的交集邊界。在線性規劃中，包絡線用於找出可行域的邊界。

對於直線 $y = mx + b$ 的集合，上包絡線定義為：
$$E_{upper}(x) = \max_i (m_i x + b_i)$$

下包絡線定義為：
$$E_{lower}(x) = \min_i (m_i x + b_i)$$

包絡線可以用**分割擊破**或**堆疊**方法在 $O(n \log n)$ 時間內建構。

### 費馬點

**費馬點（Fermat Point）**是平面上到三個定點距離之和最小的點。對於銳角三角形，費馬點滿足：
- 與三個頂點的連線兩兩夾角為 $120^\circ$
- 可用幾何作圖或數值優化找到

對於點集 $S = \{p_1, p_2, \ldots, p_n\}$，要找的是點 $F$ 使得：
$$\sum_{i=1}^{n} d(F, p_i)$$

最小，這就是著名的**韋伯問題（Weber Problem）**或**設施選址問題（Facility Location Problem）**。

## 幾何變換

### 線性變換

向量空間中的線性變換可以用矩陣表示：

- **旋轉（Rotation）**（二維，角度 $\theta$）：
$$\mathbf{R}(\theta) = \begin{pmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{pmatrix}$$

- **縮放（Scaling）**：
$$\mathbf{S}(s_x, s_y) = \begin{pmatrix} s_x & 0 \\ 0 & s_y \end{pmatrix}$$

- **剪切（Shearing）**：
$$\mathbf{SH}_x(s) = \begin{pmatrix} 1 & s \\ 0 & 1 \end{pmatrix}$$

### 仿射變換

**仿射變換（Affine Transformation）**結合了線性變換和平移：
$$\mathbf{x}' = \mathbf{A}\mathbf{x} + \mathbf{b}$$

仿射變換保持：
- 直線映射到直線
- 平行線映射到平行線
-  ratios of distances along lines（共線點的距离比例）

### 齊次座標

使用**齊次座標（Homogeneous Coordinates）**，仿射變換可以表示為單一的矩陣乘法：

$$\begin{pmatrix} x' \\ y' \\ 1 \end{pmatrix} = \begin{pmatrix} a & b & t_x \\ c & d & t_y \\ 0 & 0 & 1 \end{pmatrix} \begin{pmatrix} x \\ y \\ 1 \end{pmatrix}$$

這是電腦圖學中座標變換的基礎。

## 總結

幾何學是電腦科學的重要數學基礎。計算幾何提供了處理空間數據的有效工具，從凸包建構到點定位，每個問題都有其獨特的演算法設計技巧。透過理解這些基礎概念，我們可以更好地解決實際世界中的各種幾何問題。

在機器學習中，幾何概念也扮演著重要角色。例如，支援向量機（Support Vector Machine）是一個典型的幾何方法，通過在高維空間中尋找最大間隔超平面來進行分類。這個「間隔」的概念正是幾何學在機器學習中的直接應用。

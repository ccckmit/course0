# 微積分

**標籤**: #微積分 #微分 #積分 #導數 #梯度 #偏微分 #數值分析 #SymPy #SciPy
**日期**: 2024-01-15

微積分是數學的核心分支，主要研究變化的學問。它包含兩個核心概念：微分研究瞬時變化率，積分研究累積效應。這兩個概念互為逆運算。

## 微分的基礎

### 微分的定義

一個函數 $f(x)$ 在點 $x$ 的微分 $f'(x)$ 定義為極限：

$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$

這個極限值稱為導數，代表函數圖形在該點切線的斜率。

### 微分的基本法則

- 常數的微分：$\frac{d}{dx}(c) = 0$
- 冪函數微分：$\frac{d}{dx}(x^n) = nx^{n-1}$
- 和差微分：$\frac{d}{dx}(f \pm g) = f' \pm g'$
- 積的法則：$\frac{d}{dx}(fg) = f'g + fg'$
- 商的法則：$\frac{d}{dx}\left(\frac{f}{g}\right) = \frac{f'g - fg'}{g^2}$
- 鏈式法則：$\frac{d}{dx}(f(g(x))) = f'(g(x)) \cdot g'(x)$

### 常見函數的導數

- 三角函數：
  - $\frac{d}{dx}(\sin x) = \cos x$
  - $\frac{d}{dx}(\cos x) = -\sin x$
  - $\frac{d}{dx}(\tan x) = \sec^2 x$
- 指數函數：
  - $\frac{d}{dx}(e^x) = e^x$
  - $\frac{d}{dx}(a^x) = a^x \ln a$
- 對數函數：
  - $\frac{d}{dx}(\ln x) = \frac{1}{x}$
  - $\frac{d}{dx}(\log_a x) = \frac{1}{x \ln a}$

## Python 中的微分運算

### 使用 SymPy 進行符號微分

```python
import sympy
from sympy import symbols, sin, cos, exp

x = symbols('x')

# f(x) = x² + 2x + 1
f_x = x**2 + 2*x + 1
derivative = sympy.diff(f_x, x)
print(f"f(x) = {f_x} 的微分是：{derivative}")  # 2*x + 2

# 指數函數
g_x = exp(x)
print(f"e^x 的微分是：{sympy.diff(g_x, x)}")  # e^x

# 正弦函數
h_x = sin(x)
print(f"sin(x) 的微分是：{sympy.diff(h_x, x)}")  # cos(x)
```

### 使用 SciPy 進行數值微分

```python
import numpy as np
from scipy.misc import derivative

def f(x):
    return x**2 + 2*x + 1

# 在 x=3 處計算數值微分
derivative_at_3 = derivative(f, 3.0, dx=1e-6)
print(f"f(x) 在 x=3 的微分值是：{derivative_at_3}")  # ≈ 8
```

## 積分的基礎

### 積分的定義

積分是微分的逆運算，分為不定積分和定積分。

- **不定積分**（反導數）：$\int f(x)\,dx = F(x) + C$，其中 $F'(x) = f(x)$
- **定積分**：$\int_a^b f(x)\,dx$，計算函數曲線下的面積

### 基本積分公式

- $\int x^n\,dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$
- $\int \frac{1}{x}\,dx = \ln|x| + C$
- $\int e^x\,dx = e^x + C$
- $\int \sin x\,dx = -\cos x + C$
- $\int \cos x\,dx = \sin x + C$

## Python 中的積分運算

### 使用 SymPy 進行符號積分

```python
import sympy
from sympy import symbols, sin

x = symbols('x')

# 不定積分
g_x = 3*x**2
integral = sympy.integrate(g_x, x)
print(f"∫3x²dx = {integral}")  # x³

# 定積分
definite = sympy.integrate(x**2, (x, 0, 2))
print(f"∫[0,2] x²dx = {definite}")  # 8/3
```

### 使用 SciPy 進行數值積分

```python
import numpy as np
from scipy.integrate import quad

# 數值定積分
result, error = quad(lambda t: np.sin(t), 0, np.pi)
print(f"∫[0,π] sin(x)dx = {result}")  # ≈ 2
```

## 偏微分與梯度

### 偏導數

對於多變數函數 $f(x, y)$，偏導數是固定其他變數對某一變數的微分：

$$\frac{\partial f}{\partial x} = \lim_{h \to 0} \frac{f(x+h, y) - f(x, y)}{h}$$

### 梯度

梯度是偏導數組成的向量，指向函數增長最快的方向：

$$\nabla f = \left(\frac{\partial f}{\partial x_1}, \frac{\partial f}{\partial x_2}, \ldots, \frac{\partial f}{\partial x_n}\right)$$

```python
import sympy

x, y = sympy.symbols('x y')
f = x**2 + y**2 + 2*x*y

# 偏導數
df_dx = sympy.diff(f, x)
df_dy = sympy.diff(f, y)
print(f"∂f/∂x = {df_dx}")
print(f"∂f/∂y = {df_dy}")
```

## 梯度下降法

梯度下降法是最優化的核心演算法，用於找到函數的最小值。

### 數學表達

$$x_{n+1} = x_n - \alpha \nabla f(x_n)$$

其中 $\alpha$ 是學習率（步長）。

```python
import numpy as np

def gradient_descent(grad_func, initial_point, learning_rate=0.1, tolerance=1e-6):
    point = np.array(initial_point, dtype=float)
    
    for i in range(1000):
        gradient = grad_func(point)
        new_point = point - learning_rate * gradient
        
        if np.linalg.norm(new_point - point) < tolerance:
            print(f"在第 {i+1} 次迭代收斂")
            break
        
        point = new_point
    
    return point

# 範例：f(x,y) = x² + y²
def grad_func(point):
    x, y = point
    return np.array([2*x, 2*y])

result = gradient_descent(grad_func, [10, 10], learning_rate=0.1)
print(f"最小值位置：{result}")  # ≈ [0, 0]
```

## 反向傳播

反向傳播是神經網路訓練的核心演算法，本質上是應用梯度下降法，利用鏈式法則計算每個參數的梯度。

```python
def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def sigmoid_derivative(x):
    return sigmoid(x) * (1 - sigmoid(x))

# 網路
class SimpleNN:
    def __init__(self):
        np.random.seed(42)
        self.W1 = np.random.randn(2, 2)
        self.b1 = np.zeros((1, 2))
        self.W2 = np.random.randn(2, 1)
        self.b2 = np.zeros((1, 1))
    
    def forward(self, X):
        self.z1 = np.dot(X, self.W1) + self.b1
        self.a1 = sigmoid(self.z1)
        self.z2 = np.dot(self.a1, self.W2) + self.b2
        self.a2 = sigmoid(self.z2)
        return self.a2
    
    def backward(self, X, y, learning_rate=0.1):
        m = X.shape[0]
        
        # 輸出層誤差
        dz2 = self.a2 - y
        dW2 = (1/m) * np.dot(self.a1.T, dz2)
        db2 = (1/m) * np.sum(dz2, axis=0, keepdims=True)
        
        # 隱藏層誤差
        dz1 = np.dot(dz2, self.W2.T) * sigmoid_derivative(self.z1)
        dW1 = (1/m) * np.dot(X.T, dz1)
        db1 = (1/m) * np.sum(dz1, axis=0, keepdims=True)
        
        # 更新參數
        self.W2 -= learning_rate * dW2
        self.b2 -= learning_rate * db2
        self.W1 -= learning_rate * dW1
        self.b1 -= learning_rate * db1
```

## 數值方法

### 數值微分

- 前方差分：$f'(x) \approx \frac{f(x+h) - f(x)}{h}$
- 中央差分：$f'(x) \approx \frac{f(x+h) - f(x-h)}{2h}$

### 數值積分

- 矩形法則：$\int f(x)dx \approx \sum f(x_i)\Delta x$
- 梯形法則：$\int f(x)dx \approx \sum \frac{f(x_i) + f(x_{i+1})}{2} \Delta x$
- 辛普森法則（更高精度）：

$$\int_a^b f(x)dx \approx \frac{\Delta x}{3} \left[f(x_0) + 4f(x_1) + 2f(x_2) + \cdots + f(x_n)\right]$$

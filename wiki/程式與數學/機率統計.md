# 機率統計

**標籤**: #機率統計 #機率論 #統計學 #隨機變數 #機率分佈 #期望值 #變異數
**日期**: 2024-01-15

機率統計是數學的重要分支，研究隨機現象的規律性。機率論研究隨機事件的機率規律，統計學從數據中推斷規律。兩者密切相關：機率論是統計學的理論基礎，統計學是機率論的實際應用。

## 機率論基礎

### 樣本空間與事件

樣本空間 $S$ 是所有可能結果的集合。事件是樣本空間的子集。若 $A \cap B = \emptyset$，則稱為互斥事件。

### 機率公理

機率函數 $P$ 滿足：
1. **非負性**：對任意事件 $A$ 有 $P(A) \geq 0$
2. **規範性**：$P(S) = 1$
3. **可加性**：對於互斥事件序列 $A_1, A_2, \ldots$，有 $P(\cup A_i) = \sum P(A_i)$

### 條件機率與獨立性

條件機率：$P(A|B) = \frac{P(A \cap B)}{P(B)}$（當 $P(B) > 0$）

若 $P(A|B) = P(A)$，則 $A$ 與 $B$ 獨立。

**貝葉斯定理**：

$$P(A|B) = \frac{P(B|A)P(A)}{P(B)}$$

## 離散隨機變數

### 機率質量函數 (PMF)

離散隨機變數 $X$ 的機率質量函數 $p(x) = P(X = x)$，滿足 $p(x) \geq 0$ 且 $\sum p(x) = 1$。

### 常見離散分佈

**伯努利分佈** $Bernoulli(p)$：$X \in \{0, 1\}$，$P(X=1) = p$

**二項分佈** $Binomial(n, p)$：$P(X=k) = \binom{n}{k}p^k(1-p)^{n-k}$

**卜瓦松分佈** $Poisson(\lambda)$：$P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}$

### 期望值與變異數

- **期望值**：$E[X] = \sum x \cdot p(x)$
- **變異數**：$Var(X) = E[(X-\mu)^2] = E[X^2] - \mu^2$

| 分佈 | 期望值 | 變異數 |
|------|---------|---------|
| Bernoulli($p$) | $p$ | $p(1-p)$ |
| Binomial($n,p$) | $np$ | $np(1-p)$ |
| Poisson($\lambda$) | $\lambda$ | $\lambda$ |

## 連續隨機變數

### 機率密度函數 (PDF)

連續隨機變數 $X$ 的機率密度函數 $f(x)$，滿足 $f(x) \geq 0$ 且 $\int_{-\infty}^{\infty} f(x)\,dx = 1$。

注意 $P(X = a) = 0$，因此 $P(a \leq X \leq b) = P(a < X \leq b) = \int_a^b f(x)\,dx$。

### 常見連續分佈

**均勻分佈** $Uniform(a, b)$：$f(x) = \frac{1}{b-a}$

**指數分佈** $Exponential(\lambda)$：$f(x) = \lambda e^{-\lambda x} \quad (x \geq 0)$

**常態分佈** $Normal(\mu, \sigma^2)$：

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$

標準常態分佈 $N(0, 1)$ 是均值為 $0$、標準差為 $1$ 的常態分佈。

### 期望值與變異數（連續）

- **期望值**：$E[X] = \int_{-\infty}^{\infty} x \cdot f(x)\,dx$
- **變異數**：$Var(X) = \int_{-\infty}^{\infty} (x-\mu)^2 f(x)\,dx = E[X^2] - \mu^2$

## 多變數機率分佈

### 聯合分佈

- 離散：$P(X=x, Y=y) = p(x, y)$
- 連續：$f(x, y)$ 滿足 $\iint f(x, y)\,dx\,dy = 1$

### 共變異數與相關係數

**共變異數**：

$$Cov(X, Y) = E[(X-\mu_X)(Y-\mu_Y)] = E[XY] - \mu_X \mu_Y$$

**相關係數**：

$$\rho = \frac{Cov(X, Y)}{\sigma_X \sigma_Y}$$

$-1 \leq \rho \leq 1$：$\rho > 0$ 表示正相關，$\rho < 0$ 表示負相關，$\rho = 0$ 表示獨立（對常態分佈成立）。

## 隨機樣本與估計

### 隨機樣本

設 $X_1, X_2, \ldots, X_n$ 為獨立同分佈 (i.i.d.) 的隨機變數，則樣本均值 $\bar{X} = \frac{1}{n}\sum X_i$ 是對整體均值 $E[X]$ 的估計。

### 點估計

- **最大概似估計 (MLE)**：選擇使觀察到樣本機率最大的參數
- **動差估計 (Method of Moments)**：使用樣本動差匹配整體動差

### 區間估計

95% 信心區間：$\bar{X} \pm z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}$

## 假設檢定

### 基本概念

- **虛無假設** $H_0$：需要檢驗的假設
- **對立假設** $H_1$：$H_0$ 的替代
- **型一錯誤** $\alpha = P(\text{拒絕}H_0|H_0\text{為真})$
- **型二錯誤** $\beta = P(\text{接受}H_0|H_1\text{為真})$

### 常用檢定

- **Z 檢定**：已知整體變異數或大樣本時檢驗均值
- **t 檢定**：未知整體變異數時檢驗均值
- **卡方檢定**：檢驗類別變數的獨立性或適配度
- **ANOVA**：比較三個或以上組別的均值差異

### p 值

p 值是在 $H_0$ 為真時，觀察到與樣本相同或更極端結果的機率。若 $p < \alpha$，則拒絕 $H_0$。

## 大數法則與中央極限定理

### 大數法則

$$\bar{X} \to E[X] \quad (n \to \infty)$$

這意味著隨著樣本增加，估計越來越準確。

### 中央極限定理

$$\frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \to N(0, 1) \quad (n \to \infty)$$

無論原始分佈如何，樣本均值的分佈趨近常態分佈。

## 迴歸分析

### 簡單線性迴歸

$$Y = \beta_0 + \beta_1 X + \varepsilon$$

目標是最小化殘差平方和 $RSS = \sum (Y_i - \hat{Y}_i)^2$。

### 估計

$$\hat{\beta}_1 = \frac{Cov(X, Y)}{Var(X)}$$
$$\hat{\beta}_0 = \bar{Y} - \hat{\beta}_1 \bar{X}$$

### $R^2$

$$R^2 = 1 - \frac{RSS}{TSS}$$

代表模型解釋的變異比例，取值範圍為 $0$ 到 $1$。

## Python 實現

```python
import numpy as np
from scipy import stats

# 產生隨機樣本
np.random.seed(42)
data = np.random.normal(100, 15, 1000)

# 描述統計
mean = np.mean(data)
median = np.median(data)
std = np.std(data)
variance = np.var(data)

# 假設檢定：單樣本 t 檢定
t_stat, p_value = stats.ttest_1samp(data, 100)
print(f"t 統計量: {t_stat}, p 值: {p_value}")

# 常態性檢定
stat, p = stats.normaltest(data)
print(f"常態性檢定 p 值: {p}")
```

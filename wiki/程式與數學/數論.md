# 數論 (Number Theory)

## 概述

數論是研究整數性質的數學分支，被稱為「數學的皇后」。在電腦科學中，數論扮演著核心角色，特別是在**密碼學**、**編碼理論**、**演算法設計**和**資訊安全**等領域。現代公鑰密碼系統（如 RSA、橢圓曲線密碼學）完全建立在數論的基礎之上。

數論的研究對象是整數 $\mathbb{Z} = \{\ldots, -2, -1, 0, 1, 2, \ldots\}$，以及其子集如自然數 $\mathbb{N} = \{0, 1, 2, \ldots\}$ 和質數集合。雖然整數的定義看似簡單，但數論蘊含了數學中最深刻、最困難的問題，其中許多問題已經困擾數學家數百年。

## 整數除法與同餘

### 整除性

對於整數 $a$ 和 $b$（$b \neq 0$），如果存在整數 $q$ 使得 $a = bq$，則稱 $b$ 整除 $a$，記作 $b \mid a$。這時 $b$ 稱為 $a$ 的因數（divisor），$a$ 稱為 $b$ 的倍數（multiple）。

**除法算法**（Euclidean Division）給出：對於任意整數 $a$ 和正整數 $b$，存在唯一的整數 $q$（商）和 $r$（餘數）使得：
$$a = bq + r, \quad 0 \leq r < b$$

### 最大公因數與最小公倍數

兩個整數 $a$ 和 $b$ 的**最大公因數**（Greatest Common Divisor, GCD）記作 $\gcd(a, b)$，是同時整除 $a$ 和 $b$ 的最大正整數。**最小公倍數**（Least Common Multiple, LCM）記作 $\text{lcm}(a, b)$，是同時被 $a$ 和 $b$ 整除的最小正整數。

它們之間的關係為：
$$\gcd(a, b) \times \text{lcm}(a, b) = |a \cdot b|$$

**歐幾里得演算法**（Euclidean Algorithm）是計算 GCD 的經典方法：

```python
def gcd(a, b):
    """
    歐幾里得演算法計算最大公因數
    
    基於原理：gcd(a, b) = gcd(b, a mod b)
    """
    while b:
        a, b = b, a % b
    return abs(a)

def extended_gcd(a, b):
    """
    擴展歐幾里得演算法
    
    返回：(g, x, y) 使得 g = gcd(a, b) = ax + by
    """
    if b == 0:
        return (a, 1, 0)
    else:
        g, x1, y1 = extended_gcd(b, a % b)
        x = y1
        y = x1 - (a // b) * y1
        return (g, x, y)
```

擴展歐幾里得演算法特別重要，因為它可以計算**模反元素**：對於 $a$ 和 $m$ 互質，找出 $x$ 使得 $ax \equiv 1 \pmod{m}$。這是 RSA 加密和解密的數學基礎。

### 同餘關係

**同餘（Congruence）**是數論中的核心概念：對於整數 $a$、$b$ 和正整數 $m$，如果 $m \mid (a-b)$，則稱 $a$ 和 $b$ 同餘模 $m$，記作：
$$a \equiv b \pmod{m}$$

同餘關係具有以下性質：
1. **自反性**：$a \equiv a \pmod{m}$
2. **對稱性**：若 $a \equiv b \pmod{m}$，則 $b \equiv a \pmod{m}$
3. **傳遞性**：若 $a \equiv b \pmod{m}$ 且 $b \equiv c \pmod{m}$，則 $a \equiv c \pmod{m}$
4. **加法律**：若 $a \equiv b \pmod{m}$ 且 $c \equiv d \pmod{m}$，則 $a+c \equiv b+d \pmod{m}$
5. **乘法律**：若 $a \equiv b \pmod{m}$ 且 $c \equiv d \pmod{m}$，則 $ac \equiv bd \pmod{m}$

## 質數理論

### 質數的定義與分布

**質數（Prime）**是大於 1 且只能被 1 和自身整除的自然數。質數是乘法構建所有自然數的「原子」：根據**算術基本定理**，每個大於 1 的自然數都可以唯一分解為質數的乘積（忽略順序）。

$$n = p_1^{e_1} p_2^{e_2} \cdots p_k^{e_k}, \quad p_i \text{ 為質數}, e_i \geq 1$$

質數的分布是一個深刻的數學問題。**質數定理**給出了質數分布的漸近估計：

$$\pi(x) \sim \frac{x}{\ln x}$$

其中 $\pi(x)$ 表示小於 $x$ 的質數數量。這個定理意味著，第 $n$ 個質數約為 $n \ln n$。

### 質數檢驗

質數檢驗是密碼學的基礎操作。以下是幾種常用的質數檢驗方法：

**埃拉託斯特尼篩法（Sieve of Eratosthenes）**找出某範圍內的所有質數：

```python
def sieve_of_eratosthenes(n):
    """
    埃拉託斯特尼篩法
    
    時間複雜度：O(n log log n)
    空間複雜度：O(n)
    """
    is_prime = [True] * (n + 1)
    is_prime[0] = is_prime[1] = False
    
    for i in range(2, int(n**0.5) + 1):
        if is_prime[i]:
            for j in range(i*i, n+1, i):
                is_prime[j] = False
    
    return [i for i in range(n+1) if is_prime[i]]
```

**米勒-拉賓質數檢驗（Miller-Rabin Primality Test）**是一個機率性演算法，可以用於檢驗大數是否為質數：

```python
import random

def miller_rabin(n, k=10):
    """
    米勒-拉賓質數檢驗
    
    參數:
        n: 待檢驗的數
        k: 測試回合數（決定錯誤機率）
    
    返回:
        True 表示 n 很可能是質數
        False 表示 n 確定是合數
    """
    if n < 2:
        return False
    if n == 2:
        return True
    if n % 2 == 0:
        return False
    
    # 寫成 n-1 = 2^r * d 的形式
    r, d = 0, n - 1
    while d % 2 == 0:
        r += 1
        d //= 2
    
    # 執行 k 回合測試
    for _ in range(k):
        a = random.randrange(2, n - 1)
        x = pow(a, d, n)
        
        if x == 1 or x == n - 1:
            continue
        
        for _ in range(r - 1):
            x = pow(x, 2, n)
            if x == n - 1:
                break
        else:
            return False
    
    return True
```

米勒-拉賓檢驗的錯誤機率為 $4^{-k}$，當 $k=10$ 時錯誤機率小於百萬分之一。

### 質數生成與 RSA

RSA 公鑰密碼系統的安全性完全依賴於大質數分解的困難性。生成 RSA 金鑰的過程如下：

1. 隨機選擇兩個大質數 $p$ 和 $q$
2. 計算 $n = p \times q$（公開模數）
3. 計算 $\phi(n) = (p-1)(q-1)$（歐拉函數）
4. 選擇公開指數 $e$，通常為 65537（$2^{16} + 1$）
5. 計算私鑰 $d$，使得 $ed \equiv 1 \pmod{\phi(n)}$

**歐拉函數**（Euler's Totient Function）$\phi(n)$ 定義為小於 $n$ 且與 $n$ 互質的正整數數量。對於質數 $p$，$\phi(p) = p-1$；對於兩個質數的乘積 $n = pq$，$\phi(n) = (p-1)(q-1)$。

RSA 加密和解密的數學公式為：
- 加密：$c = m^e \pmod{n}$
- 解密：$m = c^d \pmod{n}$

其中 $m$ 是明文，$c$ 是密文，$(e, n)$ 是公鑰，$(d, n)$ 是私鑰。

## 模運算與原根

### 模反元素

在模運算中，**模反元素**（Modular Multiplicative Inverse）是滿足以下條件的數 $x$：
$$a \cdot x \equiv 1 \pmod{m}$$

模反元素存在的條件是 $\gcd(a, m) = 1$（即 $a$ 和 $m$ 互質）。

使用擴展歐幾里得演算法可以計算模反元素：

```python
def mod_inverse(a, m):
    """
    計算模反元素 a^(-1) mod m
    
    假設 a 和 m 互質
    """
    g, x, _ = extended_gcd(a, m)
    if g != 1:
        raise Exception('Modular inverse does not exist')
    return x % m
```

### 中國剩餘定理

**中國剩餘定理**（Chinese Remainder Theorem, CRT）解決了以下問題：给定一系列同餘方程，求解滿足所有方程的數。

形式化地，CRT 陳述：若 $m_1, m_2, \ldots, m_k$ 兩兩互質，則對於任意整數 $a_1, a_2, \ldots, a_k$，存在唯一解 $x$ 模 $M = m_1 m_2 \cdots m_k$ 使得：
$$x \equiv a_i \pmod{m_i}, \quad \forall i \in \{1, 2, \ldots, k\}$$

CRT 在密碼學和高效計算中有重要應用。

### 原根

對於質數 $p$，如果整數 $g$ 滿足 $g, g^2, \ldots, g^{p-1}$ 產生 $1$ 到 $p-1$ 的所有剩餘，則稱 $g$ 為 $p$ 的**原根**（Primitive Root）。

原根的存在條件：$g$ 是 $p$ 的原根當且僅當對於 $p-1$ 的每個質因數 $q$，都有 $g^{(p-1)/q} \not\equiv 1 \pmod{p}$。

原根在**迪菲-赫爾曼金鑰交換**（Diffie-Hellman Key Exchange）中扮演關鍵角色。

## 離散對數

**離散對數問題**是密碼學的另一個核心假設：给定質數 $p$、原根 $g$ 和值 $g^x \pmod{p}$，計算 $x$ 在計算上是困難的。

形式化地：已知 $g, p, y = g^x \pmod{p}$，求 $x$。

離散對數問題是以下密碼系統的數學基礎：
- **迪菲-赫爾曼金鑰交換**：允許兩方在不共享秘密的情況下建立共享金鑰
- **艾爾伽莫爾（ElGamal）加密**：基於離散對數的公鑰加密系統

## 二次剩餘

### 勒讓德符號

**勒讓德符號**（Legendre Symbol）$\left(\frac{a}{p}\right)$ 定義為：
$$\left(\frac{a}{p}\right) = \begin{cases} 1 & \text{若 } a \text{ 是 } p \text{ 的二次剩餘且 } a \not\equiv 0 \pmod{p} \\ -1 & \text{若 } a \text{ 不是 } p \text{ 的二次剩餘} \\ 0 & \text{若 } a \equiv 0 \pmod{p} \end{cases}$$

**歐拉準則**提供了計算勒讓德符號的方法：
$$\left(\frac{a}{p}\right) \equiv a^{\frac{p-1}{2}} \pmod{p}$$

### 二次互反律

**二次互反律**（Quadratic Reciprocity）是數論中最優美的定理之一：若 $p$ 和 $q$ 為奇質數，則：
$$\left(\frac{p}{q}\right) \left(\frac{q}{p}\right) = (-1)^{\frac{p-1}{2} \cdot \frac{q-1}{2}}$$

這個定理大大簡化了勒讓德符號的計算。

## 密碼學應用

### RSA 加密系統

RSA 是最廣泛使用的公鑰密碼系統之一，其安全性基於大整數分解的困難性。

```python
def generate_rsa_keys(p, q):
    """
    生成 RSA 金鑰對
    
    參數:
        p, q: 兩個大質數
    """
    n = p * q
    phi = (p - 1) * (q - 1)
    
    # 選擇公開指數 e
    e = 65537
    
    # 計算私鑰 d
    d = mod_inverse(e, phi)
    
    # 公鑰: (e, n)
    # 私鑰: (d, n)
    return (e, n), (d, n)

def rsa_encrypt(message, e, n):
    """RSA 加密"""
    return pow(message, e, n)

def rsa_decrypt(ciphertext, d, n):
    """RSA 解密"""
    return pow(ciphertext, d, n)
```

### 橢圓曲線密碼學

**橢圓曲線密碼學**（Elliptic Curve Cryptography, ECC）使用橢圓曲線上的點群運算，提供與 RSA 相似的安全性但使用更短的金鑰。

橢圓曲線定義為：
$$y^2 = x^3 + ax + b \pmod{p}$$

其中 $4a^3 + 27b^2 \neq 0 \pmod{p}$ 確保曲線是光滑的。

曲線上的點可以進行「加法」運算：對於點 $P$ 和 $Q$，定義 $P + Q$ 為曲線上第三個點的對稱點。這種運算構成一個有限阿貝爾群，其離散對數問題比整數域上的離散對數問題更困難，因此 ECC 可以使用更短的金鑰達到相同的安全等級。

## 總結

數論是現代密碼學的數學基礎。從質數檢驗到模運算，從歐拉函數到離散對數，每個數論概念都在資訊安全中扮演著關鍵角色。隨著量子計算的發展，傳統的基於質數分解和離散對數的密碼系統可能面臨挑戰，這也推動了**後量子密碼學**的發展，探索基於格（lattice）、編碼（code）或多元多項式（multivariate polynomial）的新型密碼系統。

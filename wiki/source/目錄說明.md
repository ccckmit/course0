# 原始來源

此目錄存放 Wiki 的原始來源資料。

## 存放規則

- 原始來源放入對應分類的子目錄
- 來源是不可變的（LLM 只讀不寫）
- 每個來源對應一個摘要頁（放在 sources/ 子目錄）

## 目錄結構

```
source/
├── 計算機結構/
│   └── 教材.pdf
├── 系統程式/
│   └── 教材.pdf
├── 演算法/
│   └── 教材.pdf
├── 機器學習/
│   └── 教材.pdf
├── 網頁設計/
│   └── 教材.pdf
└── 軟體工程/
    └── 教材.pdf
```

## 使用方式

1. 將來源放入對應分類子目錄
2. 告知 LLM 處理該來源
3. LLM 會自動：
   - 閱讀並摘要來源
   - 在 sources/ 建立摘要頁
   - 更新 index.md
   - 更新相關概念頁面的交叉引用
   - 追加 log.md

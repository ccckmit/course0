# c4 - 四函式 C 編譯器

**來源類型**: GitHub 專案
**來源路徑**: https://github.com/rswier/c4
**攝取日期**: 2026-04-07

## 摘要

c4 是一個僅用 4 個函式（約 500 行代碼）實現的 self-hosted C 編譯器，支持完整的 C 語言語法，包括指標、陣列、结构体、函式等。

## 關鍵要點

- 四個核心函式：lexical analyzer、parser、code generator、main
- 自我托管：能用自己編譯自己
- 生成 x86-64 Linux 可執行文件
- 極簡設計，適合作為教學工具

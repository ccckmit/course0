# xv6 - MIT 教學用 Unix 作業系統

**來源類型**: GitHub 專案
**來源路徑**: https://github.com/mit-pdos/xv6-riscv
**攝取日期**: 2026-04-07

## 摘要

xv6 是 MIT 對 Unix Version 6 的重新實作，使用 ANSI C 編寫，支援現代 RISC-V 多核心處理器。是作業系統教學的經典教材。

## 關鍵要點

- 由 MIT PDOS Group 維護
- 支援 RISC-V 架構
- 簡潔的系統呼叫介面
- 包含排程、檔案系統、虛擬記憶體等核心功能
